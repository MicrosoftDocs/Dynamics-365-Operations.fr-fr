## <a name="white-papers"></a>Livres blancs
* Le <a href="https://mbs.microsoft.com/customersource/northamerica/AX/learning/documentation/white-papers/msd365optgtstcostacc/">Microsoft Dynamics 365 pour les opérations - mise en route de la comptabilité des coûts</a> livre blanc fournit les étapes de base pour configurer Microsoft Dynamics 365 pour les opérations – contrôle de gestion pour effectuer l’analyse du coût réel. Il fournit également une série de procédures que vous pouvez effectuer dans l’ordre dans lequel ils apparaissent.
* Le <a href="https://mbs.microsoft.com/customersource/northamerica/AX/learning/documentation/white-papers/leanmanufkanban365opt/">365 Dynamics pour les opérations - Lean Manufacturing : Capable to Promise et la planification des tâches de Kanban</a> livre blanc décrit les détails de la planification des tâches de kanban et introduit le concept de prise de commande qui utilise la méthode de délai (SDD) capable de contrôle de date de livraison. Il explique également comment les CTP peut être combinée avec des kanbans d’événement pour modéliser des scénarios Assembler pour commande ou build-to-order.
* Le <a href="https://mbs.microsoft.com/customersource/northamerica/AX/learning/documentation/white-papers/365operationsbomcalsheet/">Microsoft Dynamics 365 pour les opérations de calcul de nomenclature à l’aide d’une feuille d’évaluation des coûts -</a> livre blanc explique comment configurer une feuille d’évaluation des coûts qui comprend le matériau et fabrication, et les résultats de la façon dont le programme d’installation affecte le calcul de nomenclature. Vous pouvez effectuer les exercices dans l’annexe pour les besoins de la formation et les connaissances que vous avez obtenus à partir de la lecture de ce document pour effectuer une analyse de calcul de nomenclature.
