# <a name="calculate-and-adjust-sales-tax-on-a-vendor-invoice"></a>Calculer et ajuster la taxe sur une facture fournisseur

[!include[task guide banner](../../includes/task-guide-banner.md)]

Si le document source d'origine affiche différents montants de taxe calculés, vous pouvez régler ces montants avant la validation. La société fictive DEMF est citée en exemple dans cette tâche.

1. Accédez à Comptabilité fournisseur > Factures > Journal des factures.
2. Cliquez sur Nouveau.
3. Dans la liste, marquez la ligne sélectionnée.
4. Dans le champ Nom, cliquez sur le bouton de liste déroulante pour ouvrir la recherche.
5. Dans la liste, cliquer sur le lien dans la ligne sélectionnée.
6. Cliquez sur Lignes.
7. Dans la liste, marquer la ligne sélectionnée.
8. Dans le champ Compte, spécifiez les valeurs souhaitées.
9. Tapez une valeur dans le champ Facture.
10. Entrez un numéro dans le champ Crédit.
11. Spécifiez les valeurs souhaitées dans le champ Compte de contrepartie.
12. Cliquez sur Taxe.
13. Entrez un nombre dans le champ Montant réel de taxe total.
14. Cliquez sur OK.
15. Cliquez sur Enregistrer.
16. Cliquez sur Taxe.
17. Sous l'onglet Ajustement, les montants de taxe peuvent être ajustés par code taxe.
18. Cliquez sur Réinitialiser les montants réels des montants calculés.
19. Cliquez sur OK.
20. Cliquez sur Enregistrer.
