# <a name="set-up-method-of-payment-france"></a><span data-ttu-id="f59f0-103">Paramétrer un mode de paiement (France)</span><span class="sxs-lookup"><span data-stu-id="f59f0-103">Set up method of payment (France)</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="f59f0-104">La procédure vous guide dans la configuration des modes de paiement d'établissement afin de vous aider à créer et valider la lettre de change après l'avoir approuvée dans le journal de création de lettres de change.</span><span class="sxs-lookup"><span data-stu-id="f59f0-104">The procedure walks you through setting up methods of payment so that you can create and post the bill of exchange after approving it from the draw bill of exchange journal.</span></span>

<span data-ttu-id="f59f0-105">Cette procédure a été créée à l'aide des données fictives de la société FRSI.</span><span class="sxs-lookup"><span data-stu-id="f59f0-105">This procedure was created using the demo data company FRSI.</span></span> 

<span data-ttu-id="f59f0-106">Cette fonctionnalité est disponible pour les entités juridiques dont l'adresse principale est en France.</span><span class="sxs-lookup"><span data-stu-id="f59f0-106">This functionality is available for legal entities whose primary address is in France.</span></span>

<span data-ttu-id="f59f0-107">Pour effectuer cette procédure, vous devez avoir le rôle de responsable de la comptabilité client.</span><span class="sxs-lookup"><span data-stu-id="f59f0-107">You should have a role of Accounts receivables manager to perform this procedure.</span></span>





1. <span data-ttu-id="f59f0-108">Accédez à Comptabilité client > Paramétrage des paiements > Modes de paiement.</span><span class="sxs-lookup"><span data-stu-id="f59f0-108">Go to Accounts receivable > Payments setup > Methods of payment.</span></span>
2. <span data-ttu-id="f59f0-109">Cliquez sur Nouveau.</span><span class="sxs-lookup"><span data-stu-id="f59f0-109">Click New.</span></span>
3. <span data-ttu-id="f59f0-110">Tapez une valeur dans le champ Mode de paiement.</span><span class="sxs-lookup"><span data-stu-id="f59f0-110">In the Method of payment field, type a value.</span></span>
4. <span data-ttu-id="f59f0-111">Dans le champ Description, entrez une valeur.</span><span class="sxs-lookup"><span data-stu-id="f59f0-111">In the Description field, type a value.</span></span>
5. <span data-ttu-id="f59f0-112">Dans le champ Statut de paiement, sélectionnez « Approuvé ».</span><span class="sxs-lookup"><span data-stu-id="f59f0-112">In the Payment status field, select 'Approved'.</span></span>
6. <span data-ttu-id="f59f0-113">Dans le champ Type de paiement, sélectionnez « Lettre de change ».</span><span class="sxs-lookup"><span data-stu-id="f59f0-113">In the Payment type field, select 'Bill of exchange'.</span></span>
7. <span data-ttu-id="f59f0-114">Développez ou réduisez la section Général.</span><span class="sxs-lookup"><span data-stu-id="f59f0-114">Expand or collapse the General section.</span></span>
8. <span data-ttu-id="f59f0-115">Sélectionnez Banque dans le champ Type de compte.</span><span class="sxs-lookup"><span data-stu-id="f59f0-115">In the Account type field, select 'Bank'.</span></span>
9. <span data-ttu-id="f59f0-116">Dans le champ Compte de paiement, indiquez les valeurs souhaitées.</span><span class="sxs-lookup"><span data-stu-id="f59f0-116">In the Payment account field, specify the desired values.</span></span>
    * <span data-ttu-id="f59f0-117">Par exemple : « FRSI OPER »</span><span class="sxs-lookup"><span data-stu-id="f59f0-117">For example: 'FRSI OPER'</span></span>  
10. <span data-ttu-id="f59f0-118">Développez ou réduisez la section Formats de fichier.</span><span class="sxs-lookup"><span data-stu-id="f59f0-118">Expand or collapse the File formats section.</span></span>
11. <span data-ttu-id="f59f0-119">Cliquez sur Paramétrage.</span><span class="sxs-lookup"><span data-stu-id="f59f0-119">Click Setup.</span></span>
12. <span data-ttu-id="f59f0-120">Dans la liste, recherchez et sélectionnez l'enregistrement souhaité.</span><span class="sxs-lookup"><span data-stu-id="f59f0-120">In the list, find and select the desired record.</span></span>

13. <span data-ttu-id="f59f0-122">Cliquez sur Enregistrer.</span><span class="sxs-lookup"><span data-stu-id="f59f0-122">Click Save.</span></span>
14. <span data-ttu-id="f59f0-123">Fermez la page.</span><span class="sxs-lookup"><span data-stu-id="f59f0-123">Close the page.</span></span>

    * <span data-ttu-id="f59f0-126">Sélectionnez « EnmmEAR »</span><span class="sxs-lookup"><span data-stu-id="f59f0-126">Select "EnmmEAR"</span></span>

    * <span data-ttu-id="f59f0-129">Par exemple : Sélectionnez « Lettre de change française »</span><span class="sxs-lookup"><span data-stu-id="f59f0-129">For example: Select "French bill of exchange"</span></span>

21. <span data-ttu-id="f59f0-132">Cliquez sur Enregistrer.</span><span class="sxs-lookup"><span data-stu-id="f59f0-132">Click Save.</span></span>
