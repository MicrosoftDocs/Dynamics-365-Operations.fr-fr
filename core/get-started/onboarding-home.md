# <a name="onboarding-home-page"></a>Page d'accueil de l'intégration

[!include[banner](../includes/banner.md)]


Pour démarrer avec Microsoft Dynamics 365 for Operations, choisissez un niveau d'abonnement qui vous convient, recherchez un partenaire qui connaît votre région ou secteur, puis utilisez le programme Microsoft FastTrack pour pouvoir être opérationnel rapidement.  

Les ressources suivantes fournissent une vue d'ensemble de Lifecycle Services et expliquent comment collaborer avec votre partenaire afin d'intégrer Dynamics 365 for Operations.
-   [Souscrire un abonnement](/dynamics365/operations/dev-itpro/dev-tools/sign-up-preview-subscription)
-   [Vue d'ensemble de Microsoft FastTrack for Dynamics 365](fasttrack-dynamics-365-overview.md)
-   [Introduction du client à Lifecycle Services](/dynamics365/operations/dev-itpro/lifecycle-services/lcs-works-lcs)
