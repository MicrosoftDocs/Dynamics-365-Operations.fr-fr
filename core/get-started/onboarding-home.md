# <a name="onboarding-home-page"></a>Page d'accueil de l'intégration

[!include[banner](../includes/banner.md)]

Les ressources suivantes fournissent une vue d'ensemble de Lifecycle Services et expliquent comment collaborer avec votre partenaire afin d'intégrer Finance and Operations.

-   [Vue d'ensemble de Microsoft FastTrack for Dynamics 365](fasttrack-dynamics-365-overview.md)
