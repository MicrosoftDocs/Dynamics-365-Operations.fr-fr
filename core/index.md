# <a name="introduction-to-dynamics-365-for-operations"></a>Présentation de Dynamics 365 pour les opérations

[!include[banner](includes/banner.md)]

Microsoft Dynamics 365 pour les opérations est l’application d’entreprise de Microsoft pour les entreprises. Il permet aux utilisateurs de rapidement prendre de meilleures décisions grâce à une interface d’utilisateur intelligent qui fournit l’accès à des analyses en temps réel et des données. Il permet aux entreprises de transformation et de l’innnovate en fournissant la logique métier éprouvée qui leur permet de remanier leurs processus métiers plus rapidement. Il permet aux entreprises de croissance à leur propre rythme et de faire des affaires de presque n’importe où, à tout moment, ce qui permet de les mettre à l’échelle de leurs opérations de manière globale afin de répondre aux besoins de l’entreprise. 

Ce guide peut vous aider à apprendre comment utiliser 365 Dynamics pour les opérations de votre entreprise.
