# <a name="track-commissions-in-pos-using-sales-groups"></a>Commissions de suivi dans POS utilisant des groupes de ventes

Il s'agit d'une pratique au détail courantes de suivre les ventes par l'associer travaillant à l'aide de le client approvisionnement, - vendant, croisée vendant, et traitant la transaction.

Suivi des ventes par le commercial est une mesure des associers vendant des capacités, tandis que les ventes par le caissier est une mesure de vitesse et de rendement. Les ventes suivies par le commercial sont également généralement utilisées pour calculer des commissions ou d'autres primes.

## <a name="configuring-a-worker-to-be-a-sales-representative-in-pos"></a>La configuration d'un travailleur pour être un commercial dans POS
Lorsqu'un travailleur est ajouté à un groupe de ventes, ils deviennent éligibles pour la commission et peuvent être identifiées comme commercial dans le système. Un travailleur qui n'est pas dans un groupe de ventes n'est pas admissible à la commission et n'est pas répertorié comme commercial dans l'application de (POS) du point de vente. Dans le POS, la liste des commerciaux est dérivée de tous les groupes de ventes contenant au moins un travailleur affecté au magasin. La liste est affichée dans POS comme combinaison d'identification groupe de ventes et de nom (ID : Nom). Un groupe de ventes par défaut peut être affecté aux travailleurs à des scénarios de support dans lequel le détaillant choisit de définir le commercial dans les lignes du PDV automatiquement. Les utilisateurs peuvent sélectionner n'importe quel groupe de vente auquel le travailleur est membre.

Un utilisateur peut néanmoins définir manuellement le groupe de ventes à l'aide d'un bouton de groupe de boutons du PDV.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| **Prompt for sales representative**   | Cette option a trois valeurs possibles : ** Aucun ** - Si cette option est sélectionnée, l'utilisateur ne sera pas invité à sélectionner un groupe de ventes. ## <a name="displaying-the-sales-representative-information-on-the-pos-transactions-screen"></a>Afficher les informations Commercial dans l'écran Transactions POS
La mise en page de l'écran et du contenu de transaction POS sont configurables à l'aide de le concepteur de mise en page de l'écran et mises en page de l'écran affectées aux magasins, aux registres, ou aux travailleurs. ** Commercial ** le champ peut être ajouté aux lignes onglet du volet Ticket de caisse.  Il affiche l'ID du groupe de ventes spécifié pour chaque ligne de l'écran de transaction.

## <a name="adding-sales-representative-operations-to-pos-button-grids"></a>Ajouter des opérations de commercial aux groupes de boutons du PDV
Le PDV permet aux utilisateurs de configurer les groupes de boutons, qui sont inclus dans les mises en page de l'écran pour donner accès aux opérations de PDV. Les opérations suivantes du PDV peuvent être affectées aux boutons du groupe de boutons qui concernent les commerciaux.

|                                           |                                                                                                                                                                                                                                                                                              |
|-------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Opération**                             | **Description**                                                                                                                                                                                                                                                                              |
| Définir le représentant sur la ligne          | Cette opération POS affiche une liste de groupes de ventes admissibles (ID : Nom) pour le magasin. Sélectionner un groupe de ventes de cette liste détermine la valeur de la ligne de transaction actuelle.                                                                                                            |
| Effacer le représentant sur la ligne        | Cette opération du PDV supprime la valeur du groupe de ventes de la ligne de transaction actuelle.                                                                                                                                                                                                  |
| Définissez le commercial sur la transaction   | Cette opération POS affiche une liste de groupes de ventes admissibles (ID : Nom) pour le magasin. Sélectionner un groupe de ventes de cette liste détermine la valeur par défaut sur la transaction actuelle. Toutes les lignes existantes sans groupe de ventes affecté sont définies, ainsi que toutes les lignes par la suite ajoutés. |
| Commercial clair dans la transaction | Cette opération du PDV supprime la valeur par défaut du groupe de ventes de la transaction actuelle. Elle n'a aucune ligne existant dans la transaction.                                                                                                                             |

## <a name="calculating-commissions"></a>Commissions calcul
La commission est calculée pour les travailleurs aux groupes de ventes spécifié lors de la validation du relevé ou de la validation de commande client. Montant de la commission est déterminé sur le partage de la commission du travailleur, comme défini dans le groupe de ventes et les paramètres associés de calcul de la commission pour le client et/ou les produits sur la transaction.
