# <a name="number-of-books-per-journal"></a><span data-ttu-id="b6b7f-104">Nombre de registres par journal</span><span class="sxs-lookup"><span data-stu-id="b6b7f-104">Number of books per journal</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="b6b7f-105">Cette rubrique décrit la relation entre les journaux et les registres d’immobilisations lorsque vous créez une acquisition d’immobilisations ou une proposition d’amortissement via un traitement par lots.</span><span class="sxs-lookup"><span data-stu-id="b6b7f-105">This topic describes the relationship between journals and asset books when you create a fixed asset acquisition or depreciation proposal through a batch job.</span></span> <span data-ttu-id="b6b7f-106">Vous pouvez définir le nombre maximal de registres inclus pour chaque acquisition et pour l’amortissement en utilisant les champs de la section **Nombre de registres par journal** sur l’onglet **Général** de la page **Paramètres des immobilisations** (**Immobilisations \> Configurer \> Paramètres des immobilisations**).</span><span class="sxs-lookup"><span data-stu-id="b6b7f-106">You can define the maximum number of books that are included for each acquisition and for depreciation by using the fields in the **Number of books per journal** section on the **General** tab of the **Fixed assets parameters** page (**Fixed assets \> Setup \> Fixed assets parameters**).</span></span> <span data-ttu-id="b6b7f-107">Ces champs vous permettent de répartir le nombre de registres d’immobilisation par journal d’acquisition et journal d’amortissement.</span><span class="sxs-lookup"><span data-stu-id="b6b7f-107">These fields let you distribute the number of asset books per acquisition journal and depreciation journal.</span></span>

<span data-ttu-id="b6b7f-108">Pour une proposition d’acquisition, la valeur par défaut est d’au moins 10 000 registres.</span><span class="sxs-lookup"><span data-stu-id="b6b7f-108">For an acquisition proposal, the default value is at least 10,000 books.</span></span> <span data-ttu-id="b6b7f-109">Pour une proposition d’amortissement, la valeur par défaut est d’au moins 2 000 registres.</span><span class="sxs-lookup"><span data-stu-id="b6b7f-109">For a depreciation proposal, the default value is at least 2,000 books.</span></span>

<span data-ttu-id="b6b7f-110">Par exemple, s’il y a 4 000 immobilisations et que deux registres sont associés à chaque actif, un registre sera publié dans la couche actuelle et l’autre registre sera publié dans la couche fiscale.</span><span class="sxs-lookup"><span data-stu-id="b6b7f-110">For example, if there are 4,000 fixed assets, and two books are associated with each asset, one book will be posted to the current layer, and the other book will be posted to the tax layer.</span></span> <span data-ttu-id="b6b7f-111">Si vous acquérez 4 000 immobilisations via un traitement par lots, le traitement par lots qui crée un journal d’acquisition d’immobilisations contiendra 4 000 registres d’immobilisations.</span><span class="sxs-lookup"><span data-stu-id="b6b7f-111">If you acquire 4,000 fixed assets through batch processing, the batch job that creates one fixed asset acquisition journal will contain 4,000 asset books.</span></span>

> [!NOTE]
> <span data-ttu-id="b6b7f-112">Le journal qui est créé continuera d’être utilisé jusqu’à ce que le travail par lots soit terminé.</span><span class="sxs-lookup"><span data-stu-id="b6b7f-112">The journal that is created will continue to be used until the batch job is completed.</span></span>
>
> <span data-ttu-id="b6b7f-113">Les registres dérivés ne sont pas inclus dans le nombre maximum de registres par revue.</span><span class="sxs-lookup"><span data-stu-id="b6b7f-113">The derived books aren't included in the maximum number of books per journal.</span></span>

<span data-ttu-id="b6b7f-114">Vous pouvez utiliser le traitement par lots pour exécuter l’amortissement pour le même ensemble d’immobilisations acquises.</span><span class="sxs-lookup"><span data-stu-id="b6b7f-114">You can use  batch processing to run depreciation for the same set of acquired assets.</span></span> <span data-ttu-id="b6b7f-115">Par exemple, un traitement par lots crée deux journaux d’amortissement, chacun se composant de 2 000 registres d’immobilisations.</span><span class="sxs-lookup"><span data-stu-id="b6b7f-115">For example, a batch job creates two depreciation journals, each of which consists of 2,000 asset books.</span></span> <span data-ttu-id="b6b7f-116">Le premier journal contiendra des registres associés aux immobilisations numérotées de 1 à 2 000.</span><span class="sxs-lookup"><span data-stu-id="b6b7f-116">The first journal will contain books that are associated with the fixed assets that are numbered 1 through 2,000.</span></span> <span data-ttu-id="b6b7f-117">Le second journal contiendra alors des registres associés aux immobilisations numérotées de 2 001 à 4 000.</span><span class="sxs-lookup"><span data-stu-id="b6b7f-117">The second journal will then contain books that are associated with the fixed assets that are numbered 2,001 through 4,000.</span></span>

<span data-ttu-id="b6b7f-118">Le travail de traitement par lots exclut les registres fermés.</span><span class="sxs-lookup"><span data-stu-id="b6b7f-118">The batch processing job excludes closed books.</span></span> <span data-ttu-id="b6b7f-119">Par exemple, dans un traitement par lots pour l’amortissement, 10 des 2 000 premiers registres sont fermés.</span><span class="sxs-lookup"><span data-stu-id="b6b7f-119">For example, in a batch job for depreciation, 10 of the first 2,000 books are closed.</span></span> <span data-ttu-id="b6b7f-120">Dans le cas, le premier journal contiendra des registres associés aux immobilisations numérotées de 1 à 2 011.</span><span class="sxs-lookup"><span data-stu-id="b6b7f-120">In this case, the first journal will contain books that are associated with the fixed assets that are numbered 1 through 2,011.</span></span> <span data-ttu-id="b6b7f-121">Le second journal contiendra alors des registres associés aux immobilisations numérotées de 2 012 à 4 000.</span><span class="sxs-lookup"><span data-stu-id="b6b7f-121">The second journal will then contain books that are associated with the fixed assets that are numbered 2,012 through 4,000.</span></span>

<span data-ttu-id="b6b7f-122">La limite du nombre de registres est appliquée si les ID d’éléments en double n’existent pas dans le même journal.</span><span class="sxs-lookup"><span data-stu-id="b6b7f-122">The limit on the number of books is applied if duplicate asset IDs don't exist in the same journal.</span></span> <span data-ttu-id="b6b7f-123">Cependant, si l’ID d’actif est le même que l’ID de registres, le nombre de registres par journal peut être dépassé pour conserver l’ID d’actif dans le même journal.</span><span class="sxs-lookup"><span data-stu-id="b6b7f-123">However, if the asset ID is the same as the book ID, the number of books per journal can be exceeded to keep the asset ID in the same journal.</span></span>

<span data-ttu-id="b6b7f-124">Par exemple, il existe 5 001 ID d’immobilisation, trois registres sont associés à chaque ID d’immobilisation et chaque registre d’immobilisation est enregistré dans la même couche de comptabilisation.</span><span class="sxs-lookup"><span data-stu-id="b6b7f-124">For example, there are 5,001 fixed asset IDs, three books are associated with each fixed asset ID, and each asset book is posted to the same posting layer.</span></span> <span data-ttu-id="b6b7f-125">Vous exécutez l’amortissement pendant trois mois consécutifs, sans synthèse.</span><span class="sxs-lookup"><span data-stu-id="b6b7f-125">You run depreciation for three consecutive months, without summarizing.</span></span>  <span data-ttu-id="b6b7f-126">Le journal d’amortissement sera créé via un traitement par lots, et le système créera sept journaux contenant 667 ID d’immobilisation et trois registres pour chaque ID d’immobilisation.</span><span class="sxs-lookup"><span data-stu-id="b6b7f-126">The depreciation journal will be created through a batch job, and the system will create seven journals that have 667 fixed asset IDs and three books for each fixed asset ID.</span></span> <span data-ttu-id="b6b7f-127">Le résultat sera 2 001 registres.</span><span class="sxs-lookup"><span data-stu-id="b6b7f-127">The result will be 2,001 books.</span></span> <span data-ttu-id="b6b7f-128">Par conséquent, dans trois mois, il y aura 6 003 lignes de journal pour conserver les mêmes ID d’actif dans le même journal.</span><span class="sxs-lookup"><span data-stu-id="b6b7f-128">Therefore, in three months, there will be 6,003 journal lines to maintain the same asset IDs in the same journal.</span></span> <span data-ttu-id="b6b7f-129">Le système créera également un journal contenant 332 ID d’immobilisation et trois registres pour chaque ID d’immobilisation.</span><span class="sxs-lookup"><span data-stu-id="b6b7f-129">The system will also create one journal that has 332 fixed asset IDs and three books for each fixed asset ID.</span></span> <span data-ttu-id="b6b7f-130">Dans trois mois, il y aura 2 988 lignes.</span><span class="sxs-lookup"><span > [!Note] 
> <span data-ttu-id="b6b7f-131">Si le paramètre **Cumuler l’amortissement** est activé lorsque vous créez une proposition d'amortissement, alors la valeur du champ **Nombre de registres par journal - Proposition d'amortissement** n'a aucun effet.</span><span class="sxs-lookup"><span data-stu-id="b6b7f-131">If the **Summarize depreciation** parameter is turned on when you're creating a depreciation proposal, then the value in the **Number of books per journal - Depreciation proposal** field has no effect.</span></span> <span data-ttu-id="b6b7f-132">Dans ce cas, le nombre de livres par journal est de 6000, qui est la limite interne définie.</span><span class="sxs-lookup"><span data-stu-id="b6b7f-132">In this case number of books per journal is 6000, which is the internal defined limit.</span></span>
