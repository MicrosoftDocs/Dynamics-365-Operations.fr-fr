# <a name="fixed-asset-mass-update"></a><span data-ttu-id="af8b9-103">Mise à jour collective des immobilisations</span><span class="sxs-lookup"><span data-stu-id="af8b9-103">Fixed asset mass update</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="af8b9-104">Si vous utilisez des registres, vous pouvez modifier les conventions d’amortissement pour les groupes d’immobilisations faisant partie du même registre.</span><span class="sxs-lookup"><span data-stu-id="af8b9-104">If you use books, you can change the depreciation conventions for groups of assets that are part of the same book.</span></span>

<span data-ttu-id="af8b9-105">Par exemple, si vous vous trouvez aux États-Unis et que vous placez plus de 40 pour cent de vos immobilisations en service durant le quatrième trimestre de l’année, vous devez utiliser la convention d’amortissement à mi-trimestre.</span><span class="sxs-lookup"><span data-stu-id="af8b9-105">For example, if you are in the United States, and you put more than 40 percent of your assets in service during the fourth quarter of the year, you must use the mid-quarter depreciation convention.</span></span> <span data-ttu-id="af8b9-106">Vous pouvez utiliser le processus pour effectuer une mise à jour collective afin de modifier toutes les immobilisations nécessitant la nouvelle convention d’amortissement.</span><span class="sxs-lookup"><span data-stu-id="af8b9-106">You can use the process for a mass update to change all assets that require the new depreciation convention.</span></span> 

<span data-ttu-id="af8b9-107">Lorsque vous mettez à jour la convention d’amortissement pour les actifs, vous supprimez toutes les transactions d’amortissement qui existent pour ces actifs.</span><span class="sxs-lookup"><span data-stu-id="af8b9-107">When you update the depreciation convention for assets, you delete all depreciation transactions that exist for those assets.</span></span> <span data-ttu-id="af8b9-108">Vous supprimez également toutes les transactions d’ajustement de l’amortissement, les transactions d’amortissement de la prime et les transactions d’amortissement exceptionnel pour ces actifs.</span><span class="sxs-lookup"><span data-stu-id="af8b9-108">You also delete all transactions for depreciation adjustments, transactions for bonus depreciation, and transactions for extraordinary depreciation for those assets.</span></span> 

<span data-ttu-id="af8b9-109">Pour mettre à jour la convention d’amortissement pour des immobilisations déjà cédées, vous devez commencer par supprimer les transaction de cession existantes.</span><span class="sxs-lookup"><span data-stu-id="af8b9-109">To update the depreciation convention for assets that have already been disposed of, you must first delete the existing disposal transactions.</span></span> <span data-ttu-id="af8b9-110">Vous devez également supprimer toutes les transactions générées en raison du processus de cession.</span><span class="sxs-lookup"><span data-stu-id="af8b9-110">You must also delete all transactions that were generated because of the disposal process.</span></span> 

<span data-ttu-id="af8b9-111">Une fois la convention d’amortissement mise à jour pour les immobilisations, vous pouvez traiter l’amortissement et un amortissement exceptionnel pour chaque immobilisation.</span><span class="sxs-lookup"><span data-stu-id="af8b9-111">After you update the depreciation convention for assets, you can process depreciation and extraordinary depreciation for each asset.</span></span> <span data-ttu-id="af8b9-112">Vous pouvez également opérer des ajustements d’amortissement manuels si nécessaire.</span><span class="sxs-lookup"><span data-stu-id="af8b9-112">You can also make manual depreciation adjustments, if any adjustments are required.</span></span>
