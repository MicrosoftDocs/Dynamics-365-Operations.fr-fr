# <a name="dispose-of-a-fixed-asset-using-a-free-text-invoice"></a><span data-ttu-id="180e5-103">Céder des immobilisations à l’aide d’une facture financière</span><span class="sxs-lookup"><span data-stu-id="180e5-103">Dispose of a fixed asset using a free text invoice</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="180e5-104">Cette rubrique explique comment céder une immobilisation à l’aide de la facture financière.</span><span class="sxs-lookup"><span data-stu-id="180e5-104">This topic explains how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="180e5-105">Dans le volet de navigation, allez dans **Modules > Comptabilité client > Factures > Toutes factures financières**.</span><span class="sxs-lookup"><span data-stu-id="180e5-105">In the navigation pane, go to **Modules > Accounts receivable > Invoices > All free text invoices**.</span></span>
2. <span data-ttu-id="180e5-106">Sélectionnez **Nouveau**.</span><span class="sxs-lookup"><span data-stu-id="180e5-106">Select **New**.</span></span>
3. <span data-ttu-id="180e5-107">Entrez ou sélectionnez une valeur dans le champ **Compte client**.</span><span class="sxs-lookup"><span data-stu-id="180e5-107">In the **Customer account** field, enter or select a value.</span></span>
4. <span data-ttu-id="180e5-108">Validez la date de **facture** par défaut et modifiez-la, le cas échéant.</span><span class="sxs-lookup"><span data-stu-id="180e5-108">Validate the default **Invoice** date and edit if applicable.</span></span>
5. <span data-ttu-id="180e5-109">Validez les champs d’en-tête par défaut restants, comme la **devise**, et modifiez-les, le cas échéant.</span><span class="sxs-lookup"><span data-stu-id="180e5-109">Validate remaining default header fields, such as **Currency** and edit if applicable.</span></span>
6. <span data-ttu-id="180e5-110">Dans la section **Lignes de facture**, validez les champs **Description** et **Compte principal**.</span><span class="sxs-lookup"><span data-stu-id="180e5-110">In the **Invoice lines** section, validate the **Description** and **Main account** fields.</span></span>
7. <span data-ttu-id="180e5-111">Validez les champs **Groupe de taxe** et **Groupe de taxe d’article**.</span><span class="sxs-lookup"><span data-stu-id="180e5-111">Validate the default **Sales tax group** and **Item sales tax group** fields.</span></span>
8. <span data-ttu-id="180e5-112">Entrez le **prix unitaire** ou le **montant** de la vente de l’immobilisation.</span><span class="sxs-lookup"><span data-stu-id="180e5-112">Enter the **Unit price** or the **Amount** of the sale of the fixed asset.</span></span>
9. <span data-ttu-id="180e5-113">Sélectionnez la section **Détails de ligne**.</span><span class="sxs-lookup"><span data-stu-id="180e5-113">Select the **Line details** section.</span></span>  
10. <span data-ttu-id="180e5-114">Entrez ou sélectionnez une valeur dans le champ **Immobilisation**.</span><span class="sxs-lookup"><span data-stu-id="180e5-114">Enter or select a value in the **Fixed asset** field.</span></span>
11. <span data-ttu-id="180e5-115">Sélectionnez **Valider**.</span><span class="sxs-lookup"><span data-stu-id="180e5-115">Select **Post**.</span></span>
