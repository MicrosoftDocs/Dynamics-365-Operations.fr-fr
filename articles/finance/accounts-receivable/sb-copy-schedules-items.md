# <a name="copy-a-billing-schedule"></a>Copier un échéancier de facturation

[!include [banner](../includes/banner.md)]

Vous pouvez utiliser la page **Copier le calendrier** pour créer un nouveau calendrier de facturation à partir d’un calendrier de facturation existant. Le nouveau calendrier de facturation aura les mêmes paramètres que le calendrier de facturation d’origine. Vous pouvez également utiliser la page pour copier des éléments d’un calendrier de facturation à un autre.

## <a name="copy-a-billing-schedule"></a>Copier un échéancier de facturation

Pour copier un calendrier de facturation, procédez comme suit.

1. Sur la page de liste **Tous les calendriers de facturation** ou **Calendriers de facturation actifs**, sélectionnez un calendrier de facturation existant.
2. Sur l’onglet **Calendrier de facturation**, sélectionnez **Copier le calendrier**.
3. Dans le champ **Compte client**, sélectionnez le compte client.
4. Mettez à jour les dates de début et de fin de facturation au besoin.
5. Mettez à jour les valeurs des champs facultatifs suivants, si nécessaire : **Compte utilisateur final**, **Compte partenaire**, **Référence client**, **Demande client** et **Date de début du report**.
6. Sélectionnez **Voir l’aperçu** pour afficher et sélectionner les lignes de calendrier de facturation que vous souhaitez copier.
7. Sélectionnez **Copie**.

Une fois le processus de copie terminé, le nouveau calendrier de facturation apparaît au bas de la page de liste **Tous les calendriers de facturation** ou **Calendriers de facturation actifs**, et sur la page **Calendriers de facturation**.

## <a name="copy-items-from-one-billing-schedule-to-another"></a>Copier des éléments d’un calendrier de facturation à un autre

1. Sur la page de liste **Tous les calendriers de facturation** ou **Calendriers de facturation actifs**, sélectionnez un calendrier de facturation existant.
2. Sur l’onglet **Calendrier de facturation**, sélectionnez **Copier le calendrier**.
3. Dans le champ **Compte client**, sélectionnez le compte client.
4. Dans le champ **Numéro de calendrier de facturation**, sélectionnez un numéro de calendrier de facturation.
4. Mettez à jour les dates de début et de fin de facturation.
5. Mettez à jour les valeurs des champs suivants, si nécessaire : **Compte utilisateur final**, **Compte partenaire**, **Référence client**, **Demande client** et **Date de début du report**.
6. Sélectionnez **Voir l’aperçu** pour afficher et sélectionner les lignes de calendrier de facturation que vous souhaitez copier vers un calendrier de facturation existant.
7. Sélectionnez **Copie**.

Une fois le processus de copie terminé, vérifiez le calendrier de facturation dans lequel vous avez copié les éléments et assurez-vous que les éléments ont été ajoutés comme prévu.
