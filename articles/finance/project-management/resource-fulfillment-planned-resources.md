# <a name="resource-fulfillment-for-planned-resources"></a><span data-ttu-id="a97ae-103">Traitement de ressources pour les ressources planifiées</span><span class="sxs-lookup"><span data-stu-id="a97ae-103">Resource fulfillment for planned resources</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="a97ae-104">Un chef de projet peut organiser des rôles de ressources requis pour un projet.</span><span class="sxs-lookup"><span data-stu-id="a97ae-104">A project manager can plan required resource roles for a project.</span></span> <span data-ttu-id="a97ae-105">Le gestionnaire de ressources verra ces ressources planifiées en tant que demandes dans la page **Traitement des ressources** et peut affecter des ressources réelles.</span><span class="sxs-lookup"><span data-stu-id="a97ae-105">The resource manager will see these planned resources as requests on the **Resource fulfillment** page and can assign actual resources.</span></span>

1. <span data-ttu-id="a97ae-106">Sur la page **Tous les projets**, sélectionnez le projet **Phase 2 de mise à niveau de XYZ**.</span><span class="sxs-lookup"><span data-stu-id="a97ae-106">On the **All projects** page, select the **XYZ Upgrade Phase 2** project.</span></span>
2. <span data-ttu-id="a97ae-107">Sélectionnez **Projet**, puis sélectionnez **Modifier**.</span><span class="sxs-lookup"><span data-stu-id="a97ae-107">Select **Project**, and then select **Edit**.</span></span>
3. <span data-ttu-id="a97ae-108">Sous l’onglet **Équipe de projet et planification**, sélectionnez **Ajouter**.</span><span class="sxs-lookup"><span data-stu-id="a97ae-108">On the **Project team and scheduling** tab, select **Add**.</span></span>
4. <span data-ttu-id="a97ae-109">Dans la boîte de dialogue **Ajoutez des rôles**, sélectionnez le rôle **Développeur logiciel**.</span><span class="sxs-lookup"><span data-stu-id="a97ae-109">In the **Add roles** dialog box, select the **Software developer** role.</span></span>
5. <span data-ttu-id="a97ae-110">Sélectionnez **Créer**, puis fermez la page du projet.</span><span class="sxs-lookup"><span data-stu-id="a97ae-110">Select **Create**, and then close the project page.</span></span>
6. <span data-ttu-id="a97ae-111">Sur la page **Traitement des ressources**, sélectionnez **Développeur de logiciel 1** pour le projet **Phase 2 de projet de mise à niveau de XYZ**.</span><span class="sxs-lookup"><span data-stu-id="a97ae-111">On the **Resource fulfillment** page, select **Software developer 1** for the **XYZ Upgrade project Phase 2** project.</span></span>
7. <span data-ttu-id="a97ae-112">Sélectionnez un collaborateur, puis sélectionnez **Affecter**.</span><span class="sxs-lookup"><span data-stu-id="a97ae-112">Select a worker, and then select **Assign**.</span></span>
8. <span data-ttu-id="a97ae-113">Vérifiez que la ligne pour **Programmateur logiciel 1** a été supprimée du projet **Phase 2 de projet de mise à niveau de XYZ**.</span><span class="sxs-lookup"><span data-stu-id="a97ae-113">Verify that the line for **Software developer 1** has been removed for the **XYZ Upgrade project Phase 2** project.</span></span>
9. <span data-ttu-id="a97ae-114">Sous l’onglet **Équipe de projet et planification**, pour le projet **Phase 2 de mise à niveau de XYZ**, vérifiez que le collaborateur sélectionné à l’étape précédente a été ajouté comme **Programmateur logiciel**.</span><span class="sxs-lookup"><span data-stu-id="a97ae-114">On the **Project team and scheduling** tab, for the **XYZ Upgrade Phase 2** project, verify that the worker that you selected in the previous step has been added as **Software developer**.</span></span>
