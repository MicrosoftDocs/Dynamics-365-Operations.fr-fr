[!include [banner](../includes/banner.md)]

<span data-ttu-id="f0cee-104">Après avoir créé et envoyé un devis de projet à un client, vous devez obtenir une confirmation de celui-ci avant de mettre à jour le statut du devis sur **Envoyé**.</span><span class="sxs-lookup"><span data-stu-id="f0cee-104">After you create and send a project quotation to a customer, you must get confirmation from the customer before you update the quotation status to **Sent**.</span></span> <span data-ttu-id="f0cee-105">Toute modification demandée par le client peut être mise à jour dans le devis.</span><span class="sxs-lookup"><span data-stu-id="f0cee-105">Any modifications requested by the customer can be updated in the quotation.</span></span> <span data-ttu-id="f0cee-106">Après avoir mis à jour le statut du devis sur **Envoyé**, vous ne pouvez pas le modifier.</span><span class="sxs-lookup"><span data-stu-id="f0cee-106">After you update the quotation status to **Sent**, you can’t modify it.</span></span> <span data-ttu-id="f0cee-107">La procédure suivante décrit les options permettant d'envoyer un devis pour confirmation, d'effectuer des mises à jour en fonction des commentaires reçus, puis d'envoyer le devis.</span><span class="sxs-lookup"><span data-stu-id="f0cee-107">The following procedure describes the options to send a quotation for confirmation, make updates based on feedback, and then send the quotation.</span></span>

<span data-ttu-id="f0cee-109">Vous pouvez envoyer un devis de projet existant pour confirmation par e-mail ou sous forme imprimée.</span><span class="sxs-lookup"><span data-stu-id="f0cee-109">You can send an existing project quotation for confirmation through email or as a printed copy.</span></span>

6. <span data-ttu-id="f0cee-116">Dans la page **Paramètres de destination d'impression**, sélectionnez **OK**.</span><span class="sxs-lookup"><span data-stu-id="f0cee-116">On the **Print destination settings** page, select **OK**.</span></span>
