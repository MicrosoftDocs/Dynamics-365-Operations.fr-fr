## <a name="set-up-competencies-certificates"></a><span data-ttu-id="8881f-112">Paramétrer des compétences : Certificats</span><span class="sxs-lookup"><span data-stu-id="8881f-112">Set up competencies: Certificates</span></span>

1. <span data-ttu-id="8881f-113">Dans la page de liste **Collaborateurs**, sélectionnez la ligne du collaborateur pour lequel vous ajoutez des informations de certificat.</span><span class="sxs-lookup"><span data-stu-id="8881f-113">On the **Workers** list page, select the line for the worker to add certificate information for.</span></span>

4. <span data-ttu-id="8881f-116">Dans le champ **Date de début**, sélectionnez **10/01/2015**, puis sélectionnez **Enregistrer**.</span><span class="sxs-lookup"><span data-stu-id="8881f-116">In the **Start date** field, select **10/1/2015**, and select **Save**.</span></span>

## <a name="set-up-competencies-skills"></a><span data-ttu-id="8881f-117">Paramétrer des compétences : Qualifications</span><span class="sxs-lookup"><span data-stu-id="8881f-117">Set up competencies: Skills</span></span>

2. <span data-ttu-id="8881f-120">Sélectionnez **Nouveau**.</span><span class="sxs-lookup"><span data-stu-id="8881f-120">Select **New**.</span></span>
3. <span data-ttu-id="8881f-121">Dans le champ **Qualification**, sélectionnez **Gestion de projet**.</span><span class="sxs-lookup"><span data-stu-id="8881f-121">In the **Skill** field, select **Project management**.</span></span>
4. <span data-ttu-id="8881f-122">Dans le champ **Niveau**, sélectionnez **5 Expert**.</span><span class="sxs-lookup"><span data-stu-id="8881f-122">In the **Level** field, select **5 Expert**.</span></span>

7. <span data-ttu-id="8881f-125">Sélectionnez **Sauvegarder**, puis fermez la page.</span><span class="sxs-lookup"><span data-stu-id="8881f-125">Select **Save**, and then close the page.</span></span>
