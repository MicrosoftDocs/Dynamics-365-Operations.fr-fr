# <a name="view-journal-entries-or-transactions"></a>Afficher les entrées de journal ou les transactions

[!include [banner](../../includes/banner.md)]

Cette procédure indique comment utiliser la demande de pièces comptables pour rechercher des entrées ou des transactions de journal.

1. Accédez au **Volet de navigation > Modules > Comptabilité > Recherches et états > Pièces comptables**.
2. Sélectionnez le champ pour lequel vous souhaitez définir des critères de filtre.
3. Saisissez vos critères de filtre pour le champ sélectionné. Vous pouvez filtrer selon une valeur unique ou une plage. Lorsque vous définissez une plage, vérifiez que la syntaxe appropriée est utilisée. Les valeurs doivent être séparées par deux points (..).  
4. Cliquez sur l’onglet **Jointures** pour ajouter des tables supplémentaires à partir desquelles appliquer un filtre.
5. Dans l’arborescence, sélectionnez **Tables/Entrée de journal des opérations diverses**.
6. Cliquez sur **Ajouter une jointure de table**.
7. Cliquez sur **Annuler** si vous décidez de ne pas ajouter une table supplémentaire.
8. Cliquez sur l’onglet **Plage**.
9. Cliquez sur **OK** pour exécuter la requête.
10. Dans le volet Actions, cliquez sur **Origine de la transaction**. Divers boutons relatifs à la grille peuvent être utilisés pour rechercher des informations supplémentaires sur l’enregistrement sélectionné du N° document. Il se peut que certains boutons ne soient pas disponibles, selon le type de transaction et les caractéristiques de la transaction.
11. Fermez la page.
12. Dans le volet Actions, cliquez sur **Document d’origine**.
13. Fermez la page.
