# <a name="create-and-validate-journals"></a>Créer et valider des journaux

[!include [banner](../../includes/banner.md)]

Cette procédure crée et valide les journaux et les lignes de journal. Vous pouvez essayer cette procédure à l’aide de la société fictive USMF.  

1. Accédez à **Comptabilité > Entrées de journal > Journaux des opérations diverses.**.
2. Cliquez sur **Nouveau**.
3. Dans le champ **Nom**, cliquez sur le bouton de liste déroulante pour ouvrir la recherche.
4. Dans la liste, recherchez et sélectionnez l’enregistrement souhaité.
5. Cliquez sur **Lignes**.
6. Dans le champ **Compte**, entrez un compte approprié en fonction du type de compte.
7. Tapez une valeur dans le champ **Description**.
8. Entrez un montant pour le compte, soit dans **Débit**, soit dans **Crédit**. 
9. Dans le champ **Compte de contrepartie**, entrez un compte approprié en fonction du type de compte de contrepartie.
10. Cliquez sur **Valider**.
11. Cliquez sur **Valider**.
12. Cliquez sur **Valider**.
13. Cliquez **N° document**.
