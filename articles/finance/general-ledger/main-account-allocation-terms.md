# <a name="allocation-terms"></a><span data-ttu-id="22864-103">Conditions de répartition</span><span class="sxs-lookup"><span data-stu-id="22864-103">Allocation terms</span></span>

<span data-ttu-id="22864-104">Cette rubrique fournit des informations sur l’utilisation des conditions de répartition sur un compte principal.</span><span class="sxs-lookup"><span data-stu-id="22864-104">This topic provides information about using allocation terms on a main account.</span></span> <span data-ttu-id="22864-105">Les conditions de répartition permettent de répartir des montants entre plusieurs combinaisons de comptes généraux.</span><span class="sxs-lookup"><span data-stu-id="22864-105">Allocation terms are used to distribute amounts across multiple ledger account combinations.</span></span> <span data-ttu-id="22864-106">Elles aident à garantir que les dépenses ou les revenus sont imputés à l’objet approprié dans la comptabilité.</span><span class="sxs-lookup"><span data-stu-id="22864-106">They help ensure that expenses or revenues are charged to the correct object in accounting.</span></span>

<span data-ttu-id="22864-107">Chaque condition de répartition que vous créez sur un compte principal définit le pourcentage d’un N° document à répartir à partir d’un compte principal à source unique et d’une combinaison de dimensions financières.</span><span class="sxs-lookup"><span data-stu-id="22864-107">Each allocation term that you create on a main account defines the percentage of a voucher to be allocated from a single-source main account and a combination of financial dimensions.</span></span> <span data-ttu-id="22864-108">En outre, vous définissez le compte principal de destination et les dimensions financières sur lesquels le montant va être réparti.</span><span class="sxs-lookup"><span data-stu-id="22864-108">In addition, you define the destination main account and financial dimensions where the amount will be allocated.</span></span> 

<span data-ttu-id="22864-109">Lorsque vous définissez la dimension financière source pour la répartition, vous pouvez sélectionner si chaque dimension est spécifique ou non.</span><span class="sxs-lookup"><span data-stu-id="22864-109">When you define the source financial dimension for the allocation, you can select if each dimension is specific or unspecific.</span></span> <span data-ttu-id="22864-110">Spécifique indique que la dimension financière de la transaction source doit correspondre à la dimension sélectionnée.</span><span class="sxs-lookup"><span data-stu-id="22864-110">Specific indicates that the financial dimension for the source transaction must match the selected dimension.</span></span> <span data-ttu-id="22864-111">Lorsque vous sélectionnez Non spécifique, cela indique que toute valeur de la dimension financière peut contribuer à une correspondance.</span><span class="sxs-lookup"><span data-stu-id="22864-111">When you select unspecific, it indicates that any value for the financial dimension can contribute to a match.</span></span>

<span data-ttu-id="22864-112">Lorsque vous définissez le compte général de destination pour une condition de répartition, vous devez spécifier le compte principal sur lequel vous répartissez le data-ttu-id="22864-114">Si le même compte principal est utilisé, un avertissement s’affiche lors de la sauvegarde de l’enregistrement.</span><span class="sxs-lookup"><span data-stu-id="22864-114">If <span data-ttu-id="22864-119">Il n’y a pas de limite au nombre de conditions de répartition que vous pouvez ajouter à un compte principal ; cependant, la somme des pourcentages à répartir sur une condition d’allocation ne peut pas dépasser 100.</span><span class="sxs-lookup"><span data-stu-id="22864-119">There is no limit to the number of allocations terms that you can add to a main account; however, the sum of the percent to allocate on an allocation term cannot exceed 100.</span></span> <span data-ttu-id="22864-120">Vous pouvez créer des répartitions pour moins de 100 % si une partie du montant du N° document d’origine doit rester dans les dimensions financières source.</span><span class="sxs-lookup"><span data-stu-id="22864-120">You can create allocations for less than 100 percent if a portion of the original voucher amount should remain in the source financial dimensions.</span></span> <span data-ttu-id="22864-121">Des conditions de répartition peuvent être ajoutées à un compte principal en tant que remplacement d’entité juridique.</span><span class="sxs-lookup"><span data-stu-id="22864-121">Allocations terms can be added to a main account as a legal entity override.</span></span> <span data-ttu-id="22864-122">Si vous utilisez un plan comptable partagé, les conditions de répartition doivent être définies pour chaque entité juridique.</span><span class="sxs-lookup"><span data-stu-id="22864-122">If you are using a shared chart of accounts, allocation terms must be defined for each legal entity.</span></span> <span data-ttu-id="22864-123">Pour accéder au bouton **Conditions de répartition**, vous devez d’abord activer la case à cocher **Répartition** dans la section Remplacement d’entité juridique.</span><span class="sxs-lookup"><span data-stu-id="22864-123">To access the **Allocation terms** button, you must first select the **Allocation** check box on the Legal entity override.</span></span>

## <a name="allocation-term-example"></a><span data-ttu-id="22864-124">Exemple de condition de répartition</span><span class="sxs-lookup"><span data-stu-id="22864-124">Allocation term example</span></span>
