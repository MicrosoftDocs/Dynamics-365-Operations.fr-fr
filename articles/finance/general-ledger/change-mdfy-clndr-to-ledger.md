<span data-ttu-id="1b625-108">Il existe deux scénarios pour modifier ou réaffecter un calendrier comptable.</span><span class="sxs-lookup"><span data-stu-id="1b625-108">There are two scenarios for changing or reassigning a ledger calendar.</span></span> <span data-ttu-id="1b625-109">Le premier scénario consiste à modifier un calendrier existant déjà affecté à la comptabilité.</span><span class="sxs-lookup"><span data-stu-id="1b625-109">The first scenario involves changing an existing calendar that is already assigned to the ledger.</span></span> <span data-ttu-id="1b625-110">Le deuxième scénario consiste à créer un nouveau calendrier et à l’affecter à la comptabilité.</span><span class="sxs-lookup"><span data-stu-id="1b625-110">The second scenario involves creating a new calendar and assigning it to the ledger.</span></span> <span data-ttu-id="1b625-111">Les deux modifications peuvent être effectuées à tout moment, même après que les transactions ont été validées dans la comptabilité.</span><span class="sxs-lookup"><span data-stu-id="1b625-111">Both changes can be done at any time, even after transactions have been posted to the ledger.</span></span>

### <a name="change-an-existing-fiscal-calendar"></a><span data-ttu-id="1b625-112">Modifier un calendrier fiscal existant</span><span class="sxs-lookup"><span data-stu-id="1b625-112">Change an existing fiscal calendar</span></span>

<span data-ttu-id="1b625-113">Pour modifier le calendrier affecté à votre comptabilité, accédez à **Comptabilité \> Configuration de la comptabilité \> Comptabilité**, puis sélectionnez le lien du calendrier fiscal pour ouvrir la page **Calendriers comptables**.</span><span class="sxs-lookup"><span data-stu-id="1b625-113">To change an existing calendar that is assigned to your ledger, go to **General ledger \> Ledger setup \> Ledger**, and select the link for the fiscal calendar to open the **Ledger calendars** page.</span></span>

<span data-ttu-id="1b625-114">Les modifications que vous pouvez apporter au calendrier sont soumises à certaines conditions.</span><span class="sxs-lookup"><span data-stu-id="1b625-114">There are limits to the changes that can be made to a calendar.</span></span> <span data-ttu-id="1b625-115">Par exemple, vous pouvez modifier les dates de début et de fin des *périodes* dans une année, mais vous ne pouvez pas modifier les dates de début et de fin d’une *année* dans le calendrier.</span><span class="sxs-lookup"><span data-stu-id="1b625-115">For example, you can change the start and end dates of the *periods* in a year, but you can't change the start and end dates of a *year* in the calendar.</span></span>

<span data-ttu-id="1b625-116">Pour modifier les périodes dans une année, sélectionnez le calendrier et l’année appropriés.</span><span class="sxs-lookup"><span data-stu-id="1b625-116">To change the periods in a year, select the appropriate calendar and year.</span></span> <span data-ttu-id="1b625-117">Tout d’abord, utilisez le bouton **Supprimer la période** pour supprimer tout ou partie des périodes de fonctionnement existantes.</span><span class="sxs-lookup"><span data-stu-id="1b625-117">First, use the use the **Delete period** button to delete some or all of the existing operating periods.</span></span> <span data-ttu-id="1b625-118">Toutes les périodes de fonctionnement sauf la première peuvent être supprimées.</span><span class="sxs-lookup"><span data-stu-id="1b625-118">All operating periods except the first can be deleted.</span></span> <span data-ttu-id="1b625-119">Ensuite, utilisez le bouton **Diviser la période** pour diviser la ou les périodes restantes en nouvelles périodes en saisissant une date de début appropriée pour la période suivante.</span><span class="sxs-lookup"><span data-stu-id="1b625-119">Then use the **Divide period** button to divide the remaining period or periods into new periods by entering an appropriate start date for the next period.</span></span>

<span data-ttu-id="1b625-120">Après avoir modifié les périodes dans un calendrier, vous devez exécuter le processus **Recalculer les périodes comptables** dans chaque entité juridique (entreprise) à laquelle le calendrier est attribué.</span><span class="sxs-lookup"><span data-stu-id="1b625-120">After you change the periods in a calendar, you must run the **Recalculate ledger periods** process in every legal entity (company) that the calendar is assigned to.</span></span> <span data-ttu-id="1b625-121">Bien qu’aucun message d’avertissement ne vous rappelle d’effectuer cette étape, le recalcul est nécessaire pour mettre à jour la période affectée à chaque transaction validée dans la comptabilité.</span><span class="sxs-lookup"><span data-stu-id="1b625-121">Although no warning message reminds you to complete this step, the recalculation process is required to update the period that is assigned to each posted transaction in General ledger.</span></span> <span data-ttu-id="1b625-122">Pour exécuter le recalcul, sélectionnez **Recalculer les périodes comptables** sur la page **Configuration de la comptabilité** dans chaque entreprise.</span><span class="sxs-lookup"><span data-stu-id="1b625-122">To run the recalculation process, select **Recalculate ledger periods** on the **Ledger setup** page in each company.</span></span>

<span data-ttu-id="1b625-123">Si le recalcul n’est pas exécuté, les soldes de transaction sur une balance comptable ou sur les tableaux d’analyse risquent d’être incorrectement inclus dans les périodes.</span><span class="sxs-lookup"><span data-stu-id="1b625-123">If the recalculation process isn't run, transaction balances on a trial balance or on financial statements might be incorrectly included in periods.</span></span> <span data-ttu-id="1b625-124">Dans ce cas, vous pouvez corriger les soldes à tout moment en exécutant le recalcul.</span><span class="sxs-lookup"><span data-stu-id="1b625-124">In this case, you can correct the balances at any time by running the recalculation process.</span></span>
