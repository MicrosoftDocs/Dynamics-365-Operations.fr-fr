# <a name="create-a-preliminary-budget-for-public-sector"></a>Créer un budget préliminaire pour le secteur public

[!include [banner](../../includes/banner.md)]

Vous pouvez créer des écritures de registre budgétaires préliminaires pour un modèle de budget et des valeurs de dimension spécifiques. Une fois le budget réel approuvé, vous pouvez créer des écritures de registre budgétaires d'origine. Cette procédure a été créée à l'aide des données de la société fictive PSUS dans la partition du secteur public.

1. Accédez à Budgétisation > Écritures de registre budgétaires.
2. Cliquez sur Nouveau.
3. Dans le champ Modèle de budget, cliquez sur le bouton de liste déroulante pour ouvrir la recherche.
4. Dans la liste, recherchez et sélectionnez l'enregistrement souhaité.
5. Dans le champ Code budgétaire, cliquez sur le bouton de liste déroulante pour ouvrir la recherche.
6. Dans la liste, recherchez et sélectionnez l'enregistrement souhaité.
7. Dans le champ Code motif, cliquez sur le bouton de liste déroulante pour ouvrir la recherche.
8. Dans la liste, cliquez sur l'enregistrement de votre choix.
9. Cliquez sur Enregistrer.
10. Cliquez sur Ajouter une ligne.
    * Facultatif : si vous souhaitez modifier la date figurant dans l'en-tête, spécifiez une nouvelle date. Cette date détermine la période fiscale dans laquelle le budget sera enregistré. Lorsque vous affichez le guide de tâche, et que vous souhaitez renseigner d'autres champs, cliquez sur Déverrouiller en haut de la page.  
11. Dans le champ Structure de compte, cliquez sur le bouton de liste déroulante pour ouvrir la recherche.
12. Dans la liste, recherchez et sélectionnez l'enregistrement souhaité.
13. Dans le champ Valeurs de dimension, spécifiez les valeurs souhaitées.
14. Dans le champ Montant, entrez un nombre.
    * Vous pouvez également entrer un type de montant.  
15. Dans le champ Devise, cliquez sur le bouton de liste déroulante pour ouvrir la recherche.
16. Dans la liste, recherchez et sélectionnez l'enregistrement souhaité.
17. Cliquez sur Enregistrer.
18. Cliquez sur Mettre à jour les soldes budgétaires.
19. Cliquez sur Mise à jour.
    * Pour afficher les résultats de la mise à jour, cliquez sur Détails du message dans la barre bleue.
