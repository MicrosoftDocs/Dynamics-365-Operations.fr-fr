# <a name="create-a-commitment-to-reserve-budget-funds-for-the-public-sector"></a>Créer un engagement de réserver des fonds budgétaires pour le secteur public

[!include [banner](../../includes/banner.md)]

Les engagements sont des documents source de contrôle budgétaire utilisés par les entités du secteur public en France. Ce guide de tâche a été créée en utilisant les données de la société fictive PSUS dans la partition du secteur public.

1. Accédez à Budgétisation > Engagements.
2. Cliquez sur Nouveau.
3. Dans le champ Nom, tapez un nom qui permettra d'identifier cet engagement.
4. Cliquez sur Ajouter une ligne.
5. Dans le champ Compte général, sélectionnez les fonds et le compte principal.
6. Dans le champ Montant, entrez un nombre.
    * Entrez une ligne de contrat d'achat si l'engagement concerne un contrat d'achat spécifique. Ajoutez des lignes supplémentaires si nécessaire.  
7. Dans le volet Actions, cliquez sur Options.
8. Cliquez sur Changer de vue.
9. Cliquez sur Vue de l'en-tête.
10. Dans le champ Compte fournisseur, cliquez sur le bouton de liste déroulante pour ouvrir la recherche.
11. Facultatif : dans la liste, sélectionnez le fournisseur pour cet engagement.
12. Dans le volet Actions, cliquez sur Options.
13. Cliquez sur Changer de vue.
14. Cliquez sur Affichage des lignes.
15. Cliquez sur Enregistrer.
