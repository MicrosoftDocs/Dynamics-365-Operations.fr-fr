[!include [banner](../../includes/banner.md)]

<span data-ttu-id="bac38-104">Les types de fonds sont des catégories que vous pouvez utiliser pour regrouper des fonds à des fins de suivi et de génération d'états fiscaux détaillés.</span><span class="sxs-lookup"><span data-stu-id="bac38-104">Fund types are categories that you can use to group funds for detailed fiscal tracking and reporting.</span></span> <span data-ttu-id="bac38-105">Les principes comptables généralement reconnus (GAAP) pour la comptabilité gouvernementale et locale fournissent des normes que vous pouvez suivre lorsque vous paramétrez votre système de fonds et de types de fonds.</span><span class="sxs-lookup"><span data-stu-id="bac38-105">The Generally Accepted Accounting Principles (GAAP) for state and local governmental accounting provides standards that you can follow when you set up your system of funds and fund types.</span></span>

1. <span data-ttu-id="bac38-107">Accédez à Comptabilité > Plan de comptes > Fonds > Types de fonds.</span><span class="sxs-lookup"><span data-stu-id="bac38-107">Go to General ledger > Chart of accounts > Funds > Fund types.</span></span>

3. <span data-ttu-id="bac38-109">Dans le champ Type de fonds, tapez une valeur.</span><span class="sxs-lookup"><span data-stu-id="bac38-109">In the Fund type field, type a value.</span></span>

4. <span data-ttu-id="bac38-111">Dans le champ Description, entrez une valeur.</span><span class="sxs-lookup"><span data-stu-id="bac38-111">In the Description field, type a value.</span></span>
