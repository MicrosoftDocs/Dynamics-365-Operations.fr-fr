# <a name="set-up-interest-distribution-for-cash-accounts"></a><span data-ttu-id="ffb1b-104">Paramétrer la distribution d'intérêts pour les comptes de disponibilités</span><span class="sxs-lookup"><span data-stu-id="ffb1b-104">Set up interest distribution for cash accounts</span></span>

[!include[banner](../includes/banner.md)]

<span data-ttu-id="ffb1b-105">Votre agence peut affecter (distribuer) les intérêts sur un compte bancaire à des comptes généraux spécifiques, selon le solde quotidien moyen dans les comptes de disponibilités.</span><span class="sxs-lookup"><span data-stu-id="ffb1b-105">Your agency can allocate (distribute) the interest on a bank account to specific General ledger accounts, based on the average daily balance in cash accounts.</span></span> <span data-ttu-id="ffb1b-106">Cette procédure vous permet de générer une écriture comptable avancée pour les montants d'intérêts.</span><span class="sxs-lookup"><span data-stu-id="ffb1b-106">You can use this process to generate an advanced ledger entry for the interest amounts.</span></span> <span data-ttu-id="ffb1b-107">Sinon, vous pouvez générer les montants d'intérêts pour révision, sans les valider.</span><span class="sxs-lookup"><span data-stu-id="ffb1b-107">Alternatively, you can generate the interest amounts for review, without posting them.</span></span>

<span data-ttu-id="ffb1b-108">Avant distribuiez les intérêts, vous devez paramétrer vos comptes de disponibilités participants dans la page **Règles de distribution d'intérêts**.</span><span class="sxs-lookup"><span data-stu-id="ffb1b-108">Before you distribute the interest, you must set up your participating cash accounts on the **Interest distribution rules** page.</span></span> <span data-ttu-id="ffb1b-109">Chaque combinaison de compte de disponibilités et d'une subvention peut être utilisée pour calculer les intérêts d'un compte d'intérêts différent.</span><span class="sxs-lookup"><span data-stu-id="ffb1b-109">Each combination of a cash account and a grant can be used to calculate interest for a different interest account.</span></span>

4. <span data-ttu-id="ffb1b-114">Facultatif : Dans le champ **Subvention**, entrez la subvention pour les montants de transaction associés à une subvention à inclure dans les calculs de solde des disponibilités pour le compte de disponibilités associé.</span><span class="sxs-lookup"><span data-stu-id="ffb1b-114">Optional: In the **Grant** field, enter the grant for grant-related transaction amounts that should be included in cash balance calculations for the associated cash account.</span></span> <span data-ttu-id="ffb1b-115">Si toutes les subventions du compte de disponibilités valident des intérêts dans le même compte, vous pouvez laisser ce champ vide.</span><span class="sxs-lookup"><span data-stu-id="ffb1b-115">If all grants for the cash account post interest to the same account, you can leave this field blank.</span></span> <span data-ttu-id="ffb1b-116">Toutefois, si vous entrez un compte d'intérêts pour une subvention spécifique, vous devez entrer un compte d'intérêts pour chaque subvention pour le compte de disponibilités.</span><span class="sxs-lookup"><span data-stu-id="ffb1b-116">However, if you enter an interest account for a specific grant, you must enter an interest account for each grant for the cash account.</span></span>

6. <span data-ttu-id="ffb1b-121">Entrez un compte d'intérêts ou un projet spécifique.</span><span class="sxs-lookup"><span data-stu-id="ffb1b-121">Enter either a specific interest account or a project.</span></span> <span data-ttu-id="ffb1b-122">La valeur que vous entrez détermine le compte général sur lequel les intérêts sont validés pour le compte de disponibilités associé.</span><span class="sxs-lookup"><span data-stu-id="ffb1b-122">The value that you enter determines the General ledger account that interest is posted to for the associated cash account.</span></span>

    - <span data-ttu-id="ffb1b-124">Dans le champ **ID projet**, entrez un projet à utiliser comme base pour valider les intérêts.</span><span class="sxs-lookup"><span data-stu-id="ffb1b-124">In the **Project ID** field, enter a project to use as the basis for posting interest.</span></span> <span data-ttu-id="ffb1b-125">Ce compte est utilisé avec d'autres facteurs de compte pour déterminer le compte complet auquel cela est validé.</span><span class="sxs-lookup"><span data-stu-id="ffb1b-125">This account is used in conjunction with other account factors to determine the complete account that is posted to.</span></span>

7. <span data-ttu-id="ffb1b-126">Activez la case à cocher **Intérêts négatifs** pour calculer un montant d'intérêts négatifs lorsque le compte de disponibilités a un solde négatif.</span><span class="sxs-lookup"><span data-stu-id="ffb1b-126">Select the **Negative interest** check box to calculate a negative interest amount when the cash account has a negative balance.</span></span> <span data-ttu-id="ffb1b-127">Si cette case à cocher est désactivée, le compte de disponibilités indique des intérêts de 0 (zéro) plutôt qu'un montant négatif.</span><span class="sxs-lookup"><span data-stu-id="ffb1b-127">If this check box is cleared, the cash account reports the interest as 0 (zero) instead of a negative amount.</span></span>
8. <span data-ttu-id="ffb1b-128">Les intérêts d'un compte de disponibilités peuvent recevoir tous les montants de différence minime lorsque la distribution d'intérêts est calculée.</span><span class="sxs-lookup"><span data-stu-id="ffb1b-128">The interest account for one cash account can receive any penny difference amounts when interest distribution is calculated.</span></span> <span data-ttu-id="ffb1b-129">Activez la case à cocher **Arrondi** pour ce compte de disponibilités.</span><span class="sxs-lookup"><span data-stu-id="ffb1b-129">Select the **Rounding** check box for that cash account.</span></span> <span data-ttu-id="ffb1b-130">Vous pouvez marquer un seul compte de disponibilités comme compte d'arrondi.</span><span class="sxs-lookup"><span data-stu-id="ffb1b-130">You can mark only one cash account as the rounding account.</span></span>
