# <a name="assign-number-sequences"></a><span data-ttu-id="bedeb-104">Affecter des souches de N°</span><span class="sxs-lookup"><span data-stu-id="bedeb-104">Assign number sequences</span></span>

[!include [banner](../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]

<span data-ttu-id="bedeb-105">Cette rubrique explique comment créer des souches de N° pour les ID de bail.</span><span class="sxs-lookup"><span data-stu-id="bedeb-105">This topic explains how to create number sequences for lease IDs.</span></span> <span data-ttu-id="bedeb-106">Il explique également comment créer des ID uniques qui sont utilisés dans le processus de réévaluation de l’indice.</span><span class="sxs-lookup"><span data-stu-id="bedeb-106">It also explains how to create unique IDs that are used in the index revaluation process.</span></span>

1. <span data-ttu-id="bedeb-107">Accédez à **Location d’actifs \> Configuration \> Paramètres de location d’actif**.</span><span class="sxs-lookup"><span data-stu-id="bedeb-107">Go to **Asset leasing \> Setup \> Asset leasing parameters**.</span></span>
2. <span data-ttu-id="bedeb-108">Sélectionnez l’onglet latéral **Souches de N°**.</span><span class="sxs-lookup"><span data-stu-id="bedeb-108">Select the **Number sequences** side tab.</span></span>
3. <span data-ttu-id="bedeb-109">Sélectionnez **Souches de N°** sous l’onglet latéral.</span><span class="sxs-lookup"><span data-stu-id="bedeb-109">Select **Number sequences** in the side bar.</span></span>
4. <span data-ttu-id="bedeb-110">Sélectionnez la souche de N° pour la référence **ID du bail**.</span><span class="sxs-lookup"><span data-stu-id="bedeb-110">Select the number sequence for the **Lease ID** reference.</span></span> <span data-ttu-id="bedeb-111">Cette souche de N° sera utilisée pour générer l’identificateur unique de chaque bail.</span><span class="sxs-lookup"><span data-stu-id="bedeb-111">This number sequence will be used to generate the unique identifier for each lease.</span></span>
5. <span data-ttu-id="bedeb-112">Sélectionnez la souche de N° pour la référence **ID processus**.</span><span class="sxs-lookup"><span data-stu-id="bedeb-112">Select the number sequence for the **Process ID** reference.</span></span> <span data-ttu-id="bedeb-113">Cette souche de N° sera utilisée pour générer l’identificateur unique de chaque processus de réévaluation de l’index.</span><span class="sxs-lookup"><span data-stu-id="bedeb-113">This number sequence will be used to generate the unique identifier for each index revaluation process.</span></span>
6. <span data-ttu-id="bedeb-114">Sélectionnez la souche de N° pour la référence **ID de proposition de résiliation de contrat**.</span><span class="sxs-lookup"><span data-stu-id="bedeb-114">Select the number sequence for the **Termination Proposal ID** reference.</span></span> <span data-ttu-id="bedeb-115">Cette souche de N° sera utilisée pour générer l’identificateur unique de chaque proposition de résiliation.</span><span class="sxs-lookup"><span data-stu-id="bedeb-115">This number sequence will be used to generate the unique identifier for each termination proposal.</span></span>
