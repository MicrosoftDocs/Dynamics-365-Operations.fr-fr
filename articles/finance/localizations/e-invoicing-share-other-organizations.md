# <a name="share-globalization-features-with-other-organizations"></a>Partager les fonctionnalités de globalisation avec d’autres organisations

[!include [banner](../includes/banner.md)]

Suivez ces étapes pour partager une fonctionnalité de globalisation avec une organisation externe.

1. Connectez-vous à votre compte RCS (Regulatory Configuration Service).
2. Dans l’espace de travail **Fonctionnalité de globalisation**, dans la section **Fonctionnalités**, sélectionnez la vignette **Facturation électronique**.
3. Dans la page **Fonctionnalités de facturation électronique**, sélectionnez la fonctionnalité et la version à partager. Vous ne pouvez pas partager les versions de fonctionnalité dont le statut est défini sur **Brouillon**.
4. Dans l’onglet **Organisations**, sélectionnez **Partager avec** puis, dans la boîte de dialogue déroulante, entrez le nom de domaine de l’organisation avec laquelle vous souhaitez partager la fonctionnalité.
5. Sélectionnez **Partager**.

La fonctionnalité est partagée avec l’organisation sélectionnée et lui est disponible dans le référentiel global. De là, la fonctionnalité peut être importée dans l’instance RCS de l’organisation afin de pouvoir être utilisée.
