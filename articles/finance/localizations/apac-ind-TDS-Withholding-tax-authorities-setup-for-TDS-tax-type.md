# <a name="set-up-withholding-tax-authorities-for-the-tds-tax-type"></a>Paramétrer les administrations de retenue à la source pour le type de taxe TDS

[!include [banner](../includes/banner.md)]

Cette rubrique explique comment configurer les administrations de Taxe déduite à la Source (TDS).

1. Accédez à **Taxe \> Taxes indirectes \> Administrations de la retenue à la source**.

    [![Page Administrations de la retenue à la source.](./media/apac-ind-TDS-12.png)](./media/apac-ind-TDS-12.png)

2. Dans le champ **Type de taxe**, sélectionnez **TDS** pour configurer des administrations de retenue à la source pour le type de taxe TDS.
3. Dans le volet Actions, sélectionnez **Nouveau** pour créer une ligne.
4. Dans le champ **Administration fiscale**, entrez un nom pour l'administration TDS.
5. Entrez une description dans le champ **Description**.
6. Dans le raccourci **Général**, dans le champ **Compte fournisseur**, sélectionnez le compte fournisseur pour l'administration TDS.

    > [!NOTE]
    > Vous devez définir le nom de la banque où les fonds dus au fournisseur de l'autorité TDS seront déposés. Le nom de la banque est défini sur la page **Comptes bancaires** (**Comptes à payer \> Tous les fournisseurs\> Fournisseur \> Paramétrer \> Comptes bancaires**).

7. Fermez la page.
