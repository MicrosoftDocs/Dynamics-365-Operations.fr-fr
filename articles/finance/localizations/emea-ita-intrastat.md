# <a name="italian-intrastat"></a>Déclaration d'échanges de biens italienne

[!include[banner](../includes/banner.md)]

## <a name="overview"></a>Vue d’ensemble

La page **Déclaration d'échanges de biens** sert à générer et rapporter des informations sur le commerce entre pays/régions de l'Union européenne. La page Déclaration d'échanges de biens pour l'Italie contient des informations sur le commerce de biens et de services pour les déclarations mensuelles ou trimestrielles.

Vous devez envoyer deux déclarations aux autorités. Une déclaration concerne les expéditions intracommunautaires et l'autre concerne les arrivées intracommunautaires. Chaque déclaration comprend cinq parties :

- **Page de garde**

- **Section 1. Marchandises** : cette section contient des informations sur les transactions normales et les notes de crédit liées aux factures de marchandises au cours de la même période de déclaration que la période de déclaration d'échanges de biens.

- **Section 2. Corrections marchandises** : cette section contient des informations sur les corrections et les notes de crédit liées aux factures de marchandises des périodes de déclaration d'échanges de biens antérieures.

- **Section 3. Services** : cette section contient des informations sur les transactions normales et les notes de crédit liées aux factures de services au cours de la même période de déclaration que la période de déclaration d'échanges de biens.

- **Section 4. Corrections services** : cette section contient des informations sur les corrections et les notes de crédit liées aux factures de services des périodes de déclaration d'échanges de biens antérieures.

## <a name="set-up-intrastat"></a>Configuration de la Déclaration d'échanges de biens

### <a name="preliminary-setup"></a>Paramétrage préliminaire

Les informations générales suivantes doivent être configurées avant de commencer à utiliser les déclarations d'échanges de biens :

-   Codes marchandise. Pour les services, vous devez définir des codes marchandises à six chiffres.

-   Codes transaction. Notez que l'Italie utilise des codes transaction à un chiffre.

-   Modes de transport.

-   Procédures statistiques.

-   Paramètres de commerce extérieur.

-   Entreposage.

-   Détails des produits lancés.

-   Coordonnées des agents.

Pour plus d'informations, voir [Vue d'ensemble de la déclaration d'échanges de biens](emea-intrastat.md).

### <a name="set-up-italian-intrastat"></a>Configuration de la Déclaration d'échanges de biens pour l'Italie

Suivez ces étapes pour configurer les options spécifiques à l'Italie afin de pouvoir utiliser les déclarations d'échanges de biens.

1.  Dans [Microsoft Dynamics Lifecycle Services (LCS)](https://lcs.dynamics.com/Logon/Index), dans la bibliothèque d’actifs partagés, téléchargez la dernière version des configurations d’états électroniques pour la déclaration d'échanges de biens :

    -   Modèle de déclaration d’échanges de biens

    -   État de déclaration d’échanges de biens

    -   Déclaration d'échanges de biens (IT)

Pour plus d’informations, voir [Télécharger les configurations des états électroniques à partir de Lifecycle Services](../../fin-ops-core/dev-itpro/analytics/download-electronic-reporting-configuration-lcs.md)

2.  Dans Dynamics 365 Finance, accédez à **Taxes** &gt; **Paramétrage** &gt; **Paramètres de commerce extérieur**.

3.  Sur l'onglet **Déclaration d'échange de biens**, dans le raccourci **Général**, définissez les champs suivants :

    - **Département d'origine/destination** : sélectionnez le département de votre entreprise. Ce département sera utilisé pour les expéditions.

    <!-- -->

    - **Code transaction** : sélectionnez le code transaction pour les transferts de propriété. Ce code sera utilisé pour les transactions qui entraînent un transfert de propriété réel ou prévu contre compensation, ainsi que pour les corrections.

    - **Avoir** : sélectionnez le code transaction pour le retour des marchandises. Ce code sera utilisé pour le retour des marchandises après l'enregistrement de la transaction d'origine sous le code de transaction.

    - **Période de déclaration des ventes** : sélectionnez la période de déclaration pour la déclaration d'exportation : **Mois** ou **Trimestre**. Les déclarations trimestrielles sont exportées au format simplifié.

    - **Période de déclaration des achats** : sélectionnez la période de déclaration pour la déclaration d'importation : **Mois** ou **Trimestre**. Les déclarations trimestrielles sont exportées au format simplifié.

4.  Dans le raccourci **Génération d'états électroniques**, définissez les champs suivants :

    - **Mappage de format de fichier** : sélectionnez **Déclaration d'échanges de biens (IT)**.

    <!-- -->

    - **Mappage de format d'état** : sélectionnez **Déclaration d'échanges de biens**.

5.  Dans le raccourci **Hiérarchie des codes marchandise**, dans le champ **Hiérarchie des catégories**, sélectionnez **Déclaration d'échanges de biens**.

6.  Dans le raccourci **Valeur statistique**, définissez l'option **Imprimer et exporter des données statistiques** sur **Oui** si nécessaire. Ce paramètre active le transfert de la section statistique. La section statistique comprend des données sur les poids, les unités supplémentaires, les valeurs statistiques, les conditions de livraison, les plans de livraison, les modes de transport et les régions d'origine.

>[!NOTE]
>
>Pour une déclaration trimestrielle, la déclaration d'échanges de biens n'inclura pas la partie statistique, ni les informations sur les conditions de livraison et les modes de transport. Pour plus d’informations, voir le tableau de la section [Vue d’ensemble](#overview) de cette rubrique.

7.  Sur l'onglet **Propriétés de pays/région**, répertoriez tous les pays ou régions avec lesquels votre organisation fait affaire. Pour chaque pays ou région, définissez les champs suivants :

- **Pays/région de la partie** : sélectionnez le code pays/région.

- **Code de déclaration d'échanges de biens** : saisissez le code de déclaration d'échanges de biens à deux chiffres.

- **Devise** : précisez la devise nationale du pays ou de la région. Si le fournisseur est basé dans un pays de l'UE qui n'utilise pas l'euro, les montants des factures seront indiqués à la fois dans la devise du fournisseur et en euros. Par exemple, si le vendeur est basé au Danemark, les montants déclarés pour la déclaration d'importation seront à la fois en couronnes danoises (DKK) et en euros (EUR).

- **Type de pays/région** : sélectionnez le type de pays ou de région par rapport à votre organisation. Pour le journal des déclarations d'échanges de biens, seuls les pays ou régions des types **UE** et **Intérieur spécial** seront transférés.

>[!NOTE]
>
>Pour les pays ou régions du type **Intérieur spécial**, les champs suivants sont omis du fichier de déclaration d'échanges de biens : **Poids**, **Unités supplémentaires**, **Valeur statistique**, **Conditions de livraison**, **Code transport**, **Pays/région d'origine/destination** et **Département d'origine/destination**. Par exemple, dans le champ **Pays/région de la partie**, sélectionnez **SMR (Saint-Marin)** puis, dans le champ **Type de pays/région**, sélectionnez **Intérieur spécial**.

8.  Accédez à **Comptabilité fournisseur** &gt; **Paramétrage** &gt; **Conditions de livraison**.

9.  Dans la grille, sélectionnez les conditions de livraison.

10. Dans le raccourci **Général**, dans le champ **Code de déclaration d'échanges de biens**, entrez le code à un chiffre qui sera utilisé sur la déclaration d'échanges de biens.

11. Attribuez des d'exonération fiscale aux clients et aux fournisseurs en suivant ces étapes. Ces numéros apparaîtront sur la déclaration d'échanges de biens.

-   Accédez à **Taxes** &gt; **Paramétrage** &gt; **Taxe de vente** &gt; **Numéros d'exonération fiscale** et répertoriez tous les numéros d'exonération fiscale de vos clients et fournisseurs. Pour chaque partenaire, définissez les champs suivants :

    - **Pays/région** : sélectionnez le pays ou la région du partenaire.

    - **Numéro d'exonération fiscale** : entrez le numéro d'exonération fiscale du partenaire.

    - **Nom de l'entreprise** : saisissez le nom du partenaire.

<!-- -->

-   Accédez à **Comptabilité client** &gt; **Clients** &gt; **Tous les clients**, et suivez ces étapes pour chaque client :

    1.  Sélectionnez un client.

    2.  Dans le raccourci **Facture et livraison**, dans la section **Taxe de vente**, dans le champ **Numéro d'exonération fiscale**, sélectionnez **Tous**.

    3.  Sélectionnez le numéro d'exonération fiscale du client.

-   Accédez à **Comptabilité fournisseur** &gt; **Fournisseurs** &gt; **Tous les fournisseurs**, et suivez ces étapes pour chaque fournisseur :

    1.  Sélectionnez un fournisseur.

    2.  Dans le raccourci **Facture et livraison**, dans la section **Taxe de vente**, dans le champ **Numéro d'exonération fiscale**, sélectionnez **Tous**.

    3.  Sélectionnez le numéro d'exonération fiscale du fournisseur.

12. Accédez à **Taxes** &gt; **Paramétrage** &gt; **Commerce extérieur** &gt; **Compression de déclaration d'échanges de biens** et sélectionnez les champs qui doivent être comparés lors de la synthèse des informations de la déclaration d'échanges de biens. Pour l'Italie, sélectionnez **Numéro d'exonération fiscale**, **Code transaction**, **Marchandise**, **Conditions de livraison**, **Transport**, **Pays/région**, **Pays/région d'origine**, **Département d'origine**, **Département d'origine/destination**, **Devise**, **Mois**, **Trimestre** et **Année de correction**.

## <a name="italian-vendor-invoice-journal-for-foreign-trade"></a>Journal des factures des fournisseurs italien pour le commerce extérieur

Lorsque vous utilisez le journal des factures fournisseur, il est essentiel que vous preniez en compte certains paramètres qui affectent votre déclaration d'échanges de biens. Sur l'onglet **Commerce extérieur**, vous pouvez consulter et définir les champs spécifiques à l'Italie suivants.

| **Champ**                    | **Description**                                                                                 |
|------------------------------|-------------------------------------------------------------------------------------------------|
| Code transaction             | Permet de sélectionner le code transaction.                                                                    |
| Transport                    | Permet de sélectionner le transport utilisé pour le transfert.                                             |
| Marchandise                    | Permet de sélectionner le code marchandise de l'article.                                                                 |
| Unités supplémentaires             | Entrez des unités supplémentaires, si des unités supplémentaires doivent être déclarées pour le code marchandise sélectionné. |
| Pays/région               | Permet de sélectionner le pays/la région du partenaire.                                                            |
| Département d'origine/de destination | Permet de sélectionner le département de destination.                                                                  |

## <a name="intrastat-journal"></a>Journal des déclarations d'échanges de biens

Le journal des déclarations d'échanges de biens vous permet de gérer les informations relatives au commerce entre pays de l'Union européenne.

Pour ouvrir le journal des déclarations d'échanges de biens, accédez à **Taxes** &gt; **Déclarations** &gt; **Commerce extérieur** &gt; **Déclaration d'échanges de biens**. Les informations suivantes sont affichées pour chaque transaction :

| **Champ**                              | **Description**                                                                                                                                                                                                                                                                                                                                                                                                         |
|----------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Date                                   | Date de validation de la transaction.                                                                                                                                                                                                                                                                                                                                                                                    |
| Direction                              | Ce champ est défini sur **Arrivées** pour les achats ou sur **Expéditions** pour les ventes.                                                                                                                                                                                                                                                                                                                                            |
| Type d’article                              | Ce champ peut être défini sur **Marchandises** ou **Services**.                                                                                                                                                                                                                                                                                                                                                                     |
| Correction                             | Lorsque cette option est définie sur **Oui**, la transaction est une correction. Dans ce cas, il est possible de déclarer des corrections à des transactions de déclaration d'échanges de biens déjà déclarées. Sur l'onglet **Général**, dans la section **Corrections**, utilisez les champs **Mois**, **Trimestre** et **Année de correction** pour spécifier la période de transaction d'origine. Utilisez le champ **Enregistrement de déclaration d'échanges de biens d'origine** pour spécifier l'enregistrement d'origine. |
| Numéro d’article                            | Code-barres du produit lancé.                                                                                                                                                                                                                                                                                                                                                                                      |
| Catégorie                               | Code de la catégorie d'approvisionnement.                                                                                                                                                                                                                                                                                                                                                                                   |
| Marchandise                              | Code marchandise de l'article. Cette valeur est indiquée sur la page **Détails des produits lancés** pour les produits, ou il peut être défini manuellement sur la ligne de transaction.                                                                                                                                                                                                                                                              |
| Pondération                                 | Poids de la ligne de transaction, en kilogrammes (kg). Cette valeur est indiquée sur la page **Détails des produits lancés** pour les produits, ou il peut être défini manuellement sur la ligne de transaction.                                                                                                                                                                                                                                |
| Montant de la facture                         | Montant en devise comptable                                                                                                                                                                                                                                                                                                                                                                                   |
| Montant de la facture dans la devise de la transaction | Montant dans la devise nationale du partenaire.                                                                                                                                                                                                                                                                                                                                                                     |
| Devise                               | Devise nationale du partenaire.                                                                                                                                                                                                                                                                                                                                                                                   |

>[!NOTE]
>
>Si vous recevez une correction négative (avoir) dans la même période que la période de déclaration, vous devez modifier manuellement le journal des déclarations d'échanges de biens en suivant ces étapes.
>
>1.  Accédez à **Taxes** &gt; **Déclarations** &gt; **Commerce extérieur** &gt; **Déclaration d'échanges de biens**.
>2.  Recherchez et supprimez la transaction marquée comme correction.
>3.  Recherchez la transaction d'origine et modifiez la valeur du champ **Montant de la facture** selon les besoins.
>Par exemple, vous avez une facture de 10 000 et vous recevez un avoir de -2 000. Dans ce cas, vous devez ouvrir le journal des déclarations d'échanges de biens, rechercher et supprimer la transaction pour -2 000. Recherchez ensuite la transaction d'origine pour 10 000 et définissez le montant de la facture sur 8 000 (= 10 000 - 2 000).

### <a name="intrastat-transfer"></a>Transfert des déclarations d'échanges de biens

Dans le volet Actions, vous pouvez sélectionner **Transférer** pour transférer automatiquement les informations sur le commerce intracommunautaire à partir de vos commandes clients, de vos factures financières, des commandes fournisseur, des factures fournisseurs, des accusés de réception de produits fournisseurs, des factures de projet et des ordres de transfert. Seuls les documents ayant un pays de l'UE comme pays ou région de destination (pour les expéditions) ou de consignation (pour les arrivées) seront transférés.

Vous pouvez également saisir manuellement des transactions en sélectionnant **Nouveau** dans le volet Actions.
