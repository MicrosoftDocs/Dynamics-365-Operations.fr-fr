# <a name="fr-00002-customer-and-vendor-account-statements"></a>FR-00002 Relevés de compte client et fournisseur

[!include [banner](../../includes/banner.md)]

Vous pouvez imprimer des relevés de compte pour les clients et les fournisseurs, affichant des informations pour une période sélectionnée.

Cette procédure a été créée à l’aide des données fictives de la société FRSI. 

Cette fonctionnalité est disponible pour les entités juridiques dont l’adresse principale est en France.




## <a name="print-the-customer-account-statement-report"></a>Imprimer l’état du relevé de compte client
1. Accédez à Comptabilité > Recherches et états > Clients > État de relevé de compte client - France.
2. Entrez une date dans le champ Date de début.
3. Entrez une date dans le champ Date de fin.
4. Dans le champ Évaluation, entrez une date.
5. Cliquez sur OK.

## <a name="print-the-vendor-account-statement-report"></a>Imprimer l’état du relevé de compte fournisseur
1. Accédez à Comptabilité fournisseur > Recherches et états > États fournisseur > Relevé de compte.
2. Entrez une date dans le champ Date de début.
3. Entrez une date dans le champ Date de fin.
4. Dans le champ Évaluation, entrez une date.
5. Cliquez sur OK.
