# <a name="set-up-tds-ledger-accounts"></a>Paramétrer les comptes généraux TDS

[!include [banner](../includes/banner.md)]

Cette rubrique explique comment configurer des comptes généraux pour la fonction Taxe déduite à la source (TDS). Utilisez la page **Plan comptable** pour configurer les comptes généraux pour TDS.

1. Accédez à **Comptabilité \> Plan de comptes \> Comptes \> Comptes principaux**.
2. Dans l'onglet **Aperçu**, sélectionnez **Alt+N** pour créer un compte général TDS et saisissez les détails requis.
3. Dans l'onglet **Paramétrer**, dans le champ **Type de validation**, sélectionnez **Retenue à la source**.     

    > [!NOTE]
    > Les comptes généraux dont le type de validation est **Retenue à la source** ne peuvent être définis que comme des comptes clients utilisés pour comptabiliser le montant des comptes clients TDS pour un code TVA TDS.

4. Fermez la page.
