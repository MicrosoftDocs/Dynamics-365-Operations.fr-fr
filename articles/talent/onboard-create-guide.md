# <a name="create-and-send-an-onboarding-guide"></a>Créer et envoyer un guide d'intégration

[!include [banner](includes/banner.md)]

Microsoft Dynamics 365 Talent: Onboard vous permet de créer des guides d'intégration à partir de modèles que vous créez vous-même, de modèles disponibles dans une galerie, ou de modèles que vous créez de toutes pièces.

Après avoir créé un guide d'intégration, vous pouvez l'envoyer à une nouvelle recrue. Vous pouvez aussi l'envoyer à plusieurs recrues que vous spécifiez dans un fichier Microsoft Excel téléchargée à partir de l'application Onboard.

## <a name="create-an-onboarding-guide-from-a-template-and-send-it-to-a-single-new-hire"></a>Créer un guide d'intégration à partir d'un modèle et l'envoyer à une nouvelle recrue

1. Dans le menu à gauche, sélectionnez **Modèles**.
2. Sous **Mes modèles**, sélectionnez le modèle à définir comme guide d'intégration pour la nouvelle recrue.
3. Modifiez le modèle comme vous le souhaitez. N'oubliez pas d'enregistrer votre travail au fur et à mesure.
4. Lorsque vous avez fini de modifier le modèle, sélectionnez **Créer un guide**.

    [![Création d'un guide d'intégration à partir d'un modèle](./media/onboard-create-guide.png)](./media/onboard-create-guide.png)

5. Dans la fenêtre **Créer un guide**, sous **Qui accueillez-vous ?**, entrez le nom ou l'adresse e-mail de la nouvelle recrue. Si la nouvelle recrue n'est pas encore dans le système, sélectionnez **Ajouter maintenant**, puis entrez les informations de l'employé.

    ![[Saisie d'informations pour le guide d'intégration](./media/onboard-create-a-guide-window.png)](./media/onboard-create-a-guide-window.png)

6. Sélectionnez une date de début sous **Quand vont-elles commencer ?**.
7. Si le guide d'intégration doit être envoyé automatiquement à la nouvelle recrue à une date spécifique, assurez-vous que l'option **Planifier une date d'envoi automatique** est activée, puis sélectionnez la date d'envoi automatique. Pour envoyer le guide immédiatement, désactivez l'option **Planifier une date d'envoi automatique**.
8. Sélectionnez **Terminé**.
9. Lorsque vous avez fini de modifier le guide d'intégration, sélectionnez **Envoyer** dans l'angle supérieur droit. Suivez ensuite l'une des procédures suivantes :

    - Pour envoyer un lien vers le guide d'intégration à la nouvelle recrue, sélectionnez **copier un lien**, puis **Copier**.
    - Pour personnaliser l'e-mail pour le guide d'intégration avant de l'envoyer, sélectionnez **Personnaliser l'e-mail avant l'envoi**, sélectionnez **Suivant**, personnalisez l'e-mail comme vous le souhaitez, puis sélectionnez **Envoyer**.
    - Pour envoyer l'e-mail sans le personnaliser, sélectionnez **Suivant**, puis sélectionnez **Envoyer**.

## <a name="create-an-onboarding-guide-from-a-template-and-send-it-to-multiple-new-hires"></a>Créer un guide d'intégration à partir d'un modèle et l'envoyer à plusieurs nouvelles recrues

Onboard permet d'envoyer un guide d'intégration à plusieurs nouvelles recrues simultanément.

1. Dans le menu à gauche, sélectionnez **Modèles**.
2. Sous **Mes modèles**, sélectionnez le modèle à définir comme guide d'intégration pour les nouvelles recrues.
3. Modifiez le modèle comme vous le souhaitez. N'oubliez pas d'enregistrer votre travail au fur et à mesure.
4. Lorsque vous avez fini de modifier le modèle, sélectionnez **Créer un guide**.
5. Dans la fenêtre **Créer un guide**, sélectionnez **Vous devez accueillir plusieurs personnes à la fois ?**.

    [![Création d'un guide d'intégration pour plusieurs nouvelles recrues](./media/onboard-send-guide-multiple-people.png)](./media/onboard-send-guide-multiple-people.png)

6. Sélectionnez **nouveau modèle d'embauche**.
7. Une fois le fichier .xlsx téléchargé, sélectionnez **Ouvrir**, entrez les informations des employés dans le classeur Excel, puis enregistrez le classeur.

    [![Téléchargement du modèle Excel pour envoyer le guide d'intégration à plusieurs nouvelles recrues](./media/onboard-send-guide-download-spreadsheet.png)](./media/onboard-send-guide-download-spreadsheet.png)

    > [!NOTE]
    > Avant de modifier le classeur, vous devez sélectionner **Activer la modification** dans Excel.
    > 
    > [![Activer la modification](./media/onboard-send-guide-enable-editing.png)](./media/onboard-send-guide-enable-editing.png)

8. Faites glisser le classeur Excel dans la zone indiquée dans la fenêtre **Créer plusieurs guides** ou cliquez n'importe où dans cette zone pour rechercher le fichier sur votre ordinateur.

    [![Faire glisser le classeur modifié](./media/onboard-send-guide-drag-spreadsheet.png)](./media/onboard-send-guide-drag-spreadsheet.png)

9. Lorsque vous avez fini de modifier le guide d'intégration, sélectionnez **Envoyer** dans l'angle supérieur droit. Suivez ensuite l'une des procédures suivantes :

    - Pour envoyer un lien vers le guide d'intégration aux nouvelles recrues, sélectionnez **copier un lien**, puis **Copier**.
    - Pour personnaliser l'e-mail pour le guide d'intégration avant de l'envoyer, sélectionnez **Personnaliser l'e-mail avant l'envoi**, sélectionnez **Suivant**, personnalisez l'e-mail comme vous le souhaitez, puis sélectionnez **Envoyer**.
    - Pour envoyer l'e-mail sans le personnaliser, sélectionnez **Suivant**, puis sélectionnez **Envoyer**.

## <a name="create-a-guide-without-using-a-template"></a>Créer un guide sans utiliser de modèle

Vous n'êtes pas toujours obligé de créer un guide à partir d'un modèle. Si vous le souhaitez, vous pouvez créer un guide de toutes pièces à la place.

1. Dans le menu de gauche, sélectionnez **Guides**, puis sélectionnez le bouton **Ajouter** (le signe plus \[**+**\]).
2. Dans la fenêtre **Créer un guide**, sous **Qui accueillez-vous ?**, entrez le nom ou l'adresse e-mail de la nouvelle recrue. Si la nouvelle recrue n'est pas encore dans le système, sélectionnez **Ajouter maintenant**, puis entrez les informations de l'employé.

    ![[Saisie d'informations pour le guide d'intégration](./media/onboard-create-a-guide-window.png)](./media/onboard-create-a-guide-window.png)

3. Sélectionnez une date de début sous **Quand vont-elles commencer ?**.
4. Si le guide d'intégration doit être envoyé automatiquement à la nouvelle recrue à une date spécifique, assurez-vous que l'option **Planifier une date d'envoi automatique** est activée, puis sélectionnez la date d'envoi automatique. Pour envoyer le guide immédiatement, désactivez l'option **Planifier une date d'envoi automatique**.
5. Sélectionnez **Terminé**.

## <a name="save-a-guide-as-a-template"></a>Enregistrer un guide comme modèle

Vous pouvez enregistrer un guide d'intégration en tant que modèle. De cette manière, vous pouvez gagner du temps lorsque vous devez créer des guides d'intégration ultérieurement.

1. Dans le menu à gauche, sélectionnez **Guides**.
2. Sélectionnez le bouton **Plus** (les points de suspension \[**...**\]) en regard du guide à partir duquel vous souhaitez créer un modèle, puis sélectionnez **Enregistrer en tant que modèle**.

    ![[Enregistrement d'un guide d'intégration en tant que modèle](./media/onboard-save-guide-as-template.png)](./media/onboard-save-guide-as-template.png)

3. Dans la fenêtre **Enregistrer en tant que nouveau modèle**, entrez un nom pour votre nouveau modèle, puis sélectionnez **Enregistrer**.
