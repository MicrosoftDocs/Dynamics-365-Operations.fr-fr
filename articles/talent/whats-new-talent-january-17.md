# <a name="whats-new-or-changed-in-dynamics-365-for-talent-core-hr-january-17-2019"></a>Nouveautés ou modifications dans Dynamics 365 for Talent Core HR (17 janvier 2019)

[!include [banner](includes/banner.md)]

**Version 8.1.2107**

Cette rubrique décrit les fonctionnalités qui sont nouvelles ou ont été modifiées dans Core HR.

## <a name="minor-changes-and-bug-fixes-included-in-this-release"></a>Des modifications d'ordre mineur et des correctifs de bogue ont été inclus dans cette version

### <a name="new-position-assignment-start-datetime-default-has-been-corrected"></a>Une nouvelle date/heure de début d'affectation de poste par défaut a été corrigée
Lors du transfert des employés, le système sélectionnera, par défaut, l'heure/la date au début de la journée, pour le fuseau horaire de l'utilisateur.

### <a name="all-employees-have-the-same-position-description-in-the-exited-worker-task-management-list"></a>Tous les employés ont la même description de poste dans la liste de gestion des tâches des employés ayant quitté la société.
Une mise à jour a été effectuée dans l'espace de travail **Gestion des tâches** pour afficher les descriptions de poste correctes pour tous les employés qui ont récemment quitté la société.

### <a name="action-requested-by-field-is-now-populated-on-workers-action-page"></a>Le champ « Action requise par » est désormais complété sur la page Action des employés.
Avec cette modification, la valeur par défaut pour le champ **Action demandée par** est désormais l'utilisateur qui demande les modifications.

### <a name="ideas-portal-updated"></a>Portail des idées mis à jour
Dans l'application, les liens pour soumettre une idée pour Talent ont été mis à jour avec le nouveau portail Ideas. Toutes les idées saisies dans l'ancien portail ont été migrées, de telle sorte que toutes vos idées de génie soient encore là et disponibles pour être validées par d'autres.
