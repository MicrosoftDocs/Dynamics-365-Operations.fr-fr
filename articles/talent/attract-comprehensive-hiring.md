# <a name="choose-a-version-of-attract"></a>Choisir une version d'Attract

[[!include [banner](includes/banner.md)]

La version de Microsoft Dynamics 365 Talent: Attract qui vous convient dépend de la taille et des besoins de votre entreprise. La version de base d'Attract fournit un sous-ensemble de fonctionnalités qui convient aux petits équipes ou aux entreprises dans lesquelles le responsable du recrutement est l'utilisateur principal. Vous pouvez commencer à utiliser la version de base d'Attract avec une ou plusieurs licences.

Attract, avec le module complémentaire Recrutement complet est destiné aux équipes ou aux entreprises de taille moyenne à grande, où le recruteur ou le spécialiste des talent est l'utilisateur principal. Le module complémentaire Recrutement complet nécessite d'acheter une licence auprès d'un [Partenaire Microsoft Dynamics 365](https://dynamics.microsoft.com/partners/find-a-partner/)

> [!NOTE]
> La version d'essai d'Attract propose Attract avec le module complémentaire Recrutement complet. Ce module fournit des fonctionnalités de recrutement et d'embauche complètes, telles que la capacité de publier des offres sur un site de carrière Attract ou sur un site de recrutement externe tel que LinkedIn. Si vous optez pour la version de base d'Attract, votre abonnement inclut un sous-ensemble des capacités fournies dans la version de test.

Pour plus d'informations sur la tarification et les licences, voir le [Guide des licences Microsoft Dynamics 365](https://go.microsoft.com/fwlink/?LinkId=866544).

## <a name="find-out-which-version-of-attract-you-have"></a>Découvrez de quelle la version d'Attract vous disposez

1. Dans Attract, sélectionnez le point d'interrogation (**?**) dans le coin supérieur droit de la page.
2. Sélectionnez **A propos**.

    La fenêtre qui apparaît indique le type de licence Attract dont vous disposez.

## <a name="frequently-asked-questions"></a>Forum aux questions

### <a name="why-dont-i-see-all-the-features-that-were-included-in-the-attract-trial"></a>Pourquoi ne vois-je pas toutes les fonctions incluses dans la version d'essai d'Attract ?

Il y a plusieurs raisons pour lesquelles vous ne pouvez pas afficher toutes les fonctionnalités incluses dans la version d'essai d'Attract :

- La version d'essai d'Attract présente Attract avec le module complémentaire Recrutement complet, or maintenant, vous disposez de la version de base d'Attract. La version de base d'Attract comprend un sous-ensemble de fonctions qui est destiné aux responsables du recrutement les petites équipes et les petites entreprises.
- Vous disposez d'Attract avec le module complémentaire Recrutement complet, mais vous avez pas affecté les rôles corrects pour utiliser toutes les fonctions. Pour plus d'informations voir [Gestion de la sécurité et des rôles dans Attract](security-attract.md).

### <a name="why-dont-i-see-all-the-attract-features-that-my-coworker-sees"></a>Pourquoi ne vois-je pas toutes les fonctions d'Attract que voient mes collègues ?

Il y a plusieurs raisons pour lesquelles vous ne pouvez pas afficher toutes les fonctionnalités d'Attract que voient vos collègues :

- Votre collègue travaille avec une version d'essai ou une licence Attract avec le module complémentaire Recrutement complet, alors que vous utilisez la verison de base d'Attract.
- Vous disposez d'Attract avec le module complémentaire Recrutement complet, mais vous avez pas affecté les rôles corrects pour utiliser toutes les fonctions. Pour plus d'informations voir [Gestion de la sécurité et des rôles dans Attract](security-attract.md).
