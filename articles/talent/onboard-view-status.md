# <a name="view-the-status-of-onboarding-employees-and-your-tasks"></a>Afficher le statut d'intégration des employés et vos tâches

[!include [banner](includes/banner.md)]

Microsoft Dynamics 365 for Talent : Onboard vous permet d'afficher la progression de vos nouvelles recrues dans leurs activités d'intégration. Vous pouvez également afficher les tâches qui vous ont été affectées à partir des guides d'intégration.

## <a name="view-the-status-of-onboarding-employees"></a>Afficher le statut d'intégration des employés

1. Dans le menu à gauche, sélectionnez **Guides**. Une vue d'ensemble de la progression de tous les guides d'intégration que vous avez envoyés s'affiche.

    ![[Affichage du statut des guides d'intégration](./media/onboard-guide-status.png)](./media/onboard-guide-status.png)

2. Pour explorer au niveau du détail, sélectionnez le guide à afficher.
3. Sélectionnez l'onglet **Activités** pour voir les activités que votre nouvelle recrue a terminées.

    ![[Affichage des activités terminées par la nouvelle recrue](./media/onboard-status-activities.png)](./media/onboard-status-activities.png)

## <a name="view-onboarding-tasks-that-are-assigned-to-you"></a>Afficher les tâches d'intégration qui vous sont affectées

1. Dans le menu à gauche, sélectionnez **Tâches**. Une vue d'ensemble de toutes les tâches d'intégration qui vous ont été affectées à partir des guides d'intégration envoyés aux employés s'affiche.
2. Pour explorer au niveau du détail, sélectionnez la tâche souhaitée.
3. Effectuez le suivi de la nouvelle recrue au besoin.
4. Lorsque vous avez terminé la tâche, marquez-la comme terminée en ajoutant une coche dans le cercle.
