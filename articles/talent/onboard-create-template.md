# <a name="create-an-onboarding-template-by-using-dynamics-365-talent---onboard"></a>Créer un modèle d'intégration à l'aide de Dynamics 365 Talent - Onboard

[!include [banner](includes/banner.md)]

Microsoft Dynamics 365 Talent: Onboard fournit plusieurs modèles qui peuvent vous aider à créer un guide d'intégration le plus rapidement possible. Vous pouvez utiliser un ou plusieurs de ces modèles, ou vous pouvez créer vos propres modèles. Onboard fournit un exemple de texte que vous pouvez utiliser lorsque vous créez vos propres modèles. Par conséquent, le processus est facile même si vous commencez à partir de zéro.

## <a name="create-an-onboarding-template-from-an-existing-template"></a>Créer un modèle d'intégration à partir d'un modèle existant

1. Dans le menu à gauche, sélectionnez **Modèles**.
2. Sous **Modèles populaires de la galerie** ou **Mes modèles**, sélectionnez un modèle. Pour afficher d'autres modèles, sélectionnez **Autres modèles dans la galerie de modèles**.
3. Utilisez l'une des procédures suivantes :

    - Si le modèle provient de la galerie, sélectionnez **Enregistrer comme mon modèle**, entrez un nouveau nom pour le modèle, puis sélectionnez **Enregistrer**.
    - Si le modèle provient de **Mes modèles**, sélectionnez **Enregistrer en tant que modèle**, entrez un nouveau nom pour le modèle, puis sélectionnez **Enregistrer**.

    [![Création d'un modèle à partir d'un modèle existant](./media/onboard-save-template.png)](./media/onboard-save-template.png)

## <a name="create-a-new-onboarding-template"></a>Créer un nouveau modèle d'intégration

1. Dans le menu à gauche, sélectionnez **Modèles**.
2. Sous **Mes modèles**, sélectionnez la vignette **Ajouter** (signe plus) \[**+**\].

    [![Création d'un nouveau modèle](./media/onboard-create-new-template.png)](./media/onboard-create-new-template.png)

3. Dans la boîte de dialogue **Créer un modèle de guide**, entrez un nom pour le modèle, puis sélectionnez **Enregistrer**.
