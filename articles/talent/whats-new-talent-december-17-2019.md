# <a name="whats-new-or-changed-in-dynamics-365-talent-december-17-2019"></a>Nouveautés ou modifications dans Dynamics 365 Talent (17 décembre 2019)

Cet article décrit les fonctionnalités qui sont nouvelles ou ont été modifiées dans Dynamics 365 Talent.

## <a name="changes-in-attract"></a>Modifications apportées dans Attract

Cette version inclut des correctifs de bogues mineurs pour Dynamics 365 Talent: Attract.

## <a name="changes-in-onboard"></a>Modifications apportées à Onboard

Cette version inclut des correctifs de bogues mineurs pour Dynamics 365 Talent: Onboard.

## <a name="changes-in-core-hr"></a>Modifications apportées à Core HR

Les modifications décrites dans cette section s'appliquent au numéro de version 8.1.2674. Les numéros entre parenthèses dans certains en-têtes se rapportent aux numéros de support dans Microsoft Dynamics Lifecycle Services (LCS).

### <a name="print-performance-reviews"></a>Imprimer les évaluations des performances

Les évaluations des performances d'impression sont désormais disponibles dans tous les environnements. Pour activer cette fonctionnalité, utilisez l'espace de travail **Gestion des fonctionnalités**. Pour plus d'informations sur la gestion des fonctions, voir [Présentation de la gestion des fonctions](https://docs.microsoft.com/dynamics365/fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview). Pour plus d'informations sur l'impression des évaluations des performances, voir [Entretiens d'évaluation des performances](https://docs.microsoft.com/dynamics365/talent/performance-management-overview#performance-reviews).

### <a name="retired-position-with-direct-reports-floats-in-org-diagram-as-an-open-position-385436"></a>Position retirée avec subalternes directs « flotte » dans l'organigramme comme position ouverte (385436)

La version de cette semaine corrige l'affichage des postes retirés avec des subordonnés directs dans la hiérarchie des rapports.
 
### <a name="system-unable-to-generate-calendar-days-381195"></a>Système incapable de générer des jours civils (381195) 

Les modifications de cette semaine corrigent un problème où les jours calendaires ne pouvaient pas être générés.

### <a name="current-balance-report-in-leave-pbi-doubles-employee-balance-385581"></a>Le rapport sur le solde actuel des congés PBI double le solde des employés (385581)

Cette modification corrige le rapport de solde actuel dans ce scénario.
