# <a name="close-a-job-in-microsoft-dynamics-talent---attract"></a><span data-ttu-id="aa5ad-103">Clôturer un poste dans Microsoft Dynamics pour Talent - Attract</span><span class="sxs-lookup"><span data-stu-id="aa5ad-103">Close a job in Microsoft Dynamics Talent - Attract</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="aa5ad-104">Dans Microsoft Dynamics 365 Talent: Attract, vous pouvez clôturer un poste et le marquer comme étant un **Poste annulé** ou un **Poste pourvu**.</span><span class="sxs-lookup"><span data-stu-id="aa5ad-104">In Microsoft Dynamics 365 Talent: Attract, you can close a job as either **Job cancelled** or **Position filled**.</span></span> <span data-ttu-id="aa5ad-105">Vous pouvez annuler un poste à tout moment.</span><span class="sxs-lookup"><span data-stu-id="aa5ad-105">You can cancel a job at any time.</span></span>

<span data-ttu-id="aa5ad-106">Pour indiquer que le poste a été pourvu, un candidat doit avoir accepté l'offre d'emploi.</span><span class="sxs-lookup"><span data-stu-id="aa5ad-106">If you want to indicate that the position was filled, a candidate must have accepted an offer for the job.</span></span>

## <a name="cancel-a-job"></a><span data-ttu-id="aa5ad-107">Annuler un poste</span><span class="sxs-lookup"><span data-stu-id="aa5ad-107">Cancel a job</span></span>

<span data-ttu-id="aa5ad-108">Vous pouvez annuler un poste dans Attract à tout moment.</span><span class="sxs-lookup"><span data-stu-id="aa5ad-108">You can cancel a job in Attract at any time.</span></span>

1. <span data-ttu-id="aa5ad-109">Sélectionnez le poste à annuler dans l'onglet **Postes** d'Attract.</span><span class="sxs-lookup"><span data-stu-id="aa5ad-109">In the **Jobs** tab in Attract, select the job you want to cancel.</span></span>

2. <span data-ttu-id="aa5ad-110">Sélectionnez **Clôturer le poste**.</span><span class="sxs-lookup"><span data-stu-id="aa5ad-110">Select **Close job**.</span></span>

   ![Clôturer un poste dans Attract](./media/attract-close-job.png)

3. <span data-ttu-id="aa5ad-112">Dans la boîte de dialogue **Clôturer ce poste**, sélectionnez **Poste annulé**, ajoutez des commentaires, puis sélectionnez **Clôturer le poste**.</span><span class="sxs-lookup"><span data-stu-id="aa5ad-112">In the **Close this job** dialog box, select **Job cancelled**, add any comments, and then select **Close job**.</span></span>

   ![Clôturer un poste et le marquer comme étant un Poste pourvu](./media/attract-close-job-as-cancelled.png)

## <a name="close-a-job-as-position-filled"></a><span data-ttu-id="aa5ad-114">Clôturer un poste et le marquer comme étant un Poste pourvu</span><span class="sxs-lookup"><span data-stu-id="aa5ad-114">Close a job as Position filled</span></span>

<span data-ttu-id="aa5ad-115">Vous pouvez clôturer un poste et le marquer comme étant un **Poste pourvu** après qu'au moins un prospect a terminé le stade Offre.</span><span class="sxs-lookup"><span data-stu-id="aa5ad-115">You can close a job as **Position filled** after at least one prospect has completed the Offer stage.</span></span>

1. <span data-ttu-id="aa5ad-116">Sélectionnez le poste à annuler dans l'onglet **Postes** d'Attract.</span><span class="sxs-lookup"><span data-stu-id="aa5ad-116">In the **Jobs** tab in Attract, select the job you want to cancel.</span></span>

2. <span data-ttu-id="aa5ad-117">Sélectionnez **Clôturer le poste**.</span><span class="sxs-lookup"><span data-stu-id="aa5ad-117">Select **Close job**.</span></span>

   ![Clôturer un poste dans Attract](./media/attract-close-job.png)

3. <span data-ttu-id="aa5ad-119">Dans la boîte de dialogue **Clôturer ce poste**, sélectionnez **Poste pourvu**, sélectionnez le candidat ayant accepté l'offre, ajoutez des commentaires, puis sélectionnez **Clôturer le poste**.</span><span class="sxs-lookup"><span data-stu-id="aa5ad-119">In the **Close this job** dialog box, select **Position filled**, select the candidate who accepted the offer, add any comments, and then select **Close job**.</span></span>

   > [!NOTE]
