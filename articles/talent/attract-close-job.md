# <a name="close-a-job"></a>Clôturer un poste

[!include [banner](includes/banner.md)]

Dans Microsoft Dynamics 365 Talent: Attract, vous pouvez clôturer un poste et le marquer comme étant un **Poste annulé** ou un **Poste pourvu**. Vous pouvez annuler un poste à tout moment.

Pour indiquer que le poste a été pourvu, un candidat doit avoir accepté l'offre d'emploi.

## <a name="cancel-a-job"></a>Annuler un poste

Vous pouvez annuler un poste dans Attract à tout moment.

1. Sélectionnez le poste à annuler dans l'onglet **Postes** d'Attract.

2. Sélectionnez **Clôturer le poste**.

   ![Clôturer un poste dans Attract](./media/attract-close-job.png)

3. Dans la boîte de dialogue **Clôturer ce poste**, sélectionnez **Poste annulé**, ajoutez des commentaires, puis sélectionnez **Clôturer le poste**.

   ![Clôturer un poste et le marquer comme étant un Poste pourvu](./media/attract-close-job-as-cancelled.png)

## <a name="close-a-job-as-position-filled"></a>Clôturer un poste et le marquer comme étant un Poste pourvu

Vous pouvez clôturer un poste et le marquer comme étant un **Poste pourvu** après qu'au moins un prospect a terminé le stade Offre.

1. Sélectionnez le poste à annuler dans l'onglet **Postes** d'Attract.

2. Sélectionnez **Clôturer le poste**.

   ![Clôturer un poste dans Attract](./media/attract-close-job.png)

3. Dans la boîte de dialogue **Clôturer ce poste**, sélectionnez **Poste pourvu**, sélectionnez le candidat ayant accepté l'offre, ajoutez des commentaires, puis sélectionnez **Clôturer le poste**.

   ![Clôturer un poste et le marquer comme étant un Poste pourvu](./media/attract-close-job-as-position-filled.png)

   > [!NOTE]
   > Vous pouvez uniquement clôturer un poste et le marquer comme étant un **Poste pourvu** après qu'au moins un prospect a terminé le stade Offre. Si **Offre acceptée** n'est pas sélectionnée pour l'offre d'emploi ou si la candidature n'a pas le statut **Accepté** dans Offre, l'option **Poste pourvu** ne s'affiche pas dans le menu déroulant **Motif**.
