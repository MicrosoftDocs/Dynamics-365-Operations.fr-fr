# <a name="whats-new-or-changed-in-dynamics-365-talent-october-1-2019"></a>Nouveautés ou modifications dans Dynamics 365 Talent (1er octobre 2019)

Cette rubrique décrit les fonctionnalités qui sont nouvelles ou ont été modifiées dans Dynamics 365 Talent.

## <a name="changes-in-attract"></a>Modifications apportées dans Attract

Cette version inclut des correctifs de bogues mineurs pour Dynamics 365 Talent: Attract.

## <a name="changes-in-onboard"></a>Modifications apportées à Onboard

Cette version inclut des correctifs de bogues mineurs pour Dynamics 365 Talent: Onboard.

## <a name="changes-in-core-hr"></a>Modifications apportées à Core HR

Les modifications décrites dans cette section s'appliquent au numéro de version 8.1.2509. Les numéros entre parenthèses dans certains en-têtes se rapportent aux numéros de support dans Microsoft Dynamics Lifecycle Services (LCS).

### <a name="streamlined-employee-entry-and-navigation"></a>Navigation et entrée d'employé simplifiées

Cette fonctionnalité est désormais disponible dans tous environnements. Pour activer cette fonctionnalité, naviguez jusqu'à **Administration du système > Liens > Paramétrage > Paramètres système > Fonctionnalités en version préliminaire**. Sélectionnez **Formulaire Collaborateur et navigation améliorés**. Ces modifications sont activées pour tous les utilisateurs. Vous pouvez désactiver cette option à tout moment.

Pour plus d'informations, voir [Navigation et entrée d'employé simplifiées](./streamlined-employee-entry.md). Pour présentation un vidéo de ces changements, voir [Vue d'ensemble de Dynamics 365 for Talent 2019 2e vague de lancement](https://aka.ms/ROGT19RW2ROV).

### <a name="you-can-enable-preview-features-in-sandbox-and-trial-environments"></a>Vous pouvez activer les fonctions d'aperçu dans les environnements de bac à sable et d'essai.

Lorsque vous provisionnez une nouvelle instance de Talent, vous pouvez indiquer si le type d'instance est Production ou Bac à sable. Les instances du type Bac à sable permettent de tester les nouvelles fonctions en avant-première. Si vous souhaitez que l'une de vos instances existantes soit mise à jour avec le type d'instance Bac à sable (sandbox), contactez le Support pour initier la demande de modification.

Pour plus d'informations sur la manière dont les modifications sont publiées, voir [Mettre en service Talent](./provisioning-talent.md).

### <a name="compensation-date-defaults-to-a-different-date-than-the-position-assignment-337694"></a>Date de compensation définie par défaut à une autre date que l'affectation de poste (337694)

Avec cette modification, la date de début de la compensation par défaut est définie à la date de début de l'affectation de poste.

### <a name="not-able-to-end-compensation-along-with-its-position-assignment-328993"></a>Impossible de terminer la rémunération avec l'affectation de ce poste (328993)

Avec ce changement, vous pouvez terminer la rémunération lorsque vous mettez un terme à l'affectation de poste à l'aide de **Terminer l'affectation** dans la page **Affectations de poste du collaborateur** avec les actions du personnel activées.

### <a name="bank-account-validation-requires-routing-and-account-numbers-in-all-locations-340403"></a>La validation de compte bancaire nécessite des numéros d'acheminement et de compte à tous les emplacements (340403)

Avec les modifications de cette semaine, une nouvelle option de configuration a été ajoutée pour désactiver **Numéro d'acheminement** et la validation obligatoire **Numéro de compte**. 

### <a name="attachments-are-not-enabled-in-mss-performance-journals-for-performance-feedback-341794"></a>Les pièces jointes ne sont pas activées dans les journaux de performances MSS pour les commentaires en matière de performances (341794)

Avec le lancement de cette semaine, les pièces jointes sont activées pour les éléments de commentaire sur la page du journal des performances.

### <a name="leave-request-details-dont-sync-from-common-data-service-to-talent-369608"></a>Les détails de la demande de congés ne se synchronisent pas de Common Data Service vers Talent (369608)

Avec ces modifications, les détails de la demande de congés qui sont mis à jour dans Common Data Service se synchroniseront vers Talent.

### <a name="job-description-doesnt-display-for-the-job-in-the-skill-gap-analysis-page-369398"></a>La description de la tâche ne s'affiche pas pour la tâche sur la page d'analyse des écarts de compétence (369398)

Dans la version de cette semaine, la description s'affiche lorsque vous sélectionnez la tâche sur la page **Analyse des écarts de compétence**.

## <a name="coming-soon"></a>Prochainement

### <a name="print-performance-reviews"></a>Imprimer les évaluations des performances

Les employés, les responsables et les professionnels des RH peuvent imprimer l'examen des performances d'un employé.
