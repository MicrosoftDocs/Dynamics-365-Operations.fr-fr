# <a name="share-content-with-other-contributors"></a><span data-ttu-id="61007-103">Partager du contenu avec d'autres contributeurs</span><span class="sxs-lookup"><span data-stu-id="61007-103">Share content with other contributors</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="61007-104">Microsoft Dynamics 365 Talent: Onboard vous permet de partager des guides et des modèles d'intégration avec d'autres contributeurs afin qu'ils puissent réviser les documents et apporter des modifications.</span><span class="sxs-lookup"><span data-stu-id="61007-104">Microsoft Dynamics 365 Talent: Onboard lets you share onboarding guides and templates with other contributors so that they can review the documents and make changes.</span></span>

## <a name="share-an-onboarding-guide-with-other-contributors"></a><span data-ttu-id="61007-105">Partager un guide d'intégration avec d'autres contributeurs</span><span class="sxs-lookup"><span data-stu-id="61007-105">Share an onboarding guide with other contributors</span></span>

1. <span data-ttu-id="61007-106">Dans le menu à gauche, sélectionnez **Guides**, puis sélectionnez le guide à partager.</span><span class="sxs-lookup"><span data-stu-id="61007-106">On the left menu, select **Guides**, and then select the guide that you want to share.</span></span>
2. <span data-ttu-id="61007-107">Sélectionnez le bouton **Plus** (les trois points verticaux) dans l'angle supérieur droit de la page, puis sélectionnez **Partager**.</span><span class="sxs-lookup"><span data-stu-id="61007-107">Select the **More** button (the three vertical dots) in the upper-right corner of the page, and then select **Share**.</span></span>

    ![[<span data-ttu-id="61007-108">Partage d'un guide d'intégration avec d'autres contributeurs</span><span class="sxs-lookup"><span data-stu-id="61007-108">Sharing an onboarding guide with other contributors</span></span>](./media/onboard-share-guide.png)](./media/onboard-share-guide.png)

3. <span data-ttu-id="61007-109">Sous **Partager avec**, si les destinataires doivent pouvoir réviser le guide, définissez l'option **Ces personnes peuvent-elles apporter des modifications ?** sur **Oui**.</span><span class="sxs-lookup"><span data-stu-id="61007-109">Under **Share with**, if recipients should be able to revise the guide, set the **Can these people make edits?** option to **On**.</span></span> <span data-ttu-id="61007-110">Si les destinataires doivent être informés par e-mail que vous avez partagé le guide avec eux, définissez l'option **Envoyer une notification par e-mail aux destinataires** sur **Oui**.</span><span class="sxs-lookup"><span data-stu-id="61007-110">If recipients should be notified by email that you've shared the guide with them, set the **Send email notification to recipients** option to **On**.</span></span>

    ![[<span data-ttu-id="61007-111">Définition des préférences d'accès et d'e-mail pour un guide d'intégration</span><span class="sxs-lookup"><span data-stu-id="61007-111">Setting access and email preferences for an onboarding guide</span></span>](./media/onboard-share-guide-with-contributors.png)](./media/onboard-share-guide-with-contributors.png)

4. <span data-ttu-id="61007-112">Entrez un nom ou une adresse e-mail, puis sélectionnez le destinataire souhaité dans la liste déroulante.</span><span class="sxs-lookup"><span data-stu-id="61007-112">Enter a name or email address, and select the desired recipient in the drop-down list.</span></span>
5. <span data-ttu-id="61007-113">Sous l'onglet **Gérer l'accès**, vous pouvez examiner ou modifier ce que les destinataires peuvent faire avec le guide.</span><span class="sxs-lookup"><span data-stu-id="61007-113">On the **Manage access** tab, you can review or change what recipients can do with the guide.</span></span> <span data-ttu-id="61007-114">Pour modifier l'accès d'un destinataire, dans la liste déroulante en regard du nom du destinataire, sélectionnez **Peut afficher**, **Peut modifier** ou **Supprimer**, selon le cas.</span><span class="sxs-lookup"><span data-stu-id="61007-114">To change a recipient's access, in the drop-down list next to the recipient's name, select **Can View**, **Can edit**, or **Remove**, as appropriate.</span></span>
6. <span data-ttu-id="61007-115">Lorsque vous avez fini d'ajouter ou de modifier des destinataires, sélectionnez **Mettre à jour**.</span><span class="sxs-lookup"><span data-stu-id="61007-115">When you've finished adding or modifying recipients, select **Update**.</span></span>

## <a name="share-an-onboarding-template-with-other-contributors"></a><span data-ttu-id="61007-116">Partager un modèle d'intégration avec d'autres contributeurs</span><span class="sxs-lookup"><span data-stu-id="61007-116">Share an onboarding template with other contributors</span></span>

1. <span data-ttu-id="61007-117">Dans le menu à gauche, sélectionnez **Modèles**.</span><span class="sxs-lookup"><span data-stu-id="61007-117">On the left menu, select **Templates**.</span></span>
2. <span data-ttu-id="61007-118">Sous **Mes modèles**, sélectionnez le bouton **Plus** (les points de suspension \[**...**\]) dans le coin supérieur droit du modèle à partager, puis sélectionnez **Partager**.</span><span class="sxs-lookup"><span data-stu-id="61007-118">Under **My templates**, select the **More** button (the ellipsis \[**...**\]) in the upper-right corner of the template that you want to share, and then select **Share**.</span></span>

    <span data-ttu-id="61007-119">[![Partage d'un modèle avec d'autres contributeurs](./media/onboard-share-template.png)](./media/onboard-share-template.png)</span><span class="sxs-lookup"><span data-stu-id="61007-119">[![Sharing a template with other contributors](./media/onboard-share-template.png)](./media/onboard-share-template.png)</span></span>

3. <span data-ttu-id="61007-120">Sous **Destinataire**, si les destinataires doivent pouvoir réviser le modèle, définissez l'option **Ces personnes peuvent-elles apporter des modifications ?** sur **Oui**.</span><span class="sxs-lookup"><span data-stu-id="61007-120">Under **Recipient**, if recipients should be able to revise the template, set the **Can these people make edits?** option to **On**.</span></span> <span data-ttu-id="61007-121">Si les destinataires doivent être informés par e-mail que vous avez partagé le modèle avec eux, définissez l'option **Envoyer une notification par e-mail aux destinataires** sur **Oui**.</span><span class="sxs-lookup"><span data-stu-id="61007-121">If recipients should be notified by email that you've shared the template with them, set the **Send email notification to recipients** option to **On**.</span></span>

    <span data-ttu-id="61007-122">[Définition des préférences d'accès et d'e-mail pour un modèle](./media/onboard-share-template-access-and-email.png)](./media/onboard-share-template-access-and-email.png)</span><span class="sxs-lookup"><span data-stu-id="61007-122">[!Setting access and email preferences for a template](./media/onboard-share-template-access-and-email.png)](./media/onboard-share-template-access-and-email.png)</span></span>

4. <span data-ttu-id="61007-123">Entrez un nom ou une adresse e-mail, puis sélectionnez le destinataire souhaité dans la liste déroulante.</span><span class="sxs-lookup"><span data-stu-id="61007-123">Enter a name or email address, and select the desired recipient in the drop-down list.</span></span>
5. <span data-ttu-id="61007-124">Sous l'onglet **Gérer l'accès**, vous pouvez examiner ou modifier ce que les destinataires peuvent faire avec le modèle.</span><span class="sxs-lookup"><span data-stu-id="61007-124">On the **Manage access** tab, you can review or change what recipients can do with the template.</span></span> <span data-ttu-id="61007-125">Pour modifier l'accès d'un destinataire, dans la liste déroulante en regard du nom du destinataire, sélectionnez **Peut afficher**, **Peut modifier** ou **Supprimer**, selon le cas.</span><span class="sxs-lookup"><span data-stu-id="61007-125">To change a recipient's access, in the drop-down list next to the recipient's name, select **Can View**, **Can edit**, or **Remove**, as appropriate.</span></span>
6. <span data-ttu-id="61007-126">Lorsque vous avez fini d'ajouter ou de modifier des destinataires, sélectionnez **Mettre à jour**.</span><span class="sxs-lookup"><span data-stu-id="61007-126">When you've finished adding or modifying recipients, select **Update**.</span></span>
