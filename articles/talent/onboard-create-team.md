# <a name="create-a-hiring-team-by-using-onboard"></a>Créer une équipe de recrutement à l'aide de Onboard

[!include [banner](includes/banner.md)]

Dans Microsoft Dynamics 365 Talent: Onboard, vous pouvez créer des équipes de recrutement. Vous pouvez ensuite affecter des guides et des modèles d'intégration à chaque équipe.

## <a name="create-a-hiring-team"></a>Créer une équipe de recrutement

1. Dans le menu à gauche, sélectionnez **Équipes**.
2. Sous **Équipes**, sélectionnez la vignette **Ajouter** (signe plus) \[**+**\].
3. Dans la boîte de dialogue **Créer une nouvelle équipe**, sous **Nom de l'équipe**, entrez un nom pour l'équipe de recrutement.

    ![[Création d'une nouvelle équipe dans Onboard](./media/onboard-create-team.png)](./media/onboard-create-team.png)

4. Sous **Choisir les membres de l'équipe**, entrez le nom ou l'adresse e-mail de chaque membre de l'équipe.

    Pour supprimer des membres de l'équipe, sélectionnez le **X** qui apparaît en regard de leur nom dans la zone.

5. Pour envoyer un e-mail aux nouveaux membres de l'équipe, activez l'option **Envoyer un e-mail aux nouveaux membres**.
6. Sélectionnez **Créer**.
7. Si vous avez indiqué que vous souhaitez envoyer un e-mail aux nouveaux membres de l'équipe, modifiez l'e-mail dans la boîte de dialogue suivante, puis sélectionnez **Terminé**.

## <a name="assign-onboarding-guides-to-a-hiring-team"></a>Affecter des guides d'intégration à une équipe de recrutement

1. Dans le menu à gauche, sélectionnez **Équipes**.
2. Sélectionnez l'équipe.
3. Sous l'onglet **Guides**, sélectionnez **Ajouter des guides**.

    ![[Ajout de guides d'intégration à une équipe](./media/onboard-add-guides-to-team.png)](./media/onboard-add-guides-to-team.png)

4. Activez la case à cocher pour chaque guide d'intégration que vous souhaitez affecter à l'équipe, puis sélectionnez **Ajouter**.

    ![[Sélection des guides d'intégration à ajouter à l'équipe](./media/onboard-select-guides.png)](./media/onboard-select-guides.png)

## <a name="assign-onboarding-templates-to-a-hiring-team"></a>Affecter des modèles d'intégration à une équipe de recrutement

1. Dans le menu à gauche, sélectionnez **Équipes**.
2. Sélectionnez l'équipe.
3. Sous l'onglet **Modèles**, sélectionnez **Ajouter des modèles**.

    ![[Ajout de modèles à une équipe](./media/onboard-add-templates-to-team.png)](./media/onboard-add-templates-to-team.png)

4. Activez la case à cocher pour chaque modèle que vous souhaitez affecter à l'équipe, puis sélectionnez **Ajouter**.

    ![[Sélection des modèles à ajouter à l'équipe](./media/onboard-select-templates.png)](./media/onboard-select-templates.png)
