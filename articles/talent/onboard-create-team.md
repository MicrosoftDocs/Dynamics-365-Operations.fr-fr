## <a name="create-a-hiring-team"></a><span data-ttu-id="0580b-106">Créer une équipe de recrutement</span><span class="sxs-lookup"><span data-stu-id="0580b-106">Create a hiring team</span></span>

1. <span data-ttu-id="0580b-107">Dans le menu à gauche, sélectionnez **Équipes**.</span><span class="sxs-lookup"><span data-stu-id="0580b-107">On the left menu, select **Teams**.</span></span>
2. <span data-ttu-id="0580b-108">Sous **Équipes**, sélectionnez la vignette **Ajouter** (signe plus) \[**+**\].</span><span class="sxs-lookup"><span data-stu-id="0580b-108">Under **Teams**, select the **Add** (plus sign \[**+**\]) tile.</span></span>
3. <span data-ttu-id="0580b-109">Dans la boîte de dialogue **Créer une nouvelle équipe**, sous **Nom de l'équipe**, entrez un nom pour l'équipe de recrutement.</span><span class="sxs-lookup"><span data-stu-id="0580b-109">In the **Create a new team** dialog box, under **Team name**, enter a name for the hiring team.</span></span>

    ![[<span data-ttu-id="0580b-110">Création d'une nouvelle équipe dans Onboard</span><span class="sxs-lookup"><span data-stu-id="0580b-110">Creating a new team in Onboard</span></span>](./media/onboard-create-team.png)](./media/onboard-create-team.png)

4. <span data-ttu-id="0580b-111">Sous **Choisir les membres de l'équipe**, entrez le nom ou l'adresse e-mail de chaque membre de l'équipe.</span><span class="sxs-lookup"><span data-stu-id="0580b-111">Under **Choose team members**, enter the name or email address of each team member.</span></span>

    <span data-ttu-id="0580b-112">Pour supprimer des membres de l'équipe, sélectionnez le **X** qui apparaît en regard de leur nom dans la zone.</span><span class="sxs-lookup"><span data-stu-id="0580b-112">To remove team members, select the **X** that appears next to their name under the box.</span></span>

6. <span data-ttu-id="0580b-114">Sélectionnez **Créer**.</span><span class="sxs-lookup"><span data-stu-id="0580b-114">Select **Create**.</span></span>
7. <span data-ttu-id="0580b-115">Si vous avez indiqué que vous souhaitez envoyer un e-mail aux nouveaux membres de l'équipe, modifiez l'e-mail dans la boîte de dialogue suivante, puis sélectionnez **Terminé**.</span><span class="sxs-lookup"><span data-stu-id="0580b-115">If you indicated that you want to email new team members, edit the email in the next dialog box, and then select **Done**.</span></span>
