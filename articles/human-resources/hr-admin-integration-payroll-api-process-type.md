# <a name="process-type"></a>Type de processus

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Cette rubrique décrit l’option Non Oui définie pour Dynamics 365 Human Resources.

## <a name="option-set-process-type"></a>Ensemble d’options : Type de processus

Nom physique : mshr_hrmCompProcessType

Cette énumération fournit le groupe d’options pour le type de processus dans les entités virtuelles Ressources humaines.

| Valeur  | Étiquette | Description  |
| --- | --- | --- |
| 200000000 | None | Aucun(e). |
| 200000001 | Annuel | Annuel. |
| 200000002 | Anniversaire d’ancienneté | Anniversaire d’ancienneté. |
| 200000003 | Non cyclique | Non cyclique. |
| 200000004 | ½ an | Semestriel. |
| 200000005 | Trimestriel | Trimestriel. |
| 200000006 | Tous les mois | Tous les mois. |
