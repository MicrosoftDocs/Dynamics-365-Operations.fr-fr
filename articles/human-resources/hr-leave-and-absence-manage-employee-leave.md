# <a name="manage-employee-leave"></a>Gérer les congés des employés

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Vous pouvez gérer les congés d’un employé par type de congé. Cela comprend l’expiration de l’inscription aux congés et l’ajustement des soldes des types de congés. 

## <a name="adjust-leave-balances"></a>Ajuster les soldes de congés

1. Dans le dossier de l’employé, sélectionnez **Congé**.

2. Sélectionnez **Configuration des congés et des absences**.

3. Sélectionnez **Ajuster l’équilibre**.

4. Sélectionnez le **type de congé**.

5. Saisissez un **Montant d’ajustement**. 

6. Une autre solution consiste à sélectionner une **Date**. 

Vous pouvez inclure un code motif et un commentaire lors de l’ajustement du solde de congés d’un employé. 

Désormais, lorsque vous survolez un solde de congés, les informations suivantes s'affichent :

- **Disponible** : la valeur **Total cette année** moins la valeur **Pris cette année**.
- **Total cette année** : tous les régularisations, ajustements et reports pour l’année.
- **Pris cette année** : tous les congés approuvés.
