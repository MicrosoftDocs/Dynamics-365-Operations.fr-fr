<span data-ttu-id="d2d8c-104">Vous pouvez suspendre les congés d’un employé pour interrompre le traitement des régularisations de congés pour les types de congés sélectionnés.</span><span class="sxs-lookup"><span data-stu-id="d2d8c-104">You can suspend leave for an employee to stop leave accruals from being processed for selected leave types.</span></span> 

## <a name="suspend-leave-and-absence-for-an-employee"></a><span data-ttu-id="d2d8c-105">Suspendre les congés et les absences pour un employé</span><span class="sxs-lookup"><span data-stu-id="d2d8c-105">Suspend leave and absence for an employee</span></span>

1. <span data-ttu-id="d2d8c-106">Dans le dossier de l’employé, sélectionnez **Congé**.</span><span class="sxs-lookup"><span data-stu-id="d2d8c-106">On the employee's record, select **Leave**.</span></span>

2. <span data-ttu-id="d2d8c-107">Sélectionnez **Suspendre les congés**.</span><span class="sxs-lookup"><span data-stu-id="d2d8c-107">Select **Suspend leave**.</span></span>

4. <span data-ttu-id="d2d8c-109">Dans la boîte de dialogue **Suspendre la régularisation des congés**, sélectionnez **Type de congé** avec les champs **Date de début** et **Date de fin** définis pour la suspension.</span><span class="sxs-lookup"><span data-stu-id="d2d8c-109">In the **Suspend leave accrual** dialog box, select the **Leave type** along with the **Start date** and **End date** for the suspension.</span></span>
