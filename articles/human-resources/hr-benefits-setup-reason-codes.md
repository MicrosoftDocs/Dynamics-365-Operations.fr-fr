# <a name="set-up-reason-codes"></a>Paramétrer des codes motif

[!include [banner](includes/preview-feature.md)]

Dynamics 365 Human Resources utilise des codes de motif pour expliquer pourquoi les avantages d'un employé changent. 

1. Dans l'espace de travail **Gestion des avantages**, sous **Configuration**, sélectionnez **Codes motif**.

2. Sélectionnez **Nouveau**.

3. Spécifiez les valeurs les champs suivants :

   | Champ | Description |
   | --- | --- |
   | Code motif | Nom unique pour identifier la raison pour laquelle un employé modifierait la souscription à un plan d'avantages. |
   | Description | Description du code motif. |

4. Sélectionnez **Enregistrer**.
