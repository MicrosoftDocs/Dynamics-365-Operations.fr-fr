# <a name="create-plan-types"></a><span data-ttu-id="5e069-102">Création de types de plan</span><span class="sxs-lookup"><span data-stu-id="5e069-102">Create plan types</span></span>

[!include [banner](includes/preview-feature.md)]

<span data-ttu-id="5e069-110">Si un type de plan implique des contacts, le type de plan indique si les contacts sont des bénéficiaires ou des personnes à charge.</span><span class="sxs-lookup"><span data-stu-id="5e069-110">If a plan type involves contacts, the plan type indicates whether contacts are beneficiaries or dependents.</span></span> <span data-ttu-id="5e069-111">Par exemple, un type de plan d'assurance vie de base aurait des bénéficiaires, tandis qu'un type de plan d'assurance maladie de base aurait des personnes à charge.</span><span class="sxs-lookup"><span data-stu-id="5e069-111">For example, a Basic life plan type would have beneficiaries, while a Basic medical plan type would have dependents.</span></span> <span data-ttu-id="5e069-112">Dans certains cas, un plan peut ne pas avoir de contacts personnels.</span><span class="sxs-lookup"><span data-stu-id="5e069-112">In some cases, a plan may not have any personal contacts.</span></span> <span data-ttu-id="5e069-113">Par exemple, un compte de dépenses flexible ou une allocation de stationnement.</span><span class="sxs-lookup"><span data-stu-id="5e069-113">For example, a Flexible Spending Account or Parking allowance.</span></span>

<span data-ttu-id="5e069-114">Un type de plan peut définir des options de couverture.</span><span class="sxs-lookup"><span data-stu-id="5e069-114">A plan type may define coverage options.</span></span> <span data-ttu-id="5e069-115">Les options de couverture sont définies dans le formulaire d'option de couverture.</span><span class="sxs-lookup"><span data-stu-id="5e069-115">The coverage options are defined in the Coverage option form.</span></span> <span data-ttu-id="5e069-116">Une option de couverture peut spécifier le montant de l'avantage ou les contacts éligibles pour le type de plan.</span><span class="sxs-lookup"><span data-stu-id="5e069-116">A coverage option can specify the amount of the benefit or the contacts who are eligible for the plan type.</span></span> <span data-ttu-id="5e069-117">Par exemple, si le type de contact est Bénéficiaire, l'option de couverture doit définir les modalités de ce que le bénéficiaire est en droit de recevoir lorsque l'avantage est utilisé.</span><span class="sxs-lookup"><span data-stu-id="5e069-117">For example, if the contact type is Beneficiary, the coverage option should define the terms of what the beneficiary is eligible to receive when the benefit is utilized.</span></span> <span data-ttu-id="5e069-118">Si le type de contact est Personne à charge, l'option de couverture doit définir la relation entre la personne à charge et l'employé.</span><span class="sxs-lookup"><span data-stu-id="5e069-118">If the contact type is Dependent, the coverage option should define the relationship between the dependent and employee.</span></span>
