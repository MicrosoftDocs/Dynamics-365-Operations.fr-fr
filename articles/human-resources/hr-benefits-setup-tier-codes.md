# <a name="set-up-tier-codes"></a>Paramétrage des codes de niveau

[!include [banner](includes/preview-feature.md)]

Dynamics 365 Human Resources utilise des codes de niveau avec des taux d'avantage payés par les employés et l'employeur. Vous pouvez définir un nombre illimité de codes de niveau et configurer des calculs de taux pour répondre aux besoins de votre entreprise.

1. Dans l'espace de travail **Gestion des avantages**, sous **Configuration** , sélectionnez **Codes de niveau**.

2. Sélectionnez **Nouveau**.

3. Spécifiez les valeurs les champs suivants :

   | Champ | Description |
   | --- | --- |
   | Code de niveau | Code ou numéro de niveau d'avantage unique pour identifier le code de niveau. |
   | Description | Courte description du code de niveau d'avantage. |

4. Sélectionnez **Enregistrer**.
