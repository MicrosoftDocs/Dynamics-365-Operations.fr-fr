# <a name="removed-or-deprecated-features-in-dynamics-365-human-resources"></a>Fonctionnalités supprimées ou déconseillées dans Dynamics 365 Human Resources

Cet article décrit les fonctions qui ont été supprimées, ou qu’il est prévu de supprimer de Microsoft Dynamics 365 Human Resources.

- Une fonction *supprimée* n’est plus disponible dans le produit.
- Une fonction *déconseillée* n’est pas en développement actif et peut être supprimée dans une prochaine mise à jour.

Cette liste est conçue pour vous aider à prendre en compte ces suppressions et abandons pour votre propre planification.

## <a name="features-removed-or-deprecated-in-human-resources"></a>Fonctionnalités supprimées ou déconseillées dans Human Resources 

### <a name="new-lcs-project-creation-in-human-resources-stand-alone"></a>Création d’un nouveau projet LCS dans Human Resources (autonome) 

[Mettre en service Human Resources](/hr-admin-setup-provision)

| &nbsp; | &nbsp; |
|------------|--------------------|
| **Motif de l’abandon/de la suppression** | Remplacé par la création de projet dans l’infrastructure des applications de finances et d’opérations. Pour en savoir plus sur la fusion des infrastructures, voir [FAQ sur la fusion de l’infrastructure Dynamics 365 Human Resource](/hr-infrastructure-merge-faq). |
| **Remplacé par une autre fonctionnalité ?**   | Oui |
| **Zones de produit affectées**         | Application |
| **Option de déploiement**              | Tout |
| **Status**                         | Obsolète : depuis juillet 2022, nous ne prenons plus en charge la création de nouveaux projets dans Human Resources (autonome). |
