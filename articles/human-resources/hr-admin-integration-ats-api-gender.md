# <a name="gender"></a>Sexe


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Cette rubrique décrit l’option Sexe définie pour Dynamics 365 Human Resources.

Nom physique : mshr_hcmpersongender

Cette énumération fournit le jeu d’options des sexes pour le candidat. Elle est disponible dans le jeu d’options mshr_hcmpersongender.

| Valeur | Étiquette | Description |
| --- | --- | --- |
| 200000000 | None | Aucun sexe n’a été spécifié. |
| 200000001 | Masculin | Masculin. |
| 200000002 | Féminin | Féminin. |
| 200000003 | NonSpecific | Sélection pour un sexe non spécifique. |
