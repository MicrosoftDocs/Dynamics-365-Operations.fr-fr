# <a name="blank-yes-no"></a>Vide Oui Non


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Cet article décrit l’option Vide Oui Non définie pour Dynamics 365 Human Resources.

Nom physique : mshr_hrmblankyesno

Cette énumération fournit l’option définie pour les propriétés oui/non qui autorise également un espace vide.

| Valeur | Étiquette | Description |
| --- | --- | --- |
| 200000000 | Blanc | Aucune valeur n’a été sélectionnée. |
| 200000001 | Oui | Oui. |
| 200000002 | Non | Non |
