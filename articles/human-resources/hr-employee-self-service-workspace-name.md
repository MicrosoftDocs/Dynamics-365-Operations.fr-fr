# <a name="change-employee-self-service-workspace-name"></a><span data-ttu-id="d7c40-103">Modifier le nom de l'espace de travail en libre service des employés</span><span class="sxs-lookup"><span data-stu-id="d7c40-103">Change Employee self service workspace name</span></span>

<span data-ttu-id="d7c40-104">Si vous avez des bénévoles ou d'autres personnels non-employés, vous voudrez peut-être changer le nom de l'espace de travail **Libre service des employés**.</span><span class="sxs-lookup"><span data-stu-id="d7c40-104">If you have volunteers or other non-employees, you might want to change the name of the **Employee self-service** workspace.</span></span> <span data-ttu-id="d7c40-105">Vous pouvez modifier cet espace de travail en **Libre service**.</span><span class="sxs-lookup"><span data-stu-id="d7c40-105">You can change this workspace to **Self service** instead.</span></span>

> [!NOTE]
> <span data-ttu-id="d7c40-106">Changer le nom de l'espace de travail **Libre service des employés** modifie également l'élément de menu utilisé en interne par Dynamics 365 Human Resources.</span><span class="sxs-lookup"><span data-stu-id="d7c40-106">Changing the name of the **Employee self-service** workspace also changes the menu item that is used internally by Dynamics 365 Human Resources.</span></span> <span data-ttu-id="d7c40-107">Si vous avez précédemment appliqué des personnalisations de sécurité à l'élément de menu **HcmEmployeeSelfServiceWorkspace**, nous vous recommandons d'appliquer les mêmes modifications à **HcmSelfServiceWorkspace** pour conserver la parité.</span><span class="sxs-lookup"><span data-stu-id="d7c40-107">If you previously applied security customizations to the **HcmEmployeeSelfServiceWorkspace** menu item, we recommend applying the same changes to **HcmSelfServiceWorkspace** to maintain parity.</span></span>

3. <span data-ttu-id="d7c40-110">Sous **Nom d'affichage**, sélectionnez **Libre service**.</span><span class="sxs-lookup"><span data-stu-id="d7c40-110">Under **Display name**, select **Self service**.</span></span>
