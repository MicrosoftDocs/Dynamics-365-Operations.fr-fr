# <a name="skill-level-type"></a>Type de niveau de compétence

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Cette rubrique décrit l’option Type de niveau de compétence définie pour Dynamics 365 Human Resources.

Nom physique : mshr_hrmskillleveltype

Cette énumération fournit l’ensemble d’options de valeurs qui catégorisent le niveau attribué à la compétence d’un candidat.

| Valeur | Étiquette | Description |
| --- | --- | --- |
| 200000000 | Réel | Niveau de compétence réel. |
| 200000001 | Cible | Valeur ciblée pour le niveau de compétence. |
