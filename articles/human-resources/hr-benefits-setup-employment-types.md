# <a name="set-up-employment-types"></a>Configuration des types d'emploi

[!include [banner](includes/preview-feature.md)]

Configurez les types d'emploi dans Dynamics 365 Human Resources pour qu'ils correspondent à vos collaborateurs.

1. Dans l'espace de travail **Gestion des avantages**, sous **Configuration**, sélectionnez **Types d'emploi**.

2. Sélectionnez **Nouveau**.

3. Spécifiez les valeurs les champs suivants :

   | Champ | Description |
   | --- | --- |
   | Type d'emploi | Identification unique du type d'emploi. |
   | Description | Description du type d'emploi. |

4. Sélectionnez **Enregistrer**.
