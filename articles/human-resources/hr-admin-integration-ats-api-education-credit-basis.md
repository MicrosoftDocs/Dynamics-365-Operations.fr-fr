Cette rubrique décrit l’option Base de crédit de formation définie pour Dynamics 365 Human Resources.

Nom physique : mshr_hcmeducationcreditbasis

Cette énumération fournit l’ensemble d’options de valeurs pour la base des crédits de formation du dossier de formation du candidat.

| 200000001 | Semestre | Semestre. |
| 200000002 | Trimestre | Trimestre. |
| 200000003 | Quadrimestre | Quadrimestre. |
| 200000004 | Terme | Terme. |
| 200000005 | Autres | Autre/s. |
