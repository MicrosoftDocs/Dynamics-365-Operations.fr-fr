# <a name="set-administrator-rights-for-life-events"></a>Définir les droits d’administrateur pour les événements de vie

[!include [banner](../includes/preview-banner.md)]

Les administrateurs peuvent mettre à jour les options d’événement de vie, en fonction des paramètres de configuration. Sur la page **Paramètres des ressources humaines**, vous pouvez configurer les paramètres qui permettent aux administrateurs d’apporter des modifications aux sélections de plans. Vous pouvez également empêcher complètement les administrateurs d’apporter des modifications.

Sur la page **Paramètres des ressources humaines**, vous pouvez configurer des restrictions de changement de plan pour les plans confirmés et les options d’événements de vie.

Si l’option **Plans confirmés** est sélectionnée, les modifications ne peuvent être effectuées que si une période d’inscription est active. (Des exemples de périodes d’inscription incluent les périodes d’inscription ouvertes, les périodes d’inscription des nouveaux employés et les périodes d’inscription des événements de vie.) Si cette option n’est pas sélectionnée, des modifications peuvent être apportées à tout moment.

Si l’option **Options d’événements de vie** est sélectionnée, les changements de plan pendant un événement de vie sont limités par les options d’événement de vie qui sont définies sur le type de plan. Si cette option n’est pas sélectionnée, les sélections de plan peuvent être modifiées lors d’un événement de vie.

## <a name="set-plan-change-restrictions"></a>Définir des restrictions de modification des plans

Sur la page **Paramètres des ressources humaines**, dans l’onglet **Gestion des avantages**, les champs suivants sont disponibles.

| Champ | Description |
|-------|-------------|
| Plans confirmés | Sélectionnez cette option si les modifications apportées à un plan ne sont autorisées que si une période d’inscription est active. Si cette option n’est pas sélectionnée, des modifications peuvent être apportées à tout moment. |
| Options d’événement de vie | Sélectionnez cette option si les changements de plan pendant un événement de vie sont limités par les options d’événement de vie qui sont définies sur le type de plan. Si cette option n’est pas sélectionnée, les sélections de plan peuvent être modifiées lors d’un événement de vie. |
