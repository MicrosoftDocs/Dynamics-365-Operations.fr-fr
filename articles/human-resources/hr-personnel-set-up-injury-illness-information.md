# <a name="set-up-injury-and-illness-information"></a>Paramétrer les informations sur les blessures et les maladies



Les employeurs doivent savoir si leurs employés sont victimes de blessures ou de maladies en raison de risques sur le lieu de travail. La page Blessures et maladies vous permet de paramétrer les informations qui facilitent la déclaration de blessures et de maladies survenues sur le lieu de travail. Vous pouvez paramétrer les types de blessures et de maladies, notamment les types de traitements, les coûts, et les conséquences. Les données fictives utilisées pour créer cette procédure correspondent à la société USMF.

1. Allez dans Ressources humaines > Collaborateurs > Blessures et maladies > Paramétrage des blessures et des maladies.
2. Cliquez sur Nouveau.
3. Entrez une valeur dans le champ Type de blessures et de maladies.
    * Exemple : Fracture  
4. Dans le champ Description, entrez une valeur.
    * Exemple : Fracture osseuse  
5. Cliquez sur l'onglet Parties du corps.
6. Cliquez sur Nouveau.
7. Tapez une valeur dans le champ Partie du corps.
    * Exemple : Poignet  
8. Dans le champ Description, entrez une valeur.
    * Exemple : Poignet  
9. Cliquez sur l'onglet Types de traitement.
10. Cliquez sur Nouveau.
11. Tapez une valeur dans le champ Type de traitement.
    * Exemple : Attelle  
12. Dans le champ Description, entrez une valeur.
    * Exemple : Pose d'une attelle  
13. Cliquez sur l'onglet Types de coûts.
14. Cliquez sur Nouveau.
15. Tapez une valeur dans le champ Type de coût.
    * Exemple : Rayons X  
16. Dans le champ Description, entrez une valeur.
    * Exemple : Rayons X  
17. Cliquez sur l'onglet Types de conséquences.
18. Cliquez sur Nouveau.
19. Tapez une valeur dans le champ Résultat.
    * Exemple : Thérapie  
20. Dans le champ Description, entrez une valeur.
    * Exemple : Kinésithérapie  
21. Cliquez sur l'onglet Niveaux de gravité.
    * Des niveaux de gravité personnalisables peuvent être créés. Par exemple : La gravité 1 désigne une blessure légère et la gravité 3 désigne une blessure grave.  
22. Cliquez sur l'onglet Organismes déclarants.
    * Les organismes déclarants sont les organismes auprès desquels l'incident doit être déclaré. Sélectionnez la case à cocher par défaut correspondant à l'organisme par défaut auprès duquel vous devez déclarer les blessures et la maladies.  
23. Cliquez sur Enregistrer.
