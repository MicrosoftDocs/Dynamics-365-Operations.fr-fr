# <a name="payroll---employment-type-option-set"></a>Paie - Jeu d’options du type d’emploi

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Cet article décrit le jeu d’options Type d’emploi défini pour Dynamics 365 Human Resources.

## <a name="option-set-employment-type"></a>Jeu d’options : Type d’emploi

Cette énumération fournit le jeu d’options des types d’emploi pour l’employé. Elle est disponible dans le jeu d’options mshr_hcmemploymenttype.

| Valeur | Étiquette | Description |
| --- | --- | --- |
| 200000000 | Employé | Employé |
| 200000001 | Prestataire | Prestataire |
