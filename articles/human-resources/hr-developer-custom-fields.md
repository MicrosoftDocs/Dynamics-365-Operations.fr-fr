# <a name="custom-fields"></a>Champs personnalisés

Vous pouvez ajouter des champs personnalisés aux pages dans Dynamics 365 Human Resources. Pour plus d'informations sur la création de champs personnalisés sur une page, voir [Création et utilisation des champs personnalisés](https://docs.microsoft.com/dynamics365/unified-operations/fin-and-ops/get-started/user-defined-fields).

Les entités suivantes de Common Data Service prennent en charge les champs personnalisés créés dans Human Resources :

- Collaborateur
- Origine ethnique
- Statut de vétéran
- Code langue
- Mission
- Type de poste
- Fonction du poste
- Poste
- Type de poste

Veillez à activer tous les champs personnalisés que vous créez afin de pouvoir les utiliser dans Common Data Service. Pour plus d'informations, voir [Affichage des champs personnalisés des entités de données](https://docs.microsoft.com/dynamics365/unified-operations/fin-and-ops/get-started/user-defined-fields#exposing-custom-fields-on-data-entities).
