# <a name="payroll---gender-option-set"></a>Paie – Ensemble d’options de sexe


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Cet article décrit l’option Sexe Oui Non définie pour Dynamics 365 Human Resources.

## <a name="option-set-gender-option"></a>Ensemble d’options : option de sexe

Cette énumération fournit le jeu d’options des sexes pour l’employé Elle est disponible dans le jeu d’options mshr_hcmpersongender.

| Valeur | Étiquette | Description |
| --- | --- | --- |
| 200000000 | None | Aucun sexe n’a été spécifié. |
| 200000001 | Masculin | Masculin. |
| 200000002 | Féminin | Féminin. |
| 200000003 | NonSpecific | Sélection pour un sexe non spécifique. |
