# <a name="contact-type"></a>Type de contact

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Cette rubrique décrit l’option Type de contact définie pour Dynamics 365 Human Resources.

Nom physique : mshr_logisticselectronicaddressmethodtype

Cette énumération fournit le jeu d’options des valeurs de statut pour les types de contact des candidats. 

| Valeur | Étiquette | Description |
| --- | --- | --- |
| 200000000 | None | Aucun type n’est sélectionné. |
| 200000001 | Téléphone | Numéro de téléphone |
| 200000002 | Adresse de messagerie | Adresse e-mail. |
| 200000003 | URL | URL du site Web. |
| 200000004 | Télex | Numéro de télex. |
| 200000005 | Télécopie | Numéro de télécopie. |
| 200000006 | Facebook | Compte Facebook. Identifié par ID utilisateur. |
| 200000007 | Twitter | Compte Twitter. Identifié par @username. |
| 200000008 | LinkedIn | Compte LinkedIn. Identifié par le nom d’utilisateur. |
