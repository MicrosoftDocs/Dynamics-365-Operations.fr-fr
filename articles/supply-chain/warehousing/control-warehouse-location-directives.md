## <a name="location-directives"></a><span data-ttu-id="b563c-139">Instructions d’emplacement</span><span class="sxs-lookup"><span data-stu-id="b563c-139">Location directives</span></span>

<span data-ttu-id="b563c-140">Les instructions d’emplacement sont des règles qui aident à identifier les emplacements de prélèvement et de rangement pour le mouvement du stock.</span><span class="sxs-lookup"><span data-stu-id="b563c-140">Location directives are rules that help identify pick and put locations for inventory movement.</span></span> <span data-ttu-id="b563c-141">Par exemple, dans une transaction de commande client, une instruction d’emplacement détermine où les articles seront prélevés, et où les articles prélevés seront rangés.</span><span class="sxs-lookup"><span data-stu-id="b563c-141">For example, in a sales order transaction, a location directive determines where the items will be picked, and where the picked items will be put.</span></span> <span data-ttu-id="b563c-142">Les instructions d’emplacement sont composés d’un en-tête et de lignes associées, et vous les créez dans la page **Instructions d’emplacement**.</span><span class="sxs-lookup"><span data-stu-id="b563c-142">Location directives consist of a header and associated lines, and you create them on the **Location directives** page.</span></span>

<span data-ttu-id="b563c-143">Dans l’en-tête, chaque instruction d’emplacement doit être associée à un *type d’ordre de travail* qui spécifie le type de transaction de stock pour lequel l’instruction sera utilisée, tels que les commandes client, le réapprovisionnement, ou le prélèvement de matières premières.</span><span class="sxs-lookup"><span data-stu-id="b563c-143">On the header, each location directive must be associated with a *work order type* that specifies the type of inventory transaction that the directive will be used for, such as sales orders, replenishment, or raw material picking.</span></span> <span data-ttu-id="b563c-144">Le *type de travail* indique si l’instruction d’emplacement sera utilisée pour le travail de prélèvement ou de rangement, ou pour un autre processus d’entrepôt, par exemple le comptage ou les modifications de statut du stock.</span><span class="sxs-lookup"><span data-stu-id="b563c-144">The *work type* specifies whether the location directive will be used for picking or putting work, or for some other warehouse process, such as counting or inventory status changes.</span></span> <span data-ttu-id="b563c-145">Vous devez également spécifier un *site* et un *entrepôt*.</span><span class="sxs-lookup"><span data-stu-id="b563c-145">You must also specify a *site* and a *warehouse*.</span></span> <span data-ttu-id="b563c-146">Un *code directif* que vous spécifiez dans l’en-tête peut être utilisé pour lier l’instruction d’emplacement à un ou plusieurs modèles de travail.</span><span class="sxs-lookup"><span data-stu-id="b563c-146">A *directive code* that you specify on the header can be used to link the location directive to one or more work templates.</span></span>

<span data-ttu-id="b563c-150">Les lignes d’instruction d’emplacement définissent des restrictions supplémentaires sur l’application des règles de recherche d’emplacement.</span><span class="sxs-lookup"><span data-stu-id="b563c-150">The location directive lines set additional restrictions on the application of the location finding rules.</span></span> <span data-ttu-id="b563c-151">Vous pouvez spécifier une quantité minimale et une quantité maximale auxquelles l’instruction doit s’appliquer, vous pouvez spécifier que l’instruction doit concerner une unité de stock spécifique.</span><span class="sxs-lookup"><span data-stu-id="b563c-151">You can specify a minimum quantity and a maximum quantity that the directive should apply to, and you can specify that the directive should be for a specific inventory unit.</span></span> <span data-ttu-id="b563c-152">Par exemple, si l’unité de mesure est en palettes, les articles en palettes peuvent être rangés dans un emplacement spécifique.</span><span class="sxs-lookup"><span data-stu-id="b563c-152">For example, if the unit of measure is pallets, the items in pallets can be put in a specific location.</span></span> <span data-ttu-id="b563c-153">Vous pouvez également indiquer si la quantité peut être fractionnée entre plusieurs emplacements.</span><span class="sxs-lookup"><span data-stu-id="b563c-153">You can also specify whether the quantity can be split across multiple locations.</span></span> <span data-ttu-id="b563c-154">Comme l’en-tête des instructions de l’emplacement, chaque ligne d’instruction d’emplacement possède un numéro de souche qui détermine l’ordre des lignes dans lequel les lignes sont évaluées.</span><span class="sxs-lookup"><span data-stu-id="b563c-154">Like the location directive header, each location directive line has a sequence number that determines the order that the lines are assessed in.</span></span>
