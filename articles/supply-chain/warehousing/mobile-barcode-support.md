# <a name="mobile-bar-code-support"></a><span data-ttu-id="5523d-103">Prise en charge des codes-barres mobiles</span><span class="sxs-lookup"><span data-stu-id="5523d-103">Mobile bar code support</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="5523d-104">Comme Android est un projet en open source, tout fabricant de matériel pour les scanneurs de code-barres d'entrepôt peut fabriquer un périphérique pour exécuter le système d'exploitation Android.</span><span class="sxs-lookup"><span data-stu-id="5523d-104">Because Android is an open source project, any manufacturer of hardware for warehouse bar code scanners can build a device to run the Android operating system.</span></span> <span data-ttu-id="5523d-105">Un périphérique est compatible avec Android uniquement s'il peut exécuter des applications qui sont écrites pour l'environnement d'exécution Android.</span><span class="sxs-lookup"><span data-stu-id="5523d-105">A device is only Android-compatible if it can run apps that are written for the Android execution environment.</span></span>
<span data-ttu-id="5523d-106">Toutefois, un fournisseur de matériel peut modifier et créer des superpositions de la version Android qui s'exécutent sur leur matériel.</span><span class="sxs-lookup"><span data-stu-id="5523d-106">However, a hardware vendor can modify and create overlays for the Android version that runs on their hardware.</span></span> <span data-ttu-id="5523d-107">Microsoft ne peut pas être tenu responsable de s'assurer qu'une application de lecture de code-barres mobile pour Android est compatible avec le matériel de lecture du code-barres d'un fabricant et la version d'Android qui s'exécute sur celui-ci.</span><span class="sxs-lookup"><span data-stu-id="5523d-107">Microsoft cannot take any responsibility to ensure that a mobile bar code scanning app for Android is compatible with a manufacturer’s bar code scanning hardware and the Android version that runs on it.</span></span> 

<span data-ttu-id="5523d-108">L'application Entreposage pour Microsoft Dynamics 365 for Finance and Operations a été testée avec une sélection d'appareils sous Android pour la lecture de codes-barres.</span><span class="sxs-lookup"><span data-stu-id="5523d-108">The Warehousing app for Microsoft Dynamics 365 for Finance and Operations has been tested with a selection of Android powered devices for bar code scanning.</span></span> <span data-ttu-id="5523d-109">Ces tests couvrent uniquement des exemples de périphériques disponibles sur le marché.</span><span class="sxs-lookup"><span data-stu-id="5523d-109">These tests only cover a sample of the devices that are available on the market.</span></span>

<span data-ttu-id="5523d-110">En tant que client, nous vous recommandons de tester l'application de lecture mobile Entreposage sur du matériel sélectionné avant de prendre une décision concernant le matériel à acheter.</span><span class="sxs-lookup"><span data-stu-id="5523d-110">As a customer, we recommend that you test the Warehouse mobile scanning app on selected hardware before you decide on the hardware that you want to buy.</span></span>
