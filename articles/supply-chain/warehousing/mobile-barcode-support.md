# <a name="mobile-barcode-support"></a><span data-ttu-id="fa64a-103">Prise en charge mobile des codes-barres</span><span class="sxs-lookup"><span data-stu-id="fa64a-103">Mobile barcode support</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="fa64a-104">Comme Android est un projet en open source, tout fabricant de matériel pour les scanneurs de code à barres d’entrepôt peut fabriquer un périphérique pour exécuter le système d’exploitation Android.</span><span class="sxs-lookup"><span data-stu-id="fa64a-104">Because Android is an open-source project, any manufacturer of hardware for warehouse bar code scanners can build a device to run the Android operating system.</span></span> <span data-ttu-id="fa64a-105">Un périphérique est compatible avec Android uniquement s’il peut exécuter des applications qui sont écrites pour l’environnement d’exécution Android.</span><span class="sxs-lookup"><span data-stu-id="fa64a-105">A device is only Android-compatible if it can run apps that are written for the Android execution environment.</span></span>

<span data-ttu-id="fa64a-106">Toutefois, un fournisseur de matériel peut modifier et créer des superpositions de la version Android qui s’exécutent sur leur matériel.</span><span class="sxs-lookup"><span data-stu-id="fa64a-106">However, a hardware vendor can modify and create overlays for the Android version that runs on their hardware.</span></span> <span data-ttu-id="fa64a-107">Microsoft ne peut pas être tenu responsable de s’assurer qu’une application de lecture de code à barres mobile pour Android est compatible avec le matériel de lecture du code à barres d’un fabricant et la version d’Android qui s’exécute sur celui-ci.</span><span class="sxs-lookup"><span data-stu-id="fa64a-107">Microsoft can't take any responsibility to ensure that a mobile bar code scanning app for Android is compatible with a manufacturer's bar code scanning hardware and the Android version that runs on it.</span></span>

<span data-ttu-id="fa64a-108">L’application mobile gestion des entrepôts a été testée avec une sélection d’appareils Android pour la lecture de codes à barres.</span><span class="sxs-lookup"><span data-stu-id="fa64a-108">The Warehouse Management mobile app has been tested with a selection of Android powered devices for bar code scanning.</span></span> <span data-ttu-id="fa64a-109">Ces tests couvrent uniquement des exemples de périphériques disponibles sur le marché.</span><span class="sxs-lookup"><span data-stu-id="fa64a-109">These tests only cover a sample of the devices that are available on the market.</span></span>

<span data-ttu-id="fa64a-110">En tant que client, nous vous recommandons de tester l’application de lecture mobile Entreposage sur du matériel sélectionné avant de prendre une décision concernant le matériel à acheter.</span><span class="sxs-lookup"><span data-stu-id="fa64a-110">As a customer, we recommend that you test the Warehouse mobile scanning app on selected hardware before you decide on the hardware that you want to buy.</span></span>
