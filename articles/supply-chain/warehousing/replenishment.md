# <a name="replenishment"></a><span data-ttu-id="3a225-103">Réapprovisionnement</span><span class="sxs-lookup"><span data-stu-id="3a225-103">Replenishment</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="3a225-104">Cette rubrique décrit les stratégies de réapprovisionnement disponibles pour les entrepôts qui utilisent la fonctionnalité qui est disponible dans Gestion des entrepôts.</span><span class="sxs-lookup"><span data-stu-id="3a225-104">This topic describes the replenishment strategies that are available for warehouses that use the functionality that is available in Warehouse management.</span></span> <span data-ttu-id="3a225-105">Ces informations dans cette rubrique ne s'appliquent pas à la solution de stockage disponible dans la Gestion des stocks.</span><span class="sxs-lookup"><span data-stu-id="3a225-105">The information in this topic doesn't apply to the warehousing solution that is available in Inventory management.</span></span>

<span data-ttu-id="3a225-106">Les stratégies de réapprovisionnement suivantes sont disponibles :</span><span class="sxs-lookup"><span data-stu-id="3a225-106">The following replenishment strategies are available:</span></span>

- <span data-ttu-id="3a225-107">**Réapprovisionnement de la demande de vague** : Cette stratégie crée un travail de réapprovisionnement pour les commandes ou les charges sortantes si le stock n'est pas disponible lorsque le travail est créé par la vague.</span><span class="sxs-lookup"><span data-stu-id="3a225-107">**Wave demand replenishment** – This strategy creates replenishment work for outbound orders or loads if inventory isn't available when the wave creates work.</span></span> <span data-ttu-id="3a225-108">Par exemple, le travail de réapprovisionnement peut être créé si la quantité requise pour une commande client n'est pas disponible lors du traitement d'une vague.</span><span class="sxs-lookup"><span data-stu-id="3a225-108">For example, replenishment work can be created if the quantity that is required for a sales order isn't available when a wave is processed.</span></span>
- <span data-ttu-id="3a225-109">**Réapprovisionnement minimal et maximal** : Cette stratégie utilise les limites de stockage minimales et maximales pour déterminer quand réapprovisionner les emplacements.</span><span class="sxs-lookup"><span data-stu-id="3a225-109">**Min/Max replenishment** – This strategy uses minimum and maximum stocking limits to determine when locations should be replenished.</span></span> <span data-ttu-id="3a225-110">Les critères d'article et d'emplacement définissent le stock qui est évalué pour le réapprovisionnement.</span><span class="sxs-lookup"><span data-stu-id="3a225-110">The item and location criteria define the inventory that is evaluated for replenishment.</span></span> <span data-ttu-id="3a225-111">Les modèles de réapprovisionnement minimal et maximal constituent le mécanisme principal tenir maintenir des niveaux optimaux dans les emplacements d'enlèvement.</span><span class="sxs-lookup"><span data-stu-id="3a225-111">Min/Max replenishment templates are the primary mechanism for maintaining optimal levels in picking locations.</span></span> <span data-ttu-id="3a225-112">Pour garantir la disponibilité du stock d'enlèvement pour répondre à la demande de vague, vous pouvez utiliser le réapprovisionnement comme complément entre les cycles de réapprovisionnement Min/Max.</span><span class="sxs-lookup"><span data-stu-id="3a225-112">To help guarantee that enough pick face inventory is available to meet wave demand, you can use demand replenishment as a supplement between Min/Max replenishment cycles.</span></span>
- <span data-ttu-id="3a225-113">**Réapprovisionnement de demande de chargement** : Cette stratégie additionne la demande de plusieurs chargements et crée le travail de réapprovisionnement qui est nécessaire pour stocker les emplacements d'enlèvement correspondants.</span><span class="sxs-lookup"><span data-stu-id="3a225-113">**Load demand replenishment** – This strategy sums the demand for several loads and creates the replenishment work that is required in order to stock the relevant picking locations.</span></span> <span data-ttu-id="3a225-114">Cette stratégie permet de garantir que les chargements qui sont créés peuvent être enlevés dans l’entrepôt une fois qu’ils sont disponibles.</span><span class="sxs-lookup"><span data-stu-id="3a225-114">This strategy helps guarantee that the loads that are created can be picked in the warehouse after they are released.</span></span>
- <span data-ttu-id="3a225-115">**Réapprovisionnement immédiat** – Cette stratégie réapprovisionne le stock avant l'exécution d'une vague si l'allocation échoue pour une ligne d'introduction d'emplacement disposant d'un modèle de réapprovisionnement.</span><span class="sxs-lookup"><span data-stu-id="3a225-115">**Immediate replenishment** – This strategy replenishes inventory before a wave is run if allocation fails for a location directive line that has a replenishment template.</span></span> 

<span data-ttu-id="3a225-116">Ces quatre stratégies créent un travail de réapprovisionnement, basé sur un modèle de réapprovisionnement.</span><span class="sxs-lookup"><span data-stu-id="3a225-116">All four strategies create replenishment work, based on a replenishment template.</span></span>

## <a name="wave-demand-replenishment"></a><span data-ttu-id="3a225-117">Réapprovisionnement de la demande de vague</span><span class="sxs-lookup"><span data-stu-id="3a225-117">Wave demand replenishment</span></span>
<span data-ttu-id="3a225-118">Le réapprovisionnement de la demande de vague crée un travail de réapprovisionnement basé sur la demande, si la quantité requise pour les ordres de fabrication, les kanbans, les commandes ou les chargements sortants n'est pas disponible lorsque le travail est créé par la vague.</span><span class="sxs-lookup"><span data-stu-id="3a225-118">Wave demand replenishment creates replenishment work, based on demand, if the quantity that is required for production orders, kanbans, outbound orders, or loads isn't available when a wave creates work.</span></span> <span data-ttu-id="3a225-119">Le modèle de réapprovisionnement contient des informations sur les critères de l’article, l’unité de mesure, l’incrément de la demande et l’emplacement.</span><span class="sxs-lookup"><span data-stu-id="3a225-119">The replenishment template contains information about the item criteria, the unit of measure, the demand increment, and the location.</span></span>
