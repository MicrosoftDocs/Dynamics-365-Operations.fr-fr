# <a name="report-progress-on-a-mobile-job-device"></a>Générer un état de la progression sur un périphérique mobile

[!include [banner](../../includes/banner.md)]

Cette procédure vous indique comment démarrer une tâche de production et générer un état de la progression de celle-ci dans le formulaire d'enregistrement du périphérique de tâche.



Pour que vous soyez en mesure d'exécuter cette procédure, le rôle Administrateur système ou Opérateur doit être associé au compte utilisateur.

1. Accédez à Contrôle de la production > Contrôle et suivi de la production > Périphérique pour le bon de travail.
2. Dans le champ WorkerTextField, entrez le numéro de badge d'un collaborateur. Dans la société fictive USMF, tapez « 123 » pour Christina. Portra.
3. Cliquez sur Connexion.
4. Cliquez sur le bouton de filtre.
5. Cochez ou décochez la case Appliquer le filtre de configuration. Si vous définissez un filtre, vous pouvez utiliser l'unité de production 110 dans USMF.
6. Dans le champ Unité de production, sélectionnez le groupe de ressources sur les tâches de production duquel les collaborateurs peuvent travailler.
7. Dans la liste, cliquer sur le lien dans la ligne sélectionnée.
8. Cliquez sur OK.
9. Cliquez sur le bouton Démarrer la tâche.
10. Cliquez sur OK.
11. Cliquez sur le bouton Connaître l'état de progression.
12. Cliquez sur OK.
13. Cliquez sur le bouton Tâche suivante.
14. Cliquez sur Affecté pour afficher une vue d'ensemble de tous les boutons des tâches de production.
15. Fermez la page.
16. Cliquez sur le bouton Pause.
17. Dans la liste, recherchez et sélectionnez l'enregistrement souhaité.
18. Cliquez sur OK.
19. Cliquez sur le bouton Sortie.
20. Sélectionnez cette option pour vous déconnecter.
21. Cliquez sur OK.
22. Dans le champ WorkerTextField, connectez-vous de nouveau. Vous pouvez sélectionner le travailleur « 123 » dans la société fictive USMF.
23. Cliquez sur Connexion.
24. Cliquez sur Fin de pause.
25. Cliquez sur le bouton Activité.
26. Cliquez sur Annuler.
27. Cliquez sur le bouton Sortie.
28. Sélectionnez cette option pour pointer à la sortie.
29. Cliquez sur OK.
30. Sélectionnez une raison pour laquelle vous pointez à la sortie de façon anticipée.
