## <a name="production-flows-in-lean-manufacturing"></a><span data-ttu-id="f8999-108">Flux de production dans le cadre du concept de lean manufacturing</span><span class="sxs-lookup"><span data-stu-id="f8999-108">Production flows in lean manufacturing</span></span>
<span data-ttu-id="f8999-109">Dans les scénarios de production basés sur des ordres de fabrication, les matières sont émises en fonction d'un ordre de fabrication spécifique.</span><span class="sxs-lookup"><span data-stu-id="f8999-109">In production scenarios that are based on production orders, material is issued to a specific production order.</span></span> <span data-ttu-id="f8999-110">Lors d'une séquence d'opérations, basée sur une nomenclature et des gammes, les produits sont créés et finalement reçus à l'emplacement indiqué.</span><span class="sxs-lookup"><span data-stu-id="f8999-110">During a sequence of operations that is based on a bill of materials (BOM) and routes, products are created and are finally received at the supplied location.</span></span> <span data-ttu-id="f8999-111">Le délai d'exécution des ordres de fabrication varie de quelques minutes à des semaines.</span><span class="sxs-lookup"><span data-stu-id="f8999-111">The throughput time of production orders varies from minutes to weeks.</span></span> <span data-ttu-id="f8999-112">Tous les coûts associés, les matières et le travail sont cumulés sur l'ordre de fabrication.</span><span class="sxs-lookup"><span data-stu-id="f8999-112">All related cost, material, and labor are accumulated on the production order.</span></span> 

<span data-ttu-id="f8999-113">Pour réduire les délais de livraison et le stock excédentaire entre les centres de travail provoqués par la production par lots, le concept de lean manufacturing introduit les réapprovisionnements de kanban et les supermarchés dans la fabrication et le réapprovisionnement d'entrepôt.</span><span class="sxs-lookup"><span data-stu-id="f8999-113">To reduce the delivery lead times and excess inventory between work centers that batch production causes, lean manufacturing introduces kanban replenishment and supermarkets in manufacturing and warehouse replenishment.</span></span> <span data-ttu-id="f8999-114">En général, ces fonctionnalités perturbent la production des cycles de kanban partiellement indépendants.</span><span class="sxs-lookup"><span data-stu-id="f8999-114">Typically, these features disrupt the production of partially independent kanban cycles.</span></span> <span data-ttu-id="f8999-115">Le réapprovisionnement d'un kanban pour un produit semi-fini n'est plus déclenché par une commande pour un produit fini.</span><span class="sxs-lookup"><span data-stu-id="f8999-115">The replenishment of a kanban for a semi-finished product is no longer triggered by an order for a finished product.</span></span> 

<span data-ttu-id="f8999-116">Pour rétablir un contexte de production et de coût pour les différents scénarios de kanban proposés, les flux de production basés sur l'activité sont présentés comme l'épine dorsale du lean manufacturing.</span><span class="sxs-lookup"><span data-stu-id="f8999-116">To re-establish a production and cost context for the various kanban scenarios that are proposed, activity-based production flows have been introduced as the backbone of lean manufacturing.</span></span> <span data-ttu-id="f8999-117">Toutes les règles de kanban se rapportent à cette structure prédéfinie.</span><span class="sxs-lookup"><span data-stu-id="f8999-117">All kanban rules refer to this predefined structure.</span></span> <span data-ttu-id="f8999-118">Le modèle basé sur les activités prend en charge le paramétrage d'un large éventail de scénarios.</span><span class="sxs-lookup"><span data-stu-id="f8999-118">The activity-based model supports the setup of a wide range of scenarios.</span></span> <span data-ttu-id="f8999-119">Toutefois, ce modèle ne complique pas les choses pour les employés de l'atelier, car tous les scénarios utilisent la même interface utilisateur basée sur les activités.</span><span class="sxs-lookup"><span data-stu-id="f8999-119">However, this model doesn't add complexity for the shop floor workers, because all scenarios use the same activity-based user interface.</span></span>
