<span data-ttu-id="4167d-118">Par défaut, si aucune clé de répartition par article n'est affectée aux membres du groupe de planification intersociétés, une prévision de la demande est calculée pour tous les articles affectés à toutes les clés de répartition par article de toutes les sociétés Finance and Operations.</span><span class="sxs-lookup"><span data-stu-id="4167d-118">By default, if no item allocation keys are assigned to intercompany planning group members, a demand forecast is calculated for all items that are assigned to all item allocation keys from all Finance and Operations companies.</span></span> <span data-ttu-id="4167d-119">Des options de filtrage supplémentaires pour les sociétés et les clés de répartition par article sont disponibles dans la page **Générer des prévisions de base statistiques**.</span><span class="sxs-lookup"><span data-stu-id="4167d-119">Additional filtering options for companies and item allocation keys are available on the **Generate statistical baseline forecast** page.</span></span> 

<span data-ttu-id="4167d-120">Examinez le nombre d'articles qui sont prévus.</span><span class="sxs-lookup"><span data-stu-id="4167d-120">Review the number of items that are forecasted.</span></span> <span data-ttu-id="4167d-121">Les articles inutiles peuvent entraîner des coûts augmentés lorsque vous utilisez Microsoft Azure Machine Learning.</span><span class="sxs-lookup"><span data-stu-id="4167d-121">Unnecessary items might cause increased costs when you use Microsoft Azure Machine Learning.</span></span>
