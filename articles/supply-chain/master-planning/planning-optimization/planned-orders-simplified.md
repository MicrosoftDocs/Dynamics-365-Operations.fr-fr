# <a name="planned-orders-simplified"></a>Ordres planifiés simplifiés

[!include [banner](../../includes/banner.md)]

Cet article décrit comment afficher, gérer et confirmer les ordres planifiés à l’aide de la page **Ordres planifiés simplifiés**.

La page **Ordres planifiés simplifiés** est une version simplifiée de la page **Ordres planifiés**. Elle offre moins d’options, mais est plus rapide à utiliser lorsque tout ce que vous avez à faire consiste à examiner, approuver et confirmer vos ordres planifiés.

## <a name="turn-on-this-feature-for-your-system"></a>Activez cette fonctionnalité pour votre système

Si votre système n’inclut pas déjà les fonctionnalités décrites dans cet article, accédez à [Gestion des fonctionnalités](../../../fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview.md) et activez la fonctionnalité *Ordres planifiés simplifiés*.

## <a name="view-manage-and-firm-planned-orders"></a>Afficher, gérer et confirmer les ordres prévisionnels

La page **Ordres planifiés simplifiés** est conçue principalement pour confirmer la fonctionnalité. Elle affiche tous les ordres planifiés qui ont été générés pour le plan sélectionné, tout comme la page **Ordres planifiés** standard le fait. Par conséquent, elle peut être utilisée comme une alternative optimisée pour visualiser les résultats d’une exécution de planification générale.

Pour ouvrir la page **Ordres planifiés simplifiés**, accédez à **Planification générale \> Planification générale \> Ordres planifiés simplifiés**. Vous pouvez utiliser cette page pour effectuer les tâches suivantes :

- Affichez tous les ordres planifiés pour un plan spécifique.
- Créez de nouveaux ordres planifiés.
- Supprimez un ou plusieurs ordres planifiés.
- Confirmez un ou plusieurs ordres planifiés.
- Modifiez le statut de l’ordre planifié pour un ou plusieurs ordres planifiés.
- Ouvrez en toute transparence les détails d’un ordre planifié individuel pour afficher tous les détails de l’ordre.
- Créez des vues personnalisées à l’aide de vues enregistrées.

Des scénarios supplémentaires, tels que le fractionnement, sont pris en charge par la page **Ordres planifiés** standard.
