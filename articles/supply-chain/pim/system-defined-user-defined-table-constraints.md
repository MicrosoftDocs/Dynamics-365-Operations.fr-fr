## <a name="system-defined-table-constraints"></a><span data-ttu-id="6f2f5-137">Contraintes de table définies par le système</span><span class="sxs-lookup"><span data-stu-id="6f2f5-137">System-defined table constraints</span></span>
<span data-ttu-id="6f2f5-138">Une contrainte de table définie par le système crée une mise en correspondance dynamique entre un type d'attribut et un champ dans une table.</span><span class="sxs-lookup"><span data-stu-id="6f2f5-138">A system-defined table constraint creates a dynamic mapping between an attribute type and a field in a table.</span></span> <span data-ttu-id="6f2f5-139">Lorsqu'une contrainte de table définie par le système est incluse dans un modèle de configuration de produit, la mise en correspondance garantit que les données de la table s'affichent à la place des valeurs du type d'attribut.</span><span class="sxs-lookup"><span data-stu-id="6f2f5-139">When a system-defined table constraint is included in a product configuration model, the mapping guarantees that the data in the table is shown instead of the values of the attribute type.</span></span> <span data-ttu-id="6f2f5-140">Le résultat crée une contrainte dynamique, car le contenu de la table peut être modifié, par exemple, par d'autres modules.</span><span class="sxs-lookup"><span data-stu-id="6f2f5-140">The result is a dynamic constraint, because the table contents can be modified (for example, by other modules).</span></span>  

<span data-ttu-id="6f2f5-141">Lorsque vous créez une contrainte de table définie par le système, sélectionnez une table, définissez éventuellement la requête à utiliser, puis associez les types d'attributs aux champs de la table sélectionnée.</span><span class="sxs-lookup"><span data-stu-id="6f2f5-141">When you create a system-defined table constraint, you select a table, optionally define the query to use, and then associate attribute types to the fields in the selected table.</span></span> <span data-ttu-id="6f2f5-142">Les types de champs doivent faire correspondre aux types des types d'attributs.</span><span class="sxs-lookup"><span data-stu-id="6f2f5-142">The types of fields must match the types of attribute types.</span></span>
