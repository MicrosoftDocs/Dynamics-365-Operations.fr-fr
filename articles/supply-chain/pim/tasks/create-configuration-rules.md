# <a name="create-configuration-rules"></a>Créer des règles de configuration

[!include [banner](../../includes/banner.md)]

Cette procédure crée les règles de configuration qui peuvent être utilisées pour la configuration basée sur les dimensions pour forcer ou empêcher certaines combinaisons des lignes de nomenclature. Les données fictives utilisées pour créer cette procédure correspondent à la société USMF. Il s’agit de la septième procédure (parmi les huit) qui explique comment créer des combinaisons pour la configuration basée sur les dimensions.

1. Accédez à Gestion d’informations sur les produits > Nomenclatures et formules > Nomenclature.
2. Dans la liste, recherchez et sélectionnez l’enregistrement souhaité.
    * Trouvez et sélectionnez la nomenclature pour la configuration basée sur les dimensions.  
3. Dans le volet Actions, cliquez sur Options.
4. Cliquez sur Changer de vue.
5. Cliquez sur Vue de l’en-tête.
    * Ouvrez la vue en-tête pour accéder à l’organisateur Gamme de configuration.  
6. Développez ou réduisez la section Gamme de configuration.
    * L’organisateur Gamme de configuration doit être en mode développé.  
7. Cliquez sur Règles de configuration.
8. Cliquez sur Nouveau.
9. Dans la liste, marquez la ligne sélectionnée.
10. Dans le champ Numéro d’article, cliquez sur le bouton de liste déroulante pour ouvrir la recherche.
    * Les articles présents dans le groupe de configurations actuel sont affichés. Sélectionnez celui qui représente la condition dans la règle.  
11. Dans la liste, cliquer sur le lien dans la ligne sélectionnée.
12. Sélectionnez une option dans le champ Méthode.
    * Il est possible d’appliquer une sélection ou une « désélection » d’un article d’un autre groupe de configurations.  
13. Dans le champ Groupe dérivé, cliquez sur le bouton de liste déroulante pour ouvrir la recherche.
14. Dans la liste, recherchez et sélectionnez l’enregistrement souhaité.
15. Dans la liste, cliquer sur le lien dans la ligne sélectionnée.
    * Sélectionnez le groupe de configuration souhaité.  
16. Dans le champ Numéro d’article dérivé, cliquez sur le bouton de liste déroulante pour ouvrir la recherche.
17. Dans la liste, cliquer sur le lien dans la ligne sélectionnée.
    * Sélectionnez le numéro d’article à sélectionner ou à désactiver en fonction de la méthode choisie.  
18. Fermez la page.
