<span data-ttu-id="addc8-104">Cette procédure décrit la configuration de produits liés.</span><span class="sxs-lookup"><span data-stu-id="addc8-104">This procedure walks through configuring linked products.</span></span> <span data-ttu-id="addc8-105">Cette procédure utilise les données de démonstration USRT et suppose que les deux produits ont déjà été créés et lancés.</span><span class="sxs-lookup"><span data-stu-id="addc8-105">This procedure uses the USRT demo data, and assumes both products have already been created and released.</span></span>

4. <span data-ttu-id="addc8-109">Cliquez sur Produits liés.</span><span class="sxs-lookup"><span data-stu-id="addc8-109">Click Linked products.</span></span>
