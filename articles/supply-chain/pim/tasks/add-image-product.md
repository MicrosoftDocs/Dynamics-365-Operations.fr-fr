# <a name="add-an-image-to-a-product"></a>Ajouter une image à un produit

[!include [banner](../../includes/banner.md)]

Cette procédure indique comment ajouter une image à un produit. La société fictive utilisée pour créer cette tâche est USMF.

1. Allez à Gestion des informations sur les produits > Produits > Produits lancés.
2. Augmentez le récapitulatif de l'image du produit.
3. Dans la liste, recherchez et sélectionnez l'enregistrement souhaité.
    * Par exemple, sélectionnez le numéro d'article C0002.  
4. Cliquez sur Modifier l'image.
5. Cliquez sur Nouveau.
    * Cliquez sur Parcourir. Recherchez l'image que vous souhaitez utiliser, sélectionnez-la, puis fermez la boîte de dialogue.
