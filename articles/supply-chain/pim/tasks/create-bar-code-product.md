# <a name="create-a-bar-code-for-a-product"></a><span data-ttu-id="4cf19-103">Créer un code-barres pour un produit</span><span class="sxs-lookup"><span data-stu-id="4cf19-103">Create a bar code for a product</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="4cf19-104">Cette rubrique indique comment créer manuellement un code-barres en utilisant le numéro d’article M0001 comme exemple.</span><span class="sxs-lookup"><span data-stu-id="4cf19-104">This topic shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="4cf19-105">Les données fictives utilisées pour créer cette procédure correspondent à la société USMF.</span><span class="sxs-lookup"><span data-stu-id="4cf19-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="4cf19-106">Sélectionnez **Gestion des produits lancés** sur la page d’accueil.</span><span class="sxs-lookup"><span data-stu-id="4cf19-106">Select **Released product maintenance** on the homepage.</span></span>
2. <span data-ttu-id="4cf19-107">Accédez à **Produits > Produits lancés** dans la section **Liens**.</span><span class="sxs-lookup"><span data-stu-id="4cf19-107">Go to **Products > Released products** under the **Links** section.</span></span>
3. <span data-ttu-id="4cf19-108">Dans la liste, recherchez et sélectionnez l’enregistrement souhaité.</span><span class="sxs-lookup"><span data-stu-id="4cf19-108">In the list, find and select the desired record.</span></span> <span data-ttu-id="4cf19-109">Pour cet exemple, sélectionnez le numéro d’article **M0001**.</span><span class="sxs-lookup"><span data-stu-id="4cf19-109">For this example, select item number **M0001**.</span></span>
4. <span data-ttu-id="4cf19-110">Dans le volet Actions, sélectionnez **Gérer le stock**.</span><span class="sxs-lookup"><span data-stu-id="4cf19-110">On the Action Pane, select **Manage inventory**.</span></span>
5. <span data-ttu-id="4cf19-111">Sélectionnez le **code-barres**.</span><span class="sxs-lookup"><span data-stu-id="4cf19-111">Select **Bar codes**.</span></span>
6. <span data-ttu-id="4cf19-112">Sélectionnez **Nouveau**.</span><span class="sxs-lookup"><span data-stu-id="4cf19-112">Select **New**.</span></span>
7. <span data-ttu-id="4cf19-113">Marquez la ligne sélectionnée qui est créée dans la liste ci-dessous.</span><span class="sxs-lookup"><span data-stu-id="4cf19-113">Mark the selected row that is created in the list below.</span></span>
8. <span data-ttu-id="4cf19-114">Dans le champ **Paramétrage des codes-barres**, saisissez ou sélectionnez une valeur.</span><span class="sxs-lookup"><span data-stu-id="4cf19-114">In the **Barcode setup** field, enter or select a value.</span></span>
9. <span data-ttu-id="4cf19-115">Dans le champ **Code-barres**, entrez ou sélectionnez une valeur.</span><span class="sxs-lookup"><span data-stu-id="4cf19-115">In the **Bar code** field, enter or select a value.</span></span>
10. <span data-ttu-id="4cf19-116">Dans le champ **Code-barres**, tapez une valeur.</span><span class="sxs-lookup"><span data-stu-id="4cf19-116">In the **Bar code** field, type a value.</span></span>  
11. <span data-ttu-id="4cf19-117">Entrez un nombre dans le champ **Quantité**.</span><span class="sxs-lookup"><span data-stu-id="4cf19-117">In the **Quantity** field, enter a number.</span></span>
12. <span data-ttu-id="4cf19-118">Sélectionnez **Enregistrer**.</span><span class="sxs-lookup"><span data-stu-id="4cf19-118">Select **Save**.</span></span>
13. <span data-ttu-id="4cf19-119">Fermez la page.</span><span class="sxs-lookup"><span data-stu-id="4cf19-119">Close the page.</span></span>
