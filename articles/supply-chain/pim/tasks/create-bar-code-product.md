[!include [banner](../../includes/banner.md)]

<span data-ttu-id="1c9bc-104">Cette rubrique indique comment créer manuellement un code-barres en utilisant le numéro d'article M0001 comme exemple.</span><span class="sxs-lookup"><span data-stu-id="1c9bc-104">This topic shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="1c9bc-105">Les données fictives utilisées pour créer cette procédure correspondent à la société USMF.</span><span class="sxs-lookup"><span data-stu-id="1c9bc-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="1c9bc-106">Sélectionnez **Gestion des produits lancés** sur la page d'accueil.</span><span class="sxs-lookup"><span data-stu-id="1c9bc-106">Select **Released product maintenance** on the homepage.</span></span>
2. <span data-ttu-id="1c9bc-107">Accédez à **Produits > Produits lancés** dans la section **Liens**.</span><span class="sxs-lookup"><span data-stu-id="1c9bc-107">Go to **Products > Released products** under the **Links** section.</span></span>

5. <span data-ttu-id="1c9bc-111">Sélectionnez le **code-barres**.</span><span class="sxs-lookup"><span data-stu-id="1c9bc-111">Select **Bar codes**.</span></span>

7. <span data-ttu-id="1c9bc-113">Marquez la ligne sélectionnée qui est créée dans la liste ci-dessous.</span><span class="sxs-lookup"><span data-stu-id="1c9bc-113">Mark the selected row that is created in the list below.</span></span>

9. <span data-ttu-id="1c9bc-115">Dans le champ **Code-barres**, entrez ou sélectionnez une valeur.</span><span class="sxs-lookup"><span data-stu-id="1c9bc-115">In the **Bar code** field, enter or select a value.</span></span>
10. <span data-ttu-id="1c9bc-116">Dans le champ **Code-barres**, tapez une valeur.</span><span class="sxs-lookup"><span data-stu-id="1c9bc-116">In the **Bar code** field, type a value.</span></span>  
11. <span data-ttu-id="1c9bc-117">Entrez un nombre dans le champ **Quantité**.</span><span class="sxs-lookup"><span data-stu-id="1c9bc-117">In the **Quantity** field, enter a number.</span></span>
12. <span data-ttu-id="1c9bc-118">Sélectionnez **Enregistrer**.</span><span class="sxs-lookup"><span data-stu-id="1c9bc-118">Select **Save**.</span></span>
13. <span data-ttu-id="1c9bc-119">Fermez la page.</span><span class="sxs-lookup"><span data-stu-id="1c9bc-119">Close the page.</span></span>
