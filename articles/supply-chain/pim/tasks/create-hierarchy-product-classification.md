# <a name="create-a-hierarchy-of-product-classification"></a>Créer une hiérarchie de classification de produit

[!include [banner](../../includes/banner.md)]

Cette procédure indique comment créer une nouvelle hiérarchie de catégories et affecter un type de hiérarchie des nomenclatures douanières. Les données fictives utilisées pour créer cette procédure correspondent à la société USMF. Cette procédure est destinée au gestionnaire de catégories.


## <a name="create-the-new-category-hierarchy"></a>Créer la nouvelle hiérarchie de catégories
1. Accédez à **Volet de navigation > Modules > Gestion des informations sur les produits > Configuration > Catégories et attributs > Hiérarchies de catégories**.
2. Cliquez sur **Nouveau**.
3. Tapez une valeur dans le champ **Nom**.
4. Tapez une valeur dans le champ **Description**.
5. Cliquez sur **Créer**.

## <a name="build-the-hierarchy"></a>Créer la hiérarchie
1. Cliquez sur **Nouveau** nœud de catégorie.
2. Tapez une valeur dans le champ **Nom**.
3. Dans le champ **Code**, tapez une valeur.
4. Dans le champ **Nom convivial**, saisissez une valeur.
5. Cliquez sur **Nouveau** nœud de catégorie.
6. Tapez une valeur dans le champ **Nom**.
7. Dans le champ **Code**, tapez une valeur.
8. Dans le champ **Nom convivial**, saisissez une valeur.
9. Cliquez sur **Nouveau** nœud de catégorie.
10. Tapez une valeur dans le champ **Nom**.
11. Dans le champ **Code**, tapez une valeur.
12. Dans le champ **Nom convivial**, saisissez une valeur.
13. Cliquez sur **Nouveau** nœud de catégorie.
14. Tapez une valeur dans le champ **Nom**.
15. Dans le champ **Code**, tapez une valeur.
16. Dans le champ **Nom convivial**, saisissez une valeur.
17. Fermez la page.

## <a name="classify-the-hierarchy"></a>Classifier la hiérarchie
1. Dans la liste, recherchez et sélectionnez l'enregistrement souhaité.
2. Dans le **volet Actions**, cliquez sur **Hiérarchie de catégories**.
3. Cliquez sur **Associer le type de hiérarchie**.
4. Cliquez sur **Nouveau**.
5. Dans le champ **Type de hiérarchie de catégories**, sélectionnez une option. Sélectionnez le **type de hiérarchie de catégories des nomenclatures douanières pour la classification de produit**.  
6. Dans le champ **Hiérarchie de catégories**, cliquez sur le bouton de liste déroulante pour ouvrir la recherche.
7. Dans la liste, recherchez et sélectionnez l'enregistrement souhaité.
8. Dans la liste, cliquer sur le lien dans la ligne sélectionnée.
9. Fermez la page.
