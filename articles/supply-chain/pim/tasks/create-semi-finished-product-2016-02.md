# <a name="create-a-semi-finished-product-february-2016"></a>Créer un produit semi-fini (février 2016)

[!include [task guide banner](../../includes/task-guide-banner.md)]

Cette tâche consiste à créer un produit semi-fini. Il s'agit de la deuxième tâche de la série de calculs des nomenclatures. Les données fictives utilisées pour créer cette tâche correspondent à la société USMF.

1. Allez à Gestion des informations sur les produits > Produits > Produits lancés.
2. Cliquez sur Nouveau.
3. Dans le champ Numéro du produit, saisissez une valeur.
    * Pour cet exemple, tapez BOM_2.  
4. Saisissez ou sélectionnez une valeur dans le champ Groupe de modèles d'article.
    * Sélectionnez STD. STD représente le coût standard et est le modèle le plus fréquemment utilisé lors de l'utilisation des calculs de coûts.  
5. Dans le champ Groupe d'articles, entrez ou sélectionnez une valeur.
    * Par exemple, sélectionnez Audio. Cela n'a aucun impact sur les calculs de coûts.  
6. Saisissez ou sélectionnez une valeur dans le champ Groupe de dimension de stockage.
    * Sélectionnez SiteWH. Seul Site et entrepôt sera utilisé pour cet exemple.  
7. Saisissez ou sélectionnez une valeur dans le champ Groupe de dimension de suivi.
    * Les dimensions de suivi ne seront pas utilisées pour cet exemple, sélectionnez Aucun.  
8. Cliquez sur OK.
9. Dans le volet Actions, cliquez sur Gérer le stock.
10. Cliquez sur Paramètres de commande par défaut.
11. Dans le champ Type de commande par défaut, sélectionnez Production.
    * Comme c'est un produit semi-fini qui sera produit, sélectionnez Production.  
12. Dans le champ Site d'achat, saisissez ou sélectionnez une valeur.
    * Pour cet exemple, sélectionnez Site 1.  
13. Dans le champ Site de stock, saisissez ou sélectionnez une valeur.
    * Pour cet exemple, sélectionnez Site 1.  
14. Dans le champ Site de vente, entrez ou sélectionnez une valeur.
    * Pour cet exemple, sélectionnez Site 1.  
15. Fermez la page.
16. Cliquez sur Gérer les coûts dans le volet Actions.
17. Cliquez sur Prix de l'article.
18. Cliquez sur Nouveau.
19. Dans la liste, marquez la ligne sélectionnée.
20. Dans le champ Version, saisissez ou sélectionnez une valeur.
    * Pour cet exemple, sélectionnez Version 10.  
21. Saisissez ou sélectionnez une valeur dans le champ Site.
    * Pour cet exemple, sélectionnez Site 1.  
22. Définissez le prix sur 10
    * Pour cet exemple, entrez un prix de revient de 10.  
23. Cliquez sur Enregistrer.
24. Cliquez sur Activer le ou les prix en attente.
25. Fermez la page.
26. Fermez la page.
27. Cliquez sur BOM_2 pour l'ouvrir.
28. Développez la section Gérer les coûts.
29. Dans le champ Groupe de coûts, entrez ou sélectionnez une valeur.
    * Pour cet exemple, sélectionnez Groupe de coûts M1.  
30. Utilisez le raccourci pour enregistrer un enregistrement.
31. Fermez la page.
