<span data-ttu-id="99a62-123">Vous pouvez paramétrer une alerte lorsque vous créez un devis.</span><span class="sxs-lookup"><span data-stu-id="99a62-123">When you create a quotation, you can set up an alert.</span></span> <span data-ttu-id="99a62-124">Voici certaines façons d'utiliser les alertes :</span><span class="sxs-lookup"><span data-stu-id="99a62-124">Here are some of the ways that alerts are used:</span></span>

## <a name="price-simulation-and-discounts"></a><span data-ttu-id="99a62-127">Simulation et remises de prix</span><span class="sxs-lookup"><span data-stu-id="99a62-127">Price simulation and discounts</span></span>

### <a name="types-of-discounts-in-trade-agreements"></a><span data-ttu-id="99a62-130">Types de remises dans le cadre d'accords commerciaux</span><span class="sxs-lookup"><span data-stu-id="99a62-130">Types of discounts in trade agreements</span></span>

<span data-ttu-id="99a62-131">Dans Microsoft Supply Chain Management, les accords commerciaux peuvent comprendre quatre types de remises.</span><span class="sxs-lookup"><span data-stu-id="99a62-131">Trade agreements in Supply Chain Management can have four types of price discounts.</span></span> <span data-ttu-id="99a62-132">Vous pouvez paramétrer ces remises pour différents articles, clients ou groupes de prix, et les limiter par date.</span><span class="sxs-lookup"><span data-stu-id="99a62-132">These discounts can be set up for different items, customers, or price groups, and they can be limited by date.</span></span> <span data-ttu-id="99a62-133">Afin d'éviter toute erreur de calcul, les accords commerciaux doivent être pris en compte lors des simulations de prix.</span><span class="sxs-lookup"><span data-stu-id="99a62-133">To avoid miscalculations, you must consider trade agreements when you run price simulations.</span></span> <span data-ttu-id="99a62-134">Voici les quatre types de remises inclus dans les accords commerciaux :</span><span class="sxs-lookup"><span data-stu-id="99a62-134">Here are the four types of discounts in trade agreements:</span></span>

-   <span data-ttu-id="99a62-135">**Prix de vente** : un prix de vente distinct peut être spécifié pour chaque article.</span><span class="sxs-lookup"><span data-stu-id="99a62-135">**Sales price** – Separate sales prices can be specified for items.</span></span> <span data-ttu-id="99a62-136">Lors de la création de lignes de devis, le programme recherche le prix de vente approprié d'un article et le transfère vers les lignes de devis.</span><span class="sxs-lookup"><span data-stu-id="99a62-136">When quotation lines are created, the program searches for the correct sales price for an item and transfers it to the quotation lines.</span></span> <span data-ttu-id="99a62-137">Par conséquent, un accord commercial comportant ce type de remise n'a aucune incidence sur la simulation de prix.</span><span class="sxs-lookup"><span data-stu-id="99a62-137">Therefore, a trade agreement that has this kind of discount doesn't affect the price simulation.</span></span> <span data-ttu-id="99a62-138">Le prix de vente utilisé dans la ligne de devis reflète l'accord commercial.</span><span class="sxs-lookup"><span data-stu-id="99a62-138">The sales price that is used in the quotation line reflects the trade agreement.</span></span>
