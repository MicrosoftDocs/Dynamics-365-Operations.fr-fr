# <a name="intercompany-reservations"></a>Réservations intersociétés

[!include [banner](../../includes/banner.md)]

Lorsque vous créez directement une ligne de commande fournisseur intersociétés à partir d'une ligne de commande client originale, les deux lignes sont marquées. Ce marquage contrôle la réservation, le coût et la chaîne intersociétés. Vous pouvez afficher ce marquage sur la ligne de commande client en sélectionnant **Stock** \> **Marquage**, puis en fermant la boîte de dialogue qui est affichée.

Pour supprimer le marquage, sélectionnez **Stock** \> **Supprimer le lien** sur la ligne de commande d'origine.
