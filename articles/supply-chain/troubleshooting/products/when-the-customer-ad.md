# <a name="purchase-orders-dont-show-translated-product-text"></a>Les commandes fournisseur n'affichent pas le texte du produit traduit

Numéro de la base de connaissances : 4612098

## <a name="symptoms"></a>Symptômes

Vous avez défini des traductions de texte pour les noms de produits. Cependant, les noms de produits sont toujours affichés dans la langue d'origine, quel que soit le paramètre de langue dans l'application.

## <a name="resolution"></a>Résolution

Les traductions de produits ne sont affichées que sur les documents externes. Ils ne sont pas montrés aux utilisateurs internes. Pour plus d’informations, voir [FAQ sur les traductions relatives au produit](../../pim/translations-product-related-information.md#where-can-i-view-the-translated-information).
