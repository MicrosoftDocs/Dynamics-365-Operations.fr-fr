# <a name="cant-end-a-production-order-because-of-a-bom-consumptioncost-value-error"></a>Impossible de mettre fin à un ordre de fabrication en raison d'une erreur de la valeur BOM consumptionCost

## <a name="symptoms"></a>Symptômes

Lorsque vous essayez de mettre fin à un ordre de fabrication, vous recevez le message d’erreur suivant :

> Le calcul de la valeur BOM consumptionCost doit être négatif lors de la sortie de l'inventaire.

## <a name="resolution"></a>Résolution

Ce problème a été résolu dans la version 10.0.15.
