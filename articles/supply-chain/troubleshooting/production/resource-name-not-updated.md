# <a name="resource-name-isnt-updated-after-changing-a-worker-name-in-the-address-book"></a>Impossible de mettre à jour le nom de ressource après le changement de nom d’un collaborateur dans le carnet d’adresses

## <a name="symptoms"></a>Symptômes

Si vous modifiez le nom d’un collaborateur dans le carnet d’adresses global, le nom de la ressource correspondante n’est pas mis à jour dans le générique du groupe de ressources.

## <a name="resolution"></a>Résolution

Le nom de ressource correspondant n’est actuellement pas mis à jour automatiquement. Pour résoudre le problème, vous devez mettre à jour manuellement le nom de la ressource.
