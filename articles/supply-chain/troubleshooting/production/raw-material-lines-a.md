# <a name="users-can-unpick-material-lines-for-products-that-have-been-reported-as-finished"></a>Les utilisateurs peuvent décoller les lignes de matériaux pour les produits qui ont été déclarés comme finis

Numéro de la base de connaissances : 4614721

## <a name="symptoms"></a>Symptômes

Le système permet aux utilisateurs de décoller les lignes de matières premières pour les produits qui ont été déclarés comme finis.

## <a name="resolution"></a>Résolution

Le système se comporte comme prévu. La consommation de matières premières peut être ajustée jusqu'à ce que l'ordre de fabrication atteigne le statut *Terminé*.
