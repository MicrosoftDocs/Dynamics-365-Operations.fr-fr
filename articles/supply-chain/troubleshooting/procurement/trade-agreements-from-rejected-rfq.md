# <a name="trade-agreements-can-be-created-from-rejected-rfqs"></a>Des accords commerciaux peuvent être créés à partir d’appels d’offres rejetés

## <a name="symptoms"></a>Symptômes

Des accords commerciaux peuvent être créés à partir d’appels d’offres rejetés. Par conséquent, le système n’empêche pas la création de journaux d’accords commerciaux si la ligne d’appel d’offres n’a pas été acceptée.

## <a name="resolution"></a>Résolution

Ce comportement est attendu. Vous pouvez créer des accords commerciaux pour toutes les réponses à un appel d’offres, qu’elles aient été acceptées ou rejetées. Pour plus d’informations, voir [Vue d’ensemble des appels d’offre](/dynamics365/supply-chain/procurement/request-quotations).
