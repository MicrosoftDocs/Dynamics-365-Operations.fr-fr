# <a name="you-cant-add-the-price-unit-field-to-the-purchase-agreement-page"></a>Vous ne pouvez pas ajouter le champ Unité de prix à la page Contrat d'achat

## <a name="symptoms"></a>Symptômes

Lorsque vous ouvrez la page **Contrat d’achat** en mode d’affichage de ligne, vous ne pouvez pas personnaliser la page en ajoutant le champ **Unité de prix** dans l’aperçu de la ligne.

## <a name="resolution"></a>Résolution

Certains champs partagés dans le cadre de l’accord ne peuvent pas être inclus dans les personnalisations. Cette limitation se produit en raison du modèle de données implémenté. Par conséquent, vous ne pouvez pas personnaliser le champ **Unité de prix**.
