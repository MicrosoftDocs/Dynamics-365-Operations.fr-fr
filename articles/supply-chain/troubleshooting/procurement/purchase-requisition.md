# <a name="you-cant-add-a-line-to-a-purchase-requisition-after-you-request-a-change"></a>Vous ne pouvez pas ajouter de ligne à une demande d'achat après avoir demandé une modification

Numéro de la base de connaissances : 4611211

## <a name="symptoms"></a>Symptômes

Le système ne vous permet pas d'ajouter une ligne à une demande d'achat après avoir demandé une modification.

## <a name="resolution"></a>Résolution

Vous devez rappeler le flux de travail. Une fois que la demande d'achat a le statut *Brouillon*, vous pouvez continuer à le modifier ou y ajouter une ligne.
