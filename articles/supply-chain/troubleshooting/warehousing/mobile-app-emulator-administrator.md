# <a name="can-only-use-administrator-role-to-access-the-mobile-device-app-emulator"></a>Rôle Administrateur seul utilisable pour accéder à l’émulateur d’application pour appareil mobile

## <a name="symptoms"></a>Symptômes

Lors de la configuration d'entrepôts dans Supply Chain Management, vous ne pouvez utiliser aucun rôle à l'exception du rôle Administrateur pour accéder à l'émulateur d'application pour appareil mobile.

## <a name="resolution"></a>Résolution

L’émulateur d’application pour appareil mobile est configuré pour fonctionner uniquement avec le compte administrateur. À toutes les fins de test et de processus en direct, nous vous recommandons d’utiliser l’application mobile Warehouse Management elle-même.
