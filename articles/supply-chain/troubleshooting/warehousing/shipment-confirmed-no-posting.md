# <a name="shipment-for-load-has-been-confirmed-but-no-lines-are-posted"></a>L’expédition pour chargement a été confirmée, mais aucune ligne n’est affichée

## <a name="symptoms"></a>Symptômes

Lorsque vous utilisez des opérations d’entrepôt sortantes, le message suivant peut s’afficher :

> L’expédition a été confirmée pour le chargement 1 %.

Cependant, aucune autre publication n’a eu lieu.

## <a name="resolution"></a>Résolution

Ce comportement est fait exprès. La confirmation d’expédition n’affecte pas la publication. Elle met simplement à jour le statut de l’expédition et du chargement. La publication doit avoir lieu dans un processus distinct.
