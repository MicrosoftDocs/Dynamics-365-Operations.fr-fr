# <a name="cant-move-inventory-to-a-location-that-is-license-plate-controlled"></a>Impossible de déplacer le stock vers un emplacement contrôlé par le contenant

## <a name="symptoms"></a>Symptômes

Vous ne pouvez pas réduire les quantités prélevées sur un chargement.

## <a name="resolution"></a>Résolution

Dans les versions précédentes, vous ne pouviez pas réduire les quantités prélevées sur un chargement. Cependant, vous pouvez désormais annuler le prélèvement à un emplacement contrôlé par le contenant. Vous devez spécifier à la fois une valeur **Emplacement** et une valeur **Contenant** de la ligne de charge sur la page **Réduire la quantité prélevée**.
