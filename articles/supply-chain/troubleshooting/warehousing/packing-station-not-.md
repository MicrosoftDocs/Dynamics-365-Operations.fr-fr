# <a name="packing-station-doesnt-show-product-notes"></a>La station d'emballage n'affiche pas les notes de produit

Numéro de la base de connaissances : 4614615

## <a name="symptoms"></a>Symptômes

Les notes d'emballage ne sont pas affichées dans le formulaire d'emballage lorsque les instructions d'emballage sont ajoutées en tant que pièce jointe à une fiche produit ou à une variante de produit.

## <a name="resolution"></a>Résolution

Le système se comporte comme prévu.

La logique actuelle d'affichage des notes d'emballage dans le formulaire d'emballage nécessite que les notes soient associées aux expéditions.
