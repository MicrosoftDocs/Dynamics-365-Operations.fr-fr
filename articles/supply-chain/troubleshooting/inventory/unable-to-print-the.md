# <a name="picking-list-report-doesnt-show-new-notes-that-are-attached-to-a-journal"></a>Le rapport de la liste de sélection n'affiche pas les nouvelles notes jointes à un journal

Numéro de la base de connaissances : 4611660

## <a name="symptoms"></a>Symptômes

Le rapport de prélèvement n'affiche pas les nouvelles notes jointes à un journal de prélèvement.

## <a name="resolution"></a>Résolution

Le rapport de la liste de prélèvement affiche uniquement les notes jointes à la ligne de commande client et à l'en-tête de la commande client. Il n'affiche aucune nouvelle note ajoutée dans le journal des factures.
