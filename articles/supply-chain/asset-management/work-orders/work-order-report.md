# <a name="work-order-report"></a>Rapport de l'ordre de travail


[!include [banner](../../includes/banner.md)]

[!include [banner](../../includes/preview-banner.md)]


Vous pouvez déclarer un état de l'ordre de travail qui présente des informations détaillées relatives à un ordre de travail. Vous pouvez sélectionner un ou plusieurs ordres de travail à afficher dans l'état.

1. Cliquez sur **Gestion des actifs** > **États** > **Ordres de travail** > **État de l'ordre de travail**.

2. Sélectionnez « Oui »/« Non » sur les boutons à bascule et complétez les champs comme requis afin de déterminer les détails à inclure dans l'état.  
  a. Dans la section **Paramètres d'impression**, vous pouvez choisir d'inclure les documents joints depuis le paramétrage du type de tâche de maintenance dans l'impression.

3. Sur l'organisateur **Enregistrements à inclure**, vous pouvez filtrer le contenu de l'état par **Ordre de travail**.

4. Si nécessaire, vous pouvez paramétrer la génération de l'état de l'ordre de travail comme traitement par lots en remplissant les champs sous l'organisateur **Exécuter à l'arrière-plan**.

5. Cliquez sur **OK** pour générer l'état.
