# <a name="work-order-report"></a><span data-ttu-id="38105-103">Rapport de l'ordre de travail</span><span class="sxs-lookup"><span data-stu-id="38105-103">Work order report</span></span>

[!include [banner](../../includes/banner.md)]


<span data-ttu-id="38105-104">Vous pouvez déclarer un état de l'ordre de travail qui présente des informations détaillées relatives à un ou plusieurs ordres de travail que vous sélectionnez.</span><span class="sxs-lookup"><span data-stu-id="38105-104">You can generate a work order report that shows detailed information about one or more work orders that you select.</span></span>

1. <span data-ttu-id="38105-105">Sélectionnez **Gestion d'actifs** > **États** > **Ordres de travail** > **État de l'ordre de travail**.</span><span class="sxs-lookup"><span data-stu-id="38105-105">Select **Asset management** > **Reports** > **Work orders** > **Work order report**.</span></span>

2. <span data-ttu-id="38105-106">Sélectionnez « Oui »/« Non » sur les boutons à bascule au besoin, et complétez les champs comme requis afin de déterminer les détails à inclure dans l'état.</span><span class="sxs-lookup"><span data-stu-id="38105-106">Select "Yes"/"No" on the toggle buttons as you require, to determine the details to be included in the report.</span></span>  
    - <span data-ttu-id="38105-107">Dans la section **Paramètres d'impression**, vous pouvez choisir d'inclure les documents joints depuis le paramétrage du type de tâche de maintenance.</span><span class="sxs-lookup"><span data-stu-id="38105-107">In the **Print settings** section, you can select whether to include attachments from the setup of the related maintenance job type.</span></span>

3. <span data-ttu-id="38105-108">Sur l'organisateur **Enregistrements à inclure**, vous pouvez filtrer le contenu de l'état par **Ordre de travail**.</span><span class="sxs-lookup"><span data-stu-id="38105-108">On the **Records to include** FastTab, you can filter the contents of the report by **Work order**.</span></span>

4. <span data-ttu-id="38105-109">Dans l'organisateur **Exécuter en arrière-plan**, vous pouvez paramétrer la génération de l'état de l'ordre de travail comme un traitement par lots, comme vous le souhaitez.</span><span class="sxs-lookup"><span data-stu-id="38105-109">On the **Run in the background** FastTab, you can set up generation of the work order report as a batch job, as you require.</span></span>

5. <span data-ttu-id="38105-110">Sélectionnez **OK** pour générer l'état.</span><span class="sxs-lookup"><span data-stu-id="38105-110">Select **OK** to generate the report.</span></span>
