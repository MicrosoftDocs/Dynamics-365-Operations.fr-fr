## <a name="create-workers"></a><span data-ttu-id="dd531-110">Créer des collaborateurs</span><span class="sxs-lookup"><span data-stu-id="dd531-110">Create workers</span></span>

1. <span data-ttu-id="dd531-111">Sélectionnez **Gestion des actifs** \> **Paramétrage** \> **Collaborateurs** \> **Collaborateurs**.</span><span class="sxs-lookup"><span data-stu-id="dd531-111">Select **Asset management** \> **Setup** \> **Workers** \> **Workers**.</span></span>
2. <span data-ttu-id="dd531-112">Sélectionnez **Nouveau** pour ajouter un collaborateur à la liste.</span><span class="sxs-lookup"><span data-stu-id="dd531-112">Select **New** to add a worker to the list.</span></span>
3. <span data-ttu-id="dd531-113">Sélectionnez le collaborateur dans le champ **Collaborateur**.</span><span class="sxs-lookup"><span data-stu-id="dd531-113">In the **Worker** field, select the worker.</span></span>
4. <span data-ttu-id="dd531-114">Définissez l'option **Actif** sur **Oui** pour planifier le collaborateur sur les ordres de travail.</span><span class="sxs-lookup"><span data-stu-id="dd531-114">Set the **Active** option to **Yes** to schedule the worker on work orders.</span></span>

    <span data-ttu-id="dd531-115">Dans le raccourci **Général**, les champs **Ressource** et **Description** sont automatiquement renseignés si une ressource a été sélectionnée pour le collaborateur.</span><span class="sxs-lookup"><span data-stu-id="dd531-115">On the **General** FastTab, the **Resource** and **Description** fields are automatically filled in if a resource has been selected for the worker.</span></span> <span data-ttu-id="dd531-116">Le champ **Calendrier** est automatiquement renseigné, à condition que vous ayez paramétré le collaborateur comme ressource et affecté un calendrier à cette ressource dans la page **Ressources** (**Administration d'organisation** \> **Ressources** \> **Ressources**).</span><span class="sxs-lookup"><span data-stu-id="dd531-116">The **Calendar** field is also automatically filled in, provided that you've set up the worker as a resource and allocated a calendar to that resource on the **Resources** page (**Organization administration** \> **Resources** \> **Resources**).</span></span>

5. <span data-ttu-id="dd531-117">Dans le raccourci **Groupes**, sélectionnez **Ajouter**, puis sélectionnez un groupe d'agents de maintenance pour le collaborateur.</span><span class="sxs-lookup"><span data-stu-id="dd531-117">On the **Groups** FastTab, select **Add**, and then select a maintenance worker group for the worker.</span></span> <span data-ttu-id="dd531-118">Un collaborateur peut être affilié à plusieurs groupes.</span><span class="sxs-lookup"><span data-stu-id="dd531-118">A worker can be affiliated with more than one group.</span></span>
6. <span data-ttu-id="dd531-119">Dans le paramétrage standard, l'affiliation d'un collaborateur à un groupe est effective à compter de la date d'ajout du groupe, et elle n'expire jamais.</span><span class="sxs-lookup"><span data-stu-id="dd531-119">In the standard setup, a worker's affiliation with a group is effective from the date when you add the group, and it never expires.</span></span> <span data-ttu-id="dd531-120">Cette date est affichée dans le champ **Date d'effet**.</span><span class="sxs-lookup"><span data-stu-id="dd531-120">This date is shown in the **Effective** field.</span></span> <span data-ttu-id="dd531-121">Pour afficher le champ **Date d'effet**, sélectionnez **Afficher** \> **Tous**.</span><span class="sxs-lookup"><span data-stu-id="dd531-121">To see the **Effective** field, select **View** \> **All**.</span></span> <span data-ttu-id="dd531-122">Si l'affiliation du collaborateur à un groupe doit être limitée à une période spécifique, utilisez les champs **Date d'effet** et **Date d'expiration** pour définir la période.</span><span class="sxs-lookup"><span data-stu-id="dd531-122">If the worker's affiliation with a group should be limited to a specific period, use the **Effective** and **Expiration** fields to define the period.</span></span>
7. <span data-ttu-id="dd531-123">Dans le raccourci **Postes techniques**, sélectionnez **Ajouter**, puis sélectionnez un poste technique pour l'agent de maintenance.</span><span class="sxs-lookup"><span data-stu-id="dd531-123">On the **Functional locations** FastTab, select **Add**, and then select a functional location for the maintenance worker.</span></span> <span data-ttu-id="dd531-124">Spécifiez également quel emplacement est le poste technique principal pour le collaborateur.</span><span class="sxs-lookup"><span data-stu-id="dd531-124">Also specify which location is the primary functional location for the worker.</span></span>

    > [!NOTE]
    > <span data-ttu-id="dd531-125">Lorsque vous ajoutez des postes techniques à un collaborateur, tous les actifs actifs associés à ces postes techniques apparaissent sur les différentes options de menu, telles que **Mes actifs actifs** et **Mes postes techniques actifs**.</span><span class="sxs-lookup"><span data-stu-id="dd531-125">When you add functional locations to a worker, all active assets that are related to those functional locations appear on various menu items, such as **My active assets** and **My active functional locations**.</span></span> <span data-ttu-id="dd531-126">Ils apparaissent également dans les recherches d'actif qui s'affichent lorsque vous créez un actif, une demande de maintenance ou un ordre de travail.</span><span class="sxs-lookup"><span data-stu-id="dd531-126">They also appear in the asset lookups that are shown when you create a new asset, maintenance request, or work order.</span></span>

    <span data-ttu-id="dd531-127">Les champs du raccourci **Détails** affichent le nombre de groupes d'agents de maintenance et de postes techniques auxquels est associé l'agent de maintenance sélectionné.</span><span class="sxs-lookup"><span data-stu-id="dd531-127">The fields on the **Details** FastTab show the number of maintenance worker groups and functional locations that the selected maintenance worker is related to.</span></span>

## <a name="create-worker-groups"></a><span data-ttu-id="dd531-128">Créer des groupes d'agents</span><span class="sxs-lookup"><span data-stu-id="dd531-128">Create worker groups</span></span>

1. <span data-ttu-id="dd531-129">Sélectionnez **Gestion des actifs** \> **Paramétrage** \> **Collaborateurs** \> **Groupes d'agents de maintenance**.</span><span class="sxs-lookup"><span data-stu-id="dd531-129">Select **Asset management** \> **Setup** \> **Workers** \> **Maintenance worker groups**.</span></span>
2. <span data-ttu-id="dd531-130">Sélectionnez **Nouveau** pour ajouter un groupe d'agents à la liste.</span><span class="sxs-lookup"><span data-stu-id="dd531-130">Select **New** to add a worker group to the list.</span></span>
3. <span data-ttu-id="dd531-131">Dans le champ **Groupe d'agents de maintenance**, entrez un ID groupe.</span><span class="sxs-lookup"><span data-stu-id="dd531-131">In the **Maintenance worker group** field, enter a group ID.</span></span>
4. <span data-ttu-id="dd531-132">Dans le champ **Nom**, entrez un nom.</span><span class="sxs-lookup"><span data-stu-id="dd531-132">In the **Name** field, enter a name.</span></span>

    <span data-ttu-id="dd531-139">Le champ du raccourci **Détails** affiche le nombre d'agents de maintenance qui ont été paramétrés sur le groupe d'agents de maintenance sélectionné.</span><span class="sxs-lookup"><span data-stu-id="dd531-139">The field on the **Details** FastTab shows the number of maintenance workers that have been set up on the selected maintenance worker group.</span></span>
