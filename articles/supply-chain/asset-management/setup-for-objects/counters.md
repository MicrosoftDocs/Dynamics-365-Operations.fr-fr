# <a name="asset-measures"></a><span data-ttu-id="59b4f-103">Mesures d'actifs</span><span class="sxs-lookup"><span data-stu-id="59b4f-103">Asset measures</span></span>

<span data-ttu-id="59b4f-104">La rubrique explique comment créer des types de mesures d'actifs dans le module Gestion des actifs.</span><span class="sxs-lookup"><span data-stu-id="59b4f-104">The topic explains how to create asset measure types in Asset Management.</span></span> <span data-ttu-id="59b4f-105">Les types de mesures d’actifs permettent d’enregistrer des mesures, par exemple en ce qui concerne le nombre d’heures de production ou la quantité produite sur l’actif.</span><span class="sxs-lookup"><span data-stu-id="59b4f-105">Asset measure types are used to make measurement registrations on assets, for example, regarding number of production hours, or quantity produced on the asset.</span></span> <span data-ttu-id="59b4f-106">Les types d'actifs sont liés aux types de mesures d'actifs.</span><span class="sxs-lookup"><span data-stu-id="59b4f-106">Asset types are related to the asset measure types.</span></span> <span data-ttu-id="59b4f-107">Cela signifie qu'une mesure de l'actif ne peut être utilisée sur un actif que si la mesure de l'actif est configurée sur le type d'actif utilisé sur l'actif.</span><span class="sxs-lookup"><span data-stu-id="59b4f-107">This means that an asset measure can only be used on an asset if the asset measure is set up on the asset type used on the asset.</span></span>

<span data-ttu-id="59b4f-110">Les mesures d'actifs peuvent être utilisées sur les plans de maintenance.</span><span class="sxs-lookup"><span data-stu-id="59b4f-110">Asset measures can be used on maintenance plans.</span></span> <span data-ttu-id="59b4f-111">Une ligne de plan de maintenance peut être de type « compteur », par exemple, en relation avec le nombre d'heures ou la quantité de production produite.</span><span class="sxs-lookup"><span data-stu-id="59b4f-111">A maintenance plan line can be of type "Counter", for example, relating to number of production hours or quantity produced.</span></span> 

<span data-ttu-id="59b4f-112">Un enregistrement de mesure d'actif peut être mis à jour manuellement ou automatiquement en fonction des heures ou de la quantité de production produite.</span><span class="sxs-lookup"><span data-stu-id="59b4f-112">An asset measurement registration can be updated manually or automatically based on production hours or quantity produced.</span></span> <span data-ttu-id="59b4f-113">Une mesure d'actif peut être paramétrée pour utiliser une des trois méthodes de mise à jour (que vous sélectionnez dans le champ **Mettre à jour** dans **Compteurs**) :</span><span class="sxs-lookup"><span data-stu-id="59b4f-113">An asset measure can be set up to use one of three update methods (selected in the **Update** field in **Counters**):</span></span>

- <span data-ttu-id="59b4f-115">Heures de production - le compteur est automatiquement mis à jour en fonction du nombre d'heures de production.</span><span class="sxs-lookup"><span data-stu-id="59b4f-115">Production hours - the counter is automatically updated based on number of production hours.</span></span>  
- <span data-ttu-id="59b4f-116">Quantité de production - le compteur est automatiquement mis à jour en fonction des quantités produites.</span><span class="sxs-lookup"><span data-stu-id="59b4f-116">Production quantity - the counter is automatically updated based on number of quantity produced.</span></span>  

>[!NOTE]
><span data-ttu-id="59b4f-117">Si la quantité produite est utilisée, *tous* les articles enregistrés sont inclus dans l'enregistrement de mesure, bonne quantité, ainsi que quantité d'erreur.</span><span class="sxs-lookup"><span data-stu-id="59b4f-117">If quantity produced is used, *all* registered items are included in the measurement registration, good quantity as well as error quantity.</span></span> <span data-ttu-id="59b4f-118">Il reste possible d'effectuer des enregistrements de mesure d'actifs manuels, le cas échéant.</span><span class="sxs-lookup"><span data-stu-id="59b4f-118">It is always possible to make manual asset measurement registrations, if required.</span></span>

## <a name="create-counter-types-for-asset-counter-registrations"></a><span data-ttu-id="59b4f-119">Créez les contre-types pour les enregistrements de compteurs d'actifs</span><span class="sxs-lookup"><span data-stu-id="59b4f-119">Create counter types for asset counter registrations</span></span>

1. <span data-ttu-id="59b4f-120">Sélectionnez **Gestion des actifs** > **Paramétrage** > **Types d'actifs** > **Compteurs**.</span><span class="sxs-lookup"><span data-stu-id="59b4f-120">Select **Asset management** > **Setup** > **Asset types** > **Counters**.</span></span>
2. <span data-ttu-id="59b4f-121">Sélectionnez **Nouveau** pour créer un type de mesure d'actif.</span><span class="sxs-lookup"><span data-stu-id="59b4f-121">Select **New** to create a new asset measure type.</span></span>
3. <span data-ttu-id="59b4f-122">Insérez un ID dans le champ **Compteur**, et un nom de compteur dans le champ **Nom**.</span><span class="sxs-lookup"><span data-stu-id="59b4f-122">Insert an ID in the **Counter** field, and a counter name in the **Name** field.</span></span>
4. <span data-ttu-id="59b4f-123">Dans l'organisateur **Général**, sélectionnez l'unité de mesure dans le champ **Unité**.</span><span class="sxs-lookup"><span data-stu-id="59b4f-123">On the **General** FastTab, select a measurement unit in the **Unit** field.</span></span>
5. <span data-ttu-id="59b4f-124">Dans le champ **Mettre à jour**, sélectionnez la méthode de mise à jour à utiliser pour la mesure d'actif.</span><span class="sxs-lookup"><span data-stu-id="59b4f-124">In the **Update** field, select the update method to be used for the asset measure.</span></span>
6. <span data-ttu-id="59b4f-125">Sélectionnez « Oui » sur le bouton bascule **Hériter des contre-valeurs** si les actifs enfants d'une structure d'actif doivent hériter automatiquement des enregistrements de mesures d'actifs effectués sur l'actif parent.</span><span class="sxs-lookup"><span data-stu-id="59b4f-125">Select "Yes" on the **Inherit counter values** toggle button if child assets in an asset structure should automatically inherit asset measure registrations made on the parent asset.</span></span>
7. <span data-ttu-id="59b4f-126">Dans le champ **Agrégat total**, sélectionnez la méthode de synthèse à utiliser pour une mesure d'actif à l'aide de cette mesure de type d'actif.</span><span class="sxs-lookup"><span data-stu-id="59b4f-126">In the **Total aggregate** field, select the summation method to be used for an asset measure using this asset measure type.</span></span> <span data-ttu-id="59b4f-127">La « somme » est la sélection standard utilisée pour ajouter en continu des valeurs enregistrées à la valeur totale.</span><span class="sxs-lookup"><span data-stu-id="59b4f-127">"Sum" is the standard selection used to continuously add registered values to the total value.</span></span> <span data-ttu-id="59b4f-128">La « moyenne » peut être utilisée si une mesure d'actif est configurée pour surveiller un seuil, par exemple en ce qui concerne la température, les vibrations ou l'usure d'un actif.</span><span class="sxs-lookup"><span data-stu-id="59b4f-128">"Average" can be used if an asset measure is set up to monitor a threshold, for example, regarding temperature, vibrations, or wear and tear on an asset.</span></span> 
8. <span data-ttu-id="59b4f-129">Dans le champ **Écart supérieur à**, insérez le niveau supérieur en pourcentage pour vérifier si la mesure manuelle des enregistrements d'actifs dans une plage prévue.</span><span class="sxs-lookup"><span data-stu-id="59b4f-129">In the **Deviation over** field, insert the upper level in percent for validating if manual asset measure registrations are within an expected range.</span></span> <span data-ttu-id="59b4f-130">La validation est basée sur une augmentation linéaire du nombre d’enregistrements existants de mesures d'actifs.</span><span class="sxs-lookup"><span data-stu-id="59b4f-130">The validation is based on a linear increase in existing asset measure registrations.</span></span>
9. <span data-ttu-id="59b4f-131">Dans le champ **Écart inférieur à**, insérez le niveau inférieur en pourcentage pour vérifier si la mesure manuelle des enregistrements d'actifs dans une plage prévue.</span><span class="sxs-lookup"><span data-stu-id="59b4f-131">In the **Deviation under** field, insert the lower level in percent for validating if manual asset measure registrations are within an expected range.</span></span> <span data-ttu-id="59b4f-132">La validation est basée sur une diminution linéaire du nombre d’enregistrements existants de mesures d'actifs.</span><span class="sxs-lookup"><span data-stu-id="59b4f-132">The validation is based on a linear decrease in existing asset measure registrations.</span></span>
10. <span data-ttu-id="59b4f-133">Dans le champ **Type**, sélectionnez le type de message (informations, avertissement, erreur) à afficher si des écarts en dehors de la plage définie se produisent lorsque vous enregistrez manuellement les mesures des actifs.</span><span class="sxs-lookup"><span data-stu-id="59b4f-133">In the **Type** field, select the type of message (information, warning, error) to be shown if deviations outside the defined range occur when you make manual asset measure registrations.</span></span>
11. <span data-ttu-id="59b4f-134">Dans l'organisateur **Types d'actifs**, ajoutez des types d'actifs qui doivent pouvoir utiliser la mesure des actifs.</span><span class="sxs-lookup"><span data-stu-id="59b4f-134">On the **Asset types** FastTab, add the asset types that should be able to use the asset measure.</span></span>
12. <span data-ttu-id="59b4f-135">Dans l'organisateur **Mesures d'actifs connexes**, ajoutez les mesures d'actifs que vous souhaitez mettre à jour automatiquement lorsque cette mesure d'actif est mise à jour.</span><span class="sxs-lookup"><span data-stu-id="59b4f-135">On the **Related asset measures** FastTab, add the asset measures that you want to be automatically updated when this asset measure is updated.</span></span>
