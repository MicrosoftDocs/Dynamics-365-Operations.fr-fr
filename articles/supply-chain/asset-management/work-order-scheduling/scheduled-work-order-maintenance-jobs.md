# <a name="scheduled-work-order-maintenance-jobs"></a><span data-ttu-id="f4c78-103">Tâches de maintenance de l’ordre de travail planifiées</span><span class="sxs-lookup"><span data-stu-id="f4c78-103">Scheduled work order maintenance jobs</span></span>

[!include [banner](../../includes/banner.md)]

 

<span data-ttu-id="f4c78-104">La page **Tâches de maintenance de l’ordre de travail planifiées** affiche une vue d’ensemble des ordres de travail affectés à une ressource.</span><span class="sxs-lookup"><span data-stu-id="f4c78-104">The **Scheduled work order maintenance jobs** page shows an overview of the work orders allocated to a resource.</span></span> <span data-ttu-id="f4c78-105">Les ordres de travail qui utilisent les ressources de type « Ressources humaines » , « Outil » et « Ordinateur » sont affichés.</span><span class="sxs-lookup"><span data-stu-id="f4c78-105">Work orders using resource types "Human resources", "Tool", and "Machine" are shown.</span></span> <span data-ttu-id="f4c78-106">Par exemple, si un agent de maintenance est malade, vous pouvez utiliser cette page pour rechercher rapidement d’autres ordres de travail affectés à ce collaborateur, et affecter un autre agent de maintenance à la tâche.</span><span class="sxs-lookup"><span data-stu-id="f4c78-106">For example, if a maintenance worker calls in sick, you can use this page to quickly find work orders allocated to the worker, and then allocate another maintenance worker to the job.</span></span>

## <a name="view-scheduled-work-order-maintenance-jobs"></a><span data-ttu-id="f4c78-107">Afficher les tâches de maintenance de l’ordre de travail planifiées</span><span class="sxs-lookup"><span data-stu-id="f4c78-107">View scheduled work order maintenance jobs</span></span>

1. <span data-ttu-id="f4c78-108">Cliquez sur **Gestion des actifs** > **Commun** > **Ordres de travail** > **Tâches de maintenance de l’ordre de travail planifiées**.</span><span class="sxs-lookup"><span data-stu-id="f4c78-108">Click **Asset management** > **Common** > **Work orders** > **Scheduled work order maintenance jobs**.</span></span> <span data-ttu-id="f4c78-109">Vous voyez une liste de tous les ordres de travail définis sur l’état du cycle de vie « Planifié » ou « En cours ».</span><span class="sxs-lookup"><span data-stu-id="f4c78-109">You see a list of all work orders set to work order lifecycle state "Scheduled" or "In progress".</span></span>

2. <span data-ttu-id="f4c78-110">Vous pouvez trier la liste, par exemple, par agent de maintenance.</span><span class="sxs-lookup"><span data-stu-id="f4c78-110">You can sort the list, for example, by maintenance worker.</span></span> <span data-ttu-id="f4c78-111">Vous pouvez également utiliser le filtre pour limiter la liste pour afficher les ordres de travail alloués à une ressource ou à un agent de maintenance spécifique.</span><span class="sxs-lookup"><span data-stu-id="f4c78-111">You can also use the filter to limit the list to display work orders allocated to a specific resource or maintenance worker.</span></span>

3. <span data-ttu-id="f4c78-112">Vous pouvez voir des notes sur l’ordre de travail et, le cas échéant, ajouter de nouvelles notes en sélectionnant la tâche de l’ordre de travail, puis cliquer sur **Notes**.</span><span class="sxs-lookup"><span data-stu-id="f4c78-112">You can see notes on the work order and, if required, add new notes by selecting the work order job, and then click **Notes**.</span></span>

4. <span data-ttu-id="f4c78-113">Si vous souhaitez affecter un agent de maintenance à un ordre de travail, sélectionnez l’ordre de travail, puis cliquer sur **Ordre de travail**.</span><span class="sxs-lookup"><span data-stu-id="f4c78-113">If you want to allocate one maintenance worker to a work order, select the work order, and then click **Work order**.</span></span>

5. <span data-ttu-id="f4c78-114">La page **Ordre de travail** s’ouvre.</span><span class="sxs-lookup"><span data-stu-id="f4c78-114">The **Work order** page opens.</span></span> <span data-ttu-id="f4c78-115">Cliquez sur **Répartir** pour planifier l’ordre de travail à un agent de maintenance spécifique.</span><span class="sxs-lookup"><span data-stu-id="f4c78-115">Click **Dispatch** to schedule the work order to a specific maintenance worker.</span></span>
