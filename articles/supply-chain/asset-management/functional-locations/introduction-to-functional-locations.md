# <a name="introduction-to-functional-locations"></a><span data-ttu-id="20387-103">Présentation des postes techniques</span><span class="sxs-lookup"><span data-stu-id="20387-103">Introduction to functional locations</span></span>

[!include [banner](../../includes/banner.md)]

 

<span data-ttu-id="20387-104">Cette rubrique donne une vue d'ensemble des postes techniques dans le module Gestion des actifs.</span><span class="sxs-lookup"><span data-stu-id="20387-104">This topic provides an overview of functional locations in Asset Management.</span></span> <span data-ttu-id="20387-105">Les postes techniques sont des éléments d'une structure technique, tels que les unités fonctionnelles dans un système.</span><span class="sxs-lookup"><span data-stu-id="20387-105">Functional locations are elements of a technical structure, such as the functional units in a system.</span></span> <span data-ttu-id="20387-106">Les postes techniques sont créés hiérarchiquement, et vous installez des actifs sur ceux-ci.</span><span class="sxs-lookup"><span data-stu-id="20387-106">Functional locations are created hierarchically, and you install assets on them.</span></span> <span data-ttu-id="20387-107">Le paramétrage des postes techniques dans votre société dépend des exigences de celle-ci.</span><span class="sxs-lookup"><span data-stu-id="20387-107">The setup of functional locations in your company depends on the company's requirements.</span></span>

<span data-ttu-id="20387-108">Voici quelques exemples de la manière dont vous pouvez utiliser les postes techniques :</span><span class="sxs-lookup"><span data-stu-id="20387-108">Here are some examples of how you can use functional locations:</span></span>

- <span data-ttu-id="20387-109">**Fonctionnel** – Les postes techniques peuvent être adaptés à l'utilisateur et utilisés pour gérer des actifs avec un comportement similaire.</span><span class="sxs-lookup"><span data-stu-id="20387-109">**Functional** – The functional locations can be user-oriented and used to manage assets that have similar behavior.</span></span>

<span data-ttu-id="20387-112">Chaque poste technique est géré indépendamment dans Gestion des actifs.</span><span class="sxs-lookup"><span data-stu-id="20387-112">Each functional location is managed independently in Asset Management.</span></span> <span data-ttu-id="20387-113">Voici certaines des fonctionnalités utiles des postes techniques :</span><span class="sxs-lookup"><span data-stu-id="20387-113">Here are some of the useful features of functional locations:</span></span>

- <span data-ttu-id="20387-115">Paramétrage des besoins en caractéristiques de l'actif.</span><span class="sxs-lookup"><span data-stu-id="20387-115">Set up asset specification requirements.</span></span>
- <span data-ttu-id="20387-116">Paramétrage des séquences de maintenance pour la maintenance préventive et réactive.</span><span class="sxs-lookup"><span data-stu-id="20387-116">Set up maintenance sequences for preventive and reactive maintenance.</span></span>
- <span data-ttu-id="20387-117">Gestion des actifs installés.</span><span class="sxs-lookup"><span data-stu-id="20387-117">Manage installed assets.</span></span>
- <span data-ttu-id="20387-118">Suivi des demandes et des ordres de travail actifs associés aux actifs installés.</span><span class="sxs-lookup"><span data-stu-id="20387-118">Track active requests and work orders that are related to installed assets.</span></span>
- <span data-ttu-id="20387-119">Suivi des défaillances enregistrées sur les actifs.</span><span class="sxs-lookup"><span data-stu-id="20387-119">Track faults that are registered on assets.</span></span>
- <span data-ttu-id="20387-120">Suivi des coûts de maintenance sur les actifs associés à un poste technique à un moment donné.</span><span class="sxs-lookup"><span data-stu-id="20387-120">Track maintenance costs on the assets that are related to a functional location at any given time.</span></span>

<span data-ttu-id="20387-121">Les postes techniques fournissent la traçabilité des actifs en relation avec les demandes, les ordres de travail, les enregistrements de défaillances, les évaluations de condition, les enregistrements d'arrêt de la production et des enregistrements du compteur d'actifs.</span><span class="sxs-lookup"><span data-stu-id="20387-121">Functional locations provide traceability of assets in relation to requests, work orders, fault registrations, condition assessments, production stop registrations, and asset counter registrations.</span></span>
