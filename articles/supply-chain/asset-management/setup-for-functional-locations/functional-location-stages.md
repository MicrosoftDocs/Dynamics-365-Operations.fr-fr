# <a name="functional-location-lifecycle-states"></a><span data-ttu-id="ecb90-103">États du cycle de vie du poste technique</span><span class="sxs-lookup"><span data-stu-id="ecb90-103">Functional location lifecycle states</span></span>

[!include [banner](../../includes/banner.md)]

 

<span data-ttu-id="ecb90-104">Cette rubrique décrit le paramétrage des états de cycle de vie de poste technique et des modèles de cycle de vie dans la gestion des actifs.</span><span class="sxs-lookup"><span data-stu-id="ecb90-104">This topic describes how to set up functional location lifecycle states and lifecycle models in Asset Management.</span></span> <span data-ttu-id="ecb90-105">Les états de cycle de vie de poste technique définissent les états par lesquels un poste technique peut passer, par exemple, créé, actif, puis terminé.</span><span class="sxs-lookup"><span data-stu-id="ecb90-105">Functional location lifecycle states define the states that a functional location can go through, for example, created, active, and ended.</span></span> <span data-ttu-id="ecb90-106">Vous pouvez afficher tous les postes techniques, indépendamment de leur état de cycle de vie, dans la page de liste **Tous les postes techniques**.</span><span class="sxs-lookup"><span data-stu-id="ecb90-106">You are able to view all functional locations, regardless of their lifecycle state, in the **All functional locations** list page.</span></span> <span data-ttu-id="ecb90-107">Vous pouvez modifier l’état d’un poste technique en le sélectionnant dans la page de liste **Tous les postes techniques** et en sélectionnant **Mettre à jour l’état du poste technique**.</span><span class="sxs-lookup"><span data-stu-id="ecb90-107">You can change the state of a functional location by selecting it in the **All functional locations** list page and selecting **Update functional location state**.</span></span>

## <a name="set-up-functional-location-lifecycle-states"></a><span data-ttu-id="ecb90-108">Définir des états du cycle de vie des postes techniques</span><span class="sxs-lookup"><span data-stu-id="ecb90-108">Set up functional location lifecycle states</span></span>

1. <span data-ttu-id="ecb90-109">Sélectionnez **Gestion des actifs** > **Paramétrage** > **Postes techniques** > **États du cycle de vie**.</span><span class="sxs-lookup"><span data-stu-id="ecb90-109">Select **Asset management** > **Setup** > **Functional locations** > **Lifecycle states**.</span></span>
2. <span data-ttu-id="ecb90-110">Sélectionnez **Nouveau** pour créer un état de poste technique.</span><span class="sxs-lookup"><span data-stu-id="ecb90-110">Select **New** to create a new functional location state.</span></span>
3. <span data-ttu-id="ecb90-111">Insérez l’ID d’état dans le champ **État du cycle de vie** et le nom de l’état du poste technique dans le champ **Nom**.</span><span class="sxs-lookup"><span data-stu-id="ecb90-111">Insert the state ID in the **Lifecycle state** field and a name for the functional location state in the **Name** field.</span></span> <span data-ttu-id="ecb90-112">Dans le champ **Modèles de cycle de vie**, vous pouvez afficher le nombre de modèles de cycle de vie du poste technique qui utilise l’état de poste technique.</span><span class="sxs-lookup"><span data-stu-id="ecb90-112">In the **Lifecycle models** field, you can see the number of functional location lifecycle models that uses the functional location state.</span></span>
4. <span data-ttu-id="ecb90-113">Dans l’organisateur **Général**, sélectionnez « Oui » sur le bouton de basculement **Actif** si le poste technique est actif dans cet état.</span><span class="sxs-lookup"><span data-stu-id="ecb90-113">On the **General** FastTab, select "Yes" on the **Active** toggle button if the functional location should be active at this state.</span></span>
5. <span data-ttu-id="ecb90-114">Sélectionnez « Oui » sur le bouton de basculement **Création d’actifs** s’il doit être possible de créer automatiquement un actif avec le même nom que le poste technique et de l’installer sur le poste technique dans cet état.</span><span class="sxs-lookup"><span data-stu-id="ecb90-114">Select "Yes" on the **Create assets** toggle button if it should be possible to automatically create an asset with the same name as the functional location and install it on the functional location at this state.</span></span>  
>[!NOTE]
><span data-ttu-id="ecb90-115">Ce bouton de basculement est lié au champ **Type d’actif** sur l’organisateur **Général** dans l’écran **Types de postes techniques** (**Gestion des actifs** > **Paramétrage** > **Postes techniques** > **Types de postes techniques**).</span><span class="sxs-lookup"><span data-stu-id="ecb90-115">This toggle button relates to the **Asset type** field on the **General** FastTab in the **Functional location types** form (**Asset management** > **Setup** > **Functional locations** > **Functional location types**).</span></span>
6. <span data-ttu-id="ecb90-116">Sélectionnez « Oui » sur le bouton de basculement **Renommer le poste** s’il doit être possible de modifier le nom du poste technique dans cet état.</span><span class="sxs-lookup"><span data-stu-id="ecb90-116">Select "Yes" on the **Rename location** toggle button if it should be possible to change the name of the functional location at this state.</span></span>
7. <span data-ttu-id="ecb90-117">Sélectionnez « Oui » sur le bouton de basculement **Nouveaux sous-postes** s’il doit être possible d’ajouter de nouveaux sous-postes au poste technique dans cet état.</span><span class="sxs-lookup"><span data-stu-id="ecb90-117">Select "Yes" on the **New sub locations** toggle button if it should be possible to add new sub locations to the functional location at this state.</span></span>
8. <span data-ttu-id="ecb90-118">Sélectionnez « Oui » sur le bouton de basculement **Installer des actifs** s’il doit être possible d’installer des actifs au poste technique dans cet état.</span><span class="sxs-lookup"><span data-stu-id="ecb90-118">Select "Yes" on the **Install assets** toggle button if it should be possible to install assets on the functional location at this state.</span></span>
9. <span data-ttu-id="ecb90-119">Sélectionnez « Oui » sur le bouton de basculement **Supprimer le poste technique** s’il doit être possible de supprimer le poste technique dans cet état.</span><span class="sxs-lookup"><span data-stu-id="ecb90-119">Select "Yes" on the **Delete functional location** toggle button if it should be possible to delete the functional location at this state.</span></span>
10. <span data-ttu-id="ecb90-120">Sélectionnez un état d’actif dans le champ **État du cycle de vie** si vous souhaitez que l’état du cycle de vie de l’actif pour tous les actifs installés sur le poste technique soit mis à jour automatiquement dans cet état.</span><span class="sxs-lookup"><span data-stu-id="ecb90-120">Select an asset state in the **Lifecycle state** field if you want the asset lifecycle state for all assets installed on the functional location to be automatically updated at this state.</span></span> <span data-ttu-id="ecb90-121">Exemple : Si vous fermez un poste technique, puis définissez l’état du cycle de vie du poste technique sur « Terminé », vous pouvez modifier automatiquement l’état du cycle de vie des actifs installés sur ce poste technique sur « N’est pas en cours d’utilisation ».</span><span class="sxs-lookup"><span data-stu-id="ecb90-121">Example: If you close down a functional location, and set the functional location lifecycle state to "Ended", you may want to automatically change the lifecycle state of the assets installed on that functional location to "Not in use".</span></span>


>[!NOTE]
><span data-ttu-id="ecb90-122">Les états du cycle de vie de poste technique, les modèles de cycle de vie et les types sont associés et utilisés de la même manière que les états du cycle de vie d’ordre de travail, les modèles de cycle de vie d’ordre de travail et les types d’ordres de travail.</span><span class="sxs-lookup"><span data-stu-id="ecb90-122">Functional location lifecycle states, lifecycle models, and types are related and used in the same way as work order lifecycle states, work order lifecycle models, and work order types.</span></span> 

## <a name="set-up-functional-location-lifecycle-models"></a><span data-ttu-id="ecb90-123">Définir des modèles de cycle de vie des postes techniques</span><span class="sxs-lookup"><span data-stu-id="ecb90-123">Set up functional location lifecycle models</span></span>

2. <span data-ttu-id="ecb90-128">Sélectionnez **Nouveau** pour créer un modèle de cycle de vie.</span><span class="sxs-lookup"><span data-stu-id="ecb90-128">Select **New** to create a new lifecycle model.</span></span>
3. <span data-ttu-id="ecb90-129">Insérez l’ID du modèle de cycle de vie dans le champ **Modèle de cycle de vie** et un nom pour le modèle de cycle de vie dans le champ **Nom**.</span><span class="sxs-lookup"><span data-stu-id="ecb90-129">Insert the lifecycle model ID in the **Lifecycle model** field and a name for the lifecycle model in the **Name** field.</span></span> <span data-ttu-id="ecb90-130">Dans les champs **Types de postes techniques** et **États du cycle de vie**, vous pouvez consulter le nombre de types de postes techniques qui utilise le modèle de cycle de vie et le nombre d’états sélectionnés dans le modèle de cycle de vie.</span><span class="sxs-lookup"><span data-stu-id="ecb90-130">In the **Functional location types** and **Lifecycle states** fields, you can see the number of functional location types that uses the lifecycle model and the number of states selected in the lifecycle model.</span></span>
4. <span data-ttu-id="ecb90-131">Dans le raccourci **États du cycle de vie**, sélectionnez les états qui doivent être inclus dans le modèle.</span><span class="sxs-lookup"><span data-stu-id="ecb90-131">On the **Lifecycle states** FastTab, select the states that should be included in the model.</span></span> <span data-ttu-id="ecb90-132">Cela s’effectue en cliquant sur un état dans la section **États du cycle de vie restants** et en cliquant sur le bouton ![flèche suivant](media/02-setup-for-functional-locations.png).</span><span class="sxs-lookup"><span data-stu-id="ecb90-132">This is done by clicking on a state in the **Lifecycle states remaining** section and clicking the ![forward arrow](media/02-setup-for-functional-locations.png) button.</span></span>
5. <span data-ttu-id="ecb90-133">Si vous souhaitez sélectionner tous les états disponibles pour un modèle, cliquez sur le bouton ![Sélectionner toutes les phases disponibles](media/03-setup-for-functional-locations.png).</span><span class="sxs-lookup"><span data-stu-id="ecb90-133">If you want to select all the available states for a model, click the ![select all available stages](media/03-setup-for-functional-locations.png) button.</span></span> <span data-ttu-id="ecb90-134">Tous les états sont transférés vers la section **États du cycle de vie sélectionnés**.</span><span class="sxs-lookup"><span data-stu-id="ecb90-134">All states are transferred to the **Lifecycle states selected** section.</span></span>
6. <span data-ttu-id="ecb90-135">Pour supprimer un état sélectionné à partir du modèle, sélectionnez l’état dans la section **États du cycle de vie sélectionnés** puis sélectionnez le bouton ![flèche précédent](media/04-setup-for-functional-locations.png).</span><span class="sxs-lookup"><span data-stu-id="ecb90-135">If you want to remove a selected state from the model, select the state in the **Lifecycle states selected** section and then select the ![back arrow](media/04-setup-for-functional-locations.png) button.</span></span>
7. <span data-ttu-id="ecb90-136">Sélectionnez **Mises à jour de l’état du cycle de vie** pour définir quels états du cycle de vie peuvent suivre un état sélectionné.</span><span class="sxs-lookup"><span data-stu-id="ecb90-136">Select **Lifecycle state updates** to define which lifecycle states can follow a selected state.</span></span>
