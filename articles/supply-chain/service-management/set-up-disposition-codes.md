# <a name="set-up-disposition-codes"></a><span data-ttu-id="33c34-103">Paramétrer les codes disposition</span><span class="sxs-lookup"><span data-stu-id="33c34-103">Set up disposition codes</span></span> 

[!include [banner](../includes/banner.md)]


<span data-ttu-id="33c34-104">Vous pouvez paramétrer des codes disposition pour spécifier comment traiter un article qui est retourné par un client.</span><span class="sxs-lookup"><span data-stu-id="33c34-104">You can set up disposition codes to specify how to process an item that is returned by a customer.</span></span> <span data-ttu-id="33c34-105">Par exemple, créez un code disposition nommé **Réparation et retour** pour indiquer que l'article retourné est réparé puis remis au client.</span><span class="sxs-lookup"><span data-stu-id="33c34-105">For example, create a disposition code named **Repair and return** to indicate that the returned item will be repaired and then returned to the customer.</span></span> <span data-ttu-id="33c34-106">Pour plus d'exemples de codes disposition généralement utilisés pour les retours d'articles par les clients, voir [Spécification de la procédure de cession des articles retournés](specify-how-to-dispose-of-returned-items.md).</span><span class="sxs-lookup"><span data-stu-id="33c34-106">For more examples of disposition codes that are typically used for items that are returned by customers, see [Specify how to dispose of returned items](specify-how-to-dispose-of-returned-items.md).</span></span>

<span data-ttu-id="33c34-107">Vous pouvez également paramétrer un code motif pour expliquer la raison pour laquelle un article a été retourné.</span><span class="sxs-lookup"><span data-stu-id="33c34-107">You can also set up a reason code to help explain why an item was returned.</span></span> <span data-ttu-id="33c34-108">Pour plus d'informations sur les codes motif, voir [Paramétrage de codes motif de retour](set-up-return-reason-code.md).</span><span class="sxs-lookup"><span data-stu-id="33c34-108">For more information about reason codes, see [Set up return reason codes](set-up-return-reason-code.md).</span></span>

1.  <span data-ttu-id="33c34-109">Cliquez sur **Ventes et marketing** \> **Paramétrage** \> **Commandes client** \> **Retours** \> **Codes disposition**.</span><span class="sxs-lookup"><span data-stu-id="33c34-109">Click **Sales and marketing** \> **Setup** \> **Sales orders** \> **Returns** \> **Disposition codes**.</span></span>

2.  <span data-ttu-id="33c34-110">Cliquez sur **Nouveau** ou appuyez sur Ctrl+N pour créer un code disposition.</span><span class="sxs-lookup"><span data-stu-id="33c34-110">Click **New** or press CTRL+N to create a new disposition code.</span></span>

3.  <span data-ttu-id="33c34-111">Entrez un nom descriptif unique, sélectionnez une action, puis entrez une description pour le code disposition.</span><span class="sxs-lookup"><span data-stu-id="33c34-111">Enter a unique, descriptive name, select an action, and enter a description for the disposition code.</span></span>

4.  <span data-ttu-id="33c34-112">Pour associer des frais clients au code disposition, cliquez sur le bouton **Frais** pour ouvrir l'écran **Paramétrer des frais**.</span><span class="sxs-lookup"><span data-stu-id="33c34-112">If you want to associate any customer charges with this disposition code, click the **Charges** button to open the **Set up charges** form.</span></span>

5.  <span data-ttu-id="33c34-113">Pour définir des codes externes correspondant aux codes disposition propres à la société, cliquez sur le bouton **Codes externes** pour ouvrir l'écran **Codes externes**.</span><span class="sxs-lookup"><span data-stu-id="33c34-113">If you want to define any external codes to match with the company's own disposition codes, click the **External codes** button to open the **External codes** form.</span></span>
