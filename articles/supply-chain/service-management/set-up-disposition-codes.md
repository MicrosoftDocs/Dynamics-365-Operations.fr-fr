# <a name="set-up-disposition-codes"></a>Paramétrer les codes disposition 

[!include [banner](../includes/banner.md)]


Vous pouvez paramétrer des codes disposition pour spécifier comment traiter un article qui est retourné par un client. Par exemple, créez un code disposition nommé **Réparation et retour** pour indiquer que l'article retourné est réparé puis remis au client. Pour plus d'exemples de codes disposition généralement utilisés pour les retours d'articles par les clients, voir [Spécification de la procédure de cession des articles retournés](specify-how-to-dispose-of-returned-items.md).

Vous pouvez également paramétrer un code motif pour expliquer la raison pour laquelle un article a été retourné. Pour plus d'informations sur les codes motif, voir [Paramétrage de codes motif de retour](set-up-return-reason-code.md).

1.  Cliquez sur **Ventes et marketing** \> **Paramétrage** \> **Commandes client** \> **Retours** \> **Codes disposition**.

2.  Cliquez sur **Nouveau** ou appuyez sur Ctrl+N pour créer un code disposition.

3.  Entrez un nom descriptif unique, sélectionnez une action, puis entrez une description pour le code disposition.

4.  Pour associer des frais clients au code disposition, cliquez sur le bouton **Frais** pour ouvrir l'écran **Paramétrer des frais**.

5.  Pour définir des codes externes correspondant aux codes disposition propres à la société, cliquez sur le bouton **Codes externes** pour ouvrir l'écran **Codes externes**.
