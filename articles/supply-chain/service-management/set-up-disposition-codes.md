<span data-ttu-id="926f4-104">Vous pouvez paramétrer des codes disposition pour spécifier comment traiter un article qui est retourné par un client.</span><span class="sxs-lookup"><span data-stu-id="926f4-104">You can set up disposition codes to specify how to process an item that is returned by a customer.</span></span> <span data-ttu-id="926f4-105">Par exemple, créez un code disposition nommé **Réparation et retour** pour indiquer que l’article retourné est réparé puis remis au client.</span><span class="sxs-lookup"><span data-stu-id="926f4-105">For example, create a disposition code named **Repair and return** to indicate that the returned item will be repaired and then returned to the customer.</span></span> <span data-ttu-id="926f4-106">Pour plus d’exemples de codes disposition généralement utilisés pour les retours d’articles par les clients, voir [Spécification de la procédure de cession des articles retournés](specify-how-to-dispose-of-returned-items.md).</span><span class="sxs-lookup"><span data-stu-id="926f4-106">For more examples of disposition codes that are typically used for items that are returned by customers, see [Specify how to dispose of returned items](specify-how-to-dispose-of-returned-items.md).</span></span>

<span data-ttu-id="926f4-107">Vous pouvez également paramétrer un code motif pour expliquer la raison pour laquelle un article a été retourné.</span><span class="sxs-lookup"><span data-stu-id="926f4-107">You can also set up a reason code to help explain why an item was returned.</span></span> <span data-ttu-id="926f4-108">Pour plus d’informations sur les codes motif, voir [Paramétrage de codes motif de retour](set-up-return-reason-code.md).</span><span class="sxs-lookup"><span data-stu-id="926f4-108">For more information about reason codes, see [Set up return reason codes](set-up-return-reason-code.md).</span></span>

1.  <span data-ttu-id="926f4-109">Accédez à **Ventes et marketing** \> **Paramétrage** \> **Commandes client** \> **Retours** \> **Codes disposition**.</span><span class="sxs-lookup"><span data-stu-id="926f4-109">Go to **Sales and marketing** \> **Setup** \> **Sales orders** \> **Returns** \> **Disposition codes**.</span></span>

2.  <span data-ttu-id="926f4-110">Sélectionnez **Nouveau** pour créer un code disposition.</span><span class="sxs-lookup"><span data-stu-id="926f4-110">Select **New** to create a new disposition code.</span></span>

3.  <span data-ttu-id="926f4-111">Entrez un nom descriptif unique, sélectionnez une action, puis entrez une description pour le code disposition.</span><span class="sxs-lookup"><span data-stu-id="926f4-111">Enter a unique, descriptive name, select an action, and enter a description for the disposition code.</span></span>
