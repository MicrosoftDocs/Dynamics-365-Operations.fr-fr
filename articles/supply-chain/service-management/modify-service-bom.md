# <a name="modify-a-service-bom"></a>Modifier une nomenclature des services 

[!include [banner](../includes/banner.md)]


Vous pouvez enregistrer l'historique d'un élément dans une nomenclature des services. À chaque mise à jour d'une ligne de nomenclature, une ligne d'historique est créée dans le volet **Historique**. La ligne d'historique indique l'état actuel de la ligne de nomenclature.

## <a name="update-a-service-bom-element"></a>Mise à jour d'un élément de nomenclature des services

1.  Cliquez sur **Gestion des services** \> **Commun** \> **Accords de service** \> **Accords de service**.

2.  Cliquez sur **Modifier** pour ouvrir l'écran **Accords de service**.

3.  Dans le volet **Actions**, cliquez sur **Objets du service** pour ouvrir l'écran **Objets du service**.

4.  Sélectionnez l'objet pour lequel mettre à jour une ligne de nomenclature, puis cliquez sur **Concepteur**.

5.  Dans l'écran **Concepteur**, sélectionnez la ligne de nomenclature à mettre à jour, puis cliquez sur **Modifier une ligne de nomenclature**.
    
    > [!NOTE]
    > <P>Sous l'onglet <STRONG>Paramétrage</STRONG>, activez la case à cocher <STRONG>Éditer lors de l'ajout</STRONG> si vous souhaitez que l'écran <STRONG>Modifier une ligne de nomenclature</STRONG> s'ouvre lorsque vous faites glisser une ligne dans la nomenclature des services.</P>

6.  Dans le champ **Quantité**, entrez la quantité.

7.  Pour créer une ligne de commande de service pour l'article de remplacement qui pourra alors être facturé, activez la case à cocher **Créer une ligne de commande de service**.

8.  Cliquez sur **OK** pour fermer l'écran.

## <a name="delete-a-service-bom-line"></a>Suppression d'une ligne de nomenclature des services

1.  Cliquez sur **Gestion des services** \> **Commun** \> **Accords de service** \> **Accords de service**.

2.  Cliquez sur **Modifier** pour ouvrir l'écran **Accords de service**.

3.  Dans le volet **Actions**, cliquez sur **Objets du service** pour ouvrir l'écran **Objets du service**.

4.  Sélectionnez l'objet duquel supprimer la ligne de nomenclature, puis cliquez sur **Concepteur**.

5.  Dans l'écran **Concepteur**, sélectionnez la ligne de nomenclature à supprimer, puis cliquez sur **Supprimer une ligne de nomenclature**.
