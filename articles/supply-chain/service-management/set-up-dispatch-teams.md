# <a name="set-up-dispatch-teams"></a>Paramétrer les équipes de répartition 

[!include [banner](../includes/banner.md)]


Cette rubrique décrit la procédure de création d'une équipe de répartition. Après avoir créé une équipe de répartition, vous pouvez lui affecter des collaborateurs. Lorsque vous créez une commande de service, vous pouvez lier un membre d'une équipe de répartition à une activité dans la commande de service.

1.  Cliquez sur **Gestion des services** \> **Paramétrage** \> **Équipes de répartition**.

2.  Dans le champ **Équipes de répartition**, cliquez sur **Nouveau** pour créer une équipe.

3.  Dans le champ **Équipes de répartition**, entrez un nom unique pour l'équipe de répartition.

4.  Dans le champ **Propriétaire**, sélectionnez l'employé responsable de cette équipe.
    

    > [!NOTE]
    > <P>Vous pouvez sélectionner uniquement des employés dans ce champ. Vous ne pouvez pas sélectionner des fournisseurs.</P>



5.  Facultatif : dans le champ **Description**, entrez une brève description de l'équipe de répartition.


> [!NOTE]
> <P>Vous pouvez affecter un collaborateur à cette équipe de répartition dans l'écran <STRONG>Collaborateur</STRONG>. Pour plus d'informations sur la procédure d'affectation d'un collaborateur à une équipe de répartition, voir <A href="https://technet.microsoft.com/library/dn776288(v=ax.60)">Répartir les collaborateurs (écran)</A>.</P>
