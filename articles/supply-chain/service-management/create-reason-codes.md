# <a name="create-reason-codes"></a>Créer des codes motifs

[!include [banner](../includes/banner.md)]

1. Accédez à **Gestion des services** \> **Paramétrage** \> **Commandes de service** \> **Codes motif du stade**.

1. Sélectionnez **Nouveau** pour créer un code motif.

1. Sélectionnez **Enregistrer** pour enregistrer le code motif.

Lorsque vous êtes invité à indiquer un code motif pour expliquer la mise à jour du stade d’une commande de service, vous pouvez sélectionner un des codes motif que vous créez ici.
