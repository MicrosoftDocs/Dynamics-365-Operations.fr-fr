# <a name="set-up-template-to-represent-objects-at-customer-site"></a><span data-ttu-id="358ab-103">Paramétrage d'un modèle pour représenter des objets sur le site client</span><span class="sxs-lookup"><span data-stu-id="358ab-103">Set up template to represent objects at customer site</span></span> 

[!include [banner](../includes/banner.md)]


<span data-ttu-id="358ab-104">Cette procédure permet de paramétrer un modèle représentant des objets sur un site client.</span><span class="sxs-lookup"><span data-stu-id="358ab-104">Use this procedure to set up a template that represents objects at a customer site.</span></span>

1.  <span data-ttu-id="358ab-105">Cliquez sur **Gestion des services** \> **Commun** \> **Accords de service** \> **Accords de service**.</span><span class="sxs-lookup"><span data-stu-id="358ab-105">Click **Service management** \> **Common** \> **Service agreements** \> **Service agreements**.</span></span>

2.  <span data-ttu-id="358ab-106">Double-cliquez sur un accord de service, ou cliquez sur **Accord de service** pour créer un accord de service.</span><span class="sxs-lookup"><span data-stu-id="358ab-106">Double-click a service agreement, or click **Service agreement** to create a new service agreement.</span></span>

3.  <span data-ttu-id="358ab-107">Cliquez sur l'onglet **Paramétrage**.</span><span class="sxs-lookup"><span data-stu-id="358ab-107">Click the **Setup** tab.</span></span>

4.  <span data-ttu-id="358ab-108">Cliquez sur **Objets de service** pour lier une nomenclature de modèles à l'accord de service.</span><span class="sxs-lookup"><span data-stu-id="358ab-108">Click **Service objects** to attach a template bill of materials (BOM) to the service agreement.</span></span>

5.  <span data-ttu-id="358ab-109">Dans l'écran **Objets du service**, sélectionnez un objet de service, puis cliquez sur **Fonctions** \> **Joindre une nomenclature des modèles**.</span><span class="sxs-lookup"><span data-stu-id="358ab-109">In the **Service objects** form, select a service object, and then click **Functions** \> **Attach template BOM**.</span></span>

6.  <span data-ttu-id="358ab-110">Dans l'écran **Sélectionner une nomenclature des modèles**, sélectionnez une nomenclature des modèles dans le champ **Nomenclature des modèles**, puis cliquez sur **Ajouter**.</span><span class="sxs-lookup"><span data-stu-id="358ab-110">In the **Select template BOM** form, select a template BOM in the **Template BOM** field, and then click **OK**.</span></span>

7.  <span data-ttu-id="358ab-111">Dans l'écran **Objets du service**, cliquez sur **Concepteur** pour modifier la nomenclature de modèles afin que la nomenclature représente les objets souhaités.</span><span class="sxs-lookup"><span data-stu-id="358ab-111">In the **Service objects** form, click **Designer** to modify the template BOM so that the BOM represents the objects that you want.</span></span>

8.  <span data-ttu-id="358ab-112">Fermez les écrans pour enregistrer vos modifications.</span><span class="sxs-lookup"><span data-stu-id="358ab-112">Close the forms to save your changes.</span></span>
