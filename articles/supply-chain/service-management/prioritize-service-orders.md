# <a name="prioritize-service-orders"></a>Définition de priorités pour les commandes de service 

[!include [banner](../includes/banner.md)]


1.  Cliquez sur **Gestion des services** \> **Périodique** \> **Tableau d'affectation**.

2.  Cliquez sur **Avancé** pour afficher le diagramme de Gantt en haut de la page et les onglets dans le volet inférieur.

3.  Développez chaque liste d'équipes de répartition.

4.  Sélectionnez le graphique à code de couleurs représentant une activité de service.

5.  Sous l'un des onglets **Tous**, **Équipe**, **Non réparti** ou **Associé**, modifiez la sélection dans le champ **Priorité**.

6.  Cliquez sur **Enregistrer**.
