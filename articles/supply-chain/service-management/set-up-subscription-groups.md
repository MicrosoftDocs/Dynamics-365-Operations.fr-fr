# <a name="set-up-subscription-groups"></a><span data-ttu-id="d0d2e-103">Paramétrer des groupes d’abonnements</span><span class="sxs-lookup"><span data-stu-id="d0d2e-103">Set up subscription groups</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="d0d2e-104">Accédez à **Gestion des services** \> **Paramétrage** \> **Services récurrents** \> **Groupes d’abonnements**.</span><span class="sxs-lookup"><span data-stu-id="d0d2e-104">Go to **Service management** \> **Setup** \> **Service subscriptions** \> **Subscription groups**.</span></span>

2.  <span data-ttu-id="d0d2e-105">Sélectionnez **Nouveau** pour créer un groupe d’abonnements.</span><span class="sxs-lookup"><span data-stu-id="d0d2e-105">Select **New** to create a new subscription group.</span></span>

3.  <span data-ttu-id="d0d2e-106">Dans le champ **Groupe d’abonnements**, entrez un nom pour le groupe d’abonnements.</span><span class="sxs-lookup"><span data-stu-id="d0d2e-106">In the **Subscription group** field, enter a name for the new subscription group.</span></span>

4.  <span data-ttu-id="d0d2e-107">Dans le champ **Facturation**, sélectionnez une fréquence de facturation pour l’abonnement.</span><span class="sxs-lookup"><span data-stu-id="d0d2e-107">In the **Invoicing** field, select an invoicing frequency for the subscription.</span></span> <span data-ttu-id="d0d2e-108">La fréquence de facturation est indiquée par un code période paramétré dans l’écran **Types de période**.</span><span class="sxs-lookup"><span data-stu-id="d0d2e-108">The invoicing frequency is indicated by a period code that is set up in the **Period types** form.</span></span>

5.  <span data-ttu-id="d0d2e-109">Pour provisionner le produit pour des transactions d’abonnement, activez la case à cocher **Provisionner le produit**, puis, dans le champ **Régularisation**, sélectionnez une fréquence.</span><span class="sxs-lookup"><span data-stu-id="d0d2e-109">To accrue revenue for subscription transactions, select the **Accrue revenue** check box, and then in the **Accrual** field, select a frequency.</span></span> <span data-ttu-id="d0d2e-110">La fréquence de régularisation est basée sur un code période paramétré dans l’écran **Périodes**.</span><span class="sxs-lookup"><span data-stu-id="d0d2e-110">The accrual frequency is based on a period code that is set up in the **Periods** form.</span></span>
