<span data-ttu-id="75745-104">Les informations relatives aux modèles peuvent être copiées vers un accord de service ou une commande de service.</span><span class="sxs-lookup"><span data-stu-id="75745-104">Template information can be copied to either a service agreement or a service order.</span></span>

## <a name="copy-service-template-lines-into-a-service-agreement"></a><span data-ttu-id="75745-105">Copie des lignes du modèle de service dans un accord de service</span><span class="sxs-lookup"><span data-stu-id="75745-105">Copy service template lines into a service agreement</span></span>

1. <span data-ttu-id="75745-106">Cliquez sur **Accords de service**.</span><span class="sxs-lookup"><span data-stu-id="75745-106">Click **Service agreements**.</span></span>
2. <span data-ttu-id="75745-107">Double-cliquez sur un accord de service pour ouvrir la page **Accords de service**.</span><span class="sxs-lookup"><span data-stu-id="75745-107">Double-click a service agreement to open the **Service agreements** page.</span></span>
3. <span data-ttu-id="75745-108">Cliquez sur **Lignes de modèle** pour ouvrir la page **Copier les lignes du modèle de service**.</span><span class="sxs-lookup"><span data-stu-id="75745-108">Click **Template lines** to open the **Copy service template lines** page.</span></span>
4. <span data-ttu-id="75745-109">Sous l’onglet **Contrôle de l’arbre**, développez l’arborescence, recherchez le modèle contenant les lignes à copier vers l’accord de service, puis sélectionnez le modèle de service.</span><span class="sxs-lookup"><span data-stu-id="75745-109">On the **Tree control** tab, expand the tree, find the template that contains the lines that you want to copy to the service agreement, and then select the service template.</span></span>
5. <span data-ttu-id="75745-110">Sous l’onglet **Lignes**, activez la case à cocher **Sélection** pour les lignes de modèle de service à copier.</span><span class="sxs-lookup"><span data-stu-id="75745-110">On the **Lines** tab, select the **Select** check box for the service template lines that you want to copy.</span></span>
6. <span data-ttu-id="75745-111">Cliquez sur **OK**.</span><span class="sxs-lookup"><span data-stu-id="75745-111">Click **OK**.</span></span>
7. <span data-ttu-id="75745-112">Fermez les pages pour enregistrer vos modifications.</span><span class="sxs-lookup"><span data-stu-id="75745-112">Close the pages to save your changes.</span></span>

<span data-ttu-id="75745-113">Les lignes de modèle de service sont maintenant copiées dans l’accord de service avec les relations de l’objet du service ou de la tâche de service associées.</span><span class="sxs-lookup"><span data-stu-id="75745-113">The service template lines are now copied to the service agreement, together with the attached service object and service task relations.</span></span>

4. <span data-ttu-id="75745-118">Sous l’onglet **Contrôle de l’arbre**, développez l’arborescence, recherchez le modèle contenant les lignes à copier vers la commande de service, puis sélectionnez le modèle de service.</span><span class="sxs-lookup"><span data-stu-id="75745-118">On the **Tree control** tab, expand the tree, find the template that contains the lines that you want to copy to the service order, and then select the service template.</span></span>

6. <span data-ttu-id="75745-120">Cliquez sur **OK**.</span><span class="sxs-lookup"><span data-stu-id="75745-120">Click **OK**.</span></span>
7. <span data-ttu-id="75745-121">Fermez les pages pour enregistrer vos modifications.</span><span class="sxs-lookup"><span data-stu-id="75745-121">Close the pages to save your changes.</span></span>

<span data-ttu-id="75745-122">Les lignes de modèle de service sont maintenant copiées dans la nouvelle commande de service avec les relations de l’objet du service ou de la tâche de service associées.</span><span class="sxs-lookup"><span data-stu-id="75745-122">The service template lines are now copied to the new service order, together with the attached service object and service task relations.</span></span>
