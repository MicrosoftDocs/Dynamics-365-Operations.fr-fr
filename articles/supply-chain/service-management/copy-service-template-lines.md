# <a name="copy-service-templates-lines"></a>Copie de lignes de modèles de service 

[!include [banner](../includes/banner.md)]

Les informations relatives aux modèles peuvent être copiées vers un accord de service ou une commande de service.

## <a name="copy-service-template-lines-into-a-service-agreement"></a>Copie des lignes du modèle de service dans un accord de service

1. Cliquez sur **Accords de service**.
2. Double-cliquez sur un accord de service pour ouvrir la page **Accords de service**.
3. Cliquez sur **Lignes de modèle** pour ouvrir la page **Copier les lignes du modèle de service**.
4. Sous l'onglet **Contrôle de l'arbre**, développez l'arborescence, recherchez le modèle contenant les lignes à copier vers l'accord de service, puis sélectionnez le modèle de service.
5. Sous l'onglet **Lignes**, activez la case à cocher **Sélection** pour les lignes de modèle de service à copier.
6. Cliquez sur **OK**.
7. Fermez les pages pour enregistrer vos modifications.

Les lignes de modèle de service sont maintenant copiées dans l'accord de service avec les relations de l'objet du service ou de la tâche de service associées.

## <a name="copy-service-template-lines-into-a-service-order"></a>Copie des lignes du modèle de service dans une commande de service

1. Cliquez sur **Commandes de service**.
2. Double-cliquez sur un ordre de service pour ouvrir la page **Commandes de service**.
3. Cliquez sur Copier \> Copier les lignes du modèle pour ouvrir la page **Copier les lignes du modèle de service**.
4. Sous l'onglet **Contrôle de l'arbre**, développez l'arborescence, recherchez le modèle contenant les lignes à copier vers la commande de service, puis sélectionnez le modèle de service.
5. Sous l'onglet **Lignes**, activez la case à cocher **Sélection** pour les lignes de modèle de service à copier.
6. Cliquez sur **OK**.
7. Fermez les pages pour enregistrer vos modifications.

Les lignes de modèle de service sont maintenant copiées dans la nouvelle commande de service avec les relations de l'objet du service ou de la tâche de service associées.
