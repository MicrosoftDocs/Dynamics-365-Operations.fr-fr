# <a name="set-up-service-activity-types"></a>Paramétrage de types d’activités de service 

[!include [banner](../includes/banner.md)]


1.  Cliquez sur **Ventes et marketing** \> **Paramétrage** \> **Activités** \> **Types d’activités**.
    
    - ou -
    
    Cliquez sur **Administration d’organisation** \> **Paramétrage** \> **Activités** \> **Types d’activités**.

2.  Dans l’écran **Types d’activités**, appuyez sur Ctrl+N pour ajouter une ligne de type d’activité.

3.  Dans le champ **Type**, entrez un nom pour le type d’activité.

4.  Dans le champ **Description**, entrez une description de ce type d’activité.

5.  Fermez l’écran pour enregistrer vos modifications.
