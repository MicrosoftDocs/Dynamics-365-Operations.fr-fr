# <a name="reverse-changes-in-a-dispatch-board"></a>Contrepassation de frais dans un tableau d'affectation 

[!include [banner](../includes/banner.md)]


Cette rubrique décrit la procédure de contrepassation des modifications non enregistrées dans un tableau d'affectation. Par exemple, vous affectez un collaborateur à une activité de service, vous enregistrez l'information, puis vous décidez ultérieurement d'affecter un autre collaborateur à l'activité de service. Vous modifiez alors le collaborateur dans le tableau d'affectation, mais avant d'enregistrer la modification, vous apprenez que le collaborateur que vous venez d'affecter n'est pas disponible. Vous pouvez contrepasser la modification non enregistrée afin que le collaborateur d'origine soit affecté à nouveau à la commande de service.

Utilisez les étapes suivantes pour contrepasser les modifications non enregistrées dans un tableau d'affectation :

1.  Cliquez sur **Gestion des services** \> **Périodique** \> **Tableau d'affectation**.

2.  Dans l'écran **Tableau d'affectation**, renseignez les champs appropriés, puis cliquez sur **OK**. 

3.  Pour contrepasser la modification non enregistrée la plus récente, cliquez sur **Annuler**.

4.  Pour contrepasser une série de modifications non enregistrées, continuez à cliquer sur **Annuler** jusqu'à ce que toutes les modifications à écarter soient annulées.
