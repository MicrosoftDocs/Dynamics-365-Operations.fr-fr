# <a name="subscription-groups"></a>Groupes d'abonnements    

[!include [banner](../includes/banner.md)]


Les abonnements partagent les informations suivantes définies pour le groupe d'abonnements :

  - Éventuel code souche de numéros pour l'abonnement.

  - la période de facturation pour laquelle les transactions d'abonnement sont créées. Par exemple, si le code période **Facturation** dans l'écran **Groupes d'abonnement** est défini sur **Trimestriel**, des transactions pour tous les abonnements associés au groupe d'abonnements sont créées chaque trimestre.

  - Fréquence éventuelle du provisionnement de l'abonnement.

Vous pouvez également effectuer les tâches suivantes pour tous les abonnements associés au groupe d'abonnements :

  - créer des transactions d'abonnement ;

  - mettre à jour le prix de base ou l'index pour les abonnements ;

  - valider le produit à recevoir pour les transactions d'abonnement.
