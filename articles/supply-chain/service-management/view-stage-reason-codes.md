# <a name="view-stage-reason-codes"></a>Afficher les codes motif du stade 

[!include [banner](../includes/banner.md)]


Affichez et modifiez la liste des motifs du stade pour une commande de service. Utilisez les motifs du stade pour indiquer le motif du statut d'une commande de service. Par exemple, vous pouvez créer un stade **Annulé** auquel le code motif **Annulé par le client** est affecté.

  - Cliquez sur **Gestion des services** \> **Commun** \> **Commandes de service** \> **Commandes de service**. Ouvrez une commande de service.

  - Dans l'écran **Commandes de service**, dans le volet **Actions**, sous l'onglet **Répartir**, cliquez sur **Motifs du stade**.

  - Dans l'écran **Motifs du stade**, cliquez sur **Nouveau** pour créer un code motif de stade.
