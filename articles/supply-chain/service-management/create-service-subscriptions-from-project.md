[!include [banner](../includes/banner.md)]


<span data-ttu-id="78c3f-104">Vous pouvez créer un service récurrent à partir d'un projet dans le module **Gestion et comptabilité des projets**.</span><span class="sxs-lookup"><span data-stu-id="78c3f-104">You can create a service subscription from a project in **Project management and accounting**.</span></span>

2.  <span data-ttu-id="78c3f-107">Dans l'écran **Projets**, dans le volet **Actions**, sous l'onglet **Gérer**, cliquez sur **Service** \> **Service récurrent**.</span><span class="sxs-lookup"><span data-stu-id="78c3f-107">On the **Projects** form, on the **Action Pane**, on the **Manage** tab, click **Service** \> **Service subscription**.</span></span>

> [!NOTE]
> <P><span data-ttu-id="78c3f-112">Si vous avez paramétré une souche de numéros pour le groupe d'abonnements, un ID d'abonnement est automatiquement généré pour l'abonnement lorsque vous spécifiez le groupe d'abonnements.</span><span class="sxs-lookup"><span data-stu-id="78c3f-112">If you have set up a number sequence for the subscription group, a subscription ID is generated automatically for the subscription when you specify the subscription group.</span></span></P>
