# <a name="create-service-subscriptions-from-a-project"></a><span data-ttu-id="4778e-103">Création de services récurrents à partir d'un projet</span><span class="sxs-lookup"><span data-stu-id="4778e-103">Create service subscriptions from a project</span></span>    

[!include [banner](../includes/banner.md)]


<span data-ttu-id="4778e-104">Vous pouvez créer un service récurrent à partir d'un projet dans le module **Gestion et comptabilité des projets**.</span><span class="sxs-lookup"><span data-stu-id="4778e-104">You can create a service subscription from a project in **Project management and accounting**.</span></span>

1.  <span data-ttu-id="4778e-105">Cliquez sur **Gestion et comptabilité des projets** \> **Commun** \> **Projets** \> **Tous les projets**.</span><span class="sxs-lookup"><span data-stu-id="4778e-105">Click **Project management and accounting** \> **Common** \> **Projects** \> **All projects**.</span></span> <span data-ttu-id="4778e-106">Ouvrez le projet pour lequel vous voulez créer un abonnement.</span><span class="sxs-lookup"><span data-stu-id="4778e-106">Open the project that you want to create a subscription for.</span></span>

2.  <span data-ttu-id="4778e-107">Dans l'écran **Projets**, dans le volet **Actions**, sous l'onglet **Gérer**, cliquez sur **Service** \> **Service récurrent**.</span><span class="sxs-lookup"><span data-stu-id="4778e-107">On the **Projects** form, on the **Action Pane**, on the **Manage** tab, click **Service** \> **Service subscription**.</span></span>

4.  <span data-ttu-id="4778e-110">Entrez les autres informations dans l'écran.</span><span class="sxs-lookup"><span data-stu-id="4778e-110">Enter the remaining information in the form.</span></span>

<span data-ttu-id="4778e-111">Vous pouvez désormais créer des transactions d'abonnement pour l'abonnement.</span><span class="sxs-lookup"><span data-stu-id="4778e-111">You can now create subscription transactions for the subscription.</span></span>


> [!NOTE]
> <P><span data-ttu-id="4778e-112">Si vous avez paramétré une souche de numéros pour le groupe d'abonnements, un ID d'abonnement est automatiquement généré pour l'abonnement lorsque vous spécifiez le groupe d'abonnements.</span><span class="sxs-lookup"><span data-stu-id="4778e-112">If you have set up a number sequence for the subscription group, a subscription ID is generated automatically for the subscription when you specify the subscription group.</span></span></P>
