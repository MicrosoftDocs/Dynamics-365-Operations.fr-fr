# <a name="create-service-subscriptions-from-a-project"></a>Création de services récurrents à partir d’un projet    

[!include [banner](../includes/banner.md)]


Vous pouvez créer un service récurrent à partir d’un projet dans le module **Gestion et comptabilité des projets**.

1.  Cliquez sur **Gestion et comptabilité des projets** \> **Commun** \> **Projets** \> **Tous les projets**. Ouvrez le projet pour lequel vous voulez créer un abonnement.

2.  Dans l’écran **Projets**, dans le volet **Actions**, sous l’onglet **Gérer**, cliquez sur **Service** \> **Service récurrent**.

3.  Dans l’écran **Abonnement**, dans le champ **Service récurrent**, entrez un numéro d’identification unique. Le numéro de projet est entré automatiquement dans l’abonnement lié au projet sélectionné.

4.  Entrez les autres informations dans l’écran.

Vous pouvez désormais créer des transactions d’abonnement pour l’abonnement.


> [!NOTE]
> <P>Si vous avez paramétré une souche de numéros pour le groupe d’abonnements, un ID d’abonnement est automatiquement généré pour l’abonnement lorsque vous spécifiez le groupe d’abonnements.</P>
