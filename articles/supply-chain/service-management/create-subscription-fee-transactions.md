# <a name="create-subscription-fee-transactions"></a><span data-ttu-id="66d0b-103">Créer des transactions de redevance</span><span class="sxs-lookup"><span data-stu-id="66d0b-103">Create subscription fee transactions</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="66d0b-104">Cliquez sur **Gestion des services** \> **Commun** \> **Services récurrents** \> **Tous les abonnements aux services**.</span><span class="sxs-lookup"><span data-stu-id="66d0b-104">Click **Service management** \> **Common** \> **Service subscriptions** \> **All service subscriptions**.</span></span>

2.  <span data-ttu-id="66d0b-105">Dans la page de liste **Tous les abonnements aux services**, sélectionnez l’abonnement pour lequel vous souhaitez créer une transaction de frais d’abonnement.</span><span class="sxs-lookup"><span data-stu-id="66d0b-105">On the **All service subscriptions** list page, select the subscription for which you want to create a subscription fee transaction.</span></span>

3.  <span data-ttu-id="66d0b-106">Dans le volet **Actions**, sous l’onglet **Abonnement**, cliquez sur **Frais d’abonnement**.</span><span class="sxs-lookup"><span data-stu-id="66d0b-106">On the **Action Pane**, on the **Subscription** tab, click **Subscription fees**.</span></span>

4.  <span data-ttu-id="66d0b-107">Dans l’écran **Créer des frais d’abonnement**, entrez les informations appropriées dans les champs.</span><span class="sxs-lookup"><span data-stu-id="66d0b-107">On the **Create subscription fee** form, enter the appropriate information in the fields.</span></span>

5.  <span data-ttu-id="66d0b-108">Cliquez sur **OK**.</span><span class="sxs-lookup"><span data-stu-id="66d0b-108">Click **OK**.</span></span>
