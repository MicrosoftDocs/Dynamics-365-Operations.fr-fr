# <a name="move-service-order-lines"></a><span data-ttu-id="e17d9-103">Déplacer des lignes de commande de service</span><span class="sxs-lookup"><span data-stu-id="e17d9-103">Move service order lines</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="e17d9-104">Cliquez sur **Gestion des services** \> **Commun** \> **Commandes de service** \> **Commandes de service**.</span><span class="sxs-lookup"><span data-stu-id="e17d9-104">Click **Service management** \> **Common** \> **Service orders** \> **Service orders**.</span></span>

2.  <span data-ttu-id="e17d9-105">Double-cliquez sur la commande de service qui contient les lignes à déplacer vers une autre commande de service.</span><span class="sxs-lookup"><span data-stu-id="e17d9-105">Double-click the service order that contains the lines that you want to move to another service order.</span></span>

3.  <span data-ttu-id="e17d9-106">Dans l'écran **Commandes de service**, dans le volet Actions, cliquez sur le bouton **Déplacer des lignes**.</span><span class="sxs-lookup"><span data-stu-id="e17d9-106">In the **Service orders** form, on the Action Pane, click the **Move lines** button.</span></span>

4.  <span data-ttu-id="e17d9-107">Dans l'écran **Déplacer des lignes de commande de service**, dans le champ **Commande de service**, sélectionnez la commande de service vers laquelle déplacer les lignes.</span><span class="sxs-lookup"><span data-stu-id="e17d9-107">In the **Move service order lines** form, in the **Service order** field, select the service order that you want to move the lines to.</span></span>

5.  <span data-ttu-id="e17d9-108">Activez la case à cocher **Supprimer la commande de service vide** pour supprimer la commande de service à partir de laquelle vous avez déplacé les lignes de commande de service.</span><span class="sxs-lookup"><span data-stu-id="e17d9-108">Select the **Delete empty service order** check box to delete the service order that you moved the service order lines from.</span></span>

6.  <span data-ttu-id="e17d9-109">Cliquez sur **OK**.</span><span class="sxs-lookup"><span data-stu-id="e17d9-109">Click **OK**.</span></span>


> [!NOTE]
> <P><span data-ttu-id="e17d9-110">Vous pouvez déplacer des lignes de commande de service uniquement entre des commandes de service liées au même accord de service.</span><span class="sxs-lookup"><span data-stu-id="e17d9-110">You can move service order lines only between service orders that are attached to the same service agreement.</span></span></P>
