# <a name="set-up-service-order-stages"></a><span data-ttu-id="2342c-103">Paramétrer des stades de commande de service</span><span class="sxs-lookup"><span data-stu-id="2342c-103">Set up service order stages</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="2342c-104">Cliquez sur **Gestion des services** \> **Paramétrage** \> **Commandes de service** \> **Stades du service**.</span><span class="sxs-lookup"><span data-stu-id="2342c-104">Click **Service management** \> **Setup** \> **Service orders** \> **Service stages**.</span></span>

2.  <span data-ttu-id="2342c-105">Appuyez sur Ctrl+N pour créer un enregistrement.</span><span class="sxs-lookup"><span data-stu-id="2342c-105">Press CTRL+N to create a new record.</span></span>

3.  <span data-ttu-id="2342c-106">Dans les champs **Stade du service** et **Description**, entrez l'ID et la description du stade de service.</span><span class="sxs-lookup"><span data-stu-id="2342c-106">In the **Service stage** and **Description** fields, specify a service stage ID and description.</span></span>

4.  <span data-ttu-id="2342c-107">Sélectionnez les paramètres appropriés pour le stade.</span><span class="sxs-lookup"><span data-stu-id="2342c-107">Select the appropriate parameters for the stage.</span></span>

5.  <span data-ttu-id="2342c-108">Sélectionnez le stade parent du stade actif ou laissez le champ **Parent** vide si le stade actif correspond au stade initial du paramétrage de stade.</span><span class="sxs-lookup"><span data-stu-id="2342c-108">Select the parent stage for the current stage or leave the **Parent** field blank if the stage is the initial stage in the stage setup.</span></span>


> [!NOTE]
> <P><span data-ttu-id="2342c-109">Une fois le stade enregistré, il n'est plus possible de modifier le champ <STRONG>Parent</STRONG>.</span><span class="sxs-lookup"><span data-stu-id="2342c-109">You cannot modify the <STRONG>Parent</STRONG> field after you save the stage.</span></span> <span data-ttu-id="2342c-110">Vous pouvez supprimer l'enregistrement et le créer à nouveau en sélectionnant un autre élément dans le champ <STRONG>Parent</STRONG>.</span><span class="sxs-lookup"><span data-stu-id="2342c-110">Instead, you can delete the record and create the record again with a different selection in the <STRONG>Parent</STRONG> field.</span></span></P>
> <P><span data-ttu-id="2342c-111">En outre, vous ne pouvez créer qu'un seul stade avec un champ <STRONG>Parent</STRONG> vide.</span><span class="sxs-lookup"><span data-stu-id="2342c-111">Also, you can only create one stage with a blank <STRONG>Parent</STRONG> field.</span></span> <span data-ttu-id="2342c-112">Cela signifie qu'un seul stade initial est autorisé.</span><span class="sxs-lookup"><span data-stu-id="2342c-112">That is, only one initial stage is permitted.</span></span></P>
