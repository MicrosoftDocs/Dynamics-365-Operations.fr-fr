# <a name="delete-a-return-order"></a>Suppression d'un ordre de retour 

[!include [banner](../includes/banner.md)]


L'écran **Supprimer un ordre de retour** permet de supprimer plusieurs ordres de retour.

1.  Cliquez sur **Ventes et marketing** \> **Périodique** \> **Nettoyage** \> **Supprimer des ordres de retour**.

2.  Cliquez sur **Sélectionner** pour ouvrir l'écran **Mise à jour des ventes**.

3.  Spécifiez les critères, puis cliquez sur **OK** pour revenir à l'écran **Supprimer un ordre de retour**. Les ordres de retour créés dans l'entité juridique sont affichés en fonction des critères spécifiés.

4.  Sélectionnez la ligne d'ordre de retour à supprimer, puis cliquez sur **OK**.

5.  Cliquez sur **Oui** pour supprimer les ordres de retour.
