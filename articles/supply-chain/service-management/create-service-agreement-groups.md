# <a name="create-service-agreement-groups"></a>Création de groupes d’accords de service

[!include [banner](../includes/banner.md)]

Les groupes d’accords de service permettent de trier et de filtrer les accords de service.

1. Accédez à **Gestion des services** \> **Paramétrage** \> **Accords de service** \> **Groupes d’accords de service**.
1. Dans le volet Actions, sélectionnez **Nouveau** pour un nouveau groupe d’accords de service.
1. Dans le champ **Groupe d’accords de service**, saisissez un ID pour le groupe d’accords de service.
1. Dans le champ **Destination**, entrez la description du groupe d’accords de service.
1. Dans le champ **Contrat de niveau de service**, sélectionnez l’accord qui s’applique au groupe.
1. Fermez la page pour enregistrer vos modifications.

> [!NOTE]
> Si un groupe d’accords de service est défini dans le champ **Groupe d’accords de service**, dans la page **Paramètres de gestion des services**, le programme suggère ce groupe pour tous les accords de service que vous créez.
