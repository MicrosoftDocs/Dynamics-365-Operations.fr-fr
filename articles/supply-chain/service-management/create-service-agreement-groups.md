# <a name="create-service-agreement-groups"></a>Création de groupes d’accords de service 

[!include [banner](../includes/banner.md)]

Les groupes d’accords de service permettent de trier et de filtrer les accords de service.

1. Cliquez sur **Gestion des services** \> **Paramétrage** \> **Accords de service** \> **Groupes d’accords de service**.
2. Appuyez sur **Alt+N** pour créer un groupe d’accords de service.
3. Dans le champ **Groupe d’accords de service**, saisissez un ID pour le groupe d’accords de service.
4. Dans le champ **Destination**, entrez la description du groupe d’accords de service.
5. Fermez la page pour enregistrer vos modifications.

> [!NOTE]
> Si un groupe d’accords de service est défini dans le champ **Groupe d’accords de service**, dans la page **Paramètres de gestion des services**, le programme suggère ce groupe pour tous les accords de service que vous créez.
