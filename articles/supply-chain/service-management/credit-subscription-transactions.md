## <a name="credit-subscription-transactions"></a><span data-ttu-id="666ce-104">Crédit de transactions d’abonnement</span><span class="sxs-lookup"><span data-stu-id="666ce-104">Credit subscription transactions</span></span>

1.  <span data-ttu-id="666ce-105">Cliquez sur **Gestion des services** \> **Commun** \> **Services récurrents** \> **Tous les abonnements aux services**.</span><span class="sxs-lookup"><span data-stu-id="666ce-105">Click **Service management** \> **Common** \> **Service subscriptions** \> **All service subscriptions**.</span></span>

2.  <span data-ttu-id="666ce-106">Sélectionnez l’abonnement lié à la transaction d’abonnement pour laquelle vous souhaitez créer un avoir.</span><span class="sxs-lookup"><span data-stu-id="666ce-106">Select the subscription attached to the subscription transaction for which you want to create a credit note.</span></span>

3.  <span data-ttu-id="666ce-107">Sélectionnez l’onglet **Analyser**, puis cliquez sur le bouton **Transactions de frais** dans le volet Actions.</span><span class="sxs-lookup"><span data-stu-id="666ce-107">Select the **Analyze** tab, and then click the **Fee transactions** button on the Action Pane.</span></span>

4.  <span data-ttu-id="666ce-108">À partir de l’écran **Transactions de frais**, sélectionnez la transaction pour laquelle vous souhaitez créer un avoir.</span><span class="sxs-lookup"><span data-stu-id="666ce-108">From the **Fee transactions** form, select the transaction for which you want to create a credit note.</span></span>

5.  <span data-ttu-id="666ce-109">Cliquez sur **Fonctions** \> **Sélectionner pour l’avoir**.</span><span class="sxs-lookup"><span data-stu-id="666ce-109">Click **Functions** \> **Select for credit note**.</span></span>

6.  <span data-ttu-id="666ce-110">À partir de l’écran **Sélectionner pour l’avoir**, sélectionnez la transaction que vous voulez créditer, puis cliquez sur **OK**.</span><span class="sxs-lookup"><span data-stu-id="666ce-110">From the **Select for credit note** form, select the transaction that you want to credit and then click **OK**.</span></span>


> [!NOTE]
> <P><span data-ttu-id="666ce-111">Lors de la création d’un avoir, veillez à sélectionner l’option <STRONG>Avoirs</STRONG>.</span><span class="sxs-lookup"><span data-stu-id="666ce-111">When you create the credit note, make sure that you select <STRONG>Credit notes</STRONG>.</span></span> <span data-ttu-id="666ce-112">Celle-ci se trouve dans la liste <STRONG>Méthode de facturation</STRONG> de la boîte de dialogue <STRONG>Créer une facture</STRONG>.</span><span class="sxs-lookup"><span data-stu-id="666ce-112">This is found in the <STRONG>Invoicing method</STRONG> list in the <STRONG>Create invoice</STRONG> dialog box.</span></span></P>

<span data-ttu-id="666ce-113">Si le champ **Contrepasser les régularisations lors de l’opération de crédit** dans l’écran **Paramètres de gestion des services** est défini sur **Manuel**, vous devez contrepasser individuellement chaque transaction de produit à recevoir avant de créer une proposition d’avoir pour la transaction.</span><span class="sxs-lookup"><span data-stu-id="666ce-113">If the **Reverse accruals on crediting** field in the **Service management parameters** form is set to **Manual**, you have to reverse each accrued revenue transaction individually before you create a credit note proposal for the transaction.</span></span>
