# <a name="set-up-service-intervals"></a><span data-ttu-id="40878-103">Paramétrage des intervalles de services</span><span class="sxs-lookup"><span data-stu-id="40878-103">Set up service intervals</span></span>  

[!include [banner](../includes/banner.md)]

<span data-ttu-id="40878-104">L’intervalle de services indique la fréquence de création des lignes de commande de service pour les lignes d’accord de service lorsque vous créez des commandes de service.</span><span class="sxs-lookup"><span data-stu-id="40878-104">Service interval indicates the frequency with which service order lines are created for service agreement lines when you create service orders.</span></span>

2. <span data-ttu-id="40878-106">Créez un intervalle de services.</span><span class="sxs-lookup"><span data-stu-id="40878-106">Create a new service interval.</span></span>
3. <span data-ttu-id="40878-107">Entrez l’ID et une description de l’intervalle de services.</span><span class="sxs-lookup"><span data-stu-id="40878-107">Enter the ID and description of the service interval.</span></span>
4. <span data-ttu-id="40878-108">Dans le champ **Plage**, sélectionnez la plage.</span><span class="sxs-lookup"><span data-stu-id="40878-108">In the **Range** field, select the range.</span></span>
