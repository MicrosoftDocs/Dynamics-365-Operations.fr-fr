# <a name="set-up-service-intervals"></a><span data-ttu-id="aaa87-103">Paramétrage des intervalles de services</span><span class="sxs-lookup"><span data-stu-id="aaa87-103">Set up service intervals</span></span>  

[!include [banner](../includes/banner.md)]

<span data-ttu-id="aaa87-104">L’intervalle de services indique la fréquence de création des lignes de commande de service pour les lignes d’accord de service lorsque vous créez des commandes de service.</span><span class="sxs-lookup"><span data-stu-id="aaa87-104">Service interval indicates the frequency with which service order lines are created for service agreement lines when you create service orders.</span></span>

1. <span data-ttu-id="aaa87-105">Cliquez sur **Gestion des services** \> **Paramétrage** \> **Accords de service** \> **Intervalles de services**.</span><span class="sxs-lookup"><span data-stu-id="aaa87-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Service intervals**.</span></span>
2. <span data-ttu-id="aaa87-106">Créez un intervalle de services.</span><span class="sxs-lookup"><span data-stu-id="aaa87-106">Create a new service interval.</span></span>
3. <span data-ttu-id="aaa87-107">Entrez l’ID et une description de l’intervalle de services.</span><span class="sxs-lookup"><span data-stu-id="aaa87-107">Enter the ID and description of the service interval.</span></span>
4. <span data-ttu-id="aaa87-108">Dans le champ **Plage**, sélectionnez la plage.</span><span class="sxs-lookup"><span data-stu-id="aaa87-108">In the **Range** field, select the range.</span></span>
5. <span data-ttu-id="aaa87-109">Dans le champ **Fréquence**, entrez la fréquence.</span><span class="sxs-lookup"><span data-stu-id="aaa87-109">In the **Frequency** field, type the frequency.</span></span> <span data-ttu-id="aaa87-110">La fréquence est le facteur par lequel vous devez multiplier la plage pour obtenir l’intervalle d’un accord de service.</span><span class="sxs-lookup"><span data-stu-id="aaa87-110">The frequency is the factor by which you must multiply the range to obtain the interval for a service agreement.</span></span>
6. <span data-ttu-id="aaa87-111">Appuyez sur **Alt+S** pour enregistrer l’intervalle de services.</span><span class="sxs-lookup"><span data-stu-id="aaa87-111">Press **Alt+S** to save the service interval.</span></span>
