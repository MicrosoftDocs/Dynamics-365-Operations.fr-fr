[!include [banner](../includes/banner.md)]

<span data-ttu-id="2c500-104">L'intervalle de services indique la fréquence de création des lignes de commande de service pour les lignes d'accord de service lorsque vous créez des commandes de service.</span><span class="sxs-lookup"><span data-stu-id="2c500-104">Service interval indicates the frequency with which service order lines are created for service agreement lines when you create service orders.</span></span>

3. <span data-ttu-id="2c500-107">Entrez l'ID et une description de l'intervalle de services.</span><span class="sxs-lookup"><span data-stu-id="2c500-107">Enter the ID and description of the service interval.</span></span>
