# <a name="set-up-service-intervals"></a>Paramétrage des intervalles de services  

[!include [banner](../includes/banner.md)]

L'intervalle de services indique la fréquence de création des lignes de commande de service pour les lignes d'accord de service lorsque vous créez des commandes de service.

1. Cliquez sur **Gestion des services** \> **Paramétrage** \> **Accords de service** \> **Intervalles de services**.
2. Créez un intervalle de services.
3. Entrez l'ID et une description de l'intervalle de services.
4. Dans le champ **Plage**, sélectionnez la plage.
5. Dans le champ **Fréquence**, entrez la fréquence. La fréquence est le facteur par lequel vous devez multiplier la plage pour obtenir l'intervalle d'un accord de service.
6. Appuyez sur **Alt+S** pour enregistrer l'intervalle de services.

## <a name="example"></a>Exemple

Vous voulez créer un intervalle de services de 10 jours.

**Création d'un intervalle de services de 10 jours**

1. Cliquez sur **Gestion des services** \> **Paramétrage** \> **Accords de service** \> **Intervalles de services**.
2. Créez un intervalle de services.
3. Entrez l'ID et une description de l'intervalle de services.
4. Dans le champ **Plage**, sélectionnez **Journalier**.
5. Dans le champ **Fréquence**, entrez 10.
6. Appuyez sur **Alt+S** pour enregistrer l'intervalle de services.
