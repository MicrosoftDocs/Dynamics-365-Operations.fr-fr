10. <span data-ttu-id="de3ad-129">Sélectionnez WHS dans le champ **Nom**.</span><span class="sxs-lookup"><span data-stu-id="de3ad-129">In the **Name** field, select WHS.</span></span>
11. <span data-ttu-id="de3ad-130">Dans le champ **Entrepôt**, sélectionnez l’entrepôt 24.</span><span class="sxs-lookup"><span data-stu-id="de3ad-130">In the **Warehouse** field, select warehouse 24.</span></span> <span data-ttu-id="de3ad-131">Il s’agit de l’entrepôt par défaut utilisé pour les lignes de réception enregistrées qui utilisent ce profil.</span><span class="sxs-lookup"><span data-stu-id="de3ad-131">This is the default warehouse that will be used for registered receipt lines that use this profile.</span></span>
