# <a name="search-for-vendors"></a>Rechercher des fournisseurs

[!include [banner](../../includes/banner.md)]

Découvrez comment rechercher des fournisseurs en fonction de critères spécifiques. Cet exemple décrit la manière dont vous recherchez des fournisseurs agréés pour une catégorie d’approvisionnement spécifique et qui ont une adresse principale dans un pays spécifique. Vous pouvez exécuter cette procédure dans la société fictive de démonstration USMF ou utiliser vos propres données. Cette tâche est généralement effectuée par un professionnel de l’approvisionnement.

1. Accédez à Approvisionnements > Fournisseurs > Recherche fournisseurs.
2. Cliquez sur l’icône Plus pour ouvrir la page Sélection de la catégorie d’approvisionnement.  
3. Dans l’arborescence, sélectionnez les « CATÉGORIES D’APPROVISIONNEMENT DE L’ENTREPRISE\ORDINATEURS DE BUREAU ».
    * Si vous exécutez cette procédure en tant que guide des tâches, vous devrez peut-être cliquer sur le bouton Déverrouiller avant de pouvoir sélectionner le bon nœud. Si vous n’utilisez pas USMF, sélectionnez l’une des catégories que vous avez.  
4. Cliquez sur Ajouter.
    * Il est possible de sélectionner plusieurs catégories ici. Si vous procédez ainsi, la recherche trouvera tous les fournisseurs agréés pour au moins une des catégories.  
5. Cliquez sur OK.
6. Tapez une valeur dans le champ Pays/Région.
7. Cliquez sur OK.
