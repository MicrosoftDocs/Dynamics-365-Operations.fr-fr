<span data-ttu-id="854aa-104">Cette procédure vous indique comment créer une commande fournisseur de projet.</span><span class="sxs-lookup"><span data-stu-id="854aa-104">This procedure shows you how to create a project purchase order.</span></span> <span data-ttu-id="854aa-105">Cette tâche utilise l'ensemble de données USSI.</span><span class="sxs-lookup"><span data-stu-id="854aa-105">This task uses the USSI data set.</span></span>

1. <span data-ttu-id="854aa-106">Accédez à Gestion et comptabilité des projets > Projets > Tous les projets.</span><span class="sxs-lookup"><span data-stu-id="854aa-106">Go to Project management and accounting > Projects > All projects.</span></span>
2. <span data-ttu-id="854aa-107">Dans la liste, cliquer sur le lien dans la ligne sélectionnée.</span><span class="sxs-lookup"><span data-stu-id="854aa-107">In the list, click the link in the selected row.</span></span>
3. <span data-ttu-id="854aa-108">Dans le volet Actions, cliquez sur Gérer.</span><span class="sxs-lookup"><span data-stu-id="854aa-108">On the Action Pane, click Manage.</span></span>
4. <span data-ttu-id="854aa-109">Cliquez sur Tâche article.</span><span class="sxs-lookup"><span data-stu-id="854aa-109">Click Item task.</span></span>
5. <span data-ttu-id="854aa-110">Cliquez sur Commande fournisseur.</span><span class="sxs-lookup"><span data-stu-id="854aa-110">Click Purchase order.</span></span>
6. <span data-ttu-id="854aa-111">Dans le champ Compte fournisseur, saisissez ou sélectionnez une valeur.</span><span class="sxs-lookup"><span data-stu-id="854aa-111">In the Vendor account field, enter or select a value.</span></span>
7. <span data-ttu-id="854aa-112">Saisissez ou sélectionnez une valeur dans le champ Site.</span><span class="sxs-lookup"><span data-stu-id="854aa-112">In the Site field, enter or select a value.</span></span>

9. <span data-ttu-id="854aa-115">Cliquez sur OK.</span><span class="sxs-lookup"><span data-stu-id="854aa-115">Click OK.</span></span>
10. <span data-ttu-id="854aa-116">Dans la liste, marquez la ligne sélectionnée.</span><span class="sxs-lookup"><span data-stu-id="854aa-116">In the list, mark the selected row.</span></span>
11. <span data-ttu-id="854aa-117">Entrez ou sélectionnez une valeur dans le champ Numéro d'article.</span><span class="sxs-lookup"><span data-stu-id="854aa-117">In the Item number field, enter or select a value.</span></span>

13. <span data-ttu-id="854aa-120">Cliquez sur l'onglet Projet.</span><span class="sxs-lookup"><span data-stu-id="854aa-120">Click the Project tab.</span></span>
    * <span data-ttu-id="854aa-121">Vérifiez que les ventes et les prix de revient sont disponibles.</span><span class="sxs-lookup"><span data-stu-id="854aa-121">Verify that the sales and cost prices are available.</span></span> <span data-ttu-id="854aa-122">S'ils ne le sont pas, mais requis, entrez ces éléments.</span><span class="sxs-lookup"><span data-stu-id="854aa-122">If they are not available but needed, enter the information.</span></span>
