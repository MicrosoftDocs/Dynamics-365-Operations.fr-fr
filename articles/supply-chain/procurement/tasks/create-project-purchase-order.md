# <a name="create-project-purchase-order"></a><span data-ttu-id="8b061-103">Créer une commande fournisseur de projet</span><span class="sxs-lookup"><span data-stu-id="8b061-103">Create project purchase order</span></span>

<span data-ttu-id="8b061-104">Cette procédure vous indique comment créer une commande fournisseur de projet.</span><span class="sxs-lookup"><span data-stu-id="8b061-104">This procedure shows you how to create a project purchase order.</span></span> <span data-ttu-id="8b061-105">Cette tâche utilise l'ensemble de données USSI.</span><span class="sxs-lookup"><span data-stu-id="8b061-105">This task uses the USSI data set.</span></span>

1. <span data-ttu-id="8b061-106">Accédez à Gestion et comptabilité des projets > Projets > Tous les projets.</span><span class="sxs-lookup"><span data-stu-id="8b061-106">Go to Project management and accounting > Projects > All projects.</span></span>
2. <span data-ttu-id="8b061-107">Dans la liste, cliquer sur le lien dans la ligne sélectionnée.</span><span class="sxs-lookup"><span data-stu-id="8b061-107">In the list, click the link in the selected row.</span></span>
3. <span data-ttu-id="8b061-108">Dans le volet Actions, cliquez sur Gérer.</span><span class="sxs-lookup"><span data-stu-id="8b061-108">On the Action Pane, click Manage.</span></span>
4. <span data-ttu-id="8b061-109">Cliquez sur Tâche article.</span><span class="sxs-lookup"><span data-stu-id="8b061-109">Click Item task.</span></span>
5. <span data-ttu-id="8b061-110">Cliquez sur Commande fournisseur.</span><span class="sxs-lookup"><span data-stu-id="8b061-110">Click Purchase order.</span></span>
6. <span data-ttu-id="8b061-111">Dans le champ Compte fournisseur, saisissez ou sélectionnez une valeur.</span><span class="sxs-lookup"><span data-stu-id="8b061-111">In the Vendor account field, enter or select a value.</span></span>
7. <span data-ttu-id="8b061-112">Saisissez ou sélectionnez une valeur dans le champ Site.</span><span class="sxs-lookup"><span data-stu-id="8b061-112">In the Site field, enter or select a value.</span></span>
    * <span data-ttu-id="8b061-113">Ces étapes ne sont pas obligatoires, mais elles simplifient la commande fournisseur en paramétrant un site et entrepôt par défaut pour les lignes de commande fournisseur.</span><span class="sxs-lookup"><span data-stu-id="8b061-113">These steps aren't required, but they do simplify the purchase order by setting up a default site and warehouse for the purchase order lines.</span></span>  
8. <span data-ttu-id="8b061-114">Dans le champ Entrepôt, saisissez ou sélectionnez une valeur.</span><span class="sxs-lookup"><span data-stu-id="8b061-114">In the Warehouse field, enter or select a value.</span></span>
9. <span data-ttu-id="8b061-115">Cliquez sur OK.</span><span class="sxs-lookup"><span data-stu-id="8b061-115">Click OK.</span></span>
10. <span data-ttu-id="8b061-116">Dans la liste, marquez la ligne sélectionnée.</span><span class="sxs-lookup"><span data-stu-id="8b061-116">In the list, mark the selected row.</span></span>
11. <span data-ttu-id="8b061-117">Entrez ou sélectionnez une valeur dans le champ Numéro d'article.</span><span class="sxs-lookup"><span data-stu-id="8b061-117">In the Item number field, enter or select a value.</span></span>
    * <span data-ttu-id="8b061-118">Ce peut être le numéro d'article ou une catégorie d'approvisionnement.</span><span class="sxs-lookup"><span data-stu-id="8b061-118">This can be the item number or a procurement category.</span></span>  
12. <span data-ttu-id="8b061-119">Développez la section Détails de ligne.</span><span class="sxs-lookup"><span data-stu-id="8b061-119">Expand the Line details section.</span></span>
13. <span data-ttu-id="8b061-120">Cliquez sur l'onglet Projet.</span><span class="sxs-lookup"><span data-stu-id="8b061-120">Click the Project tab.</span></span>
    * <span data-ttu-id="8b061-121">Vérifiez que les ventes et les prix de revient sont disponibles.</span><span class="sxs-lookup"><span data-stu-id="8b061-121">Verify that the sales and cost prices are available.</span></span> <span data-ttu-id="8b061-122">S'ils ne le sont pas, mais requis, entrez ces éléments.</span><span class="sxs-lookup"><span data-stu-id="8b061-122">If they are not available but needed, enter the information.</span></span>  
14. <span data-ttu-id="8b061-123">Cliquez sur Enregistrer.</span><span class="sxs-lookup"><span data-stu-id="8b061-123">Click Save.</span></span>
