# <a name="create-record-templates-to-facilitate-data-entry"></a>Créer des modèles d'enregistrement pour faciliter la saisie des données

[!include [task guide banner](../../includes/task-guide-banner.md)]

Cette procédure montre comment créer un modèle d'enregistrement pour ne pas avoir à saisir explicitement les valeurs de champ utilisées régulièrement pour chaque nouvel enregistrement. Dans cette procédure, vous allez créer un enregistrement pour de nouveaux ordinateurs portables qui doivent être ajoutés à vos immobilisations. La société fictive USMF sert d'exemple dans cette procédure.

1. Accédez à Immobilisations > Immobilisations > Immobilisations.
2. Cliquez sur Nouveau.
3. Saisissez ou sélectionnez une valeur dans le champ Groupe d'immobilisations.
4. Tapez une valeur dans le champ Nom.
    * Par exemple, entrez « Ordinateur portable entreprise de prospect ».  
5. Dans le champ Rechercher, tapez une valeur.
    * Par exemple, entrez « ordinateur portable ».  
6. Développez la section Informations techniques.
7. Dans le champ Faire, tapez une valeur.
8. Dans le champ Modèle, tapez une valeur.
9. Dans le champ Année du modèle, tapez une valeur.
10. Dans le volet Actions, cliquez sur Options.
11. Cliquez sur Infos sur l'enregistrement.
12. Cliquez sur Modèle d'utilisateur.
13. Tapez une valeur dans le champ Nom.
    * Par exemple, entrez « Ordinateur portable entreprise ».  
14. Dans le champ Description, entrez une valeur.
    * Par exemple, entrez « Ordinateur portable entreprise ».  
15. Cliquez sur OK.
16. Cliquez sur Fermer.
