# <a name="import-users-in-bulk"></a>Importer des utilisateurs en bloc

[!include [task guide banner](../../includes/task-guide-banner.md)]

Cette procédure peut être utilisée par les administrateurs système pour importer un grand nombre d'utilisateurs depuis Azure Active Directory.


## <a name="run-as-a-batch-job"></a>Exécuter comme un traitement par lots
1. Accédez à Administration système > Utilisateurs > Utilisateurs.
2. Cliquez sur Importation par lots.
3. Développez la section Exécuter à l'arrière-plan.
4. Sélectionnez Oui dans le champ Traitement par lots.
5. Dans le champ Description de la tâche, tapez une valeur.
6. Dans le champ Groupe de traitement par lots, entrez ou sélectionnez une valeur.
    * Cette étape est facultative.  
7. Sélectionnez Oui dans le champ Privé.
    * Cette étape est facultative.  
8. Sélectionnez Oui dans le champ Tâche critique.
    * Cette étape est facultative.  
9. Dans le champ Catégorie de surveillance, sélectionnez une option.
10. Cliquez sur OK.

## <a name="run-in-a-sandbox-environment"></a>Exécuter dans un environnement de bac à sable
1. Cliquez sur Importation par lots.
2. Cliquez sur OK.
