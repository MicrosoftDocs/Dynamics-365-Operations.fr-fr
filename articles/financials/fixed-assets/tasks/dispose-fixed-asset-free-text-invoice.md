[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="8de49-104">Cette procédure explique comment céder une immobilisation à l'aide de la facture financière.</span><span class="sxs-lookup"><span data-stu-id="8de49-104">This procedure shows how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="8de49-105">Accédez à Comptabilité client > Factures > Toutes factures financières.</span><span class="sxs-lookup"><span data-stu-id="8de49-105">Go to Accounts receivable > Invoices > All free text invoices.</span></span>
2. <span data-ttu-id="8de49-106">Cliquez sur Nouveau.</span><span class="sxs-lookup"><span data-stu-id="8de49-106">Click New.</span></span>
3. <span data-ttu-id="8de49-107">Entrez ou sélectionnez une valeur dans le champ Compte client.</span><span class="sxs-lookup"><span data-stu-id="8de49-107">In the Customer account field, enter or select a value.</span></span>

6. <span data-ttu-id="8de49-110">Entrez une description dans la ligne de facture.</span><span class="sxs-lookup"><span data-stu-id="8de49-110">Enter a Description into the invoice line.</span></span>

10. <span data-ttu-id="8de49-114">Cliquez sur Détails de ligne.</span><span class="sxs-lookup"><span data-stu-id="8de49-114">Click Line details.</span></span>  
11. <span data-ttu-id="8de49-115">Sélectionnez le numéro de l'immobilisation à vendre.</span><span class="sxs-lookup"><span data-stu-id="8de49-115">Select the Fixed asset number to be sold.</span></span>
12. <span data-ttu-id="8de49-116">Cliquez sur Valider.</span><span class="sxs-lookup"><span data-stu-id="8de49-116">Click Post.</span></span>
