<span data-ttu-id="46a32-104">La société fictive USMF sert d'exemple dans ce guide de tâche.</span><span class="sxs-lookup"><span data-stu-id="46a32-104">This task guide uses the USMF demo company.</span></span>  <span data-ttu-id="46a32-105">Une nouvelle immobilisation sera alors créée à l'aide de la page de liste Immobilisations.</span><span class="sxs-lookup"><span data-stu-id="46a32-105">It will create a new fixed asset using the Fixed asset list page.</span></span>

1. <span data-ttu-id="46a32-106">Accédez à Immobilisations > Immobilisations > Immobilisations.</span><span class="sxs-lookup"><span data-stu-id="46a32-106">Go to Fixed assets > Fixed assets > Fixed assets.</span></span>
2. <span data-ttu-id="46a32-107">Cliquez sur Nouveau.</span><span class="sxs-lookup"><span data-stu-id="46a32-107">Click New.</span></span>
3. <span data-ttu-id="46a32-108">Saisissez ou sélectionnez une valeur dans le champ Groupe d'immobilisations.</span><span class="sxs-lookup"><span data-stu-id="46a32-108">In the Fixed asset group field, enter or select a value.</span></span>
    * <span data-ttu-id="46a32-109">Le numéro sera défini par défaut si vous avez activé la fonctionnalité Numéroter automatiquement les immobilisations dans les paramètres Immobilisations et dans le groupe d'immobilisations.</span><span class="sxs-lookup"><span data-stu-id="46a32-109">The Number will default if you have enabled Autonumber fixed assets functionality in the Fixed assets parameters and the Fixed asset group.</span></span>  <span data-ttu-id="46a32-110">Sinon, vous devez entrer un numéro unique pour identifier l'immobilisation.</span><span class="sxs-lookup"><span data-stu-id="46a32-110">If not, you must enter a unique number to identify the fixed asset.</span></span>

    * <span data-ttu-id="46a32-116">Entrez les informations supplémentaires dont votre entreprise a besoin pour ce registre.</span><span class="sxs-lookup"><span data-stu-id="46a32-116">Enter the additional information that your business needs for this book.</span></span>  
    * <span data-ttu-id="46a32-117">Entrez les informations supplémentaires dont votre entreprise a besoin pour les registres restants.</span><span class="sxs-lookup"><span data-stu-id="46a32-117">Enter the additional information that your business needs for the remaining books.</span></span>
