# <a name="calculate-fixed-asset-depreciation-across-legal-entities"></a>Calculer l'amortissement des immobilisations dans les entités juridiques

[!include[task guide banner](../../includes/task-guide-banner.md)]

Utilisez cette procédure pour modifier le groupe d'immobilisations auquel une immobilisation est affectée. Les immobilisations doivent être affectées au groupe d'immobilisations approprié. Le groupe d'immobilisations est utilisé lorsque vous créez des recherches et des états, paramétrez de nouvelles immobilisations, intégrez des comptabilités et validez des transactions d'immobilisation dans les comptes généraux appropriés.

La société fictive USMF sert d'exemple dans cet enregistrement.

1. Accédez à Immobilisations > Immobilisations > Immobilisations.
2. Sélectionnez l'immobilisation pour laquelle modifier le groupe d'immobilisations.
3. Cliquez sur Changer de groupe d'immobilisations.
4. Dans le champ Nouveau groupe, entrez ou sélectionnez une valeur.
5. Définissez l'option Nouveau numéro d'immobilisation sur Oui pour affecter un numéro à l'immobilisation sélectionnée.
    * Le champ Numéro d'immobilisation devient disponible lorsque l'option Nouveau numéro d'immobilisation est définie sur Oui.   Si les immobilisations sont paramétrées sur la numérotation automatique, ce champ affiche le prochain numéro d'immobilisation disponible. Vous pouvez modifier celui-ci.   Si la numérotation manuelle est paramétrée pour les immobilisations, ce champ est vide et vous devez entrer le nouveau numéro d'immobilisation.  
6. Cliquez sur OK.
7. Cliquez sur Oui.
