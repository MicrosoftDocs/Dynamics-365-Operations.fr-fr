<span data-ttu-id="94fe1-113">Lorsque vous utilisez l'écran Mettre à jour les coûts de remplacement et les valeurs assurées pour recalculer le coût de remplacement et la valeur assurée pour les immobilisations, les formules suivantes sont utilisées :</span><span class="sxs-lookup"><span data-stu-id="94fe1-113">When you use the Update replacement costs and insured values form to recalculate the replacement cost and insured value for the assets, the following formulas are used:</span></span>
