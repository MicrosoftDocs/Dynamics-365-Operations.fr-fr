# <a name="set-up-a-ledger-account-alias"></a><span data-ttu-id="a1623-103">Paramétrage d'un alias de compte général</span><span class="sxs-lookup"><span data-stu-id="a1623-103">Set up a ledger account alias</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="a1623-104">Cette procédure montre comment créer un alias de compte qui fournit un raccourci pour entrer un numéro de compte.</span><span class="sxs-lookup"><span data-stu-id="a1623-104">This procedure shows how to create an account alias that provides a shortcut for entering an account number.</span></span> <span data-ttu-id="a1623-105">La société fictive USMF sert d'exemple dans cette procédure.</span><span class="sxs-lookup"><span data-stu-id="a1623-105">This procedure users demo data company USMF.</span></span>

1. <span data-ttu-id="a1623-106">Accédez à Comptabilité > Plan de comptes > Comptes > Alias de compte général.</span><span class="sxs-lookup"><span data-stu-id="a1623-106">Go to General ledger > Chart of accounts > Accounts > Ledger account alias.</span></span>
2. <span data-ttu-id="a1623-107">Cliquez sur Nouveau.</span><span class="sxs-lookup"><span data-stu-id="a1623-107">Click New.</span></span>
3. <span data-ttu-id="a1623-108">Tapez une valeur dans le champ Alias de compte général.</span><span class="sxs-lookup"><span data-stu-id="a1623-108">In the Ledger account alias field, type a value.</span></span>
4. <span data-ttu-id="a1623-109">Dans le champ Structure de compte, sélectionnez la structure à laquelle appartiennent le compte et les dimensions.</span><span class="sxs-lookup"><span data-stu-id="a1623-109">In the Account structure field, select the structure the account and dimensions belong to.</span></span>
5. <span data-ttu-id="a1623-110">Dans le champ Société, cliquez sur le bouton de liste déroulante pour ouvrir la recherche.</span><span class="sxs-lookup"><span data-stu-id="a1623-110">In the Company field, click the drop-down button to open the lookup.</span></span>
6. <span data-ttu-id="a1623-111">Dans la liste, cherchez et sélectionnez la société à laquelle s'applique l'alias.</span><span class="sxs-lookup"><span data-stu-id="a1623-111">In the list, find and select the company that the alias applies to.</span></span>
7. <span data-ttu-id="a1623-112">Dans la liste, cliquer sur le lien dans la ligne sélectionnée.</span><span class="sxs-lookup"><span data-stu-id="a1623-112">In the list, click the link in the selected row.</span></span>
8. <span data-ttu-id="a1623-113">Dans le champ Définition d'alias de compte général, spécifiez le compte et les dimensions.</span><span class="sxs-lookup"><span data-stu-id="a1623-113">In the Ledger account alias definition field, specify the account and dimensions.</span></span>
    * <span data-ttu-id="a1623-114">Le compte et les dimensions sont remplis lorsque le raccourci est utilisé.</span><span class="sxs-lookup"><span data-stu-id="a1623-114">The account and dimensions will be populated when using the shortcut.</span></span>  
9. <span data-ttu-id="a1623-115">Dans le champ Vue initiale, sélectionnez la dimension sur laquelle sera centrée la vue quand l'alias sera utilisé.</span><span class="sxs-lookup"><span data-stu-id="a1623-115">In the Initial focus field, select the dimension that will have focus when the alias is used.</span></span>
    * <span data-ttu-id="a1623-116">Après avoir tapez le raccourci et que le compte et les dimensions sont remplis, le champ Vue initiale se trouve à l'endroit où se déplace le curseur ou la vue.</span><span class="sxs-lookup"><span data-stu-id="a1623-116">After you type the shortcut, and the account and dimensions are populated, the Initial focus field is where the cursor or focus will move to.</span></span>
