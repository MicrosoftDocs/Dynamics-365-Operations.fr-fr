# <a name="view-related-intercompany-voucher-from-journal"></a>Afficher le N° document intersociétés associé à partir du journal

[!include [task guide banner](../../includes/task-guide-banner.md)]

La fenêtre associée de numéro de document affiche le numéro de document de la société de contrepartie lors de la validation d'une transaction intersociétés du journal des opérations diverses.


## <a name="post-an-intercompany-journal"></a>Valider un journal intersociétés
1. Accédez à Journaux des opérations diverses.
2. Cliquez sur Nouveau.
3. Dans la liste, marquez la ligne sélectionnée.
4. Dans le champ Nom, entrez ou sélectionnez le nom de journal intersociétés.
5. Cliquez sur Lignes.
6. Dans la liste, marquer la ligne sélectionnée.
7. Dans le champ Compte, spécifiez les valeurs souhaitées.
8. Dans le champ Description, saisissez ou sélectionnez une valeur.
9. Dans le champ Description, entrez une valeur.
10. Fermez la page.
11. Entrez un nombre dans le champ Débit.
12. Dans le champ Société de contrepartie, saisissez ou sélectionnez la société de contrepartie.
13. Dans le champ Société de contrepartie, saisissez ou sélectionnez une valeur.
14. Fermez la page.
15. Spécifiez les valeurs souhaitées dans le champ Compte de contrepartie.
16. Cliquez sur Valider.

## <a name="view-related-intercompany-voucher"></a>Afficher le N° document intersociétés associé
1. Cliquez Document.
2. Cliquez sur N° document liés.
3. Dans la liste, marquez la ligne sélectionnée.
4. Cliquez Document.
