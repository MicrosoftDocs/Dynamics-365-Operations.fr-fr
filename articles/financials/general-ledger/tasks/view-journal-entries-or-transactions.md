[!include [task guide banner](../../includes/task-guide-banner.md)]

2. <span data-ttu-id="9d4b8-106">Sélectionnez le champ pour lequel vous souhaitez définir des critères de filtre.</span><span class="sxs-lookup"><span data-stu-id="9d4b8-106">Select the field for which you want to define a filter criteria.</span></span>
3. <span data-ttu-id="9d4b8-107">Saisissez vos critères de filtre pour le champ sélectionné.</span><span class="sxs-lookup"><span data-stu-id="9d4b8-107">Enter your filter critieria for the selected field.</span></span>
    * <span data-ttu-id="9d4b8-108">Vous pouvez filtrer selon une valeur unique ou une plage.</span><span class="sxs-lookup"><span data-stu-id="9d4b8-108">You could filter on a single value or a range.</span></span> <span data-ttu-id="9d4b8-109">Lorsque vous définissez une plage, vérifiez que la syntaxe appropriée est utilisée.</span><span class="sxs-lookup"><span data-stu-id="9d4b8-109">When defining a range, make sure the correct syntax is used.</span></span> <span data-ttu-id="9d4b8-110">Les valeurs doivent être séparées par deux points (..).</span><span class="sxs-lookup"><span data-stu-id="9d4b8-110">The values should be separated by a double period (..).</span></span>

8. <span data-ttu-id="9d4b8-115">Cliquez sur l'onglet Plage.</span><span class="sxs-lookup"><span data-stu-id="9d4b8-115">Click the Range tab.</span></span>
9. <span data-ttu-id="9d4b8-116">Cliquez sur OK pour exécuter la requête.</span><span class="sxs-lookup"><span data-stu-id="9d4b8-116">Click OK to run the query.</span></span>

13. <span data-ttu-id="9d4b8-122">Fermez la page.</span><span class="sxs-lookup"><span data-stu-id="9d4b8-122">Close the page.</span></span>
