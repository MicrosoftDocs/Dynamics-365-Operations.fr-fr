# <a name="view-journal-entries-or-transactions"></a><span data-ttu-id="24eb0-103">Afficher les entrées de journal ou les transactions</span><span class="sxs-lookup"><span data-stu-id="24eb0-103">View journal entries or transactions</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="24eb0-104">Cette procédure indique comment utiliser la demande de transactions de N° document pour rechercher des entrées ou des transactions de journal.</span><span class="sxs-lookup"><span data-stu-id="24eb0-104">This procedure shows how to use the Voucher transactions inquiry to search for journal entries or transactions.</span></span>

1. <span data-ttu-id="24eb0-105">Allez dans Comptabilité > Recherches et états > Pièces comptables.</span><span class="sxs-lookup"><span data-stu-id="24eb0-105">Go to General ledger > Inquiries and reports > Voucher transactions.</span></span>
2. <span data-ttu-id="24eb0-106">Sélectionnez le champ pour lequel vous souhaitez définir des critères de filtre.</span><span class="sxs-lookup"><span data-stu-id="24eb0-106">Select the field for which you want to define a filter criteria.</span></span>
3. <span data-ttu-id="24eb0-107">Saisissez vos critères de filtre pour le champ sélectionné.</span><span class="sxs-lookup"><span data-stu-id="24eb0-107">Enter your filter criteria for the selected field.</span></span>
    * <span data-ttu-id="24eb0-108">Vous pouvez filtrer selon une valeur unique ou une plage.</span><span class="sxs-lookup"><span data-stu-id="24eb0-108">You could filter on a single value or a range.</span></span> <span data-ttu-id="24eb0-109">Lorsque vous définissez une plage, vérifiez que la syntaxe appropriée est utilisée.</span><span class="sxs-lookup"><span data-stu-id="24eb0-109">When defining a range, make sure the correct syntax is used.</span></span> <span data-ttu-id="24eb0-110">Les valeurs doivent être séparées par deux points (..).</span><span class="sxs-lookup"><span data-stu-id="24eb0-110">The values should be separated by a double period (..).</span></span>  
4. <span data-ttu-id="24eb0-111">Cliquez sur l'onglet Jointures pour ajouter des tables supplémentaires à partir desquelles appliquer un filtre.</span><span class="sxs-lookup"><span data-stu-id="24eb0-111">Click the Joins tab to add additional tables from which to filter.</span></span>
5. <span data-ttu-id="24eb0-112">Dans l'arborescence, sélectionnez Tables\Entrée de journal des opérations diverses.</span><span class="sxs-lookup"><span data-stu-id="24eb0-112">In the tree, select 'Tables\General journal entry'.</span></span>
6. <span data-ttu-id="24eb0-113">Cliquez sur Ajouter une jointure de table.</span><span class="sxs-lookup"><span data-stu-id="24eb0-113">Click Add table join.</span></span>
7. <span data-ttu-id="24eb0-114">Cliquez sur Annuler si vous décidez de ne pas ajouter une table supplémentaire.</span><span class="sxs-lookup"><span data-stu-id="24eb0-114">Click Cancel if you decide not to add an additional table.</span></span>
8. <span data-ttu-id="24eb0-115">Cliquez sur l'onglet Plage.</span><span class="sxs-lookup"><span data-stu-id="24eb0-115">Click the Range tab.</span></span>
9. <span data-ttu-id="24eb0-116">Cliquez sur OK pour exécuter la requête.</span><span class="sxs-lookup"><span data-stu-id="24eb0-116">Click OK to run the query.</span></span>
10. <span data-ttu-id="24eb0-117">Cliquez sur Origine de la transaction.</span><span class="sxs-lookup"><span data-stu-id="24eb0-117">Click Transaction origin.</span></span>
    * <span data-ttu-id="24eb0-118">Divers boutons relatifs à la grille peuvent être utilisés pour rechercher des informations supplémentaires sur l'enregistrement sélectionné du N° document.</span><span class="sxs-lookup"><span data-stu-id="24eb0-118">Various buttons about the grid can be used to research additional information about the selected record of the voucher.</span></span> <span data-ttu-id="24eb0-119">Il se peut que certains boutons ne soient pas disponibles, selon le type de transaction et les caractéristiques de la transaction.</span><span class="sxs-lookup"><span data-stu-id="24eb0-119">Some buttons may not be available, depending on the type of transaction and characteristics of the transaction.</span></span>  
11. <span data-ttu-id="24eb0-120">Fermez la page.</span><span class="sxs-lookup"><span data-stu-id="24eb0-120">Close the page.</span></span>
12. <span data-ttu-id="24eb0-121">Cliquez sur Document d'origine.</span><span class="sxs-lookup"><span data-stu-id="24eb0-121">Click Original document.</span></span>
13. <span data-ttu-id="24eb0-122">Fermez la page.</span><span class="sxs-lookup"><span data-stu-id="24eb0-122">Close the page.</span></span>
