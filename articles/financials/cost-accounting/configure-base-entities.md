# <a name="configure-base-entities"></a>Configurer des entités de base 

[!include[banner](../includes/banner.md)]


Le Contrôle de gestion est un module indépendant qui nécessite la configuration correcte des entités de base suivantes dans Dynamics 365 for Finance and Operations, Enterprise Edition.

-  Devise
-  Taux de change
-  Calendrier fiscal  

Avant d'utiliser le module Contrôle de gestion, vous devez avoir une devise, un taux de change et un calendrier fiscal défini pour votre société. 

Pour plus d'informations sur ces entités, suivez la formation eLearning [Module 2 Paramétrage des taux de change et devises](https://mbspartner.microsoft.com/AX/CourseModules/1215) (requiert un compte CustomerSource).
