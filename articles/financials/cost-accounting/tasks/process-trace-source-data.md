# <a name="process-and-trace-source-data"></a>Traiter et effectuer le suivi des données source

[!include [task guide banner](../../includes/task-guide-banner.md)]

Le traitement des données est exécuté par tâche. Pour chaque tâche et fournisseur de données, un journal est créé pour indiquer que le processus a été exécuté et que les entrées ont été traitées dans la tâche actuelle. Utilisez cette procédure pour paramétrer une source de données et suivre l'origine d'une entrée de coût spécifique. Cet enregistrement utilise la société fictive USP2. Avant d'effectuer cette tâche, veillez à lire les guides de tâche suivants : « Créer une comptabilité de contrôle de gestion, « Définir les unités de contrôle des coûts « et « Gérer la source de données pour la comptabilité de contrôle de gestion ».

1. Accédez à Contrôle de gestion > Paramétrage de la comptabilité > Comptabilités de contrôle de gestion.
2. Dans la liste, recherchez et sélectionnez l'enregistrement souhaité.
    * Sélectionnez la comptabilité de contrôle de gestion que vous avez créée précédemment.  
3. Cliquez sur Versions réelles.
4. Dans le volet Actions, cliquez sur Traitement des données source.
5. Cliquez sur Journaux de transfert d'écritures de comptabilité.
6. Dans la liste, recherchez et sélectionnez l'enregistrement souhaité.
7. Cliquez sur Entrées de journal.
8. Dans la liste, marquez la ligne sélectionnée.
9. Cliquez sur Entrées de coût.
10. Cliquez sur Entrée source.
11. Dans le volet Actions, cliquez sur Traitement des données source.
12. Cliquez sur Comptabilité.
13. Dans le champ Période de calendrier fiscal, entrez ou sélectionnez une valeur.
    * Pour cet exemple, sélectionnez Période fiscale 9 2017.  
14. Cliquez sur OK.
