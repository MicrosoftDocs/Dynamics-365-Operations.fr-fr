# <a name="configure-cost-control-workspace-parameters"></a><span data-ttu-id="e0a9c-103">Configurer les paramètres de l'espace de travail de contrôle des coûts</span><span class="sxs-lookup"><span data-stu-id="e0a9c-103">Configure cost control workspace parameters</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="e0a9c-104">Cette procédure permet de configurer l'espace de travail Contrôle des coûts afin que les gestionnaires à différents niveaux d'une organisation puissent obtenir des informations sur leurs objets de coût, tels que les centres de coût et les groupes de produits.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-104">Use this procedure to configure the Cost control workspace so that managers at various levels in an organization can gain insight into their cost objects, such as cost centers and product groups.</span></span> <span data-ttu-id="e0a9c-105">La société fictive USP2 a été utilisée pour créer cet enregistrement.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-105">The USP2 demo company was used to create this recording.</span></span>

1. <span data-ttu-id="e0a9c-106">Accédez à Contrôle de gestion > Paramétrage > Configuration de l'espace de travail de contrôle des coûts.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-106">Go to Cost accounting > Setup > Cost control workspace configuration.</span></span>
2. <span data-ttu-id="e0a9c-107">Cliquez sur Nouveau.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-107">Click New.</span></span>
3. <span data-ttu-id="e0a9c-108">Tapez une valeur dans le champ Nom.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-108">In the Name field, type a value.</span></span>
4. <span data-ttu-id="e0a9c-109">Dans le champ Description, entrez une valeur.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-109">In the Description field, type a value.</span></span>
5. <span data-ttu-id="e0a9c-110">Sélectionnez Oui dans le champ Publié.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-110">Select Yes in the Published field.</span></span>
    * <span data-ttu-id="e0a9c-111">Si vous définissez cette option sur Oui, les utilisateurs affectés à l'un des rôles suivants peuvent afficher l'état dans l'espace de travail Contrôle des coûts : gestionnaire de contrôle de gestion, comptable, commis contrôleur de gestion ou contrôleur d'objet de coût.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-111">If you set this option to Yes, users who are assigned one of these roles can see the report in the Cost control workspace: cost accounting manager, cost accountant, cost accountant clerk, or cost object controller.</span></span> <span data-ttu-id="e0a9c-112">Si vous définissez cette option sur Non, seuls les utilisateurs affectés à l'un des rôles suivants peuvent afficher l'état dans l'espace de travail Contrôle des coûts : gestionnaire de contrôle de gestion, comptable ou commis contrôleur de gestion.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-112">If you set this option to No, only users who are assigned one of these roles can see the report in the Cost control workspace: cost accounting manager, cost accountant, or cost accountant clerk.</span></span>  
6. <span data-ttu-id="e0a9c-113">Développez la section Filtrage des données.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-113">Expand the Data filtering section.</span></span>
7. <span data-ttu-id="e0a9c-114">Dans le champ Unité de contrôle des coûts, entrez ou sélectionnez une valeur.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-114">In the Cost control unit field, enter or select a value.</span></span>
8. <span data-ttu-id="e0a9c-115">Dans le champ Version originale du budget, entrez ou sélectionnez une valeur.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-115">In the Budget original version field, enter or select a value.</span></span>
9. <span data-ttu-id="e0a9c-116">Dans le champ Hiérarchie des dimensions d'élément de coût, entrez ou sélectionnez une valeur.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-116">In the Cost element dimension hierarchy field, enter or select a value.</span></span>
10. <span data-ttu-id="e0a9c-117">Dans le champ Hiérarchie des dimensions d'objet de coût, entrez ou sélectionnez une valeur.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-117">In the Cost object dimension hierarchy field, enter or select a value.</span></span>
11. <span data-ttu-id="e0a9c-118">Développez la section Affecter les enregistrements de calcul.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-118">Expand the Assign calculation records section.</span></span>
12. <span data-ttu-id="e0a9c-119">Cliquez sur Nouveau.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-119">Click New.</span></span>
13. <span data-ttu-id="e0a9c-120">Dans la liste, marquer la ligne sélectionnée.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-120">In the list, mark the selected row.</span></span>
14. <span data-ttu-id="e0a9c-121">Dans le champ Période de calendrier fiscal, entrez ou sélectionnez une valeur.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-121">In the Fiscal calendar period field, enter or select a value.</span></span>
15. <span data-ttu-id="e0a9c-122">Dans le champ Version réelle, entrez ou sélectionnez une valeur.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-122">In the Actual version field, enter or select a value.</span></span>
16. <span data-ttu-id="e0a9c-123">Développez la section Périodes fiscales par colonne.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-123">Expand the Fiscal periods per column section.</span></span>
17. <span data-ttu-id="e0a9c-124">Sélectionnez Oui dans le champ Période actuelle.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-124">Select Yes in the Current period field.</span></span>
18. <span data-ttu-id="e0a9c-125">Développez la section Colonnes à afficher pour les coûts.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-125">Expand the Columns to display for costs section.</span></span>
19. <span data-ttu-id="e0a9c-126">Sélectionnez Oui dans le champ Coût fixe.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-126">Select Yes in the Fixed cost field.</span></span>
20. <span data-ttu-id="e0a9c-127">Sélectionnez Oui dans le champ Coût variable.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-127">Select Yes in the Variable cost field.</span></span>
21. <span data-ttu-id="e0a9c-128">Sélectionnez Oui dans le champ Coût total.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-128">Select Yes in the Total cost field.</span></span>
22. <span data-ttu-id="e0a9c-129">Cliquez sur Enregistrer.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-129">Click Save.</span></span>
23. <span data-ttu-id="e0a9c-130">Fermez la page.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-130">Close the page.</span></span>
24. <span data-ttu-id="e0a9c-131">Accédez à Contrôle de gestion > Espaces de travail > Contrôle des coûts.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-131">Go to Cost accounting > Workspaces > Cost control.</span></span>
25. <span data-ttu-id="e0a9c-132">Sélectionnez un relevé pour afficher les coûts fixes, variables, totaux et non classés pour les périodes fiscales sélectionnées.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-132">Select a statement to see fixed, variable, total, and unclassified costs for the selected fiscal periods.</span></span>
26. <span data-ttu-id="e0a9c-133">Dans le champ Période de calendrier fiscal, entrez ou sélectionnez une valeur.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-133">In the Fiscal calendar period field, enter or select a value.</span></span>
27. <span data-ttu-id="e0a9c-134">Dans le champ Nœud de hiérarchie des dimensions d'objet de coût, entrez ou sélectionnez une valeur.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-134">In the Cost object dimension hierarchy node field, enter or select a value.</span></span>
    * <span data-ttu-id="e0a9c-135">Après avoir sélectionné une hiérarchie Dimension d'objet de coût, développez la hiérarchie Dimension d'élément de coût pour afficher les valeurs de coût souhaitées.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-135">After you've selected a Cost object dimension hierarchy, expand the Cost element dimension hierarchy to see the desired cost values.</span></span> <span data-ttu-id="e0a9c-136">Par exemple, vous pouvez développer la hiérarchie des frais généraux de fabrication pour afficher la valeur.</span><span class="sxs-lookup"><span data-stu-id="e0a9c-136">For example, you can expand the hierarchy to Manufacturing overhead to see the value.</span></span>
