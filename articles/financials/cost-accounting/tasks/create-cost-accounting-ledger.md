[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="98692-104">Une comptabilité de contrôle de gestion représente l'unité organisationnelle globale.</span><span class="sxs-lookup"><span data-stu-id="98692-104">A cost accounting ledger represents the overall reporting unit.</span></span> <span data-ttu-id="98692-105">Elle est définie par une dimension d'élément de coût, une dimension statistique, un calendrier fiscal et une devise.</span><span class="sxs-lookup"><span data-stu-id="98692-105">It is defined by a cost element dimension, statistical dimension, fiscal calendar, and currency.</span></span> <span data-ttu-id="98692-106">Elle est liée au concept d'entités juridiques.</span><span class="sxs-lookup"><span data-stu-id="98692-106">It is agnostic to the concept of legal entities.</span></span> <span data-ttu-id="98692-107">Une entité juridique et ses données peuvent être associées à plusieurs comptabilités de contrôle de gestion.</span><span class="sxs-lookup"><span data-stu-id="98692-107">A legal entity and its data can be associated with many cost accounting ledgers.</span></span> <span data-ttu-id="98692-108">Cet enregistrement utilise la société fictive USP2.</span><span class="sxs-lookup"><span data-stu-id="98692-108">This recording uses the USP2 demo data company.</span></span>

1. <span data-ttu-id="98692-109">Accédez à Contrôle de gestion > Paramétrage de la comptabilité > Comptabilités de contrôle de gestion.</span><span class="sxs-lookup"><span data-stu-id="98692-109">Go to Cost accounting > Ledger setup > Cost accounting ledgers.</span></span>
2. <span data-ttu-id="98692-110">Cliquez sur Nouveau.</span><span class="sxs-lookup"><span data-stu-id="98692-110">Click New.</span></span>
3. <span data-ttu-id="98692-111">Tapez une valeur dans le champ Nom.</span><span class="sxs-lookup"><span data-stu-id="98692-111">In the Name field, type a value.</span></span>
4. <span data-ttu-id="98692-112">Dans le champ Dimension d'élément de coût, entrez ou sélectionnez une valeur.</span><span class="sxs-lookup"><span data-stu-id="98692-112">In the Cost element dimension field, enter or select a value.</span></span>
5. <span data-ttu-id="98692-113">Dans le champ Calendrier fiscal, entrez ou sélectionnez une valeur.</span><span class="sxs-lookup"><span data-stu-id="98692-113">In the Fiscal calendar field, enter or select a value.</span></span>
6. <span data-ttu-id="98692-114">Dans le champ Devise comptable, entrez ou sélectionnez une valeur.</span><span class="sxs-lookup"><span data-stu-id="98692-114">In the Accounting currency field, enter or select a value.</span></span>
7. <span data-ttu-id="98692-115">Dans le champ Type de taux de change, entrez ou sélectionnez une valeur.</span><span class="sxs-lookup"><span data-stu-id="98692-115">In the Exchange rate type field, enter or select a value.</span></span>
8. <span data-ttu-id="98692-116">Dans le champ Dimension statistique, entrez ou sélectionnez une valeur.</span><span class="sxs-lookup"><span data-stu-id="98692-116">In the Statistical dimension field, enter or select a value.</span></span>
9. <span data-ttu-id="98692-117">Cliquez sur Enregistrer.</span><span class="sxs-lookup"><span data-stu-id="98692-117">Click Save.</span></span>
