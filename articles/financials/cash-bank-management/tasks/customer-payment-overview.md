# <a name="customer-payment-overview"></a><span data-ttu-id="77a3f-103">Vue d'ensemble des paiements client</span><span class="sxs-lookup"><span data-stu-id="77a3f-103">Customer payment overview</span></span>

2. <span data-ttu-id="77a3f-107">Cliquez sur Nouveau.</span><span class="sxs-lookup"><span data-stu-id="77a3f-107">Click New.</span></span>
3. <span data-ttu-id="77a3f-108">Sélectionnez le journal des paiements dans lequel les paiements client seront enregistrés.</span><span class="sxs-lookup"><span data-stu-id="77a3f-108">Select the payment journal which the customer payments will be saved.</span></span>
4. <span data-ttu-id="77a3f-109">Sélectionnez le journal ou entrez-le manuellement.</span><span class="sxs-lookup"><span data-stu-id="77a3f-109">Select or manually enter the journal.</span></span>
5. <span data-ttu-id="77a3f-110">Cliquez sur Entrer les paiements client.</span><span class="sxs-lookup"><span data-stu-id="77a3f-110">Click Enter customer payments.</span></span>

6. <span data-ttu-id="77a3f-113">Entrez le client qui a envoyé le paiement.</span><span class="sxs-lookup"><span data-stu-id="77a3f-113">Enter the customer from whom you received the payment.</span></span>

7. <span data-ttu-id="77a3f-117">Dans le champ Référence de paiement, entrez une référence de paiement.</span><span class="sxs-lookup"><span data-stu-id="77a3f-117">In the Payment reference field, enter a payment reference.</span></span>

8. <span data-ttu-id="77a3f-120">Indiquez si le paiement est inclus dans un bordereau de remise.</span><span class="sxs-lookup"><span data-stu-id="77a3f-120">Select whether the payment will be included on a deposit slip.</span></span> 
9. <span data-ttu-id="77a3f-121">Entrez le montant du paiement client.</span><span class="sxs-lookup"><span data-stu-id="77a3f-121">Enter the amount of the customer payment.</span></span>
    * <span data-ttu-id="77a3f-122">Le montant du paiement ne sera pas transféré.</span><span class="sxs-lookup"><span data-stu-id="77a3f-122">The payment amount will not default.</span></span> <span data-ttu-id="77a3f-123">Il doit être entré manuellement.</span><span class="sxs-lookup"><span data-stu-id="77a3f-123">It must be manually entered.</span></span>  
10. <span data-ttu-id="77a3f-124">Marquez les factures qui ont été payées par le client.</span><span class="sxs-lookup"><span data-stu-id="77a3f-124">Mark the invoices that were paid by the customer.</span></span>

11. <span data-ttu-id="77a3f-128">Entrez le montant du paiement qui doit être réglé sur les factures marquées.</span><span class="sxs-lookup"><span data-stu-id="77a3f-128">Enter the amount of the payment that should be settled to the marked invoice.</span></span> 
    * <span data-ttu-id="77a3f-129">Ce champ peut être utilisé lorsque le paiement concerne un paiement partiel.</span><span class="sxs-lookup"><span data-stu-id="77a3f-129">This field can be used when the payment is for a partial payment.</span></span> <span data-ttu-id="77a3f-130">Si vous n'entrez pas de montant, le montant à régler est automatiquement déterminé pour vous.</span><span class="sxs-lookup"><span data-stu-id="77a3f-130">If you don't enter an amount, the amount to settle will automatically be determined for you.</span></span>  
12. <span data-ttu-id="77a3f-131">Cliquez sur Enregistrer dans le journal.</span><span class="sxs-lookup"><span data-stu-id="77a3f-131">Click Save in journal.</span></span>
    * <span data-ttu-id="77a3f-132">Avant d'enregistrer le paiement dans le journal, assurez-vous que le compte de contrepartie est défini.</span><span class="sxs-lookup"><span data-stu-id="77a3f-132">Before you save the payment to the journal, make sure that the offset account is defined.</span></span> <span data-ttu-id="77a3f-133">Avec Sauvegarde dans le journal, vous sauvegarderez le paiement et le déplacerez dans le journal.</span><span class="sxs-lookup"><span data-stu-id="77a3f-133">Using Save in journal will save the payment and move it to the journal.</span></span> <span data-ttu-id="77a3f-134">Vous pouvez ensuite continuer et entrer le paiement suivant.</span><span class="sxs-lookup"><span data-stu-id="77a3f-134">You can then continue entering the next payment.</span></span>  
13. <span data-ttu-id="77a3f-135">Fermez la page.</span><span class="sxs-lookup"><span data-stu-id="77a3f-135">Close the page.</span></span>
14. <span data-ttu-id="77a3f-136">Cliquez sur Lignes.</span><span class="sxs-lookup"><span data-stu-id="77a3f-136">Click Lines.</span></span>
    * <span data-ttu-id="77a3f-137">Lors de l'ouverture de Lignes, vous verrez tous les paiements consignés sur la page Entrer les paiements client et enregistrés dans le journal.</span><span class="sxs-lookup"><span data-stu-id="77a3f-137">When opening Lines, you will see any payments you recorded on the Enter customer payments page and saved into the journal.</span></span> <span data-ttu-id="77a3f-138">Cette page permet également d'entrer de nouveaux paiements client ou de modifier un paiement client existant avant sa validation.</span><span class="sxs-lookup"><span data-stu-id="77a3f-138">You can also use this page to enter new customer payments, or edit existing customer payment before they are posted.</span></span>  
15. <span data-ttu-id="77a3f-139">Cliquez sur Nouveau pour créer un autre paiement.</span><span class="sxs-lookup"><span data-stu-id="77a3f-139">Click New to create another payment.</span></span> 
16. <span data-ttu-id="77a3f-140">Sélectionnez le client qui a envoyé le paiement.</span><span class="sxs-lookup"><span data-stu-id="77a3f-140">Select the customer from whom you received the payment.</span></span>
    * <span data-ttu-id="77a3f-141">Si vous ne connaissez pas le client, mais avez connaissance d'une facture payée par le paiement, utilisez le champ Facture pour entrer ou sélectionner manuellement la facture.</span><span class="sxs-lookup"><span data-stu-id="77a3f-141">If you don't know the customer but know an invoice paid by the payment, use the Invoice field to manually enter or select the invoice.</span></span> <span data-ttu-id="77a3f-142">Le client sera transféré une fois la facture sélectionnée.</span><span class="sxs-lookup"><span data-stu-id="77a3f-142">The customer will default after the invoice is selected.</span></span>

    * <span data-ttu-id="77a3f-144">Vous n'avez besoin de régler le paiement d'aucune facture.</span><span class="sxs-lookup"><span data-stu-id="77a3f-144">You are not required to settle the payment to any invoices.</span></span> <span data-ttu-id="77a3f-145">S'il s'agit d'un acompte ou si vous ne savez pas quelle facture a été payée, vous pouvez entrer et valider le paiement.</span><span class="sxs-lookup"><span data-stu-id="77a3f-145">If this is a prepayment or if you don't know what invoice was paid, you can enter and post the payment.</span></span> <span data-ttu-id="77a3f-146">Le paiement peut être réglé ultérieurement sur une facture.</span><span class="sxs-lookup"><span data-stu-id="77a3f-146">The payment can be settled to an invoice at a later point.</span></span>  
18. <span data-ttu-id="77a3f-147">Marquez les factures réglées à l'aide du paiement.</span><span class="sxs-lookup"><span data-stu-id="77a3f-147">Mark the invoices paid by the payment.</span></span> 
19. <span data-ttu-id="77a3f-148">Entrez le montant du paiement qui sera réglé sur les factures marquées.</span><span class="sxs-lookup"><span data-stu-id="77a3f-148">Enter the amount of the payment that will be settled to the invoice.</span></span>
20. <span data-ttu-id="77a3f-149">Cliquez sur OK.</span><span class="sxs-lookup"><span data-stu-id="77a3f-149">Click OK.</span></span>
21. <span data-ttu-id="77a3f-150">Dans le champ Référence de paiement, entrez une référence de paiement.</span><span class="sxs-lookup"><span data-stu-id="77a3f-150">In the Payment reference field, Enter a payment reference.</span></span> <span data-ttu-id="77a3f-151">.</span><span class="sxs-lookup"><span data-stu-id="77a3f-151">.</span></span>
    * <span data-ttu-id="77a3f-152">La référence de paiement n'est requise que si vous choisissez d'inclure le paiement sur un bordereau de remise.</span><span class="sxs-lookup"><span data-stu-id="77a3f-152">The payment reference is only required if you mark to include the payment on a deposit slip.</span></span>  
22. <span data-ttu-id="77a3f-153">Validez les paiements client.</span><span class="sxs-lookup"><span data-stu-id="77a3f-153">Post the customer payments.</span></span>
