[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="315ee-104">La classification de contrat d'achat vous permet de contrôler les informations administratives disponibles dans les contrats d'achat.</span><span class="sxs-lookup"><span data-stu-id="315ee-104">The purchase agreement classification allows you to control the administrative information that is available on purchase agreements.</span></span> 

<span data-ttu-id="315ee-105">Cette procédure a été créée pour le secteur public français, en utilisant les données de la société fictive PSUS dans la partition du secteur public.</span><span class="sxs-lookup"><span data-stu-id="315ee-105">This procedure was created for the French public sector using the PSUS demo company data in the public sector partition.</span></span>

1. <span data-ttu-id="315ee-106">Accédez à Approvisionnements > Paramétrage > ..</span><span class="sxs-lookup"><span data-stu-id="315ee-106">Go to Procurement and sourcing > Setup > ..</span></span> <span data-ttu-id="315ee-107">> Classifications du contrat d'achat.</span><span class="sxs-lookup"><span data-stu-id="315ee-107">> Purchase agreement classification.</span></span>
2. <span data-ttu-id="315ee-108">Cliquez sur Nouveau.</span><span class="sxs-lookup"><span data-stu-id="315ee-108">Click New.</span></span>
3. <span data-ttu-id="315ee-109">Tapez une valeur dans le champ Nom.</span><span class="sxs-lookup"><span data-stu-id="315ee-109">In the Name field, type a value.</span></span>
4. <span data-ttu-id="315ee-110">Dans le champ Description, entrez une valeur.</span><span class="sxs-lookup"><span data-stu-id="315ee-110">In the Description field, type a value.</span></span>
5. <span data-ttu-id="315ee-111">Facultatif : sélectionnez l'option Sous-traitants pour inclure des informations sur les sous-traitants.</span><span class="sxs-lookup"><span data-stu-id="315ee-111">Optional: Select the Subcontractors option to include information about subcontractors.</span></span>
    * <span data-ttu-id="315ee-112">Un contrat d'achat peut avoir un fournisseur principal, des fournisseurs secondaires et des sous-traitants.</span><span class="sxs-lookup"><span data-stu-id="315ee-112">A purchase agreement can have a prime contractor, co-contractors, and subcontractors.</span></span> <span data-ttu-id="315ee-113">Vous pouvez affecter le fournisseur principal au contrat d'achat parent et un fournisseur secondaire ou un sous-traitant à des contrats enfants spécifiques.</span><span class="sxs-lookup"><span data-stu-id="315ee-113">You can assign the prime contractor to the parent purchase agreement and a co-contractor or subcontractor to specific child agreements.</span></span>  
6. <span data-ttu-id="315ee-114">Facultatif : sélectionnez l'option Certifications pour inclure des informations sur les certifications des fournisseurs.</span><span class="sxs-lookup"><span data-stu-id="315ee-114">Optional: Select the Certifications option to include information about certifications for vendors.</span></span>
    * <span data-ttu-id="315ee-115">Les informations de certification permettent de générer un état que vous pouvez utiliser pour vérifier que les fournisseurs répondent bien aux exigences en matière de certification.</span><span class="sxs-lookup"><span data-stu-id="315ee-115">Certification information can be used to generate a report that lets you monitor vendor compliance with certification requirements.</span></span>

9. <span data-ttu-id="315ee-118">Cliquez sur Enregistrer.</span><span class="sxs-lookup"><span data-stu-id="315ee-118">Click Save.</span></span>
