# <a name="set-up-a-fund-in-the-public-sector"></a><span data-ttu-id="d3752-103">Paramétrer un fonds dans le secteur public</span><span class="sxs-lookup"><span data-stu-id="d3752-103">Set up a fund in the public sector</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="d3752-104">Vous devez créer des types de fonds avant de pouvoir paramétrer des fonds.</span><span class="sxs-lookup"><span data-stu-id="d3752-104">Fund types must be created before you set up funds.</span></span> <span data-ttu-id="d3752-105">Chaque fonds doit posséder un nom et un numéro unique ; un type et une classe de fonds doivent lui être affectés.</span><span class="sxs-lookup"><span data-stu-id="d3752-105">Each fund must have a name and a unique number, and must be assigned a fund type and fund class.</span></span> <span data-ttu-id="d3752-106">Cette procédure a été créée à l'aide des données de la société fictive PSUS dans la partition du secteur public.</span><span class="sxs-lookup"><span data-stu-id="d3752-106">This procedure was created using the PSUS demo company data in the public sector partition.</span></span>

1. <span data-ttu-id="d3752-107">Accédez à Comptabilité > Plan de comptes > Fonds > Fonds.</span><span class="sxs-lookup"><span data-stu-id="d3752-107">Go to General ledger > Chart of accounts > Funds > Funds.</span></span>
2. <span data-ttu-id="d3752-108">Cliquez sur Nouveau.</span><span class="sxs-lookup"><span data-stu-id="d3752-108">Click New.</span></span>
3. <span data-ttu-id="d3752-109">Dans le champ Numéro de fonds, tapez une valeur.</span><span class="sxs-lookup"><span data-stu-id="d3752-109">In the Fund number field, type a value.</span></span>
4. <span data-ttu-id="d3752-110">Dans le champ Nom du fonds, tapez une valeur.</span><span class="sxs-lookup"><span data-stu-id="d3752-110">In the Fund name field, type a value.</span></span>
5. <span data-ttu-id="d3752-111">Dans le champ Type de fonds, cliquez sur le bouton de liste déroulante pour ouvrir la recherche.</span><span class="sxs-lookup"><span data-stu-id="d3752-111">In the Fund type field, click the drop-down button to open the lookup.</span></span>
6. <span data-ttu-id="d3752-112">Dans la liste, sélectionnez le type de ce fonds.</span><span class="sxs-lookup"><span data-stu-id="d3752-112">In the list, select the fund type for this fund.</span></span>
7. <span data-ttu-id="d3752-113">Dans le champ Classe de fonds, sélectionnez une option.</span><span class="sxs-lookup"><span data-stu-id="d3752-113">In the Fund class field, select an option.</span></span>
8. <span data-ttu-id="d3752-114">Définissez l'option Fonds ne servant pas à la génération d'états à Oui ou à Non.</span><span class="sxs-lookup"><span data-stu-id="d3752-114">Set the Non-reporting fund option to Yes or No.</span></span>
9. <span data-ttu-id="d3752-115">Définissez l'option Fonds principal à Oui ou à Non.</span><span class="sxs-lookup"><span data-stu-id="d3752-115">Set the Major fund option to Yes or No.</span></span>
10. <span data-ttu-id="d3752-116">Définir l'option Ignorer l'option de fin d'exercice sélectionnée à Oui ou à Non.</span><span class="sxs-lookup"><span data-stu-id="d3752-116">Set the Override selected year-end option to Yes or No.</span></span>
11. <span data-ttu-id="d3752-117">Si vous définissez l'option Ignorer l'option de fin d'exercice sélectionnée à Oui, sélectionnez une option de fin d'exercice.</span><span class="sxs-lookup"><span data-stu-id="d3752-117">If you set the Override selected year-end option to Yes, select a year-end option.</span></span>
12. <span data-ttu-id="d3752-118">Cliquez sur Enregistrer.</span><span class="sxs-lookup"><span data-stu-id="d3752-118">Click Save.</span></span>
