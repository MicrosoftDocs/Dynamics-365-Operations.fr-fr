[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="65655-104">Les engagements sont des documents source de contrôle budgétaire utilisés par les entités du secteur public en France.</span><span class="sxs-lookup"><span data-stu-id="65655-104">Commitments are budget control source documents used by public sector entities in France.</span></span> <span data-ttu-id="65655-105">Ce guide de tâche a été créée en utilisant les données de la société fictive PSUS dans la partition du secteur public.</span><span class="sxs-lookup"><span data-stu-id="65655-105">This task guide was created using the PSUS demo company data in the public sector partition.</span></span>

1. <span data-ttu-id="65655-106">Accédez à Budgétisation > Engagements.</span><span class="sxs-lookup"><span data-stu-id="65655-106">Go to Budgeting > Commitments.</span></span>
2. <span data-ttu-id="65655-107">Cliquez sur Nouveau.</span><span class="sxs-lookup"><span data-stu-id="65655-107">Click New.</span></span>
3. <span data-ttu-id="65655-108">Dans le champ Nom, tapez un nom qui permettra d'identifier cet engagement.</span><span class="sxs-lookup"><span data-stu-id="65655-108">In the Name field, type a name that will identify this commitment.</span></span>
4. <span data-ttu-id="65655-109">Cliquez sur Ajouter une ligne.</span><span class="sxs-lookup"><span data-stu-id="65655-109">Click Add line.</span></span>
5. <span data-ttu-id="65655-110">Dans le champ Compte général, sélectionnez les fonds et le compte principal.</span><span class="sxs-lookup"><span data-stu-id="65655-110">In the Ledger account field, select the Fund and the Main account.</span></span>
6. <span data-ttu-id="65655-111">Dans le champ Montant, entrez un nombre.</span><span class="sxs-lookup"><span data-stu-id="65655-111">In the Amount field, enter a number.</span></span>
    * <span data-ttu-id="65655-112">Entrez une ligne de contrat d'achat si l'engagement concerne un contrat d'achat spécifique.</span><span class="sxs-lookup"><span data-stu-id="65655-112">Enter a purchase agreement line if the commitment is for a specific purchase agreement.</span></span> <span data-ttu-id="65655-113">Ajoutez des lignes supplémentaires si nécessaire.</span><span class="sxs-lookup"><span data-stu-id="65655-113">Add additional lines as needed.</span></span>  
7. <span data-ttu-id="65655-114">Dans le volet Actions, cliquez sur Options.</span><span class="sxs-lookup"><span data-stu-id="65655-114">On the Action Pane, click Options.</span></span>
8. <span data-ttu-id="65655-115">Cliquez sur Changer de vue.</span><span class="sxs-lookup"><span data-stu-id="65655-115">Click Change view.</span></span>
9. <span data-ttu-id="65655-116">Cliquez sur Vue de l'en-tête.</span><span class="sxs-lookup"><span data-stu-id="65655-116">Click Header view.</span></span>
10. <span data-ttu-id="65655-117">Dans le champ Compte fournisseur, cliquez sur le bouton de liste déroulante pour ouvrir la recherche.</span><span class="sxs-lookup"><span data-stu-id="65655-117">In the Vendor account field, click the drop-down button to open the lookup.</span></span>

13. <span data-ttu-id="65655-120">Cliquez sur Changer de vue.</span><span class="sxs-lookup"><span data-stu-id="65655-120">Click Change view.</span></span>
14. <span data-ttu-id="65655-121">Cliquez sur Affichage des lignes.</span><span class="sxs-lookup"><span data-stu-id="65655-121">Click Line view.</span></span>
15. <span data-ttu-id="65655-122">Cliquez sur Enregistrer.</span><span class="sxs-lookup"><span data-stu-id="65655-122">Click Save.</span></span>
