# <a name="create-a-credit-note-on-project-invoices"></a><span data-ttu-id="c9132-103">Créer un avoir sur des factures de projet</span><span class="sxs-lookup"><span data-stu-id="c9132-103">Create a credit note on project invoices</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

1. <span data-ttu-id="c9132-104">Accédez à Gestion et comptabilité des projets > Projets > Tous les projets.</span><span class="sxs-lookup"><span data-stu-id="c9132-104">Go to Project management and accounting > Projects > All projects.</span></span> 
2. <span data-ttu-id="c9132-105">Dans la liste, recherchez et sélectionnez l'enregistrement souhaité.</span><span class="sxs-lookup"><span data-stu-id="c9132-105">In the list, find and select the desired record.</span></span> 
3. <span data-ttu-id="c9132-106">Sélectionnez l'ID projet utilisé pour l'avoir.</span><span class="sxs-lookup"><span data-stu-id="c9132-106">Select the Project ID that is used for the Credit Note.</span></span> <span data-ttu-id="c9132-107">Dans la liste, cliquer sur le lien dans la ligne sélectionnée.</span><span class="sxs-lookup"><span data-stu-id="c9132-107">In the list, click the link in the selected row.</span></span> 
4. <span data-ttu-id="c9132-108">Dans le volet Actions, cliquez sur Gérer.</span><span class="sxs-lookup"><span data-stu-id="c9132-108">On the Action Pane, click Manage.</span></span> 
5. <span data-ttu-id="c9132-109">Cliquez sur Journaux des factures.</span><span class="sxs-lookup"><span data-stu-id="c9132-109">Click Invoice journals.</span></span> 
6. <span data-ttu-id="c9132-110">Sélectionnez la transaction pour l'avoir.</span><span class="sxs-lookup"><span data-stu-id="c9132-110">Select the transaction for credit note.</span></span> <span data-ttu-id="c9132-111">Dans la liste, marquez la ligne sélectionnée.</span><span class="sxs-lookup"><span data-stu-id="c9132-111">In the list, mark the selected row.</span></span> 
7. <span data-ttu-id="c9132-112">Cliquez sur Fonctions.</span><span class="sxs-lookup"><span data-stu-id="c9132-112">Click Functions.</span></span> 
8. <span data-ttu-id="c9132-113">Cliquez sur Sélectionner pour l'avoir.</span><span class="sxs-lookup"><span data-stu-id="c9132-113">Click Select for credit note.</span></span> 
9. <span data-ttu-id="c9132-114">Sélectionnez la transaction pour l'avoir en activant la case à cocher.</span><span class="sxs-lookup"><span data-stu-id="c9132-114">Select the transaction for credit note by marking the check box.</span></span>
10. <span data-ttu-id="c9132-115">Cliquez sur OK.</span><span class="sxs-lookup"><span data-stu-id="c9132-115">Click OK.</span></span> 

<span data-ttu-id="c9132-116">Après cette étape, l'avoir est disponible pour la proposition de facture.</span><span class="sxs-lookup"><span data-stu-id="c9132-116">After this step, the Credit Note will be available for Invoice Proposal.</span></span>
