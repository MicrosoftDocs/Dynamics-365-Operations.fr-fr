# <a name="project-mobile-applications-overview"></a>Vue d'ensemble des applications mobiles du projet

[!include [banner](../includes/banner.md)]

Trois applications mobiles associées aux heures de projet sont disponibles pour Microsoft Dynamics 365 : Microsoft Dynamics 365 Project Timesheet, Entrée d'heure du projet, et Mes feuilles de temps/Feuilles de temps pour révision (optimisées pour mobile).

## <a name="project-timesheet-mobile-app"></a>Application mobile Project Timesheet

L'application mobile Project Timesheet de Dynamics 365 permet aux utilisateurs d'envoyer et d'approuver des feuilles de temps pour les projets sur leur appareil mobile. Cette application mobile permet d'exploiter la fonctionnalité de feuille de temps qui réside dans la zone Gestion de projet et comptabilité de Microsoft Dynamics 365 for Finance and Operations. lle renforce la productivité et le rendement des utilisateurs, et permet la saisie et l'approbation en temps opportun des feuilles de temps de projet.

## <a name="project-time-entry-workspace"></a>Espace de travail Entrée d'heure du projet

L'espace de travail Entrée d'heure du projet permet aux utilisateurs de réaliser la saisie et de gagner du temps par rapport à un projet. Toutefois, il ne permet pas aux utilisateurs d'envoyer ou d'approuver des feuilles de temps de projet. Il ne peut être utilisé que pour la saisie d'heures. Cet espace de travail mobile est destiné à être utilisé conjointement avec l'application mobile Microsoft Dynamics 365 for Unified Operations.

## <a name="my-timesheetstimesheets-for-my-review"></a>Mes feuilles de temps/Feuilles de temps pour révision

Mes feuilles de temps (optimisée pour mobile) et Feuilles de temps pour révision (optimisée pour mobile) constituent une solution qui permet aux employés de créer, envoyer et approuver des feuilles de temps de projet sur un appareil mobile. Les employés peuvent accéder à la fonctionnalité Finance and Operations via l'URL de l'application. Cette solution ne nécessite pas l'application mobile Dynamics 365 for Unified Operations.
