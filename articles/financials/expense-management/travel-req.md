# <a name="travel-requisitions"></a>Demandes de déplacement

[!include[banner](../includes/banner.md)]

Une *demande de déplacement* est un document source qui répertorie les dépenses qui seront engagées dans le cadre d'un déplacement. Une demande de voyage est soumise à la révision et peut ensuite être utilisée pour autoriser des dépenses.

Votre organisation peut demander à ce qu'une demande de déplacement soit envoyée avant que les employés engagent des dépenses facturées à l'organisation. Cette demande s'applique que les employés utilisent une carte de crédit professionnelle pour régler les frais, qu'ils dépensent des fonds reçus d'une avance de disponibilités ou qu'ils prennent en charge les dépenses, avant d'être remboursé par l'organisation ou non.

Les demandes de voyage et les stratégies peuvent vous aider dans le cadre du contrôle budgétaire. Par exemple, si votre organisation travaille sur un projet à prix fixe qui nécessite des déplacements, les frais associés des membres de l'équipe de projet doivent entrer dans le budget du projet. En demandant à ce que les frais de déplacement soient approuvés avant leur imputation, l'organisation s'assure que le projet ne dépasse pas le budget.

Vous pouvez paramétrer des stratégies pour les demandes de déplacement sur la page **Demande de déplacement**.
