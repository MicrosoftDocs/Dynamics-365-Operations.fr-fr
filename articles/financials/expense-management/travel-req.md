# <a name="travel-requisitions"></a><span data-ttu-id="6f1db-104">Demandes de déplacement</span><span class="sxs-lookup"><span data-stu-id="6f1db-104">Travel requisitions</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="6f1db-105">Une *demande de déplacement* est un document source qui répertorie les dépenses qui seront engagées dans le cadre d'un déplacement.</span><span class="sxs-lookup"><span data-stu-id="6f1db-105">A *travel requisition* is a source document that lists the expenses that will be incurred for the purpose of travel.</span></span> <span data-ttu-id="6f1db-106">Une demande de voyage est soumise à la révision et peut ensuite être utilisée pour autoriser des dépenses.</span><span class="sxs-lookup"><span data-stu-id="6f1db-106">A travel requisition is submitted for review and can then be used to authorize expenses.</span></span>

<span data-ttu-id="6f1db-107">Votre organisation peut demander à ce qu'une demande de déplacement soit envoyée avant que les employés engagent des dépenses facturées à l'organisation.</span><span class="sxs-lookup"><span data-stu-id="6f1db-107">Your organization might require that a travel requisition be submitted before employees incur any expense that is charged to the organization.</span></span> <span data-ttu-id="6f1db-108">Cette demande s'applique que les employés utilisent une carte de crédit professionnelle pour régler les frais, qu'ils dépensent des fonds reçus d'une avance de disponibilités ou qu'ils prennent en charge les dépenses, avant d'être remboursé par l'organisation ou non.</span><span class="sxs-lookup"><span data-stu-id="6f1db-108">This requirement applies, regardless of whether employees charge expenses to a corporate credit card, spend cash that they received from a cash advance, or incur out-of-pocket expenses that will be reimbursed by the organization.</span></span>

<span data-ttu-id="6f1db-109">Les demandes de voyage et les stratégies peuvent vous aider dans le cadre du contrôle budgétaire.</span><span class="sxs-lookup"><span data-stu-id="6f1db-109">Travel requisitions and policies can be used to help with budget control.</span></span> <span data-ttu-id="6f1db-110">Par exemple, si votre organisation travaille sur un projet à prix fixe qui nécessite des déplacements, les frais associés des membres de l'équipe de projet doivent entrer dans le budget du projet.</span><span class="sxs-lookup"><span data-stu-id="6f1db-110">For example, if your organization is working on a fixed-price project that requires travel, the travel expenses of the project's team members must fit within the budget for the project.</span></span> <span data-ttu-id="6f1db-111">En demandant à ce que les frais de déplacement soient approuvés avant leur imputation, l'organisation s'assure que le projet ne dépasse pas le budget.</span><span class="sxs-lookup"><span data-stu-id="6f1db-111">By requiring that travel expenses be approved before they are incurred, the organization can help make sure that the project remains within budget.</span></span>

<span data-ttu-id="6f1db-112">Vous pouvez paramétrer des stratégies pour les demandes de déplacement sur la page **Demande de déplacement**.</span><span class="sxs-lookup"><span data-stu-id="6f1db-112">You can set up policies for travel requisitions from the **Travel requisition** page.</span></span>
