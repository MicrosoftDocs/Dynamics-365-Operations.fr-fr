# <a name="set-up-rfm-analysis"></a><span data-ttu-id="5670e-103">Paramétrer l'analyse RFM</span><span class="sxs-lookup"><span data-stu-id="5670e-103">Set up RFM analysis</span></span>

[!include[banner](includes/banner.md)]


<span data-ttu-id="5670e-104">Cette rubrique décrit comment paramétrer une analyse RFM (récence, fréquence et valeur monétaire) de vos clients.</span><span class="sxs-lookup"><span data-stu-id="5670e-104">This topic explains how to set up a Recency, Frequency, and Monetary (RFM) analysis of your customers.</span></span>

<span data-ttu-id="5670e-105">L'analyse RFM (récence, fréquence et valeur monétaire) est un outil marketing que votre organisation peut utiliser pour évaluer les données générées par les achats client.</span><span class="sxs-lookup"><span data-stu-id="5670e-105">Recency, frequency, and monetary (RFM) analysis is a marketing tool that your organization can use to evaluate the data that is generated by customer purchases.</span></span> <span data-ttu-id="5670e-106">Après avoir paramétré l'analyse RFM, les clients se voient attribuer un score RFM calculé lorsqu'ils effectuent des achats.</span><span class="sxs-lookup"><span data-stu-id="5670e-106">After you set up RFM analysis, customers are assigned a calculated RFM score as they make purchases.</span></span> <span data-ttu-id="5670e-107">Le score RFM peut être une évaluation à trois chiffres ou un nombre entier, selon la façon dont votre organisation a configuré l'analyse RFM.</span><span class="sxs-lookup"><span data-stu-id="5670e-107">The RFM score can be a three-digit rating or an aggregate number, depending on how your organization has configured RFM analysis.</span></span> <span data-ttu-id="5670e-108">Voici comment l'évaluation fonctionne si votre organisation utilise une évaluation à trois chiffres du score :</span><span class="sxs-lookup"><span data-stu-id="5670e-108">Here’s how the rating works if your organization uses a three-digit rating for the score:</span></span>

- <span data-ttu-id="5670e-111">Le troisième chiffre est une évaluation de la valeur monétaire du client (montant que le client dépense lorsqu'il effectue des achats auprès de votre organisation).</span><span class="sxs-lookup"><span data-stu-id="5670e-111">The third digit is the customer's monetary rating, which is how much the customer spends when he makes purchases from your organization.</span></span>

-   <span data-ttu-id="5670e-114">**Score de récence de 5** – Le client a récemment effectué un achat.</span><span class="sxs-lookup"><span data-stu-id="5670e-114">**Recency rating of 5** – The customer recently made a purchase.</span></span>

<span data-ttu-id="5670e-117">Si votre organisation utilise un nombre entier pour le score, les classements individuels sont additionnés.</span><span class="sxs-lookup"><span data-stu-id="5670e-117">If your organization uses an aggregate number for the score, the individual ratings are added together.</span></span> <span data-ttu-id="5670e-118">Pour le même exemple, le client a un score de 13 (5 + 3 + 5).</span><span class="sxs-lookup"><span data-stu-id="5670e-118">For the same example, the customer has a rating of 13 (5 + 3 + 5).</span></span>
