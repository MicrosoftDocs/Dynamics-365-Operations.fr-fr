# <a name="create-a-pos-visual-profile"></a> Créer un profil visuel de PDV  

[!include[task guide banner](../includes/task-guide-banner.md)]

Cette procédure décrit la création d'un profil visuel de point de vente (PDV). Un profil visuel contient des informations de base qui déterminent l'aspect des caisses enregistreuses du PDV. Vous pouvez créer plusieurs profils visuels et affecter ceux à exécuter sur des caisses enregistreuses spécifiques. La société fictive USRT sert d'exemple dans cette procédure.

1. Accédez à Commerce et vente au détail > Paramétrage du canal > Paramétrage POS > Profils PDV > Profils visuels.
2. Cliquez sur Nouveau.
3. Dans le champ Numéro de profil, tapez une valeur.
4. Dans le champ Description, entrez une valeur.
5. Dans le champ Type d'application, cliquez sur le bouton de liste déroulante pour ouvrir la recherche.
6. Dans la liste, cliquer sur le lien dans la ligne sélectionnée.
7. Dans le champ Thème, cliquez sur le bouton de liste déroulante pour ouvrir la recherche.
8. Dans la liste, cliquer sur le lien dans la ligne sélectionnée.
9. Dans le champ Couleur d'accentuation, cliquez sur le bouton de liste déroulante pour ouvrir la recherche.
10. Dans la liste, recherchez et sélectionnez l'enregistrement souhaité.
11. Dans la liste, cliquer sur le lien dans la ligne sélectionnée.
12. Activez ou désactivez l'extension de la section Arrière-plan de connexion.
13. Dans le champ ID image en mode Paysage, sélectionnez ou saisissez un ID d'image.
14. Dans le champ ID image portrait, sélectionnez ou saisissez un ID d'image.
15. Activez ou désactivez l'extension de la section Arrière-plan.
16. RequestPopup de l'ID image.
17. Dans la liste, cliquer sur le lien dans la ligne sélectionnée.
18. Cliquez sur Enregistrer.
