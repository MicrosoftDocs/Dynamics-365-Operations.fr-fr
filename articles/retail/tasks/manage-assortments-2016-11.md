# <a name="manage-assortments-november-2016"></a>Gérer les assortiments (novembre 2016)

[!include[task guide banner](../includes/task-guide-banner.md)]

Cette procédure montre comment créer et publier un nouvel assortiment de produits. Elle utilise les données de démonstration de la société fictive USRT. 


1. Cliquez sur Gestion des catégories et des produits.

## <a name="create-an-assortment"></a>Créer un assortiment.
1. Cliquez sur l'onglet Assortiments.
2. Cliquez sur Nouveau.
3. Cliquez sur Assortiment.
    * L'identification de l'assortiment est exigée et doit être une valeur unique.  
4. Dans le champ Nom de l'assortiment, saisissez une valeur.
5. Entrez une date dans le champ Date d'effet.
6. Dans le champ Date d'expiration, entrez une date.
7. Développez la section Canaux de vente au détail.
8. Cliquez sur Ajouter une ligne.
9. Dans l'arborescence, sélectionnez « Contoso Retail\Electronics\Boston ».
10. Cliquez sur Ajouter.
11. Cliquez sur OK.
12. Développez la section Produits.
13. Cliquez sur Ajouter une ligne.
14. Dans le champ Catégorie, saisissez ou sélectionnez une valeur.
15. Cliquez sur Enregistrer.

## <a name="publish-an-assortment"></a>Publier un assortiment
1. Cliquez sur Publier.
2. Cliquez sur Oui.
