# <a name="configure-credit-card-processing"></a> Configurer le traitement de carte de crédit

[!INCLUDE [task guide banner](../includes/task-guide-banner.md)]

Cette procédure décrit comment afficher la liste des fournisseurs de paiement et comment configurer un compte de paiement pour la comptabilité client. Cette procédure utilise la société USRT dans les données de démonstration et est destinée aux administrateurs et aux professionnels de l'informatique.


## <a name="view-a-list-of-payment-providers"></a>Afficher la liste des fournisseurs de paiement
1. Accédez à Comptabilité client > Paramétrage des paiements > Services de paiement.
2. Cliquez sur Afficher les fournisseurs disponibles.

## <a name="configure-payment-account"></a>Configurer le compte de paiement
1. Cliquez sur Nouveau.
2. Dans le champ Service de paiement, tapez une valeur.
3. Dans le champ Connecteur de paiement, sélectionnez une option.
4. Activez ou désactivez l'extension de la section Compte de service de paiement.
5. Dans le champ Environnement, tapez « PROD ».
6. Cliquez sur Types de carte de crédit.
7. Dans le champ Journal des paiements, cliquez sur le bouton de liste déroulante pour ouvrir la recherche.
8. Dans la liste, cliquer sur le lien dans la ligne sélectionnée.
9. Cliquez sur Ajouter.
10. Tapez une valeur dans le champ Devise.
11. Dans la liste, recherchez et sélectionnez l'enregistrement souhaité.
12. Dans le champ Journal des paiements, cliquez sur le bouton de liste déroulante pour ouvrir la recherche.
13. Dans la liste, cliquer sur le lien dans la ligne sélectionnée.
14. Cliquez sur Ajouter.
15. Tapez une valeur dans le champ Devise.
16. Dans la liste, recherchez et sélectionnez l'enregistrement souhaité.
    * Vous pouvez répéter ces étapes pour autant de types de cartes que nécessaire.  
17. Dans le champ Journal des paiements, cliquez sur le bouton de liste déroulante pour ouvrir la recherche.
18. Dans la liste, cliquer sur le lien dans la ligne sélectionnée.
19. Cliquez sur Ajouter.
20. Tapez une valeur dans le champ Devise.
21. Cliquez sur Enregistrer.
22. Fermez la page.
23. Cliquez sur Valider.
24. Activez la case à cocher Processeur par défaut pour les nouvelles cartes de crédit.
25. Cliquez sur Enregistrer.
