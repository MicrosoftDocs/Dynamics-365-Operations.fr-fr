# <a name="design-the-relationships-between-organizational-units"></a> Créer les relations entre les unités de l'organisation

[!INCLUDE [task guide banner](../includes/task-guide-banner.md)]

Cette procédure décrit comment concevoir les relations entre les unités d'organisation. Vous devez créer un objectif d'organisation avant de définir les relations ou vous pouvez utiliser l'objectif existant. La société fictive utilisée pour réaliser cette procédure est USRT. Cette tâche est destinée au rôle d'administrateur.

1. Accédez à Administration d'organisation > Organisations > Hiérarchies d'organisation.
2. Cliquez sur Nouveau.
3. Tapez une valeur dans le champ Nom.
4. Cliquez sur Affecter un objectif.
5. Dans la liste, recherchez et sélectionnez l'enregistrement souhaité.
6. Cliquez sur Ajouter.
7. Dans la liste, recherchez et sélectionnez l'enregistrement souhaité.
8. Cliquez sur OK.
    * Vous pouvez sélectionner autant d'objectifs que nécessaire pour votre organisation.  
9. Dans la liste, recherchez et sélectionnez l'enregistrement souhaité.
10. Cliquez sur Définir par défaut.
11. Fermez la page.
12. Cliquez sur Enregistrer.
13. Cliquez sur Afficher.
14. Cliquez sur Modifier.
15. Cliquez sur Insérer
16. Cliquez sur Unité commerciale.
17. Dans la liste, recherchez et sélectionnez l'enregistrement souhaité.
18. Dans la liste, cliquer sur le lien dans la ligne sélectionnée.
19. Cliquez sur Insérer
20. Cliquez sur Canal de vente au détail.
21. Dans la liste, recherchez et sélectionnez l'enregistrement souhaité.
22. Dans la liste, cliquer sur le lien dans la ligne sélectionnée.
    * Vous pouvez ajouter autant d'unités d'organisation que nécessaire.  
23. Cliquez sur Enregistrer.
24. Cliquez sur Fermer.
25. Cliquez sur Publier pour ouvrir l'écran de boîte de dialogue.
26. Entrez une date et une heure dans le champ Date d'effet.
27. Entrez une date et une heure dans le champ Date d'effet.
28. Tapez une valeur dans le champ Décrire les modifications.
29. Cliquez sur Publier.
30. Cliquez sur Fermer.
