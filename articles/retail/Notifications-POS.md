## <a name="enable-notifications-for-order-fulfillment-operations"></a><span data-ttu-id="084d3-112">Activer les notifications pour les opérations d'exécution des commandes</span><span class="sxs-lookup"><span data-stu-id="084d3-112">Enable notifications for order fulfillment operations</span></span>

<span data-ttu-id="084d3-113">Pour activer les notifications pour les opérations d'exécution des commandes, effectuez les étapes suivantes.</span><span class="sxs-lookup"><span data-stu-id="084d3-113">To enable notifications for order fulfillment operations, follow these steps.</span></span>

1. <span data-ttu-id="084d3-114">Accédez à **Retail** &gt; **Paramétrage du canal** &gt; **Paramétrage du PDV** &gt; **PDV** &gt; **Operations**.</span><span class="sxs-lookup"><span data-stu-id="084d3-114">Go to **Retail** &gt; **Channel setup** &gt; **POS setup** &gt; **POS** &gt; **Operations**.</span></span>

3. <span data-ttu-id="084d3-117">Accédez à **Retail** &gt; **Employés** &gt; **Collaborateurs** &gt;, sous l'onglet Retail, ouvrez les autorisations du PDV associées au travailleur.</span><span class="sxs-lookup"><span data-stu-id="084d3-117">Go to **Retail** &gt; **Employees** &gt; **Workers** &gt;, under Retail tab, open the POS permissions associated with the worker.</span></span> <span data-ttu-id="084d3-118">Développez l'organisateur **Notifications**, ajoutez l'opération **Exécution de l'ordre**, puis définissez le champ **Ordre d'affichage** sur **1**.</span><span class="sxs-lookup"><span data-stu-id="084d3-118">Expand the **Notifications** FastTab, add the **Order fulfillment** operation, and set the **Display order** field to **1**.</span></span> <span data-ttu-id="084d3-119">Si plusieurs notifications sont configurées, ce champ est utilisé pour réorganiser les notifications.</span><span class="sxs-lookup"><span data-stu-id="084d3-119">If more than one notification is configured, this field is used to arrange the notifications.</span></span> <span data-ttu-id="084d3-120">Les notifications ayant une valeur d'**Ordre d'affichage** inférieure s'affichent avant les notifications ayant une valeur supérieure.</span><span class="sxs-lookup"><span data-stu-id="084d3-120">Notifications that have a lower **Display order** value appear above notifications that have a higher value.</span></span> <span data-ttu-id="084d3-121">Les notifications ayant une valeur d'**Ordre d'affichage** de **1** se situent en premier.</span><span class="sxs-lookup"><span data-stu-id="084d3-121">Notifications that have a **Display order** value of **1** are at the top.</span></span>

    <span data-ttu-id="084d3-122">Les notifications sont affichées uniquement pour les opérations ajoutées dans l'organisateur **Notifications**, et vous pouvez ajouter des opérations uniquement si la case à cocher **Activer les notifications** est sélectionnée pour ces opérations sur la page **Opérations du PDV**.</span><span class="sxs-lookup"><span data-stu-id="084d3-122">Notifications are shown only for operations that are added on the **Notifications** FastTab, and you can add operations there only if the **Enable notifications** check box for those operations has been selected on the **POS operations** page.</span></span> <span data-ttu-id="084d3-123">En outre, les notifications d'une opération sont affichées pour les travailleurs uniquement si l'opération est ajoutée aux autorisations du PDV pour ceux-ci.</span><span class="sxs-lookup"><span data-stu-id="084d3-123">Additionally, notifications for an operation are shown to workers only if the operation is added to the POS permissions for those workers.</span></span>

    > [!NOTE]
    > <span data-ttu-id="084d3-124">Les notifications peuvent être remplacées au niveau de l'utilisateur.</span><span class="sxs-lookup"><span data-stu-id="084d3-124">Notifications can be overridden at the user level.</span></span> <span data-ttu-id="084d3-125">Ouvrez l'enregistrement du travailleur, sélectionnez **Autorisations PDV**, puis modifiez l'abonnement à la notification de l'utilisateur.</span><span class="sxs-lookup"><span data-stu-id="084d3-125">Open the worker's record, select **POS permissions**, and then edit the user's notification subscription.</span></span>
