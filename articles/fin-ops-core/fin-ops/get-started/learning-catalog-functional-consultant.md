# <a name="functional-application-consultant-learning-catalog"></a>Catalogue Apprentissage Consultant d'application technique

Le catalogue suivant est constitué de connaissances fondamentales de base et avancées dans des domaines spécifiques. Si le contenu existe dans plusieurs formats, nous vous le ferons savoir, de telle sorte que vous puissiez choisir le format de formation qui correspond le mieux à vos besoins. Pour en savoir plus, consultez les formats de contenu Formation. 

Pour les consultants techniques, notre recherche d'analyse de tâche a montré que chacun doit comprendre l'ensemble du contenu de base ainsi qu'un domaine.
