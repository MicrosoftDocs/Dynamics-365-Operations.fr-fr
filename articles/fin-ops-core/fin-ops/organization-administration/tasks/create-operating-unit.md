# <a name="create-an-operating-unit"></a><span data-ttu-id="627cc-103">Créer une unité opérationnelle</span><span class="sxs-lookup"><span data-stu-id="627cc-103">Create an operating unit</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="627cc-104">Une section est une organisation utilisée pour partager le contrôle des ressources économiques et des processus opérationnels dans une société.</span><span class="sxs-lookup"><span data-stu-id="627cc-104">An operating unit is an organization that is used to divide the control of economic resources and operational processes in a business.</span></span> <span data-ttu-id="627cc-105">Dans une section, les personnes ont la responsabilité d'optimiser l'utilisation des ressources rares, d'améliorer les processus et de justifier leurs performances.</span><span class="sxs-lookup"><span data-stu-id="627cc-105">People in an operating unit have a duty to maximize the use of scarce resources, improve processes, and account for their performance.</span></span> <span data-ttu-id="627cc-106">Les types de sections incluent les centres de coût, les unités commerciales, les départements et les chaînes de valeurs.</span><span class="sxs-lookup"><span data-stu-id="627cc-106">The types of operating units include cost centers, business units, departments, and value streams.</span></span> <span data-ttu-id="627cc-107">Procédez comme suit pour créer une unité opérationnelle.</span><span class="sxs-lookup"><span data-stu-id="627cc-107">Use the following procedure to create an operating unit.</span></span> <span data-ttu-id="627cc-108">Les données fictives utilisées pour créer cette procédure correspondent à la société USMF.</span><span class="sxs-lookup"><span data-stu-id="627cc-108">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="627cc-109">Accédez au **volet de navigation > Modules > Administration d'organisation > Organisations > Unités opérationnelles**.</span><span class="sxs-lookup"><span data-stu-id="627cc-109">Go to **Navigation pane > Modules > Organization administration > Organizations > Operating units.**</span></span>
2. <span data-ttu-id="627cc-110">Cliquez sur **Nouveau** pour ouvrir la boîte de dialogue.</span><span class="sxs-lookup"><span data-stu-id="627cc-110">Click **New** to open the drop dialog.</span></span>
3. <span data-ttu-id="627cc-111">Dans la liste, recherchez et sélectionnez l'enregistrement souhaité.</span><span class="sxs-lookup"><span data-stu-id="627cc-111">In the list, find and select the desired record.</span></span> <span data-ttu-id="627cc-112">Permet de sélectionner le type d'unité opérationnelle à créer.</span><span class="sxs-lookup"><span data-stu-id="627cc-112">Select the type of operating unit you want to create.</span></span>  
4. <span data-ttu-id="627cc-113">Dans la liste, cliquer sur le lien dans la ligne sélectionnée.</span><span class="sxs-lookup"><span data-stu-id="627cc-113">In the list, click the link in the selected row.</span></span>
5. <span data-ttu-id="627cc-114">Tapez une valeur dans le champ **Nom**.</span><span class="sxs-lookup"><span data-stu-id="627cc-114">In the **Name** field, type a value.</span></span>
    + <span data-ttu-id="627cc-115">Développez la section **Général**, le cas échéant.</span><span class="sxs-lookup"><span data-stu-id="627cc-115">Expand the **General** section, if necessary.</span></span>  
    + <span data-ttu-id="627cc-116">Indiquez les informations générales relatives à la section, telles qu'un numéro d'identification, un numéro DUNS et un responsable.</span><span class="sxs-lookup"><span data-stu-id="627cc-116">Provide general information about the operating unit, such as an identification number, DUNS number, and manager.</span></span>    
    + <span data-ttu-id="627cc-117">Développez la section **Adresses**, le cas échéant.</span><span class="sxs-lookup"><span data-stu-id="627cc-117">Expand the **Addresses** section, if necessary.</span></span>  
    + <span data-ttu-id="627cc-118">Entrez des informations d'adresse, telles que le nom de la rue et le numéro, le code postal et la ville.</span><span class="sxs-lookup"><span data-stu-id="627cc-118">Enter address information, such as the street name and number, postal code, and city.</span></span> <span data-ttu-id="627cc-119">Cliquez sur **Ajouter** pour entrer un nouvel enregistrement d'adresse ou sur Modifier pour modifier un enregistrement existant.</span><span class="sxs-lookup"><span data-stu-id="627cc-119">Click **Add** to enter a new address record, or click Edit to modify an existing address record.</span></span>   
    + <span data-ttu-id="627cc-120">Développez la section **Informations de contact**, le cas échéant.</span><span class="sxs-lookup"><span data-stu-id="627cc-120">Expand the **Contact information** section, if necessary.</span></span>  
    + <span data-ttu-id="627cc-121">Entrez les informations relatives aux modes de communication, notamment les adresses e-mail, les URL et les numéros de téléphone.</span><span class="sxs-lookup"><span data-stu-id="627cc-121">Enter information about methods of communication, such as email addresses, URLs, and telephone numbers.</span></span> <span data-ttu-id="627cc-122">Pour entrer un nouvel enregistrement de communication, cliquez sur Nouveau.</span><span class="sxs-lookup"><span data-stu-id="627cc-122">To enter a new communication record, click New.</span></span> <span data-ttu-id="627cc-123">Pour modifier un enregistrement de communication existant, cliquez sur **Plus d'options > Avancées**.</span><span class="sxs-lookup"><span data-stu-id="627cc-123">To modify an existing communication record, click **More options > Advanced**.</span></span>   
6. <span data-ttu-id="627cc-124">Le cas échéant, modifiez le **Numéro d'unité opérationnelle** si nécessaire.</span><span class="sxs-lookup"><span data-stu-id="627cc-124">Optionally, change the **Operating unit number** as needed.</span></span> <span data-ttu-id="627cc-125">Notez que ce numéro est un identifiant unique pour l'enregistrement **Partie** correspondant et ne peut pas être identique à une autre unité opérationnelle.</span><span class="sxs-lookup"><span data-stu-id="627cc-125">Note that this number is a unique idenitifier for the correspondng **Party** record and cannot be the same as any other operating unit.</span></span>
7. <span data-ttu-id="627cc-126">Sélectionnez **Enregistrer**.</span><span class="sxs-lookup"><span data-stu-id="627cc-126">Select **Save**.</span></span>
