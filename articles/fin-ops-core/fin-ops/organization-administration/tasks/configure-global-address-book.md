# <a name="configure-the-global-address-book"></a><span data-ttu-id="55ee2-103">Configurer le carnet d’adresses global</span><span class="sxs-lookup"><span data-stu-id="55ee2-103">Configure the global address book</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="55ee2-104">Cette procédure permet de définir les valeurs par défaut et les stratégies de sécurité du carnet d’adresses global.</span><span class="sxs-lookup"><span data-stu-id="55ee2-104">Use this procedure to set the default values and security policies for the global address book.</span></span> 

<span data-ttu-id="55ee2-105">Les données fictives utilisées pour créer cette tâche correspondent à la société USMF.</span><span class="sxs-lookup"><span data-stu-id="55ee2-105">The demo data company used to create this task is USMF.</span></span> <span data-ttu-id="55ee2-106">Cette tâche est destinée à l’équipe de planification et de configuration.</span><span class="sxs-lookup"><span data-stu-id="55ee2-106">This task is intended for the Planning and configuration team.</span></span>

1. <span data-ttu-id="55ee2-107">Dans le volet de navigation, accédez à **Modules > Administration d’organisation > Carnet d’adresses global > Paramètres du carnet d’adresses global**.</span><span class="sxs-lookup"><span data-stu-id="55ee2-107">In the Navigation pane, go to **Modules > Organization administration > Global address book > Global address book parameters**.</span></span>
2. <span data-ttu-id="55ee2-108">Dans le champ **Ordre des noms**, sélectionnez la manière dont les noms doivent être affichés.</span><span class="sxs-lookup"><span data-stu-id="55ee2-108">In the **Name sequence** field, select how names should be shown.</span></span>
3. <span data-ttu-id="55ee2-109">Dans le champ **Supprimer les parties sans rôles**, indiquez si vous souhaitez supprimer les parties avec lesquelles aucun rôle n’a été attribué.</span><span class="sxs-lookup"><span data-stu-id="55ee2-109">In **Delete parties with no roles**, select whether to delete parties with that have not been assigned a role.</span></span>
4. <span data-ttu-id="55ee2-110">Dans le champ **Utiliser la vérification des doublons**, sélectionnez s’il faut vérifier les enregistrements en double.</span><span class="sxs-lookup"><span data-stu-id="55ee2-110">In **Use duplicate check**, select whether to check for duplicate records.</span></span>
5. <span data-ttu-id="55ee2-111">Dans **Afficher le numéro DUNS sur les adresses**, Sélectionnez si vous souhaitez afficher le numéro DUNS sur les adresses.</span><span class="sxs-lookup"><span data-stu-id="55ee2-111">In **Display DUNS number on addresses**, select whether to display the DUNS number on addresses.</span></span>
6. <span data-ttu-id="55ee2-112">Dans **Contrôle unicité DUNS**, sélectionnez si vous souhaitez rechercher des numéros DUN uniques.</span><span class="sxs-lookup"><span data-stu-id="55ee2-112">In **Check for unique DUNS number**, select whether to check for unique DUNS numbers.</span></span>
7. <span data-ttu-id="55ee2-113">Dans le champ **Partie**, sélectionnez une option.</span><span class="sxs-lookup"><span data-stu-id="55ee2-113">In the **Party** field, select an option.</span></span>
8. <span data-ttu-id="55ee2-114">Dans le champ **Client**, sélectionnez une option.</span><span class="sxs-lookup"><span data-stu-id="55ee2-114">In the **Customer** field, select an option.</span></span>
9. <span data-ttu-id="55ee2-115">Dans le champ **Fournisseur**, sélectionnez une option.</span><span class="sxs-lookup"><span data-stu-id="55ee2-115">In the **Vendor** field, select an option.</span></span>
10. <span data-ttu-id="55ee2-116">Dans le champ **Prospect**, sélectionnez une option.</span><span class="sxs-lookup"><span data-stu-id="55ee2-116">In the **Prospect** field, select an option.</span></span>
11. <span data-ttu-id="55ee2-117">Dans le champ **Concurrent**, sélectionnez une option.</span><span class="sxs-lookup"><span data-stu-id="55ee2-117">In the **Competitor** field, select an option.</span></span>
12. <span data-ttu-id="55ee2-118">Cliquez sur l’onglet **Sécurité de l’emplacement privé**.</span><span class="sxs-lookup"><span data-stu-id="55ee2-118">Click the **Private location security** tab.</span></span>
13. <span data-ttu-id="55ee2-119">Dans la liste, recherchez et sélectionnez l’enregistrement souhaité.</span><span class="sxs-lookup"><span data-stu-id="55ee2-119">In the list, find and select the desired record.</span></span> <span data-ttu-id="55ee2-120">Appuyez sur la touche Shift pour sélectionner plusieurs rôles à ajouter au volet **Rôles sélectionnés**, puis cliquez sur la flèche pour ajouter les rôles sélectionnés. </span><span class="sxs-lookup"><span data-stu-id="55ee2-120">Press the Shift key to select multiple roles to add to the **Selected roles** pane and then click the arrow to add the selected roles.</span></span>  
14. <span data-ttu-id="55ee2-121">Cliquez sur **Enregistrer**.</span><span class="sxs-lookup"><span data-stu-id="55ee2-121">Click **Save**.</span></span>
