    - <span data-ttu-id="575b9-119">Définissez le format de la souche de numéros en ajoutant, supprimant et réorganisant les segments.</span><span class="sxs-lookup"><span data-stu-id="575b9-119">Define the format for the number sequence by adding, removing, and rearranging segments.</span></span>  
    - <span data-ttu-id="575b9-120">Les souches de numéros de toutes les portées peuvent contenir des *Segments constants* et des *Segments alphanumériques*.</span><span class="sxs-lookup"><span <span data-ttu-id="575b9-125">Utilisez un symbole numérique (#) pour indiquer les numéros de l’incrémentation et une esperluette (&) pour indiquer les lettres de l’incrémentation.</span><span class="sxs-lookup"><span data-stu-id="575b9-125">Use a number sign (#) to indicate incrementing numbers and an ampersand (&) to indicate incrementing letters.</span></span> <span
