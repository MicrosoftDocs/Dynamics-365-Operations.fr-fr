# <a name="enter-project-timesheets"></a>Saisir des feuilles de temps du projet


[!INCLUDE [PEAP](../../../../includes/peap-1.md)]

Cette procédure vous permet de créer une feuille de temps à l’aide d’une page Feuille de temps vide. La nouvelle feuille de temps peut être basée sur les informations issues d’une feuille de temps précédente, ou d’affectations d’activité et de projet sur la page **Mes favoris**. Par défaut, la page de liste **Toutes les feuilles de temps** présente toutes vos feuilles de temps pour la période en cours. Vous pouvez utiliser le champ **Afficher** sur la page **Mes feuilles de temps** afin de filtrer la liste des feuilles de temps par période ou par projet, ou pour afficher les feuilles de temps créées au nom d’autres collaborateurs. Les données fictives utilisées pour créer cette procédure correspondent à la société **USSI**.

1. Accédez à **Gestion et comptabilité des projets\> Feuilles de temps \> Mes feuilles de temps**.
2. Sélectionnez **Nouveau** pour entrer une nouvelle feuille de temps.

    Par défaut, le champ **Ressource** affiche le collaborateur affecté à l’utilisateur actuel.

    Si l’utilisateur est désigné comme délégué, cela répertorie les noms de sorte qu’un utilisateur puisse entrer une feuille de temps en leur nom.

3. Entrez une date dans le champ **Date**. Si cette option est sélectionnée, des lignes de feuille de temps sont créées en utilisant les paramètres de feuille de temps qui ont été configurés en tant que favoris.
4. Cliquez sur **OK**.
5. Sélectionnez **Nouvelle ligne**.
6. Dans la liste, marquer la ligne sélectionnée. Par défaut, le champ **Entité juridique** affiche l’entité juridique actuelle.
7. Dans le champ **Projet**, sélectionnez la flèche de liste déroulante pour ouvrir la recherche.
8. Dans la liste, recherchez et sélectionnez l’enregistrement souhaité.
9. Dans la liste, sélectionnez le lien dans la ligne sélectionnée.
10. Dans le champ **Numéro d’activité**, sélectionnez la flèche de liste déroulante pour ouvrir la recherche.
11. Dans la liste, recherchez et sélectionnez l’enregistrement souhaité.
12. Dans la liste, sélectionnez le lien dans la ligne sélectionnée.
13. Dans le champ **Catégorie**, sélectionnez la flèche de liste déroulante pour ouvrir la recherche.
14. Dans la liste, recherchez et sélectionnez l’enregistrement souhaité.
15. Dans la liste, sélectionnez le lien dans la ligne sélectionnée.
16. Entrez le nombre d’heures travaillées quotidiennement. Entrez les heures sous forme décimale. Par exemple, si vous avez travaillé deux heures et quinze minutes, entrez **2,25**.

    Dans la section **Détails de ligne**, les options suivantes sont disponibles :

    - Ajoutez des informations sur les taxes et les dimensions financières dans les onglets **Général** et **Dimensions financières**.
    - Ajoutez des commentaires sur la ligne de feuille de temps dans l’onglet **Commentaire**.

17. Dans le volet Action, sélectionnez **Workflow** pour ouvrir une boîte de dialogue déroulante.
18. Sélectionnez **Soumettre**.
19. Sélectionnez **Soumettre**.
