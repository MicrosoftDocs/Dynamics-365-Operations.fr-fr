# <a name="develop-and-open-job-requisition"></a>Développer et ouvrir une demande de poste

[!include [banner](../../includes/banner.md)]

> [!IMPORTANT]
> La fonctionnalité de recrutement dans cette rubrique sera appelée Projets de recrutement et se concentre sur les candidats, les candidatures et les projets de recrutement.  


Les projets de recrutement permettent de gérer le processus de recrutement. Pour chaque projet de recrutement, vous pouvez paramétrer des informations, telles que l’emploi pour lequel vous recrutez, le nom du recruteur, le statut du projet et le département dans lequel l’employé sera situé. Après avoir créé un projet de recrutement, vous pouvez rédiger une annonce d’emploi pour le projet, la publier dans les pages du **Libre service pour employés**, associer des candidatures à l’emploi au projet et suivre les activités du projet. Les données fictives utilisées pour créer cette procédure correspondent à la société **USMF**.

1. Accédez à **Ressources humaines \> Recrutement \> Projets de recrutement \> Projets de recrutement**.
2. Cliquez sur **Nouveau**.
3. Saisissez une valeur dans le champ **Projet de recrutement**.
4. Dans le champ **Description**, entrez une valeur.
5. Dans le champ **Recruteur**, cliquez sur la flèche de liste déroulante pour ouvrir la recherche.
6. Dans la liste, recherchez et sélectionnez l’enregistrement souhaité.
7. Dans la liste, sélectionnez le lien dans la ligne sélectionnée.
8. Cliquez sur **Sélectionner**.
9. Cliquez sur la flèche de liste déroulante pour ouvrir la recherche dans le champ **Département**.
10. Dans la liste, sélectionnez le lien dans la ligne sélectionnée.
11. Dans le champ **Poste**, cliquez sur la flèche de liste déroulante pour ouvrir la recherche.
12. Dans la liste, recherchez et sélectionnez l’enregistrement souhaité.
13. Dans la liste, sélectionnez le lien dans la ligne sélectionnée.
14. Entrez un nombre dans le champ **Nombre de postes à pourvoir**.
15. Cliquez sur la flèche de liste déroulante pour ouvrir la recherche dans le champ **Responsable du recrutement**.
16. Dans la liste, recherchez et sélectionnez l’enregistrement souhaité.
17. Dans la liste, sélectionnez le lien dans la ligne sélectionnée.
18. Cliquez sur **Sélectionner**.
19. Entrez une date dans le champ **Date limite de candidature**.
20. Sélectionnez **Média**.

    Les projets de recrutement offrent la possibilité de spécifier les établissements de communication qui seront utilisés pour signaler des postes vacants.

21. Cliquez sur **Nouveau**.
22. Dans le champ **Média**, cliquez sur la flèche de liste déroulante pour ouvrir la recherche.
23. Dans la liste, sélectionnez le lien dans la ligne sélectionnée.
24. Entrez une date dans le champ **Date de début**.
25. Entrez une date dans le champ **Date de fin**.
26. Sélectionnez **Enregistrer**.
27. Fermez la page.
28. Cliquez sur **Annonces d’emploi**.
29. Cliquez sur **Enregistrer**.
30. Fermez la page.
31. Cochez ou décochez la case **Afficher dans le libre service pour employés**.

    Le fait de cocher la case rend le projet de recrutement visible aux salariés sur leur page **Libre service pour employés**.

32. Sélectionnez le statut **Projet de recrutement**.
33. Sélectionnez **Démarrer**.

    Le statut **Démarré** indique qu’il est possible d’envoyer des candidatures pour le projet.

34. Cliquez sur **OK**.
