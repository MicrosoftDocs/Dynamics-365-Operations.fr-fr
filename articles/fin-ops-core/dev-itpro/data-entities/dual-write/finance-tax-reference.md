# <a name="access-to-finance-and-tax-reference-data"></a>Accès aux données de référence financières et fiscales

[!include [banner](../../includes/banner.md)]



Chaque entreprise fonctionne avec un ensemble de données financières de base, telles que l'année civile fiscale, la devise dans laquelle l'entreprise est effectuée, les comptes dans lesquels l'argent pour gérer l'entreprise entre ou sort, les taux d'imposition et les versements. Ces données résident dans les applications Finance and Operations. Cependant, elles sont exposées à Common Data Service de sorte que les applications pilotées par modèle dans Microsoft Dynamics 365 peuvent avoir une source unique pour les données financières et fiscales. De cette façon, les données sont uniformes dans tout l'écosystème commercial. 

Les données financières et fiscales sont intégrées à l'aide des mappages suivants :

+ [Comptabilité intégrée](ledger-mapping.md)
+ [Données principales fiscales intégrées](tax-mapping.md)
