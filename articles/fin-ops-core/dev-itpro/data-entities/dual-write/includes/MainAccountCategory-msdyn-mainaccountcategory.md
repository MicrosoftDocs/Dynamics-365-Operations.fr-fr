Ce modèle synchronise les données entre les applications Finance and Operations et Common Data Service.

---|---|---|---

REFERENCEID | = | msdyn_referenceid | 
DESCRIPTION | = | msdyn_description | 
DISPLAYORDER | = | msdyn_displayorder | 
CLOSED | >< | msdyn_closed | 
MAINACCOUNTTYPE | >< | msdyn_mainaccounttypevalue |
