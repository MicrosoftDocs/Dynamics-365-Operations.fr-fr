# <a name="list-of-er-functions-in-the-container-category"></a>Liste des fonctions ER dans la catégorie Conteneur

[!include [banner](../includes/banner.md)]

Les [fonctions](er-formula-language.md#Functions) de conteneur [pour la gestion des états électroniques](general-electronic-reporting.md) peuvent être utilisées pour exécuter les opérations qui impliquent les sources de données de type *Conteneur*. Ces opérations se produisent lorsque les données de traitement représentent une collection de données binaires au format BLOB (Binary Large Object). Cet article fournit un résumé de ces fonctions.

## <a name="list-of-supported-functions"></a>Liste des fonctions prises en charge

| Fonction | Description |
|----------|-------------|
| [Base64StringToContainer](er-functions-container-base64stringtocontainer.md) | Cette fonction renvoie une valeur *Conteneur* qui consiste en un contenu binaire décodé à partir de la chaîne ASCII spécifiée qui représente un groupe Base64 de schémas de codage binaire en texte. |
