- <span data-ttu-id="3c345-107">Pour l’élément de format FICHIER, vous pouvez définir une limite à la taille de fichier en tant qu’expression ER.</span><span class="sxs-lookup"><span data-stu-id="3c345-107">For the FILE format element, you can define a limit on the file size as an ER expression.</span></span> <span data-ttu-id="3c345-108">Si la limite définie est dépassée lorsqu’un état ER est généré, ER finit de créer le fichier actuel avant de passer à la création du fichier suivant.</span><span class="sxs-lookup"><span data-stu-id="3c345-108">If the defined limit is exceeded when an ER report is generated, ER finishes creating the current file and then moves on to create the next file.</span></span>

- <span data-ttu-id="3c345-111">Pour un élément de format SÉQUENCE XML, vous pouvez définir une limite au nombre d’éléments enfants en tant qu’expression ER.</span><span class="sxs-lookup"><span data-stu-id="3c345-111">For any XML SEQUENCE format element, you can define a limit on the number of child elements as an ER expression.</span></span> <span data-ttu-id="3c345-112">Si le nombre de nœuds XML imbriqués de l’élément de format dans le fichier généré dépasse la limite définie lorsqu’un état ER est exécuté, ER finit de créer le fichier actuel avant de passer à la création du fichier suivant.</span><span class="sxs-lookup"><span data-stu-id="3c345-112">If the number of nested XML nodes of the format element in the generated file exceeds the defined limit when an ER report is run, ER finishes creating the current file and then moves on to create the next file.</span></span>
- <span data-ttu-id="3c345-113">Vous pouvez marquer un élément de format ÉLÉMENT XML comme insécable.</span><span class="sxs-lookup"><span data-stu-id="3c345-113">You can mark any XML ELEMENT format element as non-breakable.</span></span> <span data-ttu-id="3c345-114">Ainsi, vous pouvez conserver les éléments imbriqués des nœuds XML générés sous l’élément de format dans un fichier généré unique.</span><span class="sxs-lookup"><span data-stu-id="3c345-114">In this way, you can keep the nested items of XML nodes that are generated under the format element in a single generated file.</span></span>

<span data-ttu-id="3c345-115">Outre l’utilisation des éléments de format ÉLÉMENT XML et SÉQUENCE XML pour ajouter des nœuds XML au fichier généré, vous pouvez utiliser l’élément de format XML BRUT.</span><span class="sxs-lookup"><span data-stu-id="3c345-115">In addition to using the XML ELEMENT and XML SEQUENCE format elements to add XML nodes to the generated file, you can use the RAW XML format element.</span></span> <span data-ttu-id="3c345-116">Toutefois, les nœuds ajoutés à l’aide de l’élément de format XML BRUT ne sont pas pris en compte lorsque le nombre de nœuds est calculé pour évaluer les limites du nombre d’éléments.</span><span class="sxs-lookup"><span data-stu-id="3c345-116">However, nodes that you add by using the RAW XML format element aren't considered when the number of nodes is calculated to evaluate the limits on the number of elements.</span></span>

<span data-ttu-id="3c345-117">Si vous avez configuré les destinations de fichier d’un élément de format FICHIER qui a été configuré pour fractionner la sortie générée lorsque des limites spécifiques sont dépassées, chaque élément de la sortie générée est envoyée à la destination de fichier configurée en tant que fichier individuel.</span><span class="sxs-lookup"><span data-stu-id="3c345-117">If you configured file destinations for a FILE format element that has been configured to split the generated output whenever specific limits are exceeded, each piece of generated output is sent to the configured file destination as an individual file.</span></span> <span data-ttu-id="3c345-118">Pour nommer uniquement les fichiers créés en fractionnant la sortie, vous devez configurer une expression ER pour l’élément de format FICHIER.</span><span class="sxs-lookup"><span data-stu-id="3c345-118">To uniquely name the files that are created by splitting the output, you must configure an ER expression for the FILE format element.</span></span> <span data-ttu-id="3c345-119">Si vous incluez une source de données ER du type SOUCHE DE NUMÉROS, la souche de numéros est incrémentée pour chaque élément de la sortie fractionnée.</span><span class="sxs-lookup"><span data-stu-id="3c345-119">If you include an ER data source of the NUMBER SEQUENCE type, the number sequence will be incremented for each piece of the split output.</span></span>
