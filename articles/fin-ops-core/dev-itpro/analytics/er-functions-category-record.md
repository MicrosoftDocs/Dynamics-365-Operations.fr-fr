# <a name="list-of-er-functions-in-the-record-category"></a>Liste des fonctions ER dans la catégorie d'enregistrement

[!include [banner](../includes/banner.md)]

Les fonctions d'enregistrement des états électroniques (ER) peuvent être utilisées pour extraire des informations et effectuer des opérations sur les sources de données du type de données *Conteneur (enregistrement)*. Cette rubrique fournit un résumé de ces fonctions.

## <a name="list-of-supported-functions"></a>Liste des fonctions prises en charge

| Fonction | Description |
|----------|-------------|
| [NullContainer](er-functions-record-nullcontainer.md) | Cette fonction renvoie une valeur de *Conteneur (enregistrement)* nulle avec la même structure que la liste d'enregistrements ou de l'enregistrement spécifiés. |
| [EmptyRecord](er-functions-record-emptyrecord.md) | Cette fonction renvoie une valeur de *Conteneur (enregistrement)* nulle avec la même structure que la liste d'enregistrements ou de l'enregistrement spécifiés. |
