# <a name="list-of-er-functions-in-the-mathematical-category"></a>Liste des fonctions ER dans la catégorie mathématique

[!include [banner](../includes/banner.md)]

Les fonctions mathématiques d’états électronique (ER) peuvent être utilisées pour effectuer de nombreux calculs mathématiques courants. Cet article fournit un résumé de ces fonctions.

## <a name="list-of-supported-functions"></a>Liste des fonctions prises en charge

| Fonction | Description |
|----------|-------------|
| [Abs](er-functions-mathematical-abs.md)             | Cette fonction renvoie la valeur absolue (module) du nombre spécifié en tant que valeur *Réelle*. En d’autres termes, elle renvoie le nombre sans son signe. |
| [Puissance](er-functions-mathematical-power.md)         | Cette fonction renvoie une valeur de *Réel* qui représente le résultat de l’augmentation du nombre positif spécifié à la puissance spécifiée. |
| [Round](er-functions-mathematical-round.md)         | Cette fonction renvoie le nombre spécifié comme valeur *Réelle* une fois qu’il a été arrondi au nombre de décimales spécifié. |
| [RoundDown](er-functions-mathematical-rounddown.md) | Cette fonction renvoie le nombre spécifié comme valeur *Réelle* une fois qu’il a été arrondi au nombre inférieur de décimales spécifié. |
| [RoundUp](er-functions-mathematical-roundup.md)     | Cette fonction renvoie le nombre spécifié comme valeur *Réelle* une fois qu’il a été arrondi au nombre supérieur de décimales spécifié. |
