# <a name="file-destination"></a>Destination du fichier

[!include [banner](../includes/banner.md)]

Vous pouvez configurer une destination de fichier pour chaque composant DOSSIER ou FICHIER d’un format de rapport électronique (ER) configuré pour générer des documents sortants. En fonction de la destination, un document généré est disponible pour téléchargement à partir du navigateur Web.

Sur la page **Paramètres de destination**, si vous définissez **Activé** sur **Oui**, une boîte de dialogue s’ouvre lorsque l’exécution de la configuration est terminée.
