# <a name="lower-er-function"></a>Fonction LOWER ER

[!include [banner](../includes/banner.md)]

La fonction `LOWER` renvoie la chaîne de texte spécifiée sous la forme *Chaîne* après avoir été convertie en lettres minuscules.

## <a name="syntax"></a>Syntaxe

```vb
LOWER (text)
```

## <a name="arguments"></a>Arguments

`text` : *Chaîne*

Valeur *Chaîne* qui spécifie le texte.

## <a name="return-values"></a>Valeurs de retour

*Chaîne*

Valeur de texte résultante.

## <a name="example"></a>Exemple

`LOWER ("Sample")` renvoie **« exemple »**.
