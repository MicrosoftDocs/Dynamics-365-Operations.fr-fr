# <a name="ScreenDestinationType">Destination d'écran</a>

[!include [banner](../includes/banner.md)]

Vous pouvez configurer une destination d'écran pour chaque composant « dossier » ou « fichier » d'un format de rapport électronique (ER) configuré pour générer des documents sortants. En fonction de la destination, un document généré est ouvert pour aperçu dans un onglet de navigateur distinct.

Si vous définissez **Activé** sur **Oui**, un aperçu de la sortie est créé. Vous pouvez afficher certains types de fichiers, tels que XML, TXT ou PDF, directement dans une fenêtre du navigateur. Pour les autres types de fichiers, tels que Microsoft Excel ou Word, le service Microsoft Office Online est utilisé.
