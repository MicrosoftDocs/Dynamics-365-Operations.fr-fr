# <a name="upper-er-function"></a>Fonction UPPER ER

[!include [banner](../includes/banner.md)]

La fonction `UPPER` renvoie la chaîne de texte spécifiée sous la forme *Chaîne* après avoir été convertie en lettres majuscules.

## <a name="syntax"></a>Syntaxe

```vb
UPPER (text )
```

## <a name="arguments"></a>Arguments

`text` : *Chaîne*

Chemin d’accès valide d’une source de données du type *Chaîne*.

## <a name="return-values"></a>Valeurs de retour

*Chaîne*

Valeur de texte résultante.

## <a name="example"></a>Exemple

`UPPER ("Sample")` renvoie **« EXEMPLE »**.
