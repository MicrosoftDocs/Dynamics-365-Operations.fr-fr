# <a name="enter-project-timesheets"></a>Saisir des feuilles de temps du projet

[!include [task guide banner](../../includes/task-guide-banner.md)]

Cette procédure vous permet de créer une feuille de temps à l'aide d'un écran Feuille de temps vide. La nouvelle feuille de temps peut être basée sur les informations issues d'une feuille de temps précédente, ou d'affectations d'activité et de projet dans la page Mes favoris. Par défaut, la page de liste Toutes les feuilles de temps affiche toutes vos feuilles de temps pour la période en cours. Vous pouvez utiliser la liste déroulante pour le champ Afficher de la page Mes feuilles de temps afin de filtrer la liste des feuilles de temps par période ou par projet, ou pour afficher les feuilles de temps créées au nom d'autres collaborateurs. Les données fictives utilisées pour créer cette procédure correspondent à la société USSI. Pour démarrer cette procédure, accédez à Gestion et comptabilité des projets > Feuilles de temps > Mes feuilles de temps

1. Pour entrer une nouvelle feuille de temps, cliquez sur Nouveau.
    * La liste déroulante Ressource indique le collaborateur affecté à l'utilisateur actuel, par défaut.  
    * Si l'utilisateur est désigné comme délégué, cela répertorie les noms de sorte qu'un utilisateur puisse entrer une feuille de temps en leur nom.  
2. Entrez une date dans le champ Date.
    * Si cette option est sélectionnée, des lignes de feuille de temps sont créées en utilisant les paramètres de feuille de temps qui ont été configurés en tant que favoris.  
3. Cliquez sur OK.
4. Cliquez sur Nouvelle ligne.
5. Dans la liste, marquez la ligne sélectionnée.
    * Le champ Entité juridique affiche l'entité juridique par défaut actuelle.   
6. Dans le champ Projet, cliquez sur le bouton de liste déroulante pour ouvrir la recherche.
7. Dans la liste, recherchez et sélectionnez l'enregistrement souhaité.
8. Dans la liste, cliquer sur le lien dans la ligne sélectionnée.
9. Dans le champ Activité, cliquez sur le bouton de liste déroulante pour ouvrir la recherche.
10. Dans la liste, recherchez et sélectionnez l'enregistrement souhaité.
11. Dans la liste, cliquer sur le lien dans la ligne sélectionnée.
12. Dans le champ Catégorie, cliquez sur le bouton de liste déroulante pour ouvrir la recherche.
13. Dans la liste, recherchez et sélectionnez l'enregistrement souhaité.
14. Dans la liste, cliquer sur le lien dans la ligne sélectionnée.
15. Spécifiez le nombre d'heures travaillées chaque jour.
    * Entrez les heures au format décimal.  Par exemple, si vous avez travaillé deux heures et quinze minutes, entrez 2,25.   
16. Spécifiez le nombre d'heures travaillées chaque jour.
    * Entrez les heures au format décimal.  Par exemple, si vous avez travaillé deux heures et quinze minutes, entrez 2,25.   
17. Spécifiez le nombre d'heures travaillées chaque jour.
    * Entrez les heures au format décimal.  Par exemple, si vous avez travaillé deux heures et quinze minutes, entrez 2,25.   
18. Spécifiez le nombre d'heures travaillées chaque jour.
    * Entrez les heures au format décimal.  Par exemple, si vous avez travaillé deux heures et quinze minutes, entrez 2,25.   
19. Spécifiez le nombre d'heures travaillées chaque jour.
    * Entrez les heures au format décimal.  Par exemple, si vous avez travaillé deux heures et quinze minutes, entrez 2,25.   
    * Dans Détails de ligne, les options suivantes sont disponibles : o Ajouter des informations sur les taxes et les dimensions financières.  o Ajouter des commentaires sur la ligne de feuille de temps.  
20. Cliquer sur Workflow pour ouvrir la boîte de dialogue.
21. Cliquez sur Soumettre.
22. Cliquez sur Soumettre.
