23. <span data-ttu-id="88e2b-134">Cliquer sur Afficher les sections pour ouvrir la boîte de dialogue.</span><span class="sxs-lookup"><span data-stu-id="88e2b-134">Click Show sections to open the drop dialog.</span></span>
