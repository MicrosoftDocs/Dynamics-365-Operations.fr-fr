# <a name="benefit-eligibility-process"></a>Processus de droit aux avantages

[!include [task guide banner](../../includes/task-guide-banner.md)]

Cette procédure décrit le fonctionnement du processus d'admissibilité aux avantages. Lorsque le processus est terminé, vous pouvez afficher les résultats. Les données fictives utilisées pour créer cette procédure correspondent à la société USMF.

1. Accédez à Ressources humaines > Avantages > Avantages.
2. Dans la liste, recherchez et sélectionnez l'enregistrement souhaité.
3. Dans la liste, cliquer sur le lien dans la ligne sélectionnée.
4. Cliquez sur Modifier.
5. Dans le champ Admissibilité, sélectionnez « Basée sur les règles ».
6. Dans le champ Type de règle, sélectionnez la règle de stratégie de droit aux avantages que vous souhaitez appliquer à l'avantage.
7. Cliquez sur Avantage dans le volet Actions.
8. Cliquez sur Créer un événement de droit pour ouvrir la boîte de dialogue.
9. Dans le champ Événement, tapez une valeur.
10. Tapez une valeur dans le champ Description.
11. Dans le champ Type d'événement, sélectionnez Inscription en cours.
12. Dans le champ Date de début de la couverture, entrez une date et une heure.
13. Dans le champ Date de début de la période d'inscription, entrez une date et une heure.
14. Dans le champ Jours avant l'inscription, entrez un nombre.
15. Cliquez sur Créer un événement.
16. Dans l'organisateur Collaborateurs, cliquez sur Ajouter.
17. Dans le champ Afficher par type, sélectionnez Employés.
18. Dans le champ Afficher par entité juridique, sélectionnez Entité juridique actuelle.
19. Dans la liste, cochez ou décochez toutes les lignes.
20. Cliquez sur OK.
21. Cliquez sur Traiter.
22. Cliquez sur OK.
23. Actualisez la page.
24. Cliquez sur Afficher les résultats.
25. Ouvrez le filtre de colonne État.
26. Trier de A à Z
