# <a name="manage-worker-accommodations"></a>Gérer les logements des collaborateurs

[!include[task guide banner](../../../includes/task-guide-banner.md)]

Cette procédure décrit le paramétrage des types d'aménagements de l'environnement de travail, tels que les chaises ergonomiques ou les pauses périodiques. Elle indique également la procédure pour affecter ces types d'aménagements à un collaborateur, et pour approuver ou refuser le type d'aménagement. Les données fictives utilisées pour créer cette procédure correspondent à la société USMF.


## <a name="setup-accommodations"></a>Configurer des aménagements
1. Accédez à Ressources humaines > Paramétrage > Types d'aménagements.
2. Cliquez sur Nouveau.
3. Dans le champ Type d'aménagement, entrez un nom pour le type d'aménagement. Exemple : Clavier.
4. Dans le champ Description, entrez une description de l'aménagement. Exemple : Clavier ergonomique.
5. Dans le champ Note, entrez les informations qui vous aident à clarifier l'aménagement.
6. Cliquez sur Enregistrer.
7. Fermez la page.

## <a name="assign-accommodations"></a>Attribuer des aménagements
1. Accédez à Ressources humaines > Collaborateurs > Employés.
2. Dans la liste, sélectionnez un employé.
3. Cliquez sur le nom d'un employé pour afficher les détails de l'employé qui demande les aménagements.
4. Développez Organisateur des informations personnelles.
5. Cliquez sur Aménagements.
6. Cliquez sur Nouveau.
7. Dans le champ Type d'aménagement, sélectionnez l'aménagement approprié. Exemple : Clavier
8. Dans le champ Tâche, entrez la tâche qui nécessite l'aménagement.
9. Dans le champ Statut, entrez le statut approprié. Exemple : Raisonnable
    * Les statuts suivants sont disponibles. Demandé indique que la demande d'aménagement a été créée mais pas encore été examinée. Raisonnable indique que l'aménagement est raisonnable et sera accordé. Difficultés anormale indique que l'aménagement entraînera des frais déraisonnables pour l'employeur et qu'il a été refusé. Dans cet exemple, la demande a été approuvée immédiatement, car la demande d'aménagement était minimale.  
10. Dans le champ Accepté, sélectionnez la personne qui a accepté la demande d'aménagement.
11. Cliquez sur Sélectionner.
12. Dans le champ Date acceptée, entrez la date et l'heure auxquelles la demande d'aménagement a été acceptée.
13. Développez la section Note.
14. Dans le champ Financier, entrez les coûts d'informations ou de ressources qui permettent de clarifier l'impact de l'aménagement.
    * Si l'aménagement a été refusé, le champ Réponse peut être utilisé pour indiquer la raison du refus.  
15. Cliquez sur Enregistrer.
