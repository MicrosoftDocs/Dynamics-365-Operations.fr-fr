# <a name="set-up-i-9-document-types"></a>Paramétrer les types de documents I-9

[!include[task guide banner](../../../includes/task-guide-banner.md)]

Cette procédure décrit comment afficher et paramétrer des types de documents utilisés pour la vérification du Formulaire I-9. Avant de paramétrer des types de documents I-9, vous devez également paramétrer les agences et les types d'identification. La société de données de démonstration utilisée pour créer cette procédure s'appelle USMF. Elle inclut des exemples d'agences et des types d'identification nécessaires pour exécuter la procédure.

1. Accédez à Ressources humaines > Collaborateurs > I-9 > Types de documents I-9.
2. Cliquez sur Nouveau.
3. Dans le champ Type de document I-9, tapez une valeur.
    * Exemple : ID école.  
4. Sélectionnez le type d'identification.  Exemple : ID école
    * Certains types d'identification comprennent un numéro de sécurité sociale (SSN), un numéro de visa, de passeport ou de permis de conduire.  
5. Sélectionnez la liste de documents I-9 utilisée pour le type de document.
    * Dans le cadre du processus I-9, les employés doivent présenter la documentation qui indique à l'employeur leur identité et leur permis de travail. Le site Web des services de citoyenneté et d'immigration des USA contient des informations sur les documents acceptables et sur la liste à laquelle ils appartiennent.  http://www.uscis.gov  
6. Dans le champ Formulaire, entrez le numéro de formulaire officiel du type de document. Exemple : ID école.
    * Les numéros de formulaires officiels peuvent être recherchés sur le site Web des services de citoyenneté et d'immigration des USA.  http://www.uscis.gov  
7. Cochez ou décochez la case Activer.
8. Dans le champ Expiration, définissez la date sur « 27-10-2019 ».
    * La date d'expiration est facultative.  
9. Sélectionnez l'agence ayant émis le type de document. Exemple : Province/territoire
10. Cliquez sur Enregistrer.
