# <a name="change-the-banner-or-logo"></a><span data-ttu-id="8c727-103">Modifier la bannière ou le logo</span><span class="sxs-lookup"><span data-stu-id="8c727-103">Change the banner or logo</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="8c727-104">La procédure suivante répertorie les étapes que les administrateurs système peuvent utiliser pour mettre à jour l'image de bannière ou de logo affichée pour une entité juridique.</span><span class="sxs-lookup"><span data-stu-id="8c727-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="8c727-105">Les données fictives utilisées pour créer cette procédure correspondent à la société USMF.</span><span class="sxs-lookup"><span data-stu-id="8c727-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="8c727-106">Accédez au **volet de navigation > Modules > Administration d'organisation > Organisations > Entités juridiques**.</span><span class="sxs-lookup"><span data-stu-id="8c727-106">Go to **Navigation pane > Modules > Organization administration > Organizations > Legal entities**.</span></span>
2. <span data-ttu-id="8c727-107">Dans la liste de gauche, sélectionnez l'entité juridique pour laquelle vous souhaitez mettre à jour la bannière ou le logo.</span><span class="sxs-lookup"><span data-stu-id="8c727-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="8c727-108">Si elle est déjà sélectionnée, passez à l'étape suivante.</span><span class="sxs-lookup"><span data-stu-id="8c727-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="8c727-109">Développez l'onglet **Image de tableau de bord**.</span><span class="sxs-lookup"><span data-stu-id="8c727-109">Expand **Dashboard image** tab.</span></span>
4. <span data-ttu-id="8c727-110">Sélectionnez **Modifier**.</span><span class="sxs-lookup"><span data-stu-id="8c727-110">Select **Change**.</span></span>
    
    <span data-ttu-id="8c727-111">La résolution idéale pour une image de bannière est 1920 x 281 pixels.</span><span class="sxs-lookup"><span data-stu-id="8c727-111">The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="8c727-112">La largeur idéale pour une image de logo est 350 pixels.</span><span class="sxs-lookup"><span data-stu-id="8c727-112">The ideal width for a logo image is 350 pixels.</span></span>
    
5. <span data-ttu-id="8c727-113">Sélectionnez **Enregistrer**.</span><span class="sxs-lookup"><span data-stu-id="8c727-113">Select **Save**.</span></span>
6. <span data-ttu-id="8c727-114">Accédez au **volet de navigation > Modules > Commun > Commun > Tableau de bord par défaut**.</span><span class="sxs-lookup"><span data-stu-id="8c727-114">Go to **Navigation pane > Modules > Common > Common > Default dashboard**.</span></span> <span data-ttu-id="8c727-115">Vous devez voir la nouvelle image de bannière ou de logo dans le tableau de bord.</span><span class="sxs-lookup"><span data-stu-id="8c727-115">You should see the new banner or logo image on the dashboard.</span></span>  
    
    <span data-ttu-id="8c727-116">Vous devez peut-être modifier votre société, à l'aide du sélecteur de société, pour faire figurer celle pour laquelle vous avez téléchargé la bannière.</span><span class="sxs-lookup"><span data-stu-id="8c727-116">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
