# <a name="change-the-banner-or-logo"></a><span data-ttu-id="00e0c-103">Modifier la bannière ou le logo</span><span class="sxs-lookup"><span data-stu-id="00e0c-103">Change the banner or logo</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="00e0c-104">La procédure suivante répertorie les étapes que les administrateurs système peuvent utiliser pour mettre à jour l'image de bannière ou de logo affichée pour une entité juridique.</span><span class="sxs-lookup"><span data-stu-id="00e0c-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="00e0c-105">Les données fictives utilisées pour créer cette procédure correspondent à la société USMF.</span><span class="sxs-lookup"><span data-stu-id="00e0c-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="00e0c-106">Accédez à Administration d'organisation > Organisations > Entités juridiques.</span><span class="sxs-lookup"><span data-stu-id="00e0c-106">Go to Organization administration > Organizations > Legal entities.</span></span>
2. <span data-ttu-id="00e0c-107">Dans la liste de gauche, sélectionnez l'entité juridique pour laquelle vous souhaitez mettre à jour la bannière ou le logo.</span><span class="sxs-lookup"><span data-stu-id="00e0c-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="00e0c-108">Si elle est déjà sélectionnée, passez à l'étape suivante.</span><span class="sxs-lookup"><span data-stu-id="00e0c-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="00e0c-109">Développez la section Images.</span><span class="sxs-lookup"><span data-stu-id="00e0c-109">Expand the Images section.</span></span> <span data-ttu-id="00e0c-110">Si elle est déjà développée, passez à l'étape suivante.</span><span class="sxs-lookup"><span data-stu-id="00e0c-110">If it is already expanded, go to the next step.</span></span>
4. <span data-ttu-id="00e0c-111">Cliquez sur Modifier.</span><span class="sxs-lookup"><span data-stu-id="00e0c-111">Click Edit.</span></span>
5. <span data-ttu-id="00e0c-112">Dans le champ « Type d'image société - Tableau de bord », sélectionnez « Bannière ».</span><span class="sxs-lookup"><span data-stu-id="00e0c-112">In the Dashboard company image type field, select 'Banner'.</span></span>
6. <span data-ttu-id="00e0c-113">Cliquez sur Modifier.</span><span class="sxs-lookup"><span data-stu-id="00e0c-113">Click Change.</span></span>
    * <span data-ttu-id="00e0c-114">Conseil : la résolution idéale pour une image de bannière est 1920 x 281 pixels.</span><span class="sxs-lookup"><span data-stu-id="00e0c-114">Tip: The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="00e0c-115">La largeur idéale pour une image de logo est 350 pixels.</span><span class="sxs-lookup"><span data-stu-id="00e0c-115">The ideal width for a logo image is 350 pixels.</span></span>  
    * <span data-ttu-id="00e0c-116">Une fois le téléchargement terminé, vous devez voir votre nouvelle image de bannière ou de logo dans la section Image de tableau de bord.</span><span class="sxs-lookup"><span data-stu-id="00e0c-116">When the upload is complete, you should see your new banner or logo image in the Dashboard Image section.</span></span>  
7. <span data-ttu-id="00e0c-117">Cliquez sur Enregistrer.</span><span class="sxs-lookup"><span data-stu-id="00e0c-117">Click Save.</span></span>
8. <span data-ttu-id="00e0c-118">Accédez à Commun > Commun > Tableau de bord par défaut.</span><span class="sxs-lookup"><span data-stu-id="00e0c-118">Go to Common > Common > Default dashboard.</span></span>
    * <span data-ttu-id="00e0c-119">Vous devez voir la nouvelle image de bannière ou de logo dans le tableau de bord.</span><span class="sxs-lookup"><span data-stu-id="00e0c-119">You should see the new banner or logo image on the dashboard.</span></span>  
    * <span data-ttu-id="00e0c-120">Vous devez peut-être modifier votre société, à l'aide du sélecteur de société, pour faire figurer celle pour laquelle vous avez téléchargé la bannière.</span><span class="sxs-lookup"><span data-stu-id="00e0c-120">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
