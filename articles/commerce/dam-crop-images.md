## <a name="overview"></a><span data-ttu-id="e2de6-105">Vue d'ensemble</span><span class="sxs-lookup"><span data-stu-id="e2de6-105">Overview</span></span>

## <a name="crop-an-image"></a><span data-ttu-id="e2de6-107">Rogner une image</span><span class="sxs-lookup"><span data-stu-id="e2de6-107">Crop an image</span></span>

<span data-ttu-id="e2de6-108">Pour rogner une image dans le générateur de site, procédez comme suit.</span><span class="sxs-lookup"><span data-stu-id="e2de6-108">To crop an image in site builder, follow these steps.</span></span>

1. <span data-ttu-id="e2de6-109">Dans le volet de navigation de gauche du générateur de site Commerce, sélectionnez **Bibliothèque multimédia**.</span><span class="sxs-lookup"><span data-stu-id="e2de6-109">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="e2de6-110">Dans la fenêtre principale, sélectionnez l'image que vous souhaitez modifier.</span><span class="sxs-lookup"><span data-stu-id="e2de6-110">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="e2de6-111">Dans la barre de commande, sélectionnez **Modifier**.</span><span class="sxs-lookup"><span data-stu-id="e2de6-111">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="e2de6-112">Sélectionnez l'image pour saisir **Mode édition**.</span><span class="sxs-lookup"><span data-stu-id="e2de6-112">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="e2de6-113">Sous **Mode édition**, sélectionnez **Modifier la vue par module**.</span><span class="sxs-lookup"><span data-stu-id="e2de6-113">Under **Edit Mode**, select **Edit View by Module**.</span></span>
1. <span data-ttu-id="e2de6-114">Depuis le menu déroulant **Module**, sélectionnez le type de module.</span><span class="sxs-lookup"><span data-stu-id="e2de6-114">From the **Module** drop-down menu, select the module type.</span></span>
1. <span data-ttu-id="e2de6-115">Depuis le menu déroulant **Type d'affichage**, sélectionnez le type d'affichage.</span><span class="sxs-lookup"><span data-stu-id="e2de6-115">From the **View type** drop-down menu, select the view type.</span></span>
1. <span data-ttu-id="e2de6-116">Dans le menu déroulant **Placement**, sélectionnez le placement de l'image.</span><span class="sxs-lookup"><span data-stu-id="e2de6-116">From the **Placement** drop-down menu, select the image placement.</span></span>
1. <span data-ttu-id="e2de6-117">Depuis le menu déroulant **Fenêtre d'affichage**, sélectionnez la taille de la fenêtre d'affichage.</span><span class="sxs-lookup"><span data-stu-id="e2de6-117">From the **Viewport** drop-down menu, select the viewport size.</span></span>
1. <span data-ttu-id="e2de6-118">L'image se superpose à la zone représentant la zone rognée.</span><span class="sxs-lookup"><span data-stu-id="e2de6-118">The image is overlaid with the area representing the crop region.</span></span> <span data-ttu-id="e2de6-119">Déplacez et redimensionnez la zone de rognage selon vos besoins.</span><span class="sxs-lookup"><span data-stu-id="e2de6-119">Move and resize the crop region as needed.</span></span> <span data-ttu-id="e2de6-120">Le rapport hauteur/largeur sera conservé automatiquement.</span><span class="sxs-lookup"><span data-stu-id="e2de6-120">The aspect ratio will be maintained automatically.</span></span>
1. <span data-ttu-id="e2de6-121">Lorsque vous avez terminé, dans la barre de commandes, sélectionnez **Enregistrer**, puis sélectionnez **Terminer l'édition**.</span><span class="sxs-lookup"><span data-stu-id="e2de6-121">When you're done, on the command bar, select **Save**, and then select **Finish editing**.</span></span> 

<span data-ttu-id="e2de6-122">Une fois le rognage personnalisé terminé, les modifications d'image prendront effet presque immédiatement.</span><span class="sxs-lookup"><span data-stu-id="e2de6-122">After custom cropping is completed, image modifications will take effect almost immediately.</span></span>
