# <a name="crop-images"></a>Rogner les images

[!include [banner](includes/banner.md)]

Cette rubrique décrit comment rogner des images dans le générateur de site Microsoft Dynamics 365 Commerce.

## <a name="overview"></a>Vue d’ensemble

La bibliothèque multimédia du générateur de site Commerce vous permet de rogner des images pour les optimiser pour différents types de module et fenêtres d'affichage.

## <a name="crop-an-image"></a>Rogner une image

Pour rogner une image dans le générateur de site, procédez comme suit.

1. Dans le volet de navigation de gauche du générateur de site Commerce, sélectionnez **Bibliothèque multimédia**.
1. Dans la fenêtre principale, sélectionnez l'image que vous souhaitez modifier.
1. Dans la barre de commande, sélectionnez **Modifier**.
1. Sélectionnez l'image pour saisir **Mode édition**.
1. Sous **Mode édition**, sélectionnez **Modifier la vue par module**.
1. Depuis le menu déroulant **Module**, sélectionnez le type de module.
1. Depuis le menu déroulant **Type d'affichage**, sélectionnez le type d'affichage.
1. Dans le menu déroulant **Placement**, sélectionnez le placement de l'image.
1. Depuis le menu déroulant **Fenêtre d'affichage**, sélectionnez la taille de la fenêtre d'affichage.
1. L'image se superpose à la zone représentant la zone rognée. Déplacez et redimensionnez la zone de rognage selon vos besoins. Le rapport hauteur/largeur sera conservé automatiquement.
1. Lorsque vous avez terminé, dans la barre de commandes, sélectionnez **Enregistrer**, puis sélectionnez **Terminer l'édition**. 

Une fois le rognage personnalisé terminé, les modifications d'image prendront effet presque immédiatement.
