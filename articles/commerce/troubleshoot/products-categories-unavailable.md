# <a name="products-and-categories-dont-appear-in-commerce-site-builder-after-a-new-site-is-mapped"></a>Les produits et les catégories n’apparaissent pas dans le générateur de site Commerce après le mappage d’un nouveau site

[!include [banner](../../includes/banner.md)]

Cette rubrique fournit des conseils de résolution des problèmes qui peuvent aider lorsque les produits et les catégories n’apparaissent pas dans le générateur de site Commerce après le mappage d’un nouveau site.

## <a name="description"></a>Description 

Les produits et les catégories n’apparaissent pas dans le générateur de site Commerce après le mappage d’un nouveau site.

## <a name="resolution"></a>Résolution

### <a name="add-products-to-channel-categories-in-commerce-headquarters"></a>Ajouter des produits aux catégories de canaux dans Commerce Headquarters

Pour ajouter des produits aux catégories de canaux dans Commerce Headquarters, procédez comme suit.

1. Accédez à **Commerce et vente au détail \> Paramétrage du canal \> Catégories de canal et attributs de produit**.
1. Dans le volet de navigation de gauche, sélectionnez la catégorie qui doit apparaître sur le site d’e-commerce.
1. Dans le raccourci **Produits**, sélectionnez **Ajouter**.
1. Dans la section **Produits disponibles**, sélectionnez les produits qui doivent apparaître, puis sélectionnez **Ajouter**.
1. Cliquez sur **OK**.
1. Une fois que les produits apparaissent sur le raccourci **Produits**, sélectionnez **Publier les mises à jour du canal**.
