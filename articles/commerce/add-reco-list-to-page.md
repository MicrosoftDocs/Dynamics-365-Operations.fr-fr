# <a name="add-recommendation-lists-to-a-site"></a>Ajouter des listes de recommandations à un site


[!include [banner](includes/banner.md)]

Cette rubrique explique comment ajouter une liste de recommandations de produit à une page de commerce électronique.

1. Sur une page de générateur de site de commerce électronique existante, sélectionnez **Ajouter un module**.
2. Recherchez le module **Collection de produits**.
3. Dans l'Assistant Module, sélectionnez la liste à ajouter à la page.
4. Enregistrez vos modifications.

Pour en savoir plus sur les types de listes que vous pouvez ajouter à une page, voir [Vue d'ensemble du module de collecte de produits](product-collection-module-overview.md).
