# <a name="commerce-chat-features-overview"></a>Présentation des fonctionnalités de conversation instantanée Commerce

[!include [banner](includes/banner.md)]

En intégrant les applications Microsoft Dynamics 365 Omnicanal pour Customer Service et Power Virtual Agents avec les sites web d’e-commerce Dynamics 365 Commerce, les entreprises peuvent responsabiliser leurs acheteurs en ligne grâce à des fonctionnalités de conversation instantanée. Cet article fournit un aperçu des fonctionnalités Commerce Chat qui sont disponibles pour les acheteurs en ligne lorsque les applications Omnicanal pour Customer Service et Power Virtual Agents sont intégrées avec les sites Web d’e-commerce Dynamics 365 Commerce.

## <a name="omnichannel-for-customer-service"></a>Omnicanal pour Customer Service

Omnicanal pour Customer Service est une application robuste qui étend la puissance de Dynamics 365 Customer Service. Elle permet aux organisations de se connecter instantanément et de dialoguer avec leurs clients via des canaux tels que le chat en direct, la voix et le service de SMS. Pour plus d’informations sur Omnicanal pour Customer Service, voir [Introduction à Omnicanal pour Customer Service](/dynamics365/customer-service/introduction-omnichannel). Pour plus d’informations sur l’intégration d’Omnicanal pour Customer Service à un site Web d’e-commerce Dynamics 365 Commerce, voir [Commerce Chat avec le module Omnicanal pour Customer Service](commerce-chat-module.md).

## <a name="power-virtual-agents"></a>Power Virtual Agents

Power Virtual Agents vous permet de créer de puissants chatbots alimentés par l’IA pour une gamme de demandes. Pour plus d’informations sur Power Virtual Agents, voir [Vue d’ensemble de Power Virtual Agents](/power-virtual-agents/fundamentals-what-is-power-virtual-agents). Pour plus d’informations sur l’intégration de Power Virtual Agents à un site web d’e-commerce Dynamics 365 Commerce, voir [Commerce Chat avec Microsoft Power Virtual Agents](chat-module-pva.md).

> [!NOTE]
> Power Virtual Agents est déjà inclus dans Omnicanal pour Customer Service. Si vous avez besoin à la fois de fonctionnalités d’agent virtuel (chatbot) et d’agent en direct, utilisez Omnicanal pour Customer Service.
