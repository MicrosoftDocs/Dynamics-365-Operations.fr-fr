# <a name="create-and-associate-a-device"></a> Créer et associer un périphérique

[!include[task guide banner](../includes/task-guide-banner.md)]

Cette procédure décrit comment créer un périphérique et l'associer à une caisse enregistreuse. On suppose que la caisse enregistreuse est déjà créée.  Cette tâche utilise la société USRT dans les données de démonstration. Cette tâche est destinée aux administrateurs, professionnels de l'informatique ou responsables informatiques Commerce, ainsi qu'aux rôles Directeur de magasin.

1. Accédez à Commerce et vente au détail > .. > Périphériques.
2. Cliquez sur Nouveau.
3. Dans le champ ID périphérique, tapez une valeur.
4. Dans le champ Type de périphérique, cliquez sur le bouton de liste déroulante pour ouvrir la recherche.
5. Dans la liste, recherchez et sélectionnez l'enregistrement souhaité.
6. Dans la liste, cliquer sur le lien dans la ligne sélectionnée.
7. Dans le champ Numéro de caisse enregistreuse, cliquez sur le bouton de liste déroulante pour ouvrir la recherche.
8. Dans la liste, recherchez et sélectionnez l'enregistrement souhaité.
9. Dans la liste, cliquer sur le lien dans la ligne sélectionnée.
10. Cliquez sur Enregistrer.
