<span data-ttu-id="6ca95-104">Cette rubrique décrit comment configurer et utiliser la fonctionnalité « Modifier le mode de livraison » dans votre environnement de point de vente (PDV).</span><span class="sxs-lookup"><span data-stu-id="6ca95-104">This topic describes how to set up and use the "change mode of delivery" functionality in your point of sale (POS) environment.</span></span>
