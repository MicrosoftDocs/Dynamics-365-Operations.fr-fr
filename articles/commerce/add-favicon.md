# <a name="add-a-favicon"></a><span data-ttu-id="74320-103">Ajouter une icône de favori</span><span class="sxs-lookup"><span data-stu-id="74320-103">Add a favicon</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="74320-104">Cette rubrique explique comment ajouter une icône de favori à votre site.</span><span class="sxs-lookup"><span data-stu-id="74320-104">This topic explains how to add a favicon to your site.</span></span>

## <a name="overview"></a><span data-ttu-id="74320-105">Vue d’ensemble</span><span class="sxs-lookup"><span data-stu-id="74320-105">Overview</span></span>

<span data-ttu-id="74320-106">Une icône de favori est un petit fichier graphique affiché sous un onglet du navigateur web, dans la barre d’adresse, dans l’historique de navigation, et dans les signets ou les favoris, entre autres.</span><span class="sxs-lookup"><span data-stu-id="74320-106">A favicon is a small graphics file that is shown on a web browser tab, in the Address bar, in the browsing history, and in bookmarks or favorites, among other places.</span></span> <span data-ttu-id="74320-107">Nous vous recommandons d’ajouter une icône de favori à votre site, car elle représente et renforce votre marque, et aide à distinguer votre site d’autres sites que vos clients visitent.</span><span class="sxs-lookup"><span data-stu-id="74320-107">We recommend that you add a favicon to your site, because it represents and reinforces your brand, and helps distinguish your site from other sites that your customers visit.</span></span>

<span data-ttu-id="74320-108">Bien que vous puissiez ajouter plusieurs icônes de favori de différentes tailles et types de fichiers à votre site, cette rubrique explique comment ajouter une icône de favori unique.</span><span class="sxs-lookup"><span data-stu-id="74320-108">Although you can add multiple favicons of various sizes and file types to your site, this topic shows how to add a single favicon.</span></span> <span data-ttu-id="74320-109">Toutefois, les mêmes processus et emplacement sont utilisés pour ajouter des une icônes de favori.</span><span class="sxs-lookup"><span data-stu-id="74320-109">However, the same process and location are used to add more favicons.</span></span>

## <a name="upload-a-favicon-to-your-sites-asset-collection"></a><span data-ttu-id="74320-110">Chargement de l’icône de favori dans la collection d’actifs du site</span><span class="sxs-lookup"><span data-stu-id="74320-110">Upload a favicon to your site's asset collection</span></span>

<span data-ttu-id="74320-111">Pour charger l’icône de favori dans la collection d’actifs du site, procédez comme suit.</span><span class="sxs-lookup"><span data-stu-id="74320-111">To upload a favicon to your site's asset collection, follow these steps.</span></span>

1. <span data-ttu-id="74320-112">Dans le volet de navigation de gauche, cliquez sur **Bibliothèque multimédia**.</span><span class="sxs-lookup"><span data-stu-id="74320-112">In the left navigation pane, select **Media Library**.</span></span>
1. <span data-ttu-id="74320-113">Dans la barre de commandes, sélectionnez **Télécharger \> Télécharger des éléments multimédias**.</span><span class="sxs-lookup"><span data-stu-id="74320-113">On the command bar, select **Upload \> Upload Media Items**.</span></span>
1. <span data-ttu-id="74320-114">Dans la fenêtre de l’Explorateur de fichiers, accédez au fichier image favicon que vous souhaitez télécharger, sélectionnez-le, puis sélectionnez **Ouvrir**.</span><span class="sxs-lookup"><span data-stu-id="74320-114">In the File Explorer window, browse to the favicon image file that you want to upload, select it, and then select **Open**.</span></span>
1. <span data-ttu-id="74320-115">Dans la boîte de dialogue **Télécharger un élément multimédia**, entrez le titre et le texte de remplacement requis.</span><span class="sxs-lookup"><span data-stu-id="74320-115">In the **Upload Media Item** dialog box, enter the required title and alt text.</span></span>
1. <span data-ttu-id="74320-116">Si vous souhaitez publier l’image immédiatement après le téléchargement, sélectionnez la case à cocher **Publier les éléments multimédia après le téléchargement**.</span><span class="sxs-lookup"><span data-stu-id="74320-116">If you want to publish the image immediately after upload, select the **Publish media items after upload** check box.</span></span>

    > [!NOTE]
    > <span data-ttu-id="74320-117">Si vous ne sélectionnez pas la case à cocher **Publier les éléments multimédia après le téléchargement**, vous devez retourner à la page **Éléments multimédia** et publier manuellement l’icône de favori ultérieurement.</span><span class="sxs-lookup"><span data-stu-id="74320-117">If you don't select the **Publish media items after upload** check box, you must return to **Media items** page and manually publish the favicon later.</span></span>

1. <span data-ttu-id="74320-118">Cliquez sur **OK**.</span><span class="sxs-lookup"><span data-stu-id="74320-118">Select **OK**.</span></span>
1. <span data-ttu-id="74320-119">Dans le volet de propriété de droite, copiez l’URL publique de l’icône de favori.</span><span class="sxs-lookup"><span data-stu-id="74320-119">In the property pane on the right, copy the public URL of the favicon.</span></span> <span data-ttu-id="74320-120">Vous utiliserez cette URL ultérieurement.</span><span class="sxs-lookup"><span data-stu-id="74320-120">You will use this URL later.</span></span>

## <a name="create-a-fragment-that-contains-a-metatag-for-your-favicon"></a><span data-ttu-id="74320-124">Créer un fragment contenant une balise META pour votre icône de favori</span><span class="sxs-lookup"><span data-stu-id="74320-124">Create a fragment that contains a metatag for your favicon</span></span>

<span data-ttu-id="74320-125">Pour créer un fragment contenant une balise META pour votre icône de favori, suivez cette procédure.</span><span class="sxs-lookup"><span data-stu-id="74320-125">To create a fragment that contains a metatag for your favicon, follow these steps.</span></span>

1. <span data-ttu-id="74320-126">Accédez à **Fragments**, et sélectionnez **Nouveau**.</span><span class="sxs-lookup"><span data-stu-id="74320-126">Go to **Fragments**, and select **New**.</span></span>
1. <span data-ttu-id="74320-127">Dans la boîte de dialogue **Nouveau fragment**, sélectionnez **Métabalises** comme module sur lequel le fragment est basé.</span><span class="sxs-lookup"><span data-stu-id="74320-127">In the **New fragment** dialog box, select **Metatags** as the module that the fragment is based on.</span></span>
1. <span data-ttu-id="74320-128">Entrez un nom pour le fragment, puis sélectionnez **OK**.</span><span class="sxs-lookup"><span data-stu-id="74320-128">Enter a name for the fragment, and then select **OK**.</span></span>
1. <span data-ttu-id="74320-129">Dans l’arborescence de la hiérarchie des fragments, sélectionnez l’enfant **Metatags par défaut**.</span><span class="sxs-lookup"><span data-stu-id="74320-129">In the fragment hierarchy tree, select the **Default metatags** child.</span></span>
1. <span data-ttu-id="74320-130">Dans le volet droit, sous **Balises Meta**, sélectionnez **Ajouter**, puis entrez la chaîne HTML que vous avez créée précédemment pour l’icône de favori.</span><span class="sxs-lookup"><span data-stu-id="74320-130">In the right pane, under **Meta Tags**, select **Add**, and then enter the HTML string that you created earlier for the favicon.</span></span> 
1. <span data-ttu-id="74320-131">Sélectionnez **Terminer la modification**, puis **Publier** pour publier le fragment.</span><span class="sxs-lookup"><span data-stu-id="74320-131">Select **Finish editing**, and then select **Publish** to publish the fragment.</span></span>

## <a name="add-the-metatag-fragment-to-the-html-head-section-of-your-pages"></a><span data-ttu-id="74320-132">Ajouter le fragment de métabalise à la section d’en-tête HTML de vos pages</span><span class="sxs-lookup"><span data-stu-id="74320-132">Add the metatag fragment to the HTML head section of your pages</span></span>

<span data-ttu-id="74320-133">Pour ajouter le fragment de métabalise à la section d’**en-tête** HTML de vos pages, suivez cette procédure.</span><span class="sxs-lookup"><span data-stu-id="74320-133">To add the metatag fragment to the HTML **head** section of your pages, follow these steps.</span></span>

1. <span data-ttu-id="74320-134">Accédez à **Modèles**, puis ouvrez le modèle pour les pages dans lesquelles vous souhaitez ajouter votre icône de favori, puis sélectionnez **Modifier**.</span><span class="sxs-lookup"><span data-stu-id="74320-134">Go to **Templates**, open the template for the pages that you want to add your favicon to, and then select **Edit**.</span></span>
1. <span data-ttu-id="74320-135">Dans l’arborescence de la hiérarchie des modèles, sélectionnez les points de suspension (**...**) à droite du conteneur **En-tête HTML**, puis sélectionnez **Ajouter un fragment**.</span><span class="sxs-lookup"><span data-stu-id="74320-135">In the template hierarchy tree, select the ellipsis (**...**) button to the right of the **HTML head** container, and then select **Add fragment**.</span></span>
1. <span data-ttu-id="74320-136">Dans la boîte de dialogue **Sélectionner un fragment**, sélectionnez le fragment de métabalise que vous avez créé au préalable, puis sélectionnez **OK**.</span><span class="sxs-lookup"><span data-stu-id="74320-136">In the **Select fragment** dialog box, select the metatag fragment that you created earlier, and then select **OK**.</span></span>
1. <span data-ttu-id="74320-137">Sélectionnez **Terminer la modification**, puis **Publier** pour publier le modèle.</span><span class="sxs-lookup"><span data-stu-id="74320-137">Select **Finish editing**, and then select **Publish** to publish the template.</span></span>

> [!NOTE]
> <span data-ttu-id="74320-138">Si votre site utilise plusieurs modèles, vous devez ajouter le fragment de métabalises à chacun d’eux.</span><span class="sxs-lookup"><span data-stu-id="74320-138">If your site uses more than one template, you must add the metatags fragment to all of them.</span></span>

<span data-ttu-id="74320-139">Lorsque vous prévisualisez des pages basées sur le modèle auquel vous avez ajouté le fragment de métabalises, vous devriez maintenant voir l’icône de favori dans l’onglet du navigateur.</span><span class="sxs-lookup"><span data-stu-id="74320-139">When you preview pages that are based on the template that you added the metatags fragment to, you should now see the favicon on the browser tab.</span></span>
