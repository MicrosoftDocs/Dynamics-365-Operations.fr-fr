<span data-ttu-id="01ce3-104">Cette rubrique couvre les modules de placement de contenu et d'article de placement de contenu et décrit comment les ajouter aux pages du site dans Microsoft Dynamics 365 Commerce.</span><span class="sxs-lookup"><span data-stu-id="01ce3-104">This topic covers content placement and content placement item modules, and describes how to add them to site pages in Microsoft Dynamics 365 Commerce.</span></span>

## <a name="overview"></a><span data-ttu-id="01ce3-105">Vue d'ensemble</span><span class="sxs-lookup"><span data-stu-id="01ce3-105">Overview</span></span>

<span data-ttu-id="01ce3-106">Un module de placement de contenu est un conteneur spécial dans lequel d'autres modules peuvent être mis pour une disposition spécifique.</span><span class="sxs-lookup"><span data-stu-id="01ce3-106">A content placement module is a special container that other modules can be put inside in a specific layout.</span></span> <span data-ttu-id="01ce3-107">Les modules de placement de contenu prennent en charge les types de modules suivants comme modules enfants : article de placement de contenu, article de bienvenue du compte, article de commande de compte, article de liste de souhaits de compte, et article de profil de compte.</span><span class="sxs-lookup"><span data-stu-id="01ce3-107">Content placement modules support the following types of modules as child modules: content placement item, account welcome item, account order item, account wish list item, and account profile item.</span></span>

<span data-ttu-id="01ce3-108">Les modules de placement de contenu dérivent les données des modules enfants qu'ils prennent en charge.</span><span class="sxs-lookup"><span data-stu-id="01ce3-108">Content placement modules derive data from the child modules that they support.</span></span> <span data-ttu-id="01ce3-109">Par conséquent, les modules de placement de contenu peuvent être utilisés dans différentes manières, en fonction des modules qui sont ajoutés.</span><span class="sxs-lookup"><span data-stu-id="01ce3-109">Therefore, content placement modules can be used in various ways, depending on the modules that are added to it.</span></span>

<span data-ttu-id="01ce3-110">Les modules d'article de placement de contenu utilisent des images et du texte pour présenter des promotions, des stratégies, et des fonctionnalités de produit.</span><span class="sxs-lookup"><span data-stu-id="01ce3-110">Content placement item modules use both images and text to showcase promotions, policies, and product features.</span></span> <span data-ttu-id="01ce3-111">Par exemple, un module d'article de placement de contenu peut être utilisé sur une page d'accueil pour afficher des stratégies ou des informations sur l'expédition.</span><span class="sxs-lookup"><span data-stu-id="01ce3-111">For example, a content placement item module can be used on a home page to show store policies or shipping information.</span></span> <span data-ttu-id="01ce3-112">Les modules de placement de contenu sont pilotés par les données du système de gestion de contenu (CMS) et peuvent être placés dans n'importe quelle page.</span><span class="sxs-lookup"><span data-stu-id="01ce3-112">Content placement item modules are driven by data from the content management system (CMS) and can be put on any page.</span></span> <span data-ttu-id="01ce3-113">Toutefois, ils peuvent être utilisés uniquement dans un module de placement de contenu.</span><span class="sxs-lookup"><span data-stu-id="01ce3-113">However, they can be used only inside a content placement module.</span></span>

* <span data-ttu-id="01ce3-116">Les modules de placement de contenu qui contiennent les modules d'article de placement de contenu peuvent être utilisés dans les pages de détails de produit pour présenter des fonctionnalités de produit.</span><span class="sxs-lookup"><span data-stu-id="01ce3-116">Content placement modules that contain content placement item modules can be used on product details pages to showcase product features.</span></span>
