<span data-ttu-id="bba44-106">Le terme *zone d'achat* désigne généralement la zone d'une page de détails de produit « au-dessus du volet », et qui héberge toutes les principales informations requises pour effectuer un achat de produit.</span><span class="sxs-lookup"><span data-stu-id="bba44-106">The term *buy box* typically refers to the area of a product details page that is "above the fold," and that hosts all the most important information that is required to make a product purchase.</span></span> <span data-ttu-id="bba44-107">(Une zone qui est « au-dessus du volet » n'est visible que si la page est d'abord chargée, afin que les utilisateurs n'aient pas à faire défiler l'écran pour la voir.)</span><span class="sxs-lookup"><span data-stu-id="bba44-107">(An area that is "above the fold" is visible when the page is first loaded, so that users don't have to scroll down to see it.)</span></span>

<span data-ttu-id="bba44-108">Un module Zone d'achat est un conteneur spécial utilisé pour héberger tous les modules qui sont affichés dans la zone de la zone d'achat d'une page de détails des produits.</span><span class="sxs-lookup"><span data-stu-id="bba44-108">A buy box module is special container that is used to host all the modules that are shown in the buy box area of a product details page.</span></span>

<span data-ttu-id="bba44-109">L'URL d'une page de détails de produit inclut l'ID produit.</span><span class="sxs-lookup"><span data-stu-id="bba44-109">The URL of a product details page includes the product ID.</span></span> <span data-ttu-id="bba44-110">Toutes les informations requises pour afficher un module de zone d'achat sont dérivées de cet ID produit.</span><span class="sxs-lookup"><span data-stu-id="bba44-110">All the information that is required to render a buy box module is derived from this product ID.</span></span> <span data-ttu-id="bba44-111">Si aucun ID produit n'est fourni, le module de zone d'achat ne s'affiche pas correctement sur une page.</span><span class="sxs-lookup"><span data-stu-id="bba44-111">If a product ID isn't provided, the buy box module won't be rendered correctly on a page.</span></span> <span data-ttu-id="bba44-112">Par conséquent, un module de zone d'achat ne peut être utilisé que sur des pages qui ont un contexte de produit.</span><span class="sxs-lookup"><span data-stu-id="bba44-112">Therefore, a buy box module can be used only on pages that have product context.</span></span> <span data-ttu-id="bba44-113">Pour l'utiliser sur une page qui n'a pas de contexte de produit (par exemple, une page d'accueil ou une page marketing), vous devez effectuer des personnalisations supplémentaires.</span><span class="sxs-lookup"><span data-stu-id="bba44-113">To use it on a page that doesn't have product context (for example, a home page or a marketing page), you must do additional customizations.</span></span>
