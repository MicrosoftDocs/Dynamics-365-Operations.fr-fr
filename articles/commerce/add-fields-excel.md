## <a name="add-fields-to-a-worksheet-in-an-excel-workbook"></a><span data-ttu-id="d3b23-108">Ajouter des champs à une feuille de calcul dans un classeur Excel</span><span class="sxs-lookup"><span data-stu-id="d3b23-108">Add fields to a worksheet in an Excel workbook</span></span>

<span data-ttu-id="d3b23-109">Pour ajouter des champs à un classeur Excel afin de pouvoir modifier les transactions de vente au détail, procédez comme suit.</span><span class="sxs-lookup"><span data-stu-id="d3b23-109">To add fields to an Excel workbook so that you can edit retail transactions, follow these steps.</span></span>

1. <span data-ttu-id="d3b23-110">Téléchargez et ouvrez le fichier Excel à partir de la page **Relevés**, de la page **Transactions de vente au détail** ou de la vignette **Échecs de validation de transaction** dans l’espace de travail **Finances du magasin**.</span><span class="sxs-lookup"><span data-stu-id="d3b23-110">Download and open the Excel file from the **Statements** page, the **Retail transactions** page, or the **Transaction validation failures** tile in the **Store financials** workspace.</span></span>
1. <span data-ttu-id="d3b23-111">Sélectionnez **Design**.</span><span class="sxs-lookup"><span data-stu-id="d3b23-111">Select **Design**.</span></span>
1. <span data-ttu-id="d3b23-112">Sélectionnez le symbole du crayon pour la table souhaitée, puis, dans la liste des champs disponibles, recherchez et sélectionnez le champ que vous souhaitez ajouter.</span><span class="sxs-lookup"><span data-stu-id="d3b23-112">Select the pencil symbol for the desired table, and then, in the list of available fields, find and select the field that you want to add.</span></span>
