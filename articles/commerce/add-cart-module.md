## <a name="cart-module-settings"></a><span data-ttu-id="0997a-144">Paramètres du module de panier</span><span class="sxs-lookup"><span data-stu-id="0997a-144">Cart module settings</span></span>

<span data-ttu-id="0997a-145">Les modules de panier ont trois paramètres qui peuvent être configurés :</span><span class="sxs-lookup"><span data-stu-id="0997a-145">Cart modules have three settings that can be configured:</span></span>

- <span data-ttu-id="0997a-146">**Quantité maximale** – Nombre maximal de chaque article qui peut être ajouté au panier.</span><span class="sxs-lookup"><span data-stu-id="0997a-146">**Maximum quantity** – The maximum number of each item that can be added to the cart.</span></span> <span data-ttu-id="0997a-147">Par exemple, un détaillant peut décider que seuls 10 % de chaque produit peuvent être vendus en une seule transaction.</span><span class="sxs-lookup"><span data-stu-id="0997a-147">For example, a retailer might decide that only 10 of each product can be sold in a single transaction.</span></span>
- <span data-ttu-id="0997a-148">**Contrôle de stock** – Lorsque la valeur est définie sur **Vrai**, un article est ajouté au panier uniquement une fois que le module zone d'achat garantit qu'il est en stock.</span><span class="sxs-lookup"><span data-stu-id="0997a-148">**Inventory check** – When the value is set to **True**, an item is added to the cart only after the buy box module makes sure that it's in stock.</span></span> <span data-ttu-id="0997a-149">Ce contrôle de stock est réalisé pour les cas où l'article sera expédié et pour les cas où il sera prélevé en magasin.</span><span class="sxs-lookup"><span data-stu-id="0997a-149">This inventory check is done both for scenarios where the item will be shipped and for scenarios where it will be picked up in the store.</span></span> <span data-ttu-id="0997a-150">Si la valeur est définie sur **Faux**, aucun contrôle de stock n'est effectué avant qu'un article soit ajouté au panier et la commande est passée.</span><span class="sxs-lookup"><span data-stu-id="0997a-150">If the value is set to **False**, no inventory check is done before an item is added to the cart and the order is placed.</span></span>
- <span data-ttu-id="0997a-151">**Marge de stock** – Le stock est tenu à jour en temps réel, et lorsque plusieurs clients passent des commandes, il peut être difficile de tenir à jour un volume de stock exact.</span><span class="sxs-lookup"><span data-stu-id="0997a-151">**Inventory buffer** – Inventory is maintained in real time, and when many customers place orders, it can be difficult to maintain an accurate inventory count.</span></span> <span data-ttu-id="0997a-152">Par conséquent, une marge peut être définie pour le stock.</span><span class="sxs-lookup"><span data-stu-id="0997a-152">Therefore, a buffer can be defined for inventory.</span></span> <span data-ttu-id="0997a-153">Lorsqu'un contrôle de stock est réalisé, si le stock est inférieur au montant de marge, le produit est traité comme épuisé.</span><span class="sxs-lookup"><span data-stu-id="0997a-153">When an inventory check is done, if the inventory is less than the buffer amount, the product is treated as out of stock.</span></span> <span data-ttu-id="0997a-154">Par conséquent, lorsque les ventes se produisent rapidement sur plusieurs canaux, de sorte que le volume de stock n'est pas complètement synchronisé, il y a moins de risque qu'un article qui épuisé soit vendu.</span><span class="sxs-lookup"><span data-stu-id="0997a-154">Therefore, when sales occur quickly through several channels, so that the inventory count isn't fully synced, there is less risk that an item that is out of stock will be sold.</span></span>
