# <a name="customize-image-focal-points"></a><span data-ttu-id="db120-103">Personnaliser les points focaux de l’image</span><span class="sxs-lookup"><span data-stu-id="db120-103">Customize image focal points</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="db120-104">Cette rubrique décrit comment personnaliser les points focaux des images dans le générateur de site Microsoft Dynamics 365 Commerce.</span><span class="sxs-lookup"><span data-stu-id="db120-104">This topic describes how to customize image focal points in Microsoft Dynamics 365 Commerce site builder.</span></span>

<span data-ttu-id="db120-105">Lorsqu’une image est téléchargée dans la bibliothèque multimédia du générateur de site Commerce, le système tente de déterminer le point focal de l’image.</span><span class="sxs-lookup"><span data-stu-id="db120-105">When an image is uploaded to the Commerce site builder Media Library, the system attempts to determine the focal point of the image.</span></span> <span data-ttu-id="db120-106">Par exemple, si l’image contient une personne, le système définira le point focal sur le visage de la personne par défaut.</span><span class="sxs-lookup"><span data-stu-id="db120-106">For example, if the image has a person on it, the system will set the focal point to the face of the person by default.</span></span> <span data-ttu-id="db120-107">Dans la plupart des cas, le point focal défini automatiquement fonctionne bien pour toutes les fenêtres, mais vous pouvez parfois souhaiter ajuster le point focal pour vous assurer qu’une partie spécifique de l’image est toujours visible.</span><span class="sxs-lookup"><span data-stu-id="db120-107">In most cases the automatically set focal point works well for all viewports, but sometimes you may want to adjust the focal point to ensure that a specific part of the image is always visible.</span></span>

### <a name="define-a-custom-focal-point-for-an-image"></a><span data-ttu-id="db120-108">Définir un point focal personnalisé pour une image</span><span class="sxs-lookup"><span data-stu-id="db120-108">Define a custom focal point for an image</span></span>

<span data-ttu-id="db120-109">Pour définir un point focal personnalisé pour une image, procédez comme suit.</span><span class="sxs-lookup"><span data-stu-id="db120-109">To define a custom focal point for an image, follow these steps.</span></span>

1. <span data-ttu-id="db120-110">Dans le volet de navigation de gauche du générateur de site Commerce, sélectionnez **Bibliothèque multimédia**.</span><span class="sxs-lookup"><span data-stu-id="db120-110">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="db120-111">Dans la fenêtre principale, sélectionnez l’image que vous souhaitez modifier.</span><span class="sxs-lookup"><span data-stu-id="db120-111">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="db120-112">Dans la barre de commande, sélectionnez **Modifier**.</span><span class="sxs-lookup"><span data-stu-id="db120-112">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="db120-113">Sélectionnez l’image pour saisir **Mode édition**.</span><span class="sxs-lookup"><span data-stu-id="db120-113">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="db120-114">Sous **Mode édition**, sélectionnez **Changer le point focal**.</span><span class="sxs-lookup"><span data-stu-id="db120-114">Under **Edit Mode**, select **Change Focal Point**.</span></span> <span data-ttu-id="db120-115">Un contrôle de point focal circulaire apparaît sur l’image.</span><span class="sxs-lookup"><span data-stu-id="db120-115">A circular focal point control appears over the image.</span></span>
1. <span data-ttu-id="db120-116">Sélectionnez le contrôle du point focal pour le déplacer sur le point focal souhaité.</span><span class="sxs-lookup"><span data-stu-id="db120-116">Select the focal point control to move it over the desired focal point.</span></span>
1. <span data-ttu-id="db120-117">Lorsque vous avez terminé, dans la barre de commandes, sélectionnez **Enregistrer**, puis sélectionnez **Terminer l’édition**.</span><span class="sxs-lookup"><span data-stu-id="db120-117">When you're done, on the command bar select **Save**, and then select **Finish editing**.</span></span>
