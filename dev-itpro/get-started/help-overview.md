# <a name="help-overview"></a>Vue d'ensemble d'aide

## <a name="task-guides"></a>Guides de tâche
Un Guide de tâche est une expérience qui vous guide dans les étapes d'une tâche, ou un processus entreprise commandée, guidée, interactif. Vous pouvez ouvrir (jeu) des instructions de tâche du volet d'aide. Lorsque vous cliquez sur pour la première fois un Guide de tâche, le volet d'aide affiche des instructions détaillées pour la tâche. Les guides trouvent des tâches sont désormais disponibles. [vue de lecture de guide de tâche d'![] (. /media/task-guide-ops-1024x742.png)](. /media/task-guide-ops.png) Pour démarrer l'expérience guidée et interactive, cliquez sur ** démarrez le Guide de tâche ** en bas du volet d'aide. Un pointeur noir apparaît et indique l'action que vous devez effectuer. Suivez les instructions qui s'affichent dans l'IU et entrez les données telles qu'elles sont indiquées. [instruction d'étape de guide de tâche d'![] (. /media/task-guide-step-1-ops.png)](. /media/task-guide-step-1-ops.png) ** important : ** Les données que vous entrez lors jouez des instructions de tâche sont remplies. Si vous êtes dans un environnement de production, les données seront saisies dans la société que vous utilisez actuellement.

### <a name="it-all-begins-with-task-recorder"></a>Tout commence avec l'enregistreur de tâches

Les guides de tâche sont créés à l'aide de l'enregistreur de tâches. Lorsque vous utilisez Enregistreur de tâches, toutes les actions effectuées dans l'IU de Dynamics 365 for Operations (cliquer sur des menus, modifier les paramètres, entrer des données) sont enregistrées. Les étapes que vous enregistrez sont collectivement appelées un enregistrement de tâche. Comme nous l'avons expliqué dans la section précédente, les enregistrements de tâche peuvent être affichés dans le volet Aide et lus en tant que guides de tâche. Il existe toutefois d'autres manières d'utiliser les enregistrements de tâches :

-   **Enregistrer les enregistrements de tâches dans BPM** – Vous pouvez enregistrer un enregistrement de tâche dans la bibliothèque BPM dans le crédit. Lorsque vous enregistrez un enregistrement de tâche dans BPM, un organigramme est généré et affiché avec les étapes de l'enregistrement. **Remarque :** pour afficher un enregistrement de tâche dans le volet Aide de Dynamics 365 for Operations et le lire comme un guide de tâche, vous devrez enregistrer l'enregistrement dans une bibliothèque BPM.
-   **Enregistrer les enregistrements de tâche en tant que documents Word** – en enregistrant un enregistrement de tâche en tant que document Microsoft Word, vous pouvez facilement créer des manuels de formation imprimables pour votre organisation.

Pour plus d'informations sur l'enregistreur de tâches, voir [enregistreur de tâches dans Dynamics 365 pour les opérations (]. /user-interface/task-recorder.md).

### <a name="creating-customized-task-recordings"></a>Création d'enregistrements de tâche personnalisés

Vous pouvez créer vos propres enregistrements de tâche, ou vous pouvez télécharger et personnaliser un enregistrement de tâche fourni par Microsoft. Par conséquent, vous pouvez créer une Aide personnalisée pour votre organisation qui reflète votre implémentation particulière de Dynamics 365 for Operations. Pour afficher une tâche de journalisation dans Dynamics 365 pour les opérations de le volet et jouez- le comme guide de tâche, vous devra enregistrer l'enregistrement à une bibliothèque de BPM dans des lettres de crédit. Si vous êtes un partenaire, et vous hausser les rapports dans une bibliothèque d'entreprise et l'inclure dans une solution, elle est disponible pour vos clients. Pour obtenir des instructions complètes, voir [par des enregistrements de tâches pour créer la documentation ou la formation (]. /user-interface/task-recorder.md).

## <a name="in-product-help"></a>Aide intégrée au produit
Pour accéder au contenu d'aide dans Dynamics 365 pour les opérations, ou cliquez sur ** aide ** (** ? **) l'icône puis choisissez l'aide ou appuyez sur Ctrl+Shift+ ?. Dans les deux cas, le volet Aide s'ouvre. Dans le volet d'aide, vous pouvez accéder aux articles ou aux guides de tâche. [![](./media/help-pane-wiki-1024x684.png)](./media/help-pane-wiki.png)

### <a name="accessing-articles-from-the-help-pane"></a>Navigation dans des articles du volet d'aide

Dans le volet d'aide, vous pouvez accéder aux articles qui s'appliquent à Dynamics 365 pour le client des opérations. Lorsque vous commencez ouvrez le volet d'aide et cliquez sur ** wiki ** l'onglet, vous obtenez les articles qui s'appliquent à la page à laquelle vous êtes actuellement dans Dynamics 365 pour les opérations. Si aucun article n'est trouvé, vous pouvez entrer les mots clés pour affiner votre recherche. Lorsque vous cliquez sur un élément dans le volet d'aide, un nouvel onglet s'ouvre dans votre navigateur et affiche l'article. 

### <a name="accessing-task-guides-from-the-help-pane"></a>Accéder aux guides de tâche du volet d'aide

Avant de pouvoir accéder aux guides de tâche du volet d'aide, un administrateur système doit accéder au ** des paramètres système ** page dans Dynamics 365 pour les opérations et configurent certains paramètres. **Remarques :**

-   Pour configurer l'aide, vous devez être connecté avec un compte dans le même locataire que le locataire que celui dans lequel Dynamics 365 for Operations est déployé.
-   Il n’est pas possible de se connecter à une bibliothèque LCS depuis une instance de Dynamics 365 for Operations s’exécutant sur un disque dur virtuel (VHD) local.

[écran Paramètres système![avec les paramètres d'aide (]. /media/system-parameters_ops-1024x437.png)](. /media/system-parameters_ops.png) Sous ** des paramètres système ** la page, procédez comme suit :

1.  **Important : **la première fois que vous ouvrez l'onglet Aide, vous devez vous connecter à Lifecycle Services. Veillez à cliquer sur le lien au milieu de l'écran, attendez la connexion, fermez la boîte de dialogue, puis cliquez sur OK pour obtenir à l'écran Paramètres. [![se connectent au crédit (]. /media/connect-to-lcs-crop-1024x365.png)](. /media/connect-to-lcs-crop.png)
2.  Sélectionnez le projet Lifecycle Services auquel se connecter.
3.  Sélectionnez les bibliothèques BPM (dans le projet sélectionné) à partir desquelles récupérer les enregistrements de tâche.
4.  Sélectionnez l'ordre d'affichage des bibliothèques BPM. Cela déterminer l'ordre dans lequel les enregistrements de tâche des bibliothèques s'affichent dans le volet Aide.

Une fois qu'un administrateur système a terminé ces étapes, vous pouvez ouvrir le volet Aide puis cliquer sur l'onglet **Guides de tâche**. Vous obtenez désormais les guides de tâches qui s'appliquent à la page à laquelle vous êtes actuellement dans Dynamics 365 pour les opérations. Si aucun Guide de tâche n'est trouvé, vous pouvez entrer les mots clés pour affiner votre recherche. Après avoir cliqué sur des instructions des tâches dans le volet d'aide, les affiches du volet d'aide les instructions détaillées, et vous pouvez jouer le Guide de tâche. [vue de lecture de guide de tâche d'![] (. /media/task-guide-ops-1024x742.png)](. /media/task-guide-ops.png)

### <a name="where-are-the-translated-task-guides"></a>Appartenance correspondent aux guides traduits de tâche ?

Les guides traduits de tâche sont supprimées dans les bibliothèques avec « toutes les langues » au titre. Dans Dynamics 365 pour les opérations, pour voir l'aide située de guide de tâche, vérifiez que vous êtes connecté à une bibliothèque d'apppropriate. Langue qui un Guide de tâche apparaît dans est contrôlée pour chaque utilisateur par les paramètres de langue sous ** des options ** &gt; ** des préférences **. 
-   Si un Guide de tâche a été traduits, lorsque vous ouvrez que Guide de tâche que tout le texte du Guide de tâche s'affichera dans votre langue sélectionnée.
-   Si un Guide de tâche n'a pas encore été traduit, lorsque vous l'ouvrez, seule une partie du texte (le texte des contrôles) apparaît dans votre langue sélectionnée.
